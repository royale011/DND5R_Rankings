# DND5R Rankings

这是一个面向 D&D 5.5e / 2024 / 5R 的职业、子职与构筑评分仓库。

当前仓库主要收录以下内容：

- `.codex`：项目级 Codex skills，用于固定评分流程、资料核查规则、文件结构和更新记录规则。
- `tools`：用于批量生成、校准或整理评分文件的本地脚本。
- `Rankings`：官方、UA、合作方与第三方公开内容的评分与评审产物。
- `DND5e_chm`：资料 submodule，指向 `https://github.com/DND5eChm/DND5e_chm`。这个目录用于核查最新 5.5e / 2024 内容，应保持更新。
- `DND2014_legacy`：资料 submodule，同样指向 `https://github.com/DND5eChm/DND5e_chm.git`，但固定在当前 revision。这个目录用于保留完整、纯 5e 时代资料，尤其是未来复核 5e UA 子职时使用；不要为了 5e UA 评审而更新它。

`homebrews` 目录暂不纳入本仓库。该目录目前属于个人创作与迭代区，不作为公开评分产物发布；与 homebrew 评审相关的流程规则仍保留在 `.codex` skills 中，以便本地继续使用。
