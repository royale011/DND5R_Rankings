# 勇士

所属职业：战士（Fighter）
英文名 / 社区检索名：勇士
来源 / 版本：已生成基准 / 2024 或当前采用版本

## 队伍定位调整
勇士是最简单的战士子职：它把底盘的命中、暴击、先攻和体能推高，但几乎不提供新的队伍功能。

## 详细评价
3级暴击范围扩展是直观增伤，但数学上并不等于大爆发。它和优势、`干扰`、勇士自带的稳定性、暴击触发装备有协同；问题是暴击不稳定，且不能主动解决控场、保护、反制或多目标问题。

3级 `卓越运动员` 和 `精通重击` 同时进场，前者给先攻优势、运动优势和暴击后的半速移动。先攻优势很实用，尤其战士需要先占位、先贴后排或先用 `武器精通` 制造状态；但运动优势在 2024 底盘已有 `战术头脑` 后不再独特，它更像把战士已经能做的事做得更稳。

7级 `额外战斗风格` 很实用，能让弓手补防御、盾战补盲斗或近战补双手输出。10级 `英雄斗士` 每回合开头补 `英雄激励`，稳定性明显高于旧版勇士的空白感；它的好处是真正稳定，坏处是仍没有改变队伍职责。

15级进一步扩大暴击范围，18级生存恢复让它后期更耐打。综合看，勇士不是陷阱，但它是“低操作稳定强度”，不是高上限子职。社区对它的评价也常是新手友好、数学干净，但优化上限不如战斗大师、奥法骑士、回音骑士或符文骑士。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | B | 勇士低等级把重击、运动员式能力和底盘武器输出做得很顺，但子职基本不提供新职责；它强在简单可靠，不强在改变队伍功能。 |
| Tier 2（5-10） | B | Tier 2 继续强化暴击、先攻/运动和基础战斗表现，适合不想管理资源的玩家；但同阶段战斗大师、奥法骑士、符文骑士能提供更多控制或工具。 |
| Tier 3（11-16） | C | Tier 3 的勇士仍能打，但子职增量偏数值化，高等级缺少位移、反制、群体保护或稳定减益，综合被功能型战士压下去。 |
| Tier 4（17-20） | C | Tier 4 的生存补强让勇士不容易倒，但这只是把战士底盘加厚；终局横向比较看解题面，勇士仍是最窄的官方路线之一。 |

## 不确定点
2024 基准内容按新版职业节奏评价：3 级入口、动作经济和高等级扩展都应能与新版核心规则咬合。

本评分不按社区热度给分，而按同 Tier 队伍功能、资源效率、动作经济和规则可重复性校准。若实桌环境持续奖励 近战伤害者、远程伤害者、防御和生存，该子职会显得更强；若战役重点落在 召唤 / 伙伴、制造、法术伤害者，它的短板会被放大。
