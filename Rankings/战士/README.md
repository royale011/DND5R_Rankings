## 队伍定位总结

战士是队伍最稳定的武器执行者，可以按构筑成为前排锁线、远程火力、机动突击或保护型护卫。它依靠 `战斗风格`、`武器精通`、专长密度、`动作如潮` 和多次攻击把普通攻击转化为可靠输出、位置压力与关键回合爆发；短板是缺少自带法术级反制、传送、复活、召唤和大范围控场，高阶必须靠子职、装备或队友补功能面。

## 来源与版本

本文件按 XPHB 2024 版战士底盘重评，核对来源为 `5etools-src/data/class/class-fighter.json`、`5etools-cn/data/class/class-fighter.json` 与 `DND5e_chm\玩家手册2024\角色职业\战士\战士.htm`。核心特性按本地译名使用：`战斗风格`、`回气`、`武器精通`、`动作如潮`、`战术思维`、`战术转进`、`不屈`、`战术主宰`、`究明攻击`、`传奇恩惠`。`动作如潮`明确不能执行 `魔法` 动作，因此不再按旧版多动作施法爆发计分。

## 详细评价

1级 `战斗风格`、`回气` 和 `武器精通` 决定了战士的早期身份。战斗风格让盾牌、双手、远程、双武器、保护或防御路线立刻成型；回气提供短休续航，2级后还能通过 `战术思维` 补能力检定，若检定仍失败则不消耗回气次数；武器精通从3种武器起步并随等级扩展，把普通攻击接上推离、失衡、减速、干扰或追击，给队伍创造位置和命中窗口。

2级 `动作如潮` 是战士最重要的爆发按钮，能把一次关键回合变成多次攻击、疾走、撤离、互动或其他非魔法动作组合。5级 `额外攻击` 与 `战术转进` 让武器路线稳定成型，并把回气接上无借机半速移动；9级 `不屈` 以战士等级作为重投加值，`战术主宰` 则把已会用的武器精通临时替换为推离、干扰或减速。11级第三次攻击开始把战士和多数半武职拉开。

高等级的战士靠 13级 `究明攻击`、17级第二次 `动作如潮`、19级 `传奇恩惠`、20级第四次攻击和极高专长密度继续保持武器端压迫。限制同样清楚：基础职业没有反制、复活、传送链、召唤、墙或大范围战场重写，遇到高机动施法者、飞行阵地和跨场景问题时，需要子职、装备或队友补齐。

## 分项理由

近战与远程都按 A 处理：战士拥有完整武器熟练、`战斗风格`、`武器精通`、额外专长、`动作如潮`、多次攻击和高阶 `究明攻击`，是武器端最可靠的职业之一；但基础职业本身不提供常态优势源、自动接敌、反飞行或高阶规则改写，所以不单靠底盘进入 S 档。

法术伤害、召唤、制造和反制施法者保持低档。战士底盘不施法，`动作如潮`还排除了 `魔法` 动作；所谓反制主要来自更快击杀、破坏专注、`战术主宰` 的干扰/减速和高等级更多攻击窗口，而不是 `法术反制`、`解除魔法` 或沉默封锁。

防御和生存为 A：重甲/盾牌路线、d10生命骰、`回气`、`战术转进` 与带等级加值的 `不屈` 给了战士很稳定的个人承压能力。治疗只计为 D，因为 `回气` 主要是自我急救，不能替代队伍治疗、防倒地链或临时生命防线。

团队增益、减益、战场分隔和技能使用处于 C 左右。`武器精通` 与 `战术主宰` 能给推离、倒地、减速、Sap 等战术窗口，`战术思维` 让失败检定有一次1d10补救，但这些都是局部、命中/检定驱动的工具，不是大范围控制、群体增益或法术级探索。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | A | `战斗风格`、`回气`、`武器精通` 与2级 `动作如潮` 让低级战士很早就能承担稳定输出、前线压力和一次关键爆发；限制是除 `战术思维` 的检定补救外，治疗、反制和大范围控制都要靠队友。 |
| Tier 2（5-10） | A | 5级 `额外攻击`、更多专长、`战术转进`、`不屈` 与 `战术主宰` 让 Tier 2 战士成为武器职业里的成熟执行者；若子职不补法术、机动或支援，功能宽度仍会落在满施法者之后。 |
| Tier 3（11-16） | A- | 11级第三次攻击、13级 `究明攻击`、带等级加值的 `不屈` 与高专长数量让战士在持久武器战里很可靠；但 Tier 3 遭遇更常考验传送、反制、墙、召唤和群体豁免保护，基础职业只能处理其中一部分。 |
| Tier 4（17-20） | A- | 17级双 `动作如潮`、19级 `传奇恩惠` 和20级第四次攻击提供终局武器爆发与续航；它能稳定打穿许多正面遭遇，却不能单独解决复活链、跨地图移动、反魔法和战役级场景问题。 |

| T1 | A | 奥法骑士 | 3级法术让奥法骑士能反复使用`护盾术`、魔宠、戏法和低环工具，低级生存与侦察明显更好；法术位少和属性压力限制了控制强度。 |

| T1 | A | 武士 | `斗战之魂`的优势加临时生命让低级武士能在关键回合打得准、站得住；限制是次数有限且队伍只获得间接收益。 |

| T1 | B | 勇士 | 低级勇士有19-20重击、先攻优势、运动优势和重击后半速移动，武器手感顺；但它没有新增控制、保护、机动资源或队伍支援，所以仍是朴素武器手。 |

| T1 | B | 骑兵 | `坚定之印`让低级骁骑士有明确保护身份；它必须近战命中并让敌人愿意在防线内行动，场景限制很明显。 |
| T1 | B | 魔射手 | 低级远程安全性不错，两发`奥术射击`能用`放逐矢`、`缠绕矢`、`遮影矢`等在关键目标上打出控制或伤害；次数太少，资源用完后存在感明显下降。 |

## 设计相关评分
| 设计项 | 评分 | 具体理由 |
|---|---|---|
| 设计质量 | A | 2024战士把`武器精通`、`回气`、`战术思维`、`战术转进`、`动作如潮`和`不屈`串成清楚的武器专家框架，低复杂度下仍有战术选择；扣分点是高等级仍缺少战役级工具和非战斗身份扩展。 |
| 主题 | B | “最会使用武器的人”足够清楚，也方便承载各种子职；但基础职业刻意通用，叙事个性弱于圣武士誓言、武僧修行或野蛮人狂怒。 |

## 校准说明

本轮按 XPHB 2024 `战士` 底盘、对应来源子职文本、`不全书.wcp` 的合作内容分割线和当前项目评分标准重写；社区意见只作为强度校准，不覆盖本地规则文本。公开讨论通常把2024战士视为明显比2014版更有战术选择的武器职业，焦点集中在`武器精通`、`战术思维`、`战术转进`、强化后的`不屈`和`动作如潮`禁用`魔法`动作的边界；本文件采纳这些讨论焦点作为校准背景，但最终评分以本地XPHB文本和项目横向标准为准。战士评分按非满施法者标准处理：不因没有高环法术自动扣分，也不因武器 DPR 自动升档；重点看目标接触、动作经济、武器精通、资源续航、队伍杠杆、技能/探索和高等级遭遇适应。

本轮补评的三个非第一方正式子职横向校准如下：UA二期`魔射手（UA）`比 XGE 正式版强一档，主要来自智力调整值次短休`奥术射击`、射击骰成长、`魔箭之矢`和18级反应移动反击，因此稳定停在 A 线；`骁骑士（UA）`比旧骁骑士更顺，`果决印记`不限次数让护卫身份更可靠，但仍受近战触及、防线地形、远程/飞行/传送敌人限制，所以维持 B 到 B+ 线；`附灵（胧忆岛）`比骁骑士更宽，有精华骰攻防修正和多种引灵工具，Tier 2-3 达到 A，但终局没有队伍级传送、反制或大型战场答案，故 T4 收束为 A-。这三项都不进入战士顶层，是因为它们没有 Commander 式队友行动经济、破锋者式大规模拆阵、英雄式群体数学或顶级机动/反制工具。
