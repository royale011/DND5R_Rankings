# 法外狂徒（谦卑林）

所属职业：战士（Fighter）
英文名 / 社区检索名：Scofflaw
来源 / 版本：Humblewood / 谦卑林，第三方

## 队伍定位调整
法外狂徒把战士推向街斗、欺诈和机会主义爆发：它使用临时武器、力量/敏捷社交检定、误导敌人反应攻击、偷袭式 Blindside 伤害和恶名压迫。它比勇士更有战术事件，但比战斗大师、符文骑士和奥法骑士更依赖近距离与敌人吃控制。

## 详细评价
3级附赠熟练项给欺瞒、洞悉、威吓、巧手、隐匿或盗贼黑话，威吓讥讽让战斗中魅力检定可改用力量或敏捷，确实给战士一点非战斗/社交入口。残暴斗殴让临时武器熟练并具灵巧，命中后可用附赠动作毁掉临时武器，把该次武器伤害骰取满；10级再给额外2d6，18级让临时武器攻击有优势。7级弄虚作假用附赠动作逼5尺内敌人智力豁免，失败就用反应攻击另一个目标或浪费反应，能制造队友保护和敌方资源浪费。10级 Blindside 对未行动目标、被你误导过的目标或你有优势的攻击追加5d6，15级7d6，18级9d6，属于强爆发但有休息限制。15级恶名远扬偏叙事压迫，18级一刀不够两刀来创伤提高爆发上限。它强在脏招和爆发，弱在大量能力要求近距离、敌人可理解你的行为，并且临时武器经济不是每桌都稳定。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | B | 临时武器取满骰和力量/敏捷社交检定很有特色，但真正强控制要到7级，低等级仍主要是有风格的近战战士。 |
| Tier 2（5-10） | A | 弄虚作假和 Blindside 让它能制造敌人误击、浪费反应和5d6爆发，Tier 2 的战术存在感明显高于勇士。 |
| Tier 3（11-16） | B | Blindside 伤害提高，恶名远扬增强主题压迫；但高等级敌人更常绕过近身脏招，且没有传送、反制或大范围控制。 |
| Tier 4（17-20） | B | 18级临时武器优势和9d6 Blindside 仍有爆发，但终局综合看解题面，它仍是近身机会主义路线而非战役级工具。 |
