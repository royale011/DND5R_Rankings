# 旗将

所属职业：战士（Fighter）
英文名 / 社区检索名：Banneret / Purple Dragon Knight
来源 / 版本：FRHoF

## 队伍定位调整
旗将想做领袖战士，但旧版数值和触发条件太保守，实际很难承担团队增益主责。

## 详细评价
3级 `激励呐喊` 绑定 `回气` 给队友回血，概念正确但数值小、距离和目标限制明显。它能在低等级救急，却不能和牧师、吟游诗人、圣武士光环或现代临时生命引擎竞争。

7级社交熟练让它更像骑士/军官，但对战斗评分影响很小。10级 `Inspiring Surge` 让队友用反应攻击，是子职最像样的团队输出能力；问题是次数少，且要队友有合适反应和武器回合。

15级 Bulwark 试图补豁免，但触发条件窄。18级没有足够强的队伍级终局能力。社区长期把 Banneret/Purple Dragon Knight 视为低强度设计，原因不是它不能帮助队友，而是它每个帮助都太少、太晚、太受限。

因此旗将的主题可以，但功能兑现不够。若重做，应把 `回气`、`动作如潮`、`威仪风范` 和队友反应形成完整指挥资源，而不是零散小收益。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | B | 旗将低等级至少把回气扩展成队伍治疗/鼓舞，给战士少见的支援身份；但数值和频率都保守，不能和真正辅助职业比较。 |
| Tier 2（5-10） | C | Tier 2 旗将的问题变明显：同级子职开始有战技、法术、符文、回音或灵能，旗将仍主要是把战士底盘附带一点团队修补。 |
| Tier 3（11-16） | C | Tier 3 虽有更多领导型能力，但影响通常依赖队友位置和短资源触发，没有足够强的控场或防护来支撑高评分。 |
| Tier 4（17-20） | C | Tier 4 仍能提供主题化支援，但它的团队收益太小，无法补上战士高等级功能缺口。 |

## 不确定点
FRHoF 内容需要同时考虑 Circle Magic、地区主题和新资源入口对高等级生态的影响。

本评分不按社区热度给分，而按同 Tier 队伍功能、资源效率、动作经济和规则可重复性校准。若实桌环境持续奖励 近战伤害者、远程伤害者、防御和生存，该子职会显得更强；若战役重点落在 召唤 / 伙伴、制造、法术伤害者，它的短板会被放大。
