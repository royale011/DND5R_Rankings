## 队伍定位

邪狱使是围绕武器命中、短休缄印、禁令恩惠和大魔鬼契约运转的第三方武器职业。它不是圣武士换皮：圣武士靠光环、法术表和斩击爆发保护队伍；邪狱使靠 `极恶禁令（Baleful Interdict）` 把资源放到目标身上，再把这些缄印焚灭为伤害、减豁免、传送、减伤、受擒、反制、额外攻击或集火增伤。

当前评估采用项目兼容补丁：1级视为拥有 `武器精通`，19级按2024终局结构获得 `传奇恩惠`。这两个补丁只让它和2024武器职业站在同一基线，不代表它得到完整现代化重写。它仍然是 Revised 1.0 文本：没有官方2024战士/圣武士那种完整底盘更新，也没有高环施法生态。

它的强度来自“资源密度”，不是永久免费伤害。5级 `额外攻击` 后是标准双攻击，11级 `骇怖伟力（Terrorizing Force）` 给每次武器命中+1d8，17级 `炼狱威仪（Infernal Majesty）` 期间升到+2d8并给60尺飞行与火焰/寒冷/暗蚀抗性。缄印本身从1d6、2d6、3d6到20级4d6，短休恢复；13级禁令恩惠还能加入附赠动作攻击、短时加速、受禁目标集火增伤、AC压低、真实视觉或近似反制窗口。问题是这些都在抢同一套资源：缄印既是伤害，也是控制、防御、位移、续航和生存保险。

## 详细评价

1级 `极恶禁令` 是职业循环。每个你的回合一次，你能在30尺内目标上施加一枚隐形缄印：武器命中时无动作施加，或用附赠动作给30尺内可见目标施加。缄印持续1分钟，目标死亡后你还可以用附赠动作把所有缄印转移到其30尺内另一个可见生物。受禁目标受到非邪狱使焚印来源的任意伤害后，你可无动作焚灭任意数量自有缄印，每枚造成火焰或暗蚀伤害；失能时不能焚灭。这里的强点是短休资源能由自己或队友伤害触发，弱点是每回合只上一次缄印，附赠动作也很紧。

2级 `战斗精通（Combat Mastery）` 决定基础构筑。`欺诈（Lies）` 让一类近战武器用魅力攻伤，显著降低筑灭师/狱言家的多属性压力；`无拘（Unfettered）` 把上印/焚印范围扩到60尺，并让6级 `炼狱交移（Infernal Conduit）` 变成30尺，支持远程路线；`蛮横（Brutal）`、`轻盈（Lissome）`、`无阻（Inexorable）`、`傲慢（Bravado）`则分别服务推移、游击、被围生存和魅力AC。2级 `禁令（Interdiction）` 给第一枚禁令恩惠，也正式把“焚灭缄印”和“消耗未施加缄印”分成两类资源开销。`凌虐（Bedevil）` 是早期最重要的队伍向工具：焚印后让目标下一次豁免降低熟练加值；`缓弱缄印（Abating Seal）`、`噬魂夺魄（Soul Eater）`、`冥河寂寒（Styx's Apathy）`、`极速惩戒（Swift Retribution）`则分别给减伤、临时生命、封反应和借机压迫。

T2开始，5级 `额外攻击` 与2d6缄印让武器循环站稳；7级恩惠补传送、束缚、感官共享、+2 AC、范围溅射和远程反击等实战选项。6级 `炼狱交移` 是动作治疗或吸血，11级后 `吞噬（Devour）` 失败目标还会获得一级力竭；它强在救急和压单体，但吃动作、长休骰池和体质豁免，不是治疗职业。10级 `血之代价（Blood Price）` 允许失败豁免后花生命骰并把骰值加入结果，是优秀的 bounded accuracy 防线，但会和短休回血资源竞争。

T3的关键是13级禁令恩惠池。`迪斯的猛攻（Dis's Onslaught）` 让附赠动作施加/转移缄印顺带一次武器攻击；`炼狱狂怒（Hellish Frenzy）` 消耗一枚未施加缄印换速度翻倍、AC+2和攻击动作额外一击；`魄劫（Soul's Doom）` 让受禁目标1分钟内每次受到伤害都额外吃熟练加值；`流星一条（Impaling Shot）` 能压AC，`狱视力（Hellsight）` 给1小时真实视觉，`铁狱（Iron Gaol）` 提供昂贵单体移除，`终言（Last Word）` 是生存保险。完全启动时邪狱使能做出高质量单体回合，但这些强回合不是常驻，因为缄印、附赠动作和反应会同时被输出、防御、控制和逃生占用。

17级 `炼狱威仪` 是终局前最大个人增幅：附赠动作启动10分钟，给火焰/寒冷/暗蚀抗性、60尺飞行、`血之代价`反伤、每击 `骇怖伟力` 由1d8升到2d8，并提供死亡后1d6日在九狱复活的保险。20级 `九狱之主（Master of Hell）` 是长休一次、150尺射程、50尺半径的大范围地狱风暴：炼狱给5d10火焰+5d10暗蚀和持续燃烧，瘟疫给5d10毒素+5d10暗蚀并中毒，深暗给10d10寒冷并留下1分钟敌方目盲区域。它是很好的终局按钮，但仍是一次动作和一次长休，不等于9环施法或战役级改写。
