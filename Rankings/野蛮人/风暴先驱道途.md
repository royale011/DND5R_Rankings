## 队伍定位摘要

XGE风暴先驱道途主题很好，但旧版`风暴灵光`的数值、10尺半径和附赠动作触发节奏都偏保守。放到XPHB 2024野蛮人底盘后，它不再像旧评那样在高阶掉到C档，因为新版底盘的`凶蛮打击`、`坚韧狂暴`和终局数值能托住前排执行力；但子职本身仍更像有元素氛围的小范围前排，而不是能主导战场的风暴化身。后续UA风暴先驱有单独清单项，本条只评价XGE正式版。

## 详细评价

3级`风暴灵光`在狂暴期间形成10尺魔法光环，不穿透全掩护；进入狂暴时触发一次，之后每个自己回合可用附赠动作再触发。你选择沙漠、海洋或冻原，且只有升级时才能更改。沙漠让灵光内所有其他生物受固定火焰伤害：3级2点、5级3点、10级4点、15级5点、20级6点；它会伤到其他生物而非只伤敌人，且数值极小。海洋选择光环内一名可见其他生物，敏捷豁免失败受1d6闪电伤害、成功半伤，10级2d6、15级3d6、20级4d6；这是三者里最像单体附赠动作伤害的路线，但豁免半伤与低骰让它远低于现代子职的稳定输出。冻原则让光环内自选生物获得少量临时生命：3级2点、5级3点、10级4点、15级5点、20级6点；可多人，但每轮小额临时生命很难跟高等级伤害预算同步。

6级`风暴之魂`给对应环境的常驻抗性和小环境适应：沙漠给火焰抗性、无视酷热，并可用动作点燃未被穿戴/携带的易燃物；海洋给闪电抗性、水下呼吸和30尺游泳速度；冻原给寒冷抗性、无视严寒，并可用动作把5尺水立方冻成1分钟冰块，若立方内有生物则失败。10级`风暴之盾`让你光环内自选生物也获得`风暴之魂`的伤害抗性。这个团队抗性在火焰、闪电或寒冷主题战役中可能很好用，但它锁定一种伤害类型、范围只有10尺，还要求队友愿意挤在前排附近；默认遭遇里不如先祖守卫的标记减伤或世界树的位移控位稳定。

14级`狂怒风暴`按环境给最后一个强化。沙漠是在灵光内生物命中你后，你用反应迫使其敏捷豁免，失败受等于野蛮人等级一半的火焰伤害；反应、受击触发、豁免失败和半等级火伤让它仍偏轻。海洋是在你命中灵光内生物时用反应迫使其力量豁免，失败倒地；这比伤害更有战术价值，但消耗反应、要求目标在10尺内且可被倒地有效影响。冻原在每次触发`风暴灵光`时，选择光环内一名可见生物，力量豁免失败速度降为0直到你下回合开始；这是三者中最像控场的结果，但来得太晚，范围窄，且仍被强壮敌人、传送、远程和飞行绕开。公开讨论检索有限，常见印象是XGE风暴先驱主题强但数值保守；本轮仅把这个趋势作背景，最终按本地XGE文本和2024底盘评分。

## 分项理由

`近战伤害者`主要由2024野蛮人底盘支撑；海洋灵光和沙漠灵光提供的伤害太轻，且要消耗附赠动作或有友伤问题，不能作为高档DPR增量。`防御和生存`来自底盘、6级个人抗性、10级队友抗性和高阶`坚韧狂暴`，但抗性类型锁死，且团队抗性要求10尺站位。`团队增益`在特定元素战役中会升值，默认则只是小范围单类型抗性或冻原小额临时生命。`减益`和`战场分隔`到14级后因海洋倒地、冻原速度0才明显改善；之前主要是风暴氛围而非稳定控制。`技能使用`主要仍靠2024`原初学识`，海洋游泳/水下呼吸与冻原冰块有少量探索价值。

## 校准说明

本次重评按XGE正式文本接入XPHB 2024野蛮人底盘，并未采用后续UA风暴先驱更新。旧评的高阶`C / C`上调到`B / B`，是因为2024底盘本身提高了高阶前排下限，且14级海洋/冻原路线确实给出一点控制；但低中阶仍不进入A档，因为光环数值、范围、附赠动作和环境锁定都太保守。综合校准为`B+ / B / B / B`，设计维持`C / A / C / A-`。
