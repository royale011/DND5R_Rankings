## 队伍定位摘要

狂野魔法道途把野蛮人的狂暴变成不可预测的魔法事件，并用`扶持魔法`给队伍提供少见的攻击/检定加值或低环法术位返还。它不是纯娱乐子职：随机表里有重复传送、队友AC光环、敌方困难地形、反伤、致盲射线和力场投掷武器等真实战斗价值；`扶持魔法`也能实打实延长施法者冒险日。它的短板同样清楚：3-13级不能选择涌动结果，关键效果经常和当场需求错位，14级前很难作为稳定控位、保护或反制方案来规划队伍战术。

## 详细评价

3级`魔法感知`用动作开启，直到你下回合结束前，得知60尺内未被全掩护遮挡的法术或魔法物品位置；若感知到法术，还知道其魔法学派。次数等于熟练加值，长休恢复。这不是`侦测魔法`的完整替代品，持续很短，也吃动作；但对野蛮人来说，它提供了罕见的魔法对象定位能力，能帮助队伍确认陷阱、伏击、魔法物品或正在运行的法术效果。

同级`狂野涌动`在进入狂暴时掷d8产生一个持续到狂暴结束的魔法效果，豁免DC为8 + 熟练加值 + 体质调整值。表项范围很宽：30尺内自选生物体质豁免失败受1d12黯蚀伤害并让你获得1d12临时生命；立刻传送30尺且之后每回合可用附赠动作再传送；召唤5尺爆炸灵体，失败受1d6力场伤害，之后每回合附赠动作可再召；让一把武器变力场伤害、获得轻型和投掷20/60尺并回手；被命中时让攻击者受1d6力场伤害；自己和10尺内盟友AC +1；15尺内地面对敌人为困难地形；或30尺内一名其他生物体质豁免失败受1d6光耀伤害并目盲到你下回合开始，之后每回合附赠动作可再用。这个表的上限很好，尤其是重复传送、AC光环、困难地形和重复致盲；问题是低中等级只能随机取得，不能在“需要追后排”时保证传送，也不能在“需要保护队友”时保证AC光环。

6级`扶持魔法`是本子职最稳定的队伍价值。你用动作触碰一名生物，选择10分钟内每次攻击掷骰或属性检定都可加d3，或掷d3让目标恢复一个已消耗、环阶不高于结果的法术位；同一生物每长休只能接受一次法术位返还。使用次数等于熟练加值，长休恢复。d3攻击/检定加值能帮助武器队友、擒抱/推撞/潜行/工具检定，也可给自己；返还1-3环法术位对队伍施法者非常实在，尤其是`护盾术`、`吸收元素`、`治愈真言`、`次等复原术`、`蛛网术`、`法术反制`、`火球术`、`回生术`等关键低中环资源紧张时。它吃动作和触碰，战中启动不轻，但很多时候可以预先使用。

10级`不稳反冲`让你在狂暴期间受到伤害或豁免失败后，用反应重新掷狂野魔法表并立即产生新效果，替换当前涌动。这提高了低结果的纠错率，也让受到压力时有机会换到防御、位移或控场项；代价是消耗反应，且仍然随机。14级`受控涌动`才真正把随机性压下来：每次掷表可以掷两次选一项，若两个结果相同则可无视点数选择任意效果。公开讨论中，狂野魔法道途常被评价为主题有趣、`扶持魔法`被低估、但战斗涌动不够可控；本次采纳这个方向作社区校准背景，最终仍按TCE本地文本和2024野蛮人底盘独立评分。

## 分项理由

`近战伤害者`主要仍来自2024野蛮人底盘；狂野表能偶尔给力场武器、反伤、范围小伤害或附赠动作致盲射线，但不稳定到足以按固定DPR路线计入。`团队增益`是本子职真实亮点：`扶持魔法`返还低环法术位或给攻击/检定加d3，6级起就能改善队伍资源；随机AC光环也可能带来整队防御。`防御和生存`受益于临时生命、AC光环、传送、困难地形和反伤，但其中多数需要掷中。`减益`和`战场分隔`在14级后因双掷选项明显变可靠，能更常拿到致盲、困难地形或传送站位；仍不能等同于稳定法术控场。`技能使用`随XPHB`原初学识`和`扶持魔法`d3加值提高，但魔法感知窗口短，不能替代完整侦察法术。

## 综合评分

| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | A | 2024野蛮人低级底盘已经强，`狂野涌动`有机会给传送、AC光环、困难地形、反伤或致盲；不可控性让它不高于稳定A档前排。 |
| Tier 2（5-10） | B+ | `扶持魔法`提供真实队伍资源价值，能补低环法术位或给攻击/检定加d3；`不稳反冲`提高随机表纠错率。限制是动作、次数、触碰和关键战斗效果仍不可预测。 |
| Tier 3（11-16） | B+ | 14级`受控涌动`让每次涌动双掷择一，显著提高拿到传送、AC光环、困难地形或致盲的概率；但11-13级仍只是随机表，且全阶段没有反制、整队传送或法术级战场改写。 |
| Tier 4（17-20） | B | 终局靠2024底盘、`扶持魔法`、双掷涌动和偶发强结果仍有贡献；但没有17级后新子职爆点，稳定性和工具层级落后于世界树、先祖守卫、巨人等功能型路线。 |

| 主题与能力关联度 | A- | 随机魔法爆发能表达主题；扣分在玩家常常是被动接受结果，而不是主动塑造狂野魔法。 |

## 校准说明

本次重评按TCE正式文本接入XPHB 2024野蛮人底盘。旧评的高阶`C / C`被上调，因为新版底盘本身已托到B档，且`扶持魔法`的法术位返还与d3加值是稳定队伍资源，不应只按“随机好玩”处理。它仍不进入A档：3-13级不能选择涌动结果，14级双掷也只是提高概率，不是稳定选表；高阶没有反制、队伍传送、保护标记或法术级战场处理。因此综合校准为`A / B+ / B+ / B`，设计维持`B / A / B / A-`。
