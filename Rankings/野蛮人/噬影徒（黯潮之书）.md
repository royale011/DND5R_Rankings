# 噬影徒（黯潮之书）

所属职业：野蛮人（Barbarian）
英文名 / 社区检索名：Shadow Gnawer
来源 / 版本：第三方 / 黯潮之书

## 队伍定位调整
噬影徒是暗影烟雾型野蛮人，功能集中在近身减益、短距传送、回复和致盲。它不像腐鸦那样越界成完整工具箱，也不像蔑法道途那样专门反施法；它更像一个把野蛮人周围10尺变成危险暗影区的前排。

## 详细评价
3级 `暗影之烟（Shadow Smoke）` 在狂暴期间散发10尺暗影烟雾，任何在烟雾中开始回合的生物下一次攻击掷骰劣势。这个效果强在无动作、可重复；弱点是范围小且会影响站位，队友如果贴得太近也可能被牵连。它对单体精英和近战怪很烦，但不能阻止豁免法术、传送或远程敌人绕开。

6级 `蔓行之雾（Creeping Fog）` 提供60尺可见未占据空间的魔法传送，并允许传送前后各进行一次近战攻击。这个特性很强：它解决了野蛮人常见的“摸不到人”问题，也让噬影徒可以跨过前排打后排。它不是无限制法术位传送，但对武器前排来说已经是巨大补强。

10级 `吞噬黑暗（Consume Darkness）` 在狂暴期间用附赠动作恢复1d12+体质调整值，并在随后1分钟每回合开始再回1点，每短休一次。它提高生存但数值不离谱，更多是让暗影前排多撑一轮。14级 `蚀雾（Corrosive Haze）` 让在暗影烟雾中开始回合的敌人进行体质豁免，失败则目盲到下回合开始；有盲视或真实视觉者免疫。这个能力把早期攻击劣势升级成真正近身控制，但仍受10尺范围和体质豁免限制。

客观评价：噬影徒是强 B 到 A 档第三方子职。它的传送很实用，14级致盲也有压迫力；但它的核心仍需要敌人留在10尺范围，且没有反制、召唤、团队大增益或战役工具。
