# 实验道途（歪曲之月）

所属职业：野蛮人（Barbarian）
英文名 / 社区检索名：Path of the Experiment
来源 / 版本：第三方 / 歪曲之月

## 队伍定位调整
实验道途偏第三方炼金改造风格，卖点不是单纯“身体变怪”，而是用血清和植入物把野蛮人的狂暴改造成可选战斗模块。按最新校准，第三方身份不再压低强度评分；它要按实际功能评价，而实际文本给出的近战、回复、控场和异常处理都明显高于普通野蛮人子职。

## 详细评价
3级 `炼金实验（Alchemical Experiments）` 给炼金工具熟练，并能用炼金工具制作治疗药水。这不是让野蛮人变成真正制造职业，但它给了本职少见的战斗外工具入口。真正的核心是同级 `不稳定血清（Volatile Serum）`：`狂怒（Ferocity）` 让近战攻击附带额外横扫式伤害，`怪物（Monstrosity）` 给触及、体型和失手补伤，`复原（Reconstruction）` 则在激活和每回合开始给临时生命，并把溢出治疗转成临时生命。这里的评价必须直说：这不是“有趣但保守”的第三方子职，低等级就已经把野蛮人的输出和生存都推高。

6级 `增强化合物（Augmentation Compounds）` 继续抬高上限。`刺激感官（Stimulated Senses）` 给盲视，直接改善黑暗、隐形和近距反潜；`疯狂烟雾（Maddening Fumes）` 是30尺光环，能减速并让敌人对你以外目标攻击具有劣势，这已经接近保护/控场子职的核心职责；`腐蚀喷射（Corrosive Expulsion）` 用一次攻击替换成15尺锥形或30尺线形的力场伤害，让野蛮人获得少见的范围输出。这个等级使实验道途在 Tier 2 不只是强近战，而是带有实质战场压缩能力。

10级 `反应催化剂（Reaction Catalyst）` 很强：受到伤害或陷入目盲、震慑、中毒等状态时可以用反应在伤害或状态生效前狂暴，并在狂暴开始时结束身上的这些状态；狂暴期间还对体质豁免有优势。它解决了野蛮人最怕的“还没开狂暴就被控住”问题，也让本职的异常抗性比多数官方子职更主动。14级 `生化混合（Biochemical Admixture）` 允许同时激活两个血清，并给5尺内近战攻击者每回合一次1d12力场反伤。双血清意味着 `复原` 可以和 `怪物` 或 `狂怒` 同时工作，高等级近战和生存都继续上涨。

客观结论：实验道途在实际强度上应按强子职评价，不能因为第三方来源而故意压低。它没有反制施法、召唤、传送和高环法术，所以不能冲到全生态 S+；但在野蛮人内部，它的回复、近战、范围压力、保护性光环和状态处理已经明显越过普通官方子职边界。设计层面反而要扣分，因为同一子职压入了输出、临时生命、盲视、范围伤害、减速、攻击劣势、反应开狂暴和状态清除，功能密度偏高，容易成为“什么都给一点但数值也不低”的第三方强子职。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | A | 实验道途低等级通常能提供突变式的进攻或防御增益，若抽到/选择到强战斗选项，近战压迫很高；但第三方文本的边界和稳定性比官方子职更需要 DM 校准。 |
| Tier 2（5-10） | A | 6级增强化合物让实验道途拥有盲视、减速/攻击劣势光环或范围力场喷射，队伍功能不再只是个人近战；加上复原血清的临时生命，它在 Tier 2 已经按强野蛮人子职评价。 |
| Tier 3（11-16） | A | 10级反应催化剂能在伤害或目盲、震慑、中毒等状态生效前开狂暴并清除状态，这是野蛮人很少见的主动抗控；它仍无反制和法术解题，但个人可靠性与近战控制足以进入 A。 |
| Tier 4（17-20） | B | 14级生化混合允许双血清并提供力场反伤，终局近战和生存很强；但它仍不能处理传送、反制、召唤、墙和战役级法术问题，所以不应因为第三方数值密度而给到 S 或 S+。 |

## 设计相关评分
| 设计项 | 评分 | 具体理由 |
|---|---|---|
| 设计质量 | C | 主题和战斗路线清楚，但同一子职同时给回复、范围伤害、减速、攻击劣势、盲视、反应开狂暴和状态清除，功能预算明显偏满；强度可以高评，设计不能同样高评。 |
| 主题 | A | 身体实验、变异和野性改造有鲜明辨识度。 |
| 能力设计质量 | C | 血清和化合物选项很有可玩性，但数值、动作经济和功能覆盖面都偏慷慨，尤其复原血清加疯狂烟雾会让前排保护与自保同时过强。 |
| 主题与能力关联度 | A | 用身体变化表达野蛮人力量，主题与能力基本一致。 |

## 不确定点
评分按当前采用版本和 2024 兼容环境统一校准。

本评分不按社区热度给分，而按同 Tier 队伍功能、资源效率、动作经济和规则可重复性校准。若实桌环境持续奖励 近战伤害者、防御和生存，该子职会显得更强；若战役重点落在 法术伤害者、反制施法者、召唤 / 伙伴，它的短板会被放大。
