## 队伍定位摘要

野蛮人是2024规则中最直接的近战承压与力量压迫底盘。它在低中等级靠 `狂暴`、`鲁莽攻击`、`武器精通`、高生命值和力量检定优势把敌人拖进武器交换；到了高等级，它仍能稳定接敌、承伤、推拉、倒地和破坏地形，但没有满施法者的反制、传送、召唤、信息与战役级改写能力。

## 详细评价

1级的 `狂暴` 决定职业节奏：力量攻击附伤、力量检定优势、力量豁免优势和常见物理伤害抗性，让野蛮人在低级能同时承担输出和承伤。2024版持续与维持条件更宽，减少旧版因为目标断档而浪费资源的问题。`武器精通` 同级取得也很关键，`失衡`、`推离`、`削弱`、`顺劈` 等让普通攻击能附带战术效果，而不只是换血。

2级 `鲁莽攻击` 是野蛮人低中等级的命中引擎。它让 `巨武器大师`、暴击收益、`失衡` 和子职附伤更可靠，也会把敌人的攻击优势引到自己身上，形成一种很粗暴但有效的拉仇恨。`危机感应`、高生命和抗性让敏捷伤害场景好处理；真正麻烦的是魅惑、恐慌、放逐、墙、飞行、传送、隐形和远程施法者，这些不是单靠生命值能解决的。

3级 `原初学识` 是新版底盘最重要的非战斗补丁。狂暴期间用力量处理运动、威吓、察觉、潜行、生存等检查，让野蛮人终于能把主属性带进探索和社交压力场景。限制也清楚：它消耗狂暴次数，窗口是10分钟，不能替代游荡者的可靠技能、吟游诗人的社交工具或施法者的信息法术。

5级 `额外攻击`、`快速移动` 和9级 `凶蛮打击` 让野蛮人进入成熟武器职业曲线。`凶蛮打击` 的好处是把 `鲁莽攻击` 的优势转换成额外伤害与控制，当命中率已经足够时很有价值；坏处是面对高AC、必须命中的敌人或有限回合爆发时，放弃优势仍是实际成本。

11级 `坚韧狂暴` 以后，野蛮人更难被打倒，15级后狂暴维持更稳定，20级 `原初斗士` 把力量和体质推到极高数值。这些让它在高等级仍是硬前排，但高等级战斗的核心问题往往是敌人不愿站在地面和你换血：飞行、传送、力场、墙、召唤、反制、豁免失效和战役级机动会把野蛮人的答案面压窄。因此它的高阶评价不能只看能打多少，而要看子职是否补目标接入、队伍保护、控场或反施法。

## 综合评分

| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | A | `狂暴`、`鲁莽攻击` 和 `武器精通` 在低级共同形成高命中、高承伤和稳定近战压迫；此时敌人法术生态尚未完全展开，野蛮人的远程、反制和团队工具短板还没有被放大。 |
| Tier 2（5-10） | B | `额外攻击`、专长、魔法武器和 `凶蛮打击` 让武器回合更有战术选择，但同级圣武士开始用灵光保护队伍，游侠有半施法与技能工具，满施法者开始主导控场；野蛮人仍主要解决能接触的目标。 |
| Tier 3（11-16） | C | `坚韧狂暴`、更稳定的狂暴和子职能力让自己很难倒下，但这一阶段飞行、传送、墙、召唤、反制与大范围控制成为常态；若子职没有额外队伍杠杆，底盘本身难以影响整场遭遇。 |
| Tier 4（17-20） | C | `原初斗士` 让数值非常漂亮，近战命中、伤害和生命仍有现实价值；问题是终局队伍更看重跨场景解题、反制与行动经济压缩，野蛮人默认仍是优秀执行者而不是战役级引擎。 |

| T1 | A | 兽心道途 | 野蛮人低级底盘已经强，3级模式让它可按遭遇补防御、机动或协同；此时适应性很实用。 |
| T1 | A | 图腾武者道途 | 熊图腾让狂暴抗性覆盖大量伤害类型，低级敌人很难通过伤害交换击穿你；但子职主要吸收自己受到的压力，尚未提供队伍位移或控制。 |
| T1 | A | 巨人道途 | 低级仍以野蛮人底盘近战为主，体型和触及让威胁范围更好；子职已经改善站位，但核心元素投掷尚未完全展开。 |
| T1 | A | 狂战士道途 | `狂怒` 直接挂在野蛮人低级高命中攻击上，输出干净可靠；此时缺少复杂队伍工具的问题还不致命。 |
| T1 | A | 狂热者道途 | `神性之怒` 稳定附伤且不抢动作，直接叠在野蛮人底盘近战上。 |
| T1 | A | 野兽道途 | 自然武器低级输出和防御都实用，爪击尤其能提高攻击次数并放大狂暴附伤。 |
| T1 | B | 狂野魔法道途 | 野蛮人底盘保证可玩，低级随机涌动有时很强；但结果不可控，不能按战术需要稳定兑现。 |
| T1 | B | 风暴先驱道途 | 野蛮人底盘保证低级可玩，风暴光环有主题但数值小、半径窄，子职增量有限。 |
| T1 | C | 战狂道途 | 野蛮人底盘仍能打，低级铠刺给一点额外攻击感；但装备锁定和伤害效率已经明显拖后腿。 |

| 阶段 | 评级 | 子职 | 具体理由 |

| T2 | A | 先祖守卫道途 | `精魂护盾` 让反应稳定削减队友伤害，子职增量补足普通野蛮人缺少保护他人的问题；限制是反应次数和站位。 |
| T2 | A | 图腾武者道途 | 广谱抗性在龙息、元素伤害和怪物附伤增多后仍有高价值；限制是主动控场、保护队友和目标接入少。 |
| T2 | A | 巨人道途 | `元素战刃` 补伤害类型和中距投掷，配合 `额外攻击` 后比普通野蛮人更不怕距离；限制是仍靠武器命中。 |
| T2 | A | 狂战士道途 | `无我狂暴` 解决魅惑/恐慌痛点，让近战循环更不容易被控制拆掉；限制是它仍主要提升自己，队伍操作少。 |
| T2 | A | 狂热者道途 | `专心炽志` 补关键豁免，降低野蛮人被控制踢出战斗的风险；输出和可靠性都比普通底盘好。 |
| T2 | A- | 兽心道途 | 中等级模式与探索适应让它比普通野蛮人宽，但子职增量主要服务自己，不如世界树的位移或先祖守卫的保护能改变队伍回合。 |
| T2 | B | 狂野魔法道途 | `扶持魔法` 提供真实队伍资源价值，子职不只是娱乐；限制是次数、随机性和对关键回合的不可预测。 |
| T2 | B | 野兽道途 | 移动适应和野兽形态让它比普通野蛮人更自给，但队伍保护、反制和控位工具少。 |
| T2 | B | 风暴先驱道途 | 抗性和环境适应在特定战役有价值，但默认遭遇中不如保护、传送或强制移动路线稳定。 |
| T2 | D | 战狂道途 | 同级子职开始提供保护、传送、抗性或控位，战狂仍围绕低收益铠刺循环，几乎没有补职业短板。 |

| T3 | C | 狂野魔法道途 | 随机表强化仍不足以解决飞行、反制、传送和法术级战场分割，高等级可靠性明显落后。 |
| T3 | C | 野兽道途 | 爪、尾、咬和机动适应仍有辨识度；到高等级后，它们没有升级成可靠控场、队伍机动或反施法入口。 |
| T3 | C | 风暴先驱道途 | 较有力的反应控制来得偏晚，且仍是小范围元素前排，不能解决高等级核心问题。 |
| T3 | E | 战狂道途 | 高等级铠刺没有升级成有效队伍功能，底盘也不足以掩盖子职空转。 |

| T4 | C | 野兽道途 | 终局自给近战仍可玩，能处理一部分地形和生存问题；综合功能仍停在中低档。 |
| T4 | C | 风暴先驱道途 | 终局主题仍清楚，但数值和作用范围没有跟上全生态；面对飞行、传送和施法者时答案不足。 |
| T4 | E | 战狂道途 | 终局基本只剩角色概念和野蛮人底盘，作为子职增量在同职业横向垫底。 |
