# 野蛮人总评

## 队伍定位摘要

野蛮人是 2024 规则下更健康的近战前排：低中等级靠 `狂暴`、`鲁莽攻击`、`武器精通` 和高生命值形成强压迫；高等级仍被功能面限制，主要负责接敌、承伤、推拉、倒地和力量检定，而不是法术级解题。

## 详细评价

1级 `狂暴` 是职业核心：提供力量攻击伤害、力量检定优势、力量豁免优势和常见武器伤害抗性。2024 版持续时间和维持条件更宽，减少了旧版“打不到人就断资源”的挫败。`武器精通` 同级取得也很重要，`失衡`、`推离`、`削弱`、`顺劈` 等效果让普通攻击附带倒地、推位、压低敌方命中或清小怪功能。`无甲防御` 主题强，但优化上中甲经常更省属性，所以不能把它当作稳定高 AC 来源。

2级 `鲁莽攻击` 是野蛮人 Tier 1 近战高分的核心。稳定优势让 `巨武器大师`、暴击收益、`失衡` 命中和子职附伤都更可靠；代价是敌人攻击你也有优势，因此它同时是增伤和拉仇恨机制。`危机感应` 改善敏捷豁免，配合高生命和抗性让低中等级生存很硬，但精神控制、放逐、困惑、墙、飞行和传送仍是野蛮人的结构性弱点。

3级 `原初学识` 是 2024 底盘最重要的非战斗补强：狂暴期间若干技能能改用力量，运动、威吓、察觉、潜行、生存等终于能和主属性挂钩。它改善了野蛮人战斗外无事可做的问题，但受狂暴次数和10分钟窗口限制，不能和游荡者、吟游诗人或满施法者的信息/工具能力相比。

5级 `额外攻击` 和 `快速移动` 进入正常武器职业曲线。9级 `凶蛮打击` 牺牲 `鲁莽攻击` 的优势来换额外伤害和控制选项，是新版最好的设计之一：当命中已经足够时，野蛮人能把优势转化为推、慢、干扰或战术位移。限制是面对高 AC 或必须命中的目标时，放弃优势会让输出可靠性下降。

11级 `坚韧狂暴`、15级 `持久狂暴`、18级 `不屈勇武`、20级 `原初斗士` 都在强化“我还能站着、我还能做力量事”。这符合职业幻想，但高等级横向比较看的是传送、反制、召唤、墙、信息法术、复活和现实改写。野蛮人能把可接触目标打碎，也能承受很多伤害；它不能独立解决高等级战役题。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | A | `狂暴`、`鲁莽攻击` 和 `武器精通` 在低级非常强，缺少高阶法术生态时近战和承伤价值直接；短板是远程、法术、反制和团队工具基本空白。 |
| Tier 2（5-10） | B | 5级 `额外攻击`、专长、魔法武器和9级 `凶蛮打击` 让武器回合更有战术，但队伍级功能仍明显少于圣武士、游侠和满施法者。 |
| Tier 3（11-16） | C | `坚韧狂暴` 与子职能力让自己更难倒下，但 Tier 3 开始飞行、传送、墙、反制和召唤成为常态，野蛮人的答案面明显不足。 |
| Tier 4（17-20） | C | `原初斗士` 很符合主题，近战数值仍能看；但 Tier 4 比拼战役级工具，野蛮人仍主要解决能摸到的目标。 |
