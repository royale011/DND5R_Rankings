## 队伍定位

铳士是以远程武器、枪械、扩展重击范围、短休 `风险骰（Risk Dice）` 与战技为核心的第三方武器职业。它的主职能是先手远程压制、单体伤害、用重击触发的 `破脏弹（Gut Shot）` 给目标速度减半和攻击劣势，并用通用战技补机动、防御、失手补伤和心智豁免补救。

它不是“有枪的战士”。战士靠更多攻击次数、动作爆发和强通用专长经济滚上限；游荡者靠技能/专精和一次高价值攻击维持场景存在感。铳士介于两者之间：它有强先攻、短休资源和枪械骰，但没有战士 11/20 级的第三/第四次攻击，也没有游荡者的可靠才能/专精体系。其高阶价值主要来自更宽的远程重击范围、11级补上枪械属性伤害、18级额外战技附赠动作，以及20级 `爆头（Headshot）` 的处决威胁。

默认起手 `Revolver` 是重要校正点：它是 2d6 穿刺、射程 30/120、`Firearm`、`Recoil`、`Reload (6)`、`Slow` 精通的简单远程武器。由于 `Firearm` 默认不加敏捷调整值，低级枪械的纸面伤害没有“2d6+敏捷”那么高；但 `Reload (6)` 也意味着它并不会像 `Loading` 武器那样每回合天然卡住 `额外攻击`，只是六发后要用动作或附赠动作换弹。铳士通用战技 `翻滚闪避（Dodge Roll）` 还能用附赠动作移动并换弹，所以本职业的真实限制是附赠动作竞争、弹药/价格/魔法枪械可得性、枪械时代准入，以及缺少施法级解题。

## 详细评价

1级给 `战斗风格（Fighting Style）`、`快速拔枪（Quick Draw）` 与 `武器精通（Weapon Mastery）`。`快速拔枪` 的先攻优势非常实用，低等级常常等于先处理一名威胁；双拔/收武器让多枪、备用武器和近距离切换更顺。`战斗风格` 可以把通常要求近战武器的风格用于远程武器，扩大构筑空间。`武器精通` 起步两项，后续升到四项，能覆盖左轮的 `Slow`、近距/特殊枪械的 `Scatter`、长距枪械的 `Sighted` 或普通远程武器的既有精通。

2级 `夺命重弹（Critical Shot）` 让远程武器攻击在19-20重击，9级变18-20，17级变17-20。它适用于远程武器而非只限枪械，所以铳士在缺弹、枪械受限或需要魔法武器时仍可回到长弓/弩路线。`风险（Risk）` 提供短休恢复的风险骰：2级4d8，6级5d8，10级5d10，14级6d10，18级6d12。通用战技包括临时生命、短时盲视、移动并换弹、失手补伤、心智属性检定/豁免补救、反应加AC。这一组让铳士比普通弓手更能自救，但风险骰同时承担输出、防御、移动和豁免，不能无脑全部转成伤害。

5级 `额外攻击（Extra Attack）` 是武器职业标准断点；`Revolver` 的6发弹容足以支撑数轮多攻，之后需要动作或附赠动作换弹。`枪手（Gunner）` 对这里的帮助有限，因为它忽略的是 `Loading` 而不是 `Reload`；更关键的是职业自己的 `Dodge Roll` 与战术换弹。5级 `破脏弹` 是铳士最有价值的控制来源：远程武器重击命中不超过大型的生物后，目标速度减半并攻击劣势，直到它用一次攻击剥离弹体。配合19-20/18-20重击，铳士能把个人暴击转成队伍防御收益。

7级 `反射闪避（Evasion）` 抬高生存。11级 `斩尽杀绝（Overkill）` 是枪械伤害的真正成熟点：如果远程武器本来不加属性调整值，就补上；如果已经加了，则额外1d8。这让 Valda 枪械从“高骰但无敏捷”转成合格高阶远程武器，也让非枪械路线得到补偿。13级 `死亡欺诈（Cheat Death）` 是短休一次的倒地保险；15级 `极险决策（Dire Gambit）` 通过先攻和重击回收风险骰；18级 `灵巧战技（Deft Maneuver）` 每回合给一个只能用于战技的额外附赠动作，明显缓解换弹、移动、防御和补伤的动作拥堵。

20级 `爆头（Headshot）` 在远程武器重击时可对低于100生命值的生物直接击杀，否则追加10d10同类型伤害；短休恢复，也可消耗三枚风险骰重置。它的终局辨识度很强，且17-20重击范围让触发不再罕见；但它仍依赖命中、重击、目标生命值门槛、传奇/首领生态和武器射击环境，不能等同于高环法术的战役级解题。

综合看，铳士的强项是清楚、可靠、资源短休、个人枪战体验完整；弱项是团队功能窄、没有施法级信息/传送/反制/大范围控制，且武器包的桌面准入会显著影响体验。按当前源文本给它比旧“只按DMG文艺复兴枪械”更高的中高阶评价，但仍不把现代枪械或魔法枪械经济当成默认上限。
