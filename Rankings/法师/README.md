# 法师（Wizard）总评

## 队伍定位摘要
法师是学术型奥术满施法者，也是高等级规则解题能力的标杆。它不靠单个职业特性打数字，而靠法术书、仪式、短休换准备法术和完整法师法术表覆盖几乎所有非治疗问题。

## 详细评价
2024法师的基础强度来自法术书，而不是子职。1级`施法`给法术书、长休更换准备法术和每级两道新法术；`仪式学家`允许你以仪式施展法术书中带仪式标签的法师法术，即使没有准备。`奥术回想`在短休后恢复总环阶不超过法师等级一半向上取整、且单个低于六环的法术位，使法师的每日有效法术位高于表面数字。

2级`学者`让一个已熟练的学术技能获得专精，法师在奥秘、历史、调查、自然、宗教等知识位天然强。3级子职之后，5级`记忆法术`又允许每次短休研究法术书，把一道已准备的一环以上法术替换为书中另一道。这是2024法师极重要的质量提升：它不要求玩家早晨完全猜中当天全部问题，给了法术书真正的临场价值。

18级`法术精通`把一道动作施法的一环法术和一道二环法术变成最低环无限施放且总是准备；20级`招牌法术`给两道三环法术各一次免费三环施放并短休/长休恢复。高等级法师因此同时拥有九环法术、低环常驻工具和短休三环补给。弱点仍然是d6生命骰、无护甲、武器差、治疗差；但队伍只要能保护法师，法师就是最宽的奥术解题核心。

## 分项理由
- **战场分隔 / 减益**：法师法术表拥有同阶段最强的控制和分隔工具，子职只是改变兑现方式。
- **技能使用**：`学者`给学术技能专精，仪式法术和预备法术使信息处理很强。
- **制造**：抄录、卷轴、法术书与大量工具法术让法师比多数职业更能利用时间和金币。
- **治疗和临时生命**：基础法师没有稳定治疗职责，通常只能靠特定子职、法术、专长或物品补。

## 校准说明
本轮依据`DND5e_chm\玩家手册2024\角色职业\法师`评估。社区校准参考2024法师与子职评论：法师底盘仍被视作顶级满施法者，预言/防护/幻术评价高，塑能更偏专精爆破。

## 分阶段子职排行榜

Tier 1（1-4）:
- S:
  - 时间魔法（EGW）：2级旧特性按兼容视为3级；`时间变幻`重骰和智力加先攻已经是顶级低级控制底盘。
  - 预言师：3级`预兆`能改写关键d20，低级即能决定命中、豁免或检定结果。
- A:
  - 书士会：按2024兼容，2级`法师之笔`和`觉醒魔典`视为3级获得，抄录与伤害类型转换极实用。
  - 剑咏者：3级`剑歌`把智力加入AC和专注，且武器攻击可用智力，低级防线明显强。
  - 幻术师：3级`强化幻术`移除幻术言语成分、延长射程并强化`次级幻象`，低级欺瞒/控场很强。
  - 战争魔法：按2024兼容，2级特性视为3级获得；`奥术偏斜`和`战术之智`让低级防线与先攻显著提升。
  - 重力魔法（EGW）：2级旧特性按兼容视为3级；`调节密度`能改速度和力量检定/豁免，但专注和动作成本明显。
  - 防护师：3级`奥术守御`直接补法师最薄的承伤面，低级比基础法师稳得多。
- B:
  - 咒法学派：按2024兼容，2级特性视为3级；`次级咒术`很有用但偏创造小物件。
  - 塑能师：3级`强力戏法`提高低级伤害下限，但子职功能集中于伤害。
  - 惑控学派：按2024兼容，`催眠凝视`视为3级获得，控制强但5尺距离和动作续持很危险。
  - 死灵学派：按2024兼容，`惧怖收割`视为3级获得；杀敌回血有用但不能稳定规划。
- C:
  - 变化学派：按2024兼容，`次级炼金`视为3级获得；低级偏工具玩法，战斗影响小。

Tier 2（5-10）:
- S+:
  - 时间魔法（EGW）：10级前已靠先攻、重骰和`瞬时静止`很强；10级`奥法暂滞`把四环以下法术交给他人释放，触及全体系上限。
- S:
  - 书士会：6级`神识显现`让法师从灵体位置施法和侦查，安全性与战术角度很强。
  - 剑咏者：6级`额外攻击`可把一次攻击替换为动作戏法，近战与施法循环成形。
  - 幻术师：6级`魅影生灵`给`野兽召唤术`和`妖精召唤术`，并各有一次半血无槽幻术版本，功能面大增。
  - 战争魔法：6级`魔力潮涌`虽小，但战争魔法已靠高先攻和反应防御进入强势。
  - 防护师：6级`投射守御`把结界变成队友减伤，10级前后的反制/解除能力进入顶级。
  - 预言师：6级`专业预言`让二环以上预言法术回收低环法术位，信息/资源循环很强。
- A:
  - 咒法学派：6级`王车易位`给30尺传送/换位，机动实用但动作成本较旧。
  - 塑能师：6级`法术塑形`让范围塑能法术不伤队友，爆破实战可用性明显上升。
  - 惑控学派：6级`直觉魅惑`能转移攻击，防御实用但对免疫魅惑敌人无效。
  - 死灵学派：6级`不死奴仆`让`活化死尸`更强，召唤/仆从路线成形。
  - 重力魔法（EGW）：6级`重力井`给法术命中/豁免失败后的5尺移动，控制细节很强但不破格。
- B:
  - 变化学派：6级`变化师之石`能给体质豁免、速度或抗性，实用但不惊艳。

Tier 3（11-16）:
- S+:
  - 时间魔法（EGW）：`奥法暂滞`允许队友释放你的法术，改变专注、动作和施法者位置的分工，S+成立。
- S:
  - 书士会：10级`书士宗师`每天制造强化低环卷轴，并半价半时制造卷轴，资源经济优秀。
  - 剑咏者：10级`守御之歌`可用法术位减伤，维持近战法师安全。
  - 幻术师：10级`幻影化形`可反应令一次命中自动失手，防御补齐后维持S。
  - 战争魔法：10级`耐久魔法`在专注时AC和所有豁免+2，维持关键法术极强。
  - 防护师：10级`破法者`让`法术反制`和`解除魔法`更可靠，团队安全网非常强。
  - 预言师：10级`天眼通`补黑暗视觉、读懂语言或`识破隐形`，高环法师工具箱更完整。
- A:
  - 咒法学派：10级咒法专注不会因伤害中断，召唤和持续咒法法术稳定性提高。
  - 塑能师：10级`强效塑能`加智力调整值，伤害稳定但不扩大法师解题面。
  - 惑控学派：10级`分裂惑控`双目标单体惑控法术很强，终于进入核心强点。
  - 重力魔法（EGW）：10级`狂暴牵引`强化武器命中或坠落伤害，团队配合价值上升。
- B:
  - 变化学派：10级自我`变形术`限制为CR1野兽，强度偏保守。
  - 死灵学派：10级暗蚀抗性和生命上限保护偏窄，综合不如其他高影响法师子职。

Tier 4（17-20）:
- S+:
  - 时间魔法（EGW）：14级`收束未来`可保证成功或失败，虽有力竭代价，但高等级关键检定价值压过代价。
- S:
  - 书士会：14级`书人合一`可用反应避免全部伤害，代价沉重但终局保命强。
  - 剑咏者：14级`胜利之歌`在动作法术后附赠动作武器攻击，终局动作经济很强。
  - 幻术师：14级`亦真亦幻`把幻术物件变真实，解题与战场分隔上限极高。
  - 战争魔法：14级`偏斜罩幕`给偏斜附带力场伤害，终局仍是最硬法师之一。
  - 重力魔法（EGW）：14级`事件视界`形成30尺重力场，减速、力场伤害和移动惩罚让终局进入S。
  - 防护师：14级`法术抗性`给法术豁免优势和法术伤害抗性，高等级反施法环境中仍是S。
  - 预言师：14级三枚`预兆`使关键结果操控次数增加，终局仍是最强正式法师子职之一。
- A:
  - 变化学派：14级`变化宗师`能治疗、复活或物质转化，终于有高影响团队价值。
  - 咒法学派：14级召唤生物获得30点临时生命值，召唤路线可靠但不如UA二期全面。
  - 塑能师：14级`超限导能`可最大化一至五环伤害法术，爆发强但有暗蚀反噬，综合仍为A。
  - 惑控学派：14级`编演记忆`偏社交/叙事，强度很看战役。
  - 死灵学派：14级`命令亡灵`遇到强亡灵时很强，但高度依赖战役敌人类型。

## UA/合作方/第三方子职分阶段排行榜

Tier 1（1-4）:
- S:
  - 司书（克苏鲁）：3级`编写法术`可给选定法术DC+1、改伤害或改为智力豁免，低级已是S级控制兑现。
  - 时间魔法（EGW）：2级旧特性按兼容视为3级；`时间变幻`重骰和智力加先攻已经是顶级低级控制底盘。
- A:
  - 变化师（UA二期）：3级`变化师之石`给体质豁免熟练和可调增益，低级比旧变化强很多。
  - 咒法师（UA二期）：3级`王车易位`附赠动作传送且次数按智力，低级机动显著强于旧版。
  - 地脉魔法（谦卑林）：`自然调谐`按环境给额外始终准备法术，低级已经明显扩大法师工具箱。
  - 惑控师（UA二期）：3级智力加社交技能和新版`催眠仪态`，低级比旧版更安全但仍需近距离。
  - 折纸魔法师（胧忆岛）：`奥术纸艺`和`折纸魔宠`低级就给技能、AC、目盲和侦查，功能面很宽。
  - 暗影（黯潮之书）：`暗影共生`给魔法黑暗视物和黑暗视觉，但阳光敏感让低级环境波动很大。
  - 死灵师（UA二期）：3级暗蚀抗性、亡灵魔宠和死灵法术治疗亡灵，低级已有明确亡灵体系。
  - 神秘学者（歪曲之月）：`侵扰`可让法术目标首次豁免劣势或法术攻击优势，低级控制兑现率很高但有风险。
  - 贤者（歪曲之月）：`精粹结晶`能替代材料、抄录和制造成本，并把伤害改为力场，低级资源经济很强。
  - 远古魔杖学（狮鹫鞍包）：`精髓魔杖`可把充能加到攻击、豁免或单次AC，低级已明显突破普通防御下限。
  - 重力魔法（EGW）：2级旧特性按兼容视为3级；`调节密度`能改速度和力量检定/豁免，但专注和动作成本明显。
- B:
  - 血魔法（塔尔多雷）：2级旧特性按兼容视为3级；`绯红爆发`用生命换伤害重骰，低级提升有限且风险真实。
