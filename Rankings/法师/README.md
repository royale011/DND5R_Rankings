## 队伍定位摘要
法师是学术型奥术满施法者，也是高等级规则解题能力的标杆。它不靠单个职业特性打数字，而靠法术书、仪式、短休换准备法术和完整法师法术表覆盖几乎所有非治疗问题。

## 详细评价
2024法师的基础强度来自法术书，而不是子职。1级`施法`给法术书、长休更换准备法术和每级两道新法术；`仪式学家`允许你以仪式施展法术书中带仪式标签的法师法术，即使没有准备。`奥术回想`在短休后恢复总环阶不超过法师等级一半向上取整、且单个低于六环的法术位，使法师的每日有效法术位高于表面数字。

2级`学者`让一个已熟练的学术技能获得专精，法师在奥秘、历史、调查、自然、宗教等知识位天然强。3级子职之后，5级`记忆法术`又允许每次短休研究法术书，把一道已准备的一环以上法术替换为书中另一道。这是2024法师极重要的质量提升：它不要求玩家早晨完全猜中当天全部问题，给了法术书真正的临场价值。

18级`法术精通`把一道动作施法的一环法术和一道二环法术变成最低环无限施放且总是准备；19级`史诗恩惠`默认推荐`法术回想恩惠`，20级`招牌法术`给两道三环法术各一次免费三环施放并短休/长休恢复。高等级法师因此同时拥有九环法术、低环常驻工具、史诗专长和短休三环补给。弱点仍然是d6生命骰、无护甲、武器差、治疗差；但队伍只要能保护法师，法师就是最宽的奥术解题核心。

## 分项理由
- **战场分隔 / 减益**：法师法术表拥有同阶段最强的控制和分隔工具，子职只是改变兑现方式。
- **技能使用**：`学者`给学术技能专精，仪式法术和预备法术使信息处理很强。
- **制造**：抄录、卷轴、法术书与大量工具法术让法师比多数职业更能利用时间和金币。
- **治疗和临时生命**：基础法师没有稳定治疗职责，通常只能靠特定子职、法术、专长或物品补。

## 校准说明
横向校准后，法师仍是满施法 S 标杆：法术书、仪式、奥术回想、短休记忆法术、高环法师表、法术精通和招牌法术共同提供最宽的奥术答案库。子职评级不再按 法师底盘强所以全员 S处理：预言、防护、幻术等能提高关键结果或团队防线的子职保留高档；塑能、变化、死灵等窄增量子职即使成品角色强，也会因子职贡献范围受限停在 A+/S- 边界。EGW与UA子职按实际强度评估，时间魔法、学识掌控这类真正改写 d20、施法投放或法术参数的选项才保留 EX。

2026-06-15重评依据`5etools-src/data/class/class-wizard.json`与`5etools-cn/data/class/class-wizard.json`核对XPHB文本。社区校准参考RPGBOT 2024 Wizard Handbook与2024法师公开讨论：外部评价普遍把法术书、仪式、`记忆法术`和法师法表视为新版法师继续站在满施法顶层的原因，同时也指出身体防线、治疗缺口、抄录/卷轴环境和高环法术桌裁会显著影响体感。本轮综合评分维持`A / S / S / S`。

2026-06-15官方法师横向校准确认：本轮已逐项重评XPHB法师底盘、FRHoF剑咏者、TCE书士会、XGE战争魔法、XPHB防护师/预言师/塑能师/幻术师、PHB 2014遗留咒法学派/惑控学派/死灵学派/变化学派，以及EGW例外时间魔法/重力魔法。校准后官方梯队为：时间魔法从Tier 2起为EX，因`奥法暂滞`和`收束未来`改写施法递送、专注归属和d20成败；预言师、防护师、幻术师是S+层，分别代表d20替换、队伍防线/反魔法、现实塑形；重力魔法高阶为S，强在空间压迫和秘迹学法术但成本真实；剑咏者、书士会、战争魔法与PHB咒法/惑控/死灵的成品阶段处于S-层，但前三者更偏个人稳定/法术书运营/防御，后三者分别受召唤生态、魅惑免疫和亡灵后勤限制；塑能师与变化学派维持A+顶点，前者增量集中在伤害，后者偏适配和高阶救急。官方结论不混入UA、合作方、第三方与神圣奇术映射条目。

2026-06-16 Strixhaven法师UA横向校准确认：`衡鉴魔法师（5E UA）`、`粹丽魔法师（5E UA）`、`量析魔法师（5E UA）`、`银毫魔法师（5E UA）`均已按3/6/10/14完整映射重评。量析是该批最高，T2后进入S，因为它同时给免费入书扩表、`函数概率`、`速度转移`、`空值方程`和`量子隧穿`；衡鉴、粹丽、银毫均维持S-，分别代表伙伴/跨表支援、机动/抗性/敏捷豁免稳定性、成功后反应重掷与短时情绪控制。该批整体低于`学识掌控（5E UA）`、`时间魔法（EGW）`等结果或法术生态改写者，也没有改变2026-06-15官方法师与官方基准职业校准结论。
