# 剑咏者

## 队伍定位摘要
剑咏者是法师中最成功的战斗法师路线之一。FRHoF版本把智力武器攻击和奥术回想回剑歌次数写进文本，使它比旧版更像真正单属性近战法师。

## 详细评价
3级`剑歌`用附赠动作开启1分钟，要求不穿护甲、不用盾、不双手使用武器攻击。期间AC加智力调整值、速度+10、敏捷（特技）优势；使用熟练武器攻击时可用智力代替力量或敏捷；维持专注的体质豁免可加智力调整值。次数等于智力调整值，长休恢复，且使用`奥术回想`时恢复一次。`战歌训练`给不具双手和重型词条的近战军用武器熟练，并可用熟练近战武器作法器，还给一项技能熟练。

6级`额外攻击`允许攻击动作攻击两次，并可把一次攻击替换为施法时间为动作的法师戏法。这是剑咏者的核心战斗循环：武器、戏法和防御法术并行。10级`守御之歌`在剑歌期间受伤时用反应消耗法术位，降低等于法术位环阶5倍的伤害，能把致命打击压成可承受伤害。

14级`胜利之歌`在你施展施法时间为动作的法术后，可用附赠动作进行一次武器攻击。这个版本的剑咏者高等级动作经济非常好，但它仍然是法师：真正上限来自满施法者法术表，而不是武器DPR。评分给S是因为它把法师底盘和个人防御结合得极好，不是因为它比专职武者更会砍。

综合判断：剑咏者不再和预言师、防护师并列纯S。它很强，尤其在AC、专注、近战/远程武器+戏法循环上极稳；但这些多数是个人防御和个人行动顺滑，不等于把队伍胜率提升到同等层级。

## 分项理由
- **近战伤害者**：能近战并有智力武器攻击，但主要仍是法师，不是纯武者。
- **防御和生存**：`剑歌`、专注加值和`守御之歌`极强。
- **法术伤害者**：满法师底盘不受损，14级还可动作法术后附赠攻击。
- **团队增益**：主要靠法师法术表，子职本身偏个人战斗。

| 阶段 | 综合强度 | 具体理由 |

| Tier 1（1-4） | A | 3级`剑歌`把智力加入AC和专注，并改善移动；低级防线很强，但主要是个人生存。 |
| Tier 2（5-10） | S- | 6级`额外攻击`可把一次攻击替换为戏法，个人输出和动作顺滑度优秀；仍不如预言/防护那样直接改变全队成败。 |
| Tier 3（11-16） | S- | 10级`守御之歌`用法术位减伤，14级前后个人专注与生存极强；综合为S-，因为贡献集中在自己不倒和自己输出。 |
| Tier 4（17-20） | S- | 终局仍是完整法师加顶级个人防线；但没有全队d20操控、反魔法可靠性或法术投送破格，因此保持S-。 |
