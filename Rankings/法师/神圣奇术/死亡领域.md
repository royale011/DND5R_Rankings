# 神圣奇术（5E UA） - 死亡领域

## 队伍定位调整
选择`死亡领域`后，这个神圣奇术法师的队伍位置由“法师控制/解题核心”向该领域的神术功能倾斜。以下只评价`死亡领域`的领域法术、映射特性，以及它们和法师+神圣奇术底盘的协同或冲突；神圣奇术通用规则见本目录`README.md`。

## 详细评价
**领域法术与法师法术书。** 领域法术包偏暗蚀、疾病、反生命和死亡伤害，能给法师补暗蚀输出角度；但它没有明显扩大法师的解题广度，更多是把伤害路线变窄且变凶。

**映射特性与协同。**
死亡领域给神圣奇术法师额外死灵伤害和近战/暗蚀爆发角度，但许多价值原本假设牧师站得更近。
3级引导神力可转化为单次暗蚀爆发，低级对法师有真实增量，但距离和近战站位仍危险。
6级取得死亡领域3级非引导神力后，死灵戏法/双目标类收益与法师法表有重叠，强但不宽。
10级取得6级忽略抗性类能力后，死灵伤害可靠性改善；然而法师更常靠控制赢战斗，不靠暗蚀DPR。
14级取得17级扩散类能力后，伤害上限提高；终局仍是强法师加伤害子职，而不是S级现实改写。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | A+ | 低级暗蚀爆发有用，但站位和伤害专精限制明显，评A+。 |
| Tier 2（5-10） | S- | 中期法师底盘进入S级，死亡领域补可靠暗蚀输出；子职窄，结果为S-。 |
| Tier 3（11-16） | S- | 高环阶段主要靠法师控制，死亡领域改善伤害而非战役解题，保持S-。 |
| Tier 4（17-20） | S- | 终局扩散伤害可用，但不是全队防线、反制或法术投送破界。 |

## 设计相关评分

|---|---|---|
| 设计质量 | B- | 死亡领域给法师伤害主题很清楚，但不少收益要求接近敌人或围绕暗蚀输出，和法师强控底盘关联不够自然。 |
| 主题 | A | `死亡领域`为神圣奇术提供了清楚的神祇/领域身份，能解释为什么这个法师会把奥术研究导向相应神术主题。 |
| 能力设计质量 | B- | 该映射的好坏取决于领域法术是否值得3-12级强制写入，以及3/6/10/14级特性是否能在法师动作经济中兑现。 |
| 主题与能力关联度 | B+ | 领域法术和领域能力能表达`死亡领域`主题；若该领域依赖武器、护甲、前线站位或特定敌人类型，关联度会受到法师底盘限制。 |
