# 术士（Sorcerer）总评

## 队伍定位摘要
术士是爆发型奥术满施法者。2024版通过`先天术法`、更多准备法术、短休恢复术法点和更清晰的超魔循环，把它从窄法术表职业提升为可靠的战斗控制/爆发核心。

## 详细评价
2024 术士的本职重心比2014版更明确。1级`施法`已经改为准备法术体系，术士不再被极少已知法术严重锁死；同级`先天术法`用附赠动作开启1分钟，使术士法术豁免 DC +1，并让术士法术攻击检定具有优势。这个能力把术士的爆发窗口写在本职里，尤其强化控制法术和攻击法术的兑现率。

2级`魔力泉涌`和`超魔法`仍是职业身份核心。2024文本中`谨慎法术`能让被保护目标通过豁免且在成功半伤时不受伤害，`升阶法术`只需2点术法点就能让一个目标豁免劣势，`精妙法术`能去除言语、姿势和大多数无价值材料成分；这些都让术士在控制、反制和潜入施法上有独特角度。5级`术法复苏`在短休恢复不超过术士等级一半的术法点，每长休一次，实际缓解了中期术法点压力。

7级`术法化身`是2024术士强度提升的关键：`先天术法`次数耗尽后可花2点术法点继续激活，并且激活期间每道法术可应用最多两个超魔选项。20级`奥术化神`进一步让`先天术法`期间每回合一次免费使用超魔。术士的短板仍然存在：技能、制造、常规治疗和法师式泛用解题不如对应专家。但作为战斗奥术核心，它在2024环境已经从“法术少但会改法术”变成“法术容量可接受、爆发窗口明确、超魔资源可持续”的强满施法者。

## 分项理由
- **法术伤害者**：`先天术法`提高 DC/法术攻击，`强效法术`、`瞬发法术`和高环法术让术士从 Tier 2 起进入顶级输出施法者。
- **减益 / 战场分隔**：`升阶法术`、高 DC 和术士控制法术让关键豁免更难通过；高等级满施法者自然上升到 S。
- **反制施法者**：Tier 2 获得`法术反制`后，`精妙法术`和高魅力施法使其很适合反制对局。
- **治疗和临时生命**：基础术士没有稳定治疗入口，只有个别法术、专长或物品能补，因此保持低分。

## 校准说明
本轮依据`DND5e_chm\玩家手册2024\角色职业\术士`的2024术士文本校准，并参考社区对2024术士普遍强化、时械/畸变高评价、狂野仍受随机性限制的讨论。

## 分阶段子职排行榜

Tier 1（1-4）:
- A:
  - 咒火术法：3级额外治疗/火焰法术和`咒火迸发`让它比基础术士多出支援与小额压血工具。
  - 幽影魔法：按2024兼容，1级`幽暗之瞳`和`终焉之力`视为3级获得；早期黑暗视觉、保命和`黑暗术`可用，但无完整扩展法术表。
  - 时械术法：`时械法术`和`归复平衡`在低级就同时补防护、辅助和优势/劣势干预。
  - 月之术法：低级`月之化身`给额外法术和`护盾术`等关键选项，明显强于无扩表子职。
  - 畸变术法：3级`灵能法术`直接扩表，并用传心谈话补信息位，低级已经不是单纯爆破术士。
  - 神圣之魂：按2024兼容，1级特性视为3级获得；牧师法术表入口让低级支援和治疗直接越过基础术士短板。
  - 龙族术法：3级`龙族体魄`和扩展法术让低级更耐打，但功能宽度仍不如畸变、时械。
- B:
  - 狂野术法：低级有`混乱之潮`和基础术士强度，但没有扩展法术，随机表也不能稳定当队伍计划。
  - 风暴术法：低级`风语者`和`暴风魔法`有风味与脱离能力，但缺少扩展法术，战斗贡献主要靠本体。

Tier 2（5-10）:
- S:
  - 咒火术法：6级始终准备`法术反制`且反制失败豁免可回术法点，反施法定位非常突出。
  - 时械术法：5-10级拿到`援助术`、`防护能量`、`构装召唤术`和`力场墙`，同阶段功能密度极高。
  - 畸变术法：6级`灵能术法`用术法点无言语/姿势施放核心法术，控制和潜入施法稳定性达到术士子职顶层。
  - 神圣之魂：5-10级神术表、超魔和`强效治疗`结合，能同时担任治疗、增益和奥术控制。
- A:
  - 幽影魔法：6级`凶兆猎犬`能让目标对你的法术豁免劣势，是强单体压制工具，但猎犬行动受限且仍可被处理。
  - 月之术法：6级相位切换与超魔减费提升资源效率，但操作复杂，实际强度看玩家规划。
  - 狂野术法：6级`扭曲幸运`好用，基础术士进入强势期，但子职仍缺少可控扩表。
  - 龙族术法：6级元素增伤和抗性稳定，能强化爆破定位，但没有把控制或支援推到顶层。
- B:
  - 风暴术法：6级雷鸣/闪电抗性和近身小范围伤害可用，但要求站位靠前，和术士防御并不完全匹配。

Tier 3（11-16）:
- S:
  - 咒火术法：14级临时生命成长后更像持续支援术士，高环本体仍保留顶级控制。
  - 时械术法：高环术士本体加`力场墙`、团队防护和稳定检定，使其成为最稳的术士子职之一。
  - 月之术法：14级相位增益和高环术士法术叠加，功能宽度进入 S。
  - 畸变术法：14级`血肉启示`补飞行、隐形侦测和机动，和高环控制叠加后几乎没有明显战斗短板。
  - 神圣之魂：高等级神术选择与术士高 DC/超魔并存，综合功能宽度接近顶级。
- A:
  - 幽影魔法：14级`幽影漫步`提供暗光传送，高环术士本体推高控制表现，但子职仍窄于时械/畸变。
  - 狂野术法：14级`受控混沌`降低随机风险，综合可靠性提升到 A。
  - 龙族术法：14级飞行优秀，整体仍是强术士底盘上的直接型子职。
- B:
  - 风暴术法：14级反击推离有防身价值，仍没有改变其功能宽度不足的问题。

Tier 4（17-20）:
- S:
  - 咒火术法：18级`咒火冠冕`提供飞行、减伤和对法术/魔法效果的闪避式防御，终局非常硬。
  - 时械术法：18级`时械矩阵`能治疗、修复并解除六环以下法术，终局仍保留强支援身份。
  - 月之术法：18级`月之异象`提供治疗、致盲、控速、隐形、传送和抗性，终局支援/控制价值很高。
  - 畸变术法：18级`扭曲内爆`不是最强终局特性，但本体九环法术加灵能施法足以维持 S。
  - 神圣之魂：18级`神赐痊愈`提高自保，九环术士底盘加神术表仍保持 S。
- A:
  - 幽影魔法：18级`幽暗之形`给除力场/光耀外广谱抗性，终局自保强但仍不是顶级综合术士。
  - 狂野术法：18级`驯服浪涌`能主动选择一次浪涌效果，但仍不是比时械、畸变更稳的 S。
  - 风暴术法：18级闪电/雷鸣免疫与团队飞行终于给出明确高等级价值，综合上升到 A。
  - 龙族术法：18级无材料、短时无专注`龙类召唤术`很强，但终局综合仍略低于顶级控制子职。

## UA/合作方/第三方子职分阶段排行榜

Tier 1（1-4）:
- A:
  - 亵渎术法（UA）：3级`亵渎，强化`用生命骰换伤害，低级爆发高但消耗真实防御资源。
  - 先祖术法（UA）：低级有知识检定补强、额外法术和`先祖面相`影响交涉，但真正爆点尚未到来。
  - 幽影术法（UA）：3级扩展法术、120尺黑暗视觉、10尺盲视和看穿自造黑暗，低级功能密度高。
  - 猩红术法（歪曲之月）：3级`血源`、治疗/伤害扩表和`援助术`让它低级就能同时补伤害与治疗。
  - 符文之子（塔尔多雷）：3级按兼容获得`本源符文`、`符文魔法`和`坚盾刻文`，扩表与减伤都强，但早期符文管理吃术法点。
  - 饥欲之暗（克苏鲁）：3级扩展黑暗法术、黑暗视觉和看穿自身魔法黑暗，低级就能建立暗场战术。
- B:
  - 烈焰士（Plane Shift）：按2024兼容，1级`火焰之心`视为3级获得；低级只给近身小额火焰附伤，缺扩表。
  - 织光者（黯潮之书）：低级能制造光照或黑暗并看穿黑暗，但没有正式扩展法术表，主要靠本体术士。
  - 荒漠之魂（狮鹫鞍包）：低级`飞沙走石`提供附赠动作推位，但风暴体积小、无扩展法术，主要仍靠术士底盘。
  - 鬼之血脉（胧忆岛）：低级鬼之力随术法点消耗解锁，社交、角击和自愈有潜力，但缺扩展法术且启动不稳定。

Tier 2（5-10）:
- S:
  - 先祖术法（UA）：6级免费`法术反制`/`解除魔法`、反制豁免劣势和解除检定优势让它成为顶级反施法者。
  - 幽影术法（UA）：6级无材料`亡灵召唤术`且可无专注一分钟，召唤与控制价值非常高。
- A:
  - 亵渎术法（UA）：中期`污染施术`把造槽转为临时生命，强度上升但仍受生命骰和动作规划限制。
  - 烈焰士（Plane Shift）：6级`火之血脉`给火焰抗性并让法术无视火焰抗性，火焰专精终于稳定。
  - 猩红术法（歪曲之月）：6级血池形态防御强但不能施法，`殷红生机`补抗性与力竭处理，仍是 A 档强子职。
  - 符文之子（塔尔多雷）：6级`符印强化`和`显现刻印`补防御与探索，综合强于多数旧子职但还不到时械/畸变级别。
  - 荒漠之魂（狮鹫鞍包）：6级`镜影术`入口和旋风伤害补足自保与小额输出，进入稳定 A。
  - 饥欲之暗（克苏鲁）：6级让队友看穿你创造的魔法黑暗，黑暗战术从自利变成团队工具。
  - 鬼之血脉（胧忆岛）：6级`食人掌控`能短时指定鬼之力，修正早期不稳定问题，综合进入 A。
- B:
  - 织光者（黯潮之书）：6级`折曜之盾`只惩罚5尺内近战命中者，防御可用但不改变施法者站位逻辑。

Tier 3（11-16）:
- S:
  - 亵渎术法（UA）：14级`凋零灵光`和高环术士法术叠加后，伤害、防御和资源回收达到 S。
  - 先祖术法（UA）：14级`八风不动`使伤害不能打断术士法术专注，防御和控制稳定性极高。
  - 幽影术法（UA）：14级暗光传送让机动和潜入显著增强，高环控制继续保持顶级。
  - 猩红术法（歪曲之月）：14级用生命骰刷新`血源`或免费超魔，使治疗、伤害和资源效率进入 S。
  - 符文之子（塔尔多雷）：14级`符文涌流`能把法术转力场并附带倒地/推离，和高环法术叠加后进入 S。

  - 烈焰士（Plane Shift）：14级反击火焰伤害可用，但依旧要求被近战命中，综合仍靠术士本体维持 A。

- B:
  - 织光者（黯潮之书）：14级`灵光增效`能制造阳光或魔法黑暗并解除光暗法术，场景价值高但战斗宽度仍窄。

Tier 4（17-20）:
- S:
  - 亵渎术法（UA）：18级免疫中毒/力竭、扩大光环并阻止敌人回血，终局战斗压制很强但不是全体系 S+。
  - 先祖术法（UA）：18级对法术豁免优势和一次失败转成功继续强化反施法防线，但未构成 S+。

  - 饥欲之暗（克苏鲁）：18级在自己回合隐形且不可被真实视觉/盲视感知，并能传送，终局生存与潜入非常强。

  - 烈焰士（Plane Shift）：18级火焰免疫且把火焰免疫视为抗性，火焰爆破终局很强但功能面仍窄。

  - 荒漠之魂（狮鹫鞍包）：18级`飞沙形态`给穿缝、免擒抱和反应全伤害抗性，终局自保强但不改写队伍结构。
