## 队伍定位摘要
术士是爆发型奥术满施法者。2024版通过`先天术法`、更多准备法术、短休恢复术法点和更清晰的超魔循环，把它从窄法术表职业提升为可靠的战斗控制/爆发核心；但它仍不是法师式全域工具箱。

## 详细评价
2024术士的本职重心比2014版更明确。1级`施法`改为准备法术体系，术士不再被极少已知法术严重锁死；同级`先天术法`用附赠动作开启1分钟，使术士法术豁免 DC +1，并让术士法术攻击检定具有优势。这个能力把术士的爆发窗口写在本职里，尤其强化控制法术和攻击法术的兑现率。

2级`魔力泉涌`和`超魔法`仍是职业身份核心。`谨慎法术`能保护队友免受己方法术误伤，`升阶法术`让一个目标豁免劣势，`精妙法术`能去除言语、姿势和大多数无价值材料成分；这些让术士在关键控制、潜入施法和反制博弈中有独特角度。5级`术法复苏`每长休一次在短休恢复术法点，7级`术法化身`让`先天术法`耗尽后仍可用术法点续开，并允许同一法术叠两个超魔选项。

这轮校准不把术士高等级自动写成普通 S。术士当然是强满施法者，但它缺少法师的法术书和仪式经济，缺少牧师/德鲁伊的日备全表与稳定治疗生态，也没有默认`祈愿术`。它的优势是把较窄但足够强的法术选择用超魔压到更高兑现率；因此本职 Tier 2-4 采用 S-，而真正能靠扩表、反制、隐蔽施法、专注保护或团队防御补掉术士短板的子职才进入 S。20级`奥术化神`每回合一次免费超魔很强，但它出现在终局末端，不足以把整个 Tier 4 默认抬成 S。

## 分项理由
- **法术伤害者**：`先天术法`、`强效法术`、`瞬发法术`和高环奥术表使术士从 Tier 2 起是顶级法术伤害者。
- **减益 / 战场分隔**：`升阶法术`、高 DC 和术士控制法术能压关键豁免，但可携带法术压力让它不如法师泛用。
- **反制施法者**：获得`法术反制`后，`精妙法术`和高魅力施法很适合施法对抗；没有子职强化时仍不等于防护师或咒火术法。
- **治疗和临时生命**：基础术士缺少稳定治疗/临时生命生态，需要子职、专长或物品补足。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | A | 1-4级已有完整奥术施法、`先天术法`提高 DC/法术攻击，低级控制和爆发强于多数职业，但法术数量、技能和防御仍受限。 |
| Tier 2（5-10） | S- | 5-10级三环以上法术、`术法复苏`和7级`术法化身`让术士进入顶级奥术战斗核心；但没有法师式法术书/仪式经济，也没有日备全表，因此为 S- 而非 S。 |
| Tier 3（11-16） | S- | 11-16级高环法术、`升阶法术`和双超魔能稳定压关键豁免；短板仍是已准备法术压力、非战斗广度和缺少`祈愿术`式底盘重写。 |
| Tier 4（17-20） | S- | 17-20级九环法术与20级`奥术化神`让超魔资源很强，但综合解题宽度仍窄于法师/牧师/德鲁伊顶级底盘，默认压缩为 S-。 |

## 校准说明
本轮横向比较确认术士不是法师的窄版，而是以先天术法、超魔和高 DC 窗口为核心的爆发型满施法者。高阶默认 S-：它可能取得强高环法术和终局超魔资源，但缺少法师法术书/仪式经济、牧师/德鲁伊日备全表与稳定治疗生态，准备法术压力仍真实存在。能进入 S 的术法必须补足关键短板，例如扩展法术包、隐蔽施法、反制可靠性、豁免压制或团队防护；单纯元素伤害或个人防御通常停在 A+ / S-。
