# 神圣之魂

## 队伍定位摘要
神圣之魂的核心不是翅膀，而是“牧师法术也可以成为术士法术”。在2024术士底盘上，这会显著补齐治疗、增益和防护，是最强的旧版兼容子职之一。

## 详细评价
依照本项目兼容规则，5e/2014 子职在低于3级获得的特性统一视为3级获得。`神圣魔法`允许术士在学习或替换术士戏法/法术时从术士和牧师法术列表中选择，并把这些法术视为术士法术；这意味着`祝福术`、`疗伤术`、`次等复原术`、`复活`类法术和许多防护神术都能被术士的魅力、超魔和`先天术法`体系承接。

`众神眷恩`在失败豁免或攻击失手时加2d4，短休或长休恢复，低级保命价值很高。6级`强效治疗`允许消耗1点术法点重骰治疗法术的治疗骰，数值不夸张，但和神术表入口共同让术士真正能承担治疗职责。14级`异界光翼`是永久可开关飞行，18级`神赐痊愈`在半血以下用附赠动作恢复半数生命上限，进一步补自保。

神圣之魂的风险是法术准备/选择压力：即使2024术士准备法术数量增加，也不可能同时携带所有优秀神术和奥术。实际强度取决于玩家是否清楚自己是补治疗、补增益、补反制，还是当常规奥术控制者。按优化视角，它的上限很高，原因是法术表入口，而不是后期翅膀本身。

综合判断：本轮从默认 S 压到 S-：神圣之魂的法术表入口极强，但没有自动扩展准备列表；优秀神术与奥术控制会互相挤占准备位。

## 分项理由
- **治疗和临时生命**：牧师法术表入口加`强效治疗`让它是真治疗者，不只是会一两道治疗法术。
- **团队增益**：`祝福术`等神术可被术士资源承接，补齐基础术士最缺的团队支援。
- **反制施法者**：仍靠术士表的`法术反制`与超魔；神术表主要补防护和恢复。
- **防御和生存**：`众神眷恩`、飞行和18级半血恢复显著提高自保。

|---|---|---|
| Tier 1（1-4） | A | 按2024兼容，低于3级特性视为3级获得；牧师法术表入口让低级治疗/增益明显强于基础术士。 |
| Tier 2（5-10） | S- | 5-10级神术表、超魔和`强效治疗`能做副治疗/副支援，但准备法术压力极重，不能当成牧师日备全表。 |
| Tier 3（11-16） | S- | 高等级能把部分神术、防护和术士控制结合；强度主要来自双表入口，受可准备法术数量限制。 |
| Tier 4（17-20） | S- | 18级`神赐痊愈`是个人自保，不是队伍级终局引擎；九环术士底盘加神术选择维持 S-。 |

| 设计项 | 评分 | 具体理由 |

| 设计质量 | A | 把牧师法术表开放给术士非常强且简洁，但也把大量强度压进选法术环节，对新玩家不够直观。 |
| 主题 | A | 神圣血脉、命运眷顾、治疗和光翼主题清楚，但比2024新子职略少机械整合。 |
| 能力设计质量 | A | 旧版1级入口按兼容规则移至3级后仍足以定义子职，后续特性大多是补强；曲线有效但前重后轻。 |
| 主题与能力关联度 | A | 神术表、眷恩、治疗和光翼都符合神圣主题，只是玩法核心更多来自法术表而非独特特性。 |

评分采用2024术士本职作为底盘；正式旧版子职低于3级获得的特性按本项目规则视为3级获得。
