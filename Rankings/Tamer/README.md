## 队伍定位摘要

Tamer 是《Heliana's Guide to Monster Hunting》的怪物伙伴半施法新基础职业。本轮依据本地 `5etools-homebrew\collection\Loot Tavern; Heliana's Guide To Monster Hunting.json` 重查；`DND5e_chm`、`5etools-cn` 与既有 `Rankings` 未定位可靠中文正式译名，因此本职业和四个 Training Paradigm 暂保留英文名。

这个职业的共同底盘是 d8 生命骰、轻甲/中甲/盾牌、简单武器与网、体质和魅力豁免、2级起半施法进展，以及以容器收纳、召唤、命令和训练怪物伙伴为核心的职业机制。它不是固定数据板宠物职业，而是直接读取合法怪物数据板：1级从小型或更小、CR 1/2或更低、非类人生物/巨人/集群开始，逐步提高到19级六只绑定伙伴、超大型、CR 6。伙伴使用你的熟练加值，获得训练改良、额外生命骰和高等级 Multiattack 入口，但战斗中通常只有一只被召唤，并且需要你用动作或附赠动作命令才会攻击或使用其他动作。

强度上，Tamer 的上限来自“怪物数据板 + 熟练加值替换 + 半施法 + 多伙伴工具箱”。低级已经有额外身体、侦察、承伤、特殊感官和少量法术；T2开始，多重攻击、魔法攻击、CR 2-3伙伴和子职动作把它推到强宠物职业线；T3后 CR 4-5 伙伴、换位防御、更多备选伙伴和强子职按钮让它有很高战术上限。限制同样清楚：强弱极吃 DM 给什么怪、是否允许非战斗驯服、如何裁定凝视/充能/永久光环等怪物能力；源文虽有 Lost Traits 与 Powerful Companion Abilities 侧栏，但仍把大量平衡责任交给 DM。

## 详细评价

1级 `Pocket Familiar` 建立核心伙伴规则：伙伴在容器中休眠，召唤需要动作，30尺可见空位或自己5尺内出现；同一时间只能召唤一只。战斗中伙伴与你同先攻并在你之后行动，可自行移动和反应，但若没有你的动作或附赠动作命令，只会 Dodge；你失能时才可自由行动。这让 Tamer 低级行动经济比普通角色强，但不是无成本双角色，因为命令会长期占用附赠动作或动作。

1级 `Monster Trainer` 让伙伴使用你的熟练加值，并在你升级时获得训练改良；ASI等级还会给所有伙伴额外生命骰。训练选项包括速度、额外生命骰、属性、自然武器/徒手攻击命中与伤害、豁免熟练、护甲/武器熟练。这是本职业最容易越界的地方：很多低CR怪物原本按怪物熟练和一次性遭遇设计，换成玩家熟练加值和长期训练后，攻击、豁免DC、速度、AC、生命和特殊动作都会被抬高。

1级 `Tame Creature` 用动作向30尺内目标投掷空容器。目标若是类人生物、巨人、集群，超过尺寸/CR/容器上限，或生命值超过一半，都会自动成功；若不自动成功且生命低于10或四分之一生命，则自动失败。失败后进入容器，并在你下次长休后成为伙伴。`Lost Traits` 会移除施法、召唤/造物动作、再生/复生/传奇抗性、传奇/巢穴/神话能力等；但源文也承认充能动作和常驻凝视等能力需要逐案调整。

2级 `Spellcasting` 是半施法者进展，已知法术很少，最高5环。源文写法允许你在获得1级时选择智力、感知或魅力作为 Tamer 法术施法属性，但同段又出现“用魅力设定法术豁免DC和法术攻击”的文本错位；项目按表中能力DC/攻击可用 Int/Wis/Cha 的结构理解。2级 `Psychic Bond` 给100尺心灵沟通、借感官，以及关键的 `Spell Swap`：伙伴已知的法术会加入你的已知法术，并且你施放范围为自身或接触的法术时可以让伙伴成为目标。这一条极吃伙伴来源，也解释了为什么源文禁止伙伴施法但仍保留其已知法术。

5级 `Multiattack` 允许命令有 Multiattack 的伙伴使用其 Multiattack。6级 `Alpha Strike` 让你在召唤伙伴的动作中顺便命令它行动，次数等于施法属性调整值；`Wilful Blows` 让伙伴攻击视为魔法。这是T2强度转折：许多怪物的 Multiattack、特殊伤害、条件、飞行、潜行、感官或范围动作都能成为玩家可反复调度的战术资源。

5级 `Malleable Presence` 和17级 `Magnificent Presence` 是伙伴吸引火力或降低被打概率的气势工具。前者短休一次，对单体做1分钟的攻击劣势压制；后者长休一次，扩大为15尺附近的类光环效果。它们都要敌人看见伙伴，且吃豁免、站位和敌人目标逻辑；但在保护后排或让高AC/高血伙伴承伤时很实用。

13级 `Switcheroo` 是强防御/站位按钮：你与100尺内伙伴可用附赠动作交换位置，也可在你或伙伴成为可见攻击者的攻击目标时用反应交换，并让出现于目标空间的生物成为新目标。它能救人、挡刀、越过地形或把敌人攻击转给更硬的伙伴；限制是次数等于施法属性调整值，且只针对“成为攻击目标”的反应窗口。

20级 `Summon the Horde` 终于允许一次召唤最多三只伙伴，持续1分钟，并用后续动作统一命令所有伙伴。这个终局很有排面，也让多个CR 6/训练伙伴同时上场；但要动作启动、动作命令、长休一次，结束后自己到下回合末不能移动或行动。终局高阶生态里，它是强战术爆发，却仍不是9环法术、现实改写或无限召唤。

## 本轮重评

本轮社区校准检索覆盖 `"Tamer" "Heliana's Guide to Monster Hunting" review`、`"Heliana's Guide" "Pocket Familiar"`、`"Loot Tavern" "Tamer" "Infuser"`、`"Tamer" "Summon the Horde" "D&D"`、Reddit 与 D&D Beyond 论坛方向。未定位到稳定、同版本、可逐条引用的 Tamer 强度讨论；可见公开材料更多是 Heliana 资源介绍、购买页或泛用评论。因此本轮只把外部检索结论作为“社区缺少同版共识”的校准，最终评分按本地 Heliana 文本、当前2024生态、已完成第三方新基础职业和宠物/召唤类样本独立判断。

横向上，Tamer 比普通驯兽师/伙伴子职强得多，因为它不是固定宠物表，而是可随战役更换多个怪物数据板。它也比多数半施法武职更宽，因为伙伴可承伤、侦察、飞行、特殊感官、条件和动作，同时本体仍有半施法。它低于真正高阶满施法者和强Apothecary线，原因是法术最高5环、战役级解题有限，且最强能力高度依赖DM允许的怪物来源和逐案平衡。综合定为 `A / A+ / S- / A+`。

## 校准说明

Tamer 按第三方新基础职业评分，不因非WotC来源自动降权；但它的设计风险要明确计入。与 TCE 协力者相比，Tamer更像PC本体，强在可更换怪物工具箱和半施法；与2024游侠驯兽路线相比，Tamer的伙伴上限和工具宽度明显更高；与满施法者、Apothecary或强构筑相比，它缺少高环法术和战役级确定性。

家族内部排序按“行动经济与工具宽度优先”处理：Splicer最高是因为长期改造能解决移动、感官、AC、尺寸和控制；Leader最高峰来自额外动作与双伙伴同场；Infuser稳定但偏数值；Necromancer上限高却更吃尸体、亡灵生态和桌面管理。全家族不按最极端怪物数据库套利给分，默认DM会按源文侧栏处理明显不适合PC长期使用的能力。
