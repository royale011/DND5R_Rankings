# 魔契师（Warlock）总评

## 队伍定位摘要
魔契师是短休法术位、`魔能爆`、`魔能祈唤`和三种魔契路线拼出的模块化奥术职业。它不是满施法者的资源池，也不是纯武器职业；它更像一个可按队伍缺口改装的远程输出/控制/补位角色。

## 详细评价
1级的`魔能祈唤`让职业起跑就有构筑分歧：`苦痛魔爆`与`推斥魔爆`把`魔能爆`做成稳定远程火力和推位控制，`刃之魔契`让近战魔契师真正能用魅力攻击，`链之魔契`与`书之魔契`分别补侦察、仪式和戏法广度。`契约魔法`的短休恢复是强点也是风险：短休充足时，3-5环槽位的重复爆发很高；短休稀少时，它比法师、牧师、德鲁伊更容易在关键场景前空仓。`魔法狡诈`缓解了这个问题，但它不是完整短休替代品，只是给一次最低限度的续航保险。

Tier 1 的魔契师已经有同级很强的无消耗远程回合，3级拿到宗主以后角色身份开始明确。它的低级控制主要靠`脆弱诅咒`、`魅惑类人`、推拉位移和少量短休法术位，强度稳定但不如满施法者能连续处理多个不同问题。Tier 2 是职业真正成型期：5级以后`魔能爆`射线增加，`饥渴魔刃`让刃契近战开始有实战意义，3环槽位配合`法术反制`、`催眠图纹`类控制或召唤法术可以承担队伍主控制的一部分。Tier 3-4 的`秘法奥秘`让它拿到6-9环解题权，但每环只能选定一法，灵活度低于满施法者；社区对2024魔契师的常见评价也集中在“好上手、定制强、但高环选择少且短休依赖桌风”。

它的排名不应只看`魔能爆`伤害。真正的队伍价值来自三点：第一，远程输出几乎不吃资源，能把法术位留给关键控制；第二，祈唤把角色定制到侦察、仪式、近战、召唤或位移；第三，宗主子职能明显改变职责，例如天界宗主补治疗，旧日支配者宗主补减益和无成分施法，巨灵宗主补飞行、容器和高等级拟愿。缺点同样直接：制造几乎没有职业入口，治疗默认很弱，技能也不如游荡者/吟游诗人可靠；若队伍和DM不给短休，它会从A档掉到更普通的施法输出。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | A | 1-4级靠`魔能爆`、初始祈唤和3级宗主已经能稳定承担远程伤害与少量控制，但法术位少，治疗和技能补位有限。 |
| Tier 2（5-10） | A | 5-10级射线数、祈唤数量、3-5环`契约魔法`和`法术反制`等关键法术同时到位，是魔契师最稳定的队伍功能期。 |
| Tier 3（11-16） | A | 11-16级`秘法奥秘`补上6-8环高等级解题，但每环选择固定，不能像满施法者那样随日程大幅重配。 |
| Tier 4（17-20） | A | 17-20级有9环`秘法奥秘`与`魔能大师`，终局可参与高等级法术问题，但仍受已选奥秘和短休节奏限制。 |

## 分阶段子职排行榜

Tier 1（1-4）:
- A:
  - 咒剑：3-4级`咒剑诅咒`、护甲/盾牌和额外法术包里的`护盾术`让刃契路线明显强于普通近战魔契师；但2024`刃之魔契`和通用法术入口已吸收旧版部分独占性。
  - 天界宗主：3-4级`治疗之光`让魔契师获得可靠倒地救援，队伍价值明显高于普通输出魔契师。
  - 巨灵宗主：3-4级容器与稳定附伤已经很强，但飞行和团队短休尚未到位。
  - 旧日支配者宗主：3-4级已拥有无成分惑控/幻术与心灵伤害转换，低级非战斗和减益价值很高。
  - 死灵宗主：3-4级`战栗形态`让远程攻击附带恐慌压力，早期战斗控制很强。
  - 深海意志：3-4级`深海触手`提供稳定附赠动作减速，和`魔能爆`形成早期控场。
  - 至高妖精宗主：3-4级靠多次`迷踪步`式位移显著提升生存和战术站位，但主要输出仍是基础魔契师。
  - 邪魔宗主：3-4级靠`黑暗赐福`获得击杀续航，输出和耐久都比基础魔契师更稳。
- B:
  - 不朽者：3-4级子职贡献很窄，主要靠魔契师底盘撑住A档边缘以下表现。

Tier 2（5-10）:
- S:
  - 巨灵宗主：6级飞行与抗性让它在中等级机动/防御上压过多数正式宗主。
- A:
  - 咒剑：5-10级武器输出和防御都稳定，但很多2014优势已被2024刃契底盘吸收。
  - 天界宗主：6级前后能兼顾稳定远程伤害和副治疗，但控制面不如旧日支配者。
  - 旧日支配者宗主：6级后心灵链接开始产生战斗优势，控制稳定性超过普通魔契师。
  - 死灵宗主：6级后暗蚀伤害和恐惧循环成型，但遇到免疫恐慌或暗蚀环境会下滑。
  - 深海意志：6级后可用反应减伤保护队友，控制和防护都比普通魔契师更完整。
  - 至高妖精宗主：6级后反应位移让它比普通魔契师更难被锁死，控制法术也更容易找到施放位置。
  - 邪魔宗主：6级`黑暗强运`补关键检定，爆破法术和临时生命形成简单可靠战斗循环。
- B:
  - 不朽者：6级`反抗死亡`治疗不足且触发窄，和同级巨灵、死灵、天界差距明显。

Tier 3（11-16）:
- S:
  - 巨灵宗主：10级`庇护器皿`强化队伍短休，和魔契师资源机制形成实质协同。
  - 旧日支配者宗主：10级精神防护叠加高环控制，让它在减益、反制环境和潜入施法上进入正式宗主顶层。
- A:
  - 咒剑：10级后对诅咒目标的防御不错，仍缺少顶级控制或团队功能。
  - 天界宗主：10级`天界韧性`提供全队临时生命，团队耐久价值上升。
  - 死灵宗主：10级防猝死和抗性增强生存，仍主要是战斗强项。
  - 深海意志：10级强化`艾伐黑触手`，主控场价值上升，但仍依赖专注和地形。
  - 至高妖精宗主：10级后对魅惑和精神干扰的防护更完整，但功能仍集中在机动控制，未达到全局压制。
  - 邪魔宗主：10级抗性让它在高伤害环境更能站住，但功能广度仍不如巨灵或旧日支配者。
- B:
  - 不朽者：10级偏风味生存能力无法解决高等级战斗和解题需求。

Tier 4（17-20）:
- S:
  - 巨灵宗主：14级`有限祈愿`提供跨职业高价值解题，仍受1d4长休冷却限制而非S+。
  - 旧日支配者宗主：14级操纵能力加9环`秘法奥秘`，终局仍保持强控制和信息战身份。
- A:
  - 咒剑：14级可转移诅咒，终局作为近战魔契师仍可用，但不再是跨职业天花板。
  - 天界宗主：14级`灼热复仇`提供强防猝死能力，但仍不是牧师级主治疗或满施法解题者。
  - 死灵宗主：14级`灵体投射`拓展探索和续航，但复杂度和裁定依赖阻止它进入S档。
  - 深海意志：14级传送有团队价值，但地点限制明显，终局不如巨灵或旧日支配者泛用。
  - 至高妖精宗主：14级把惑控/幻术与位移绑定，终局仍是优秀机动控制宗主，但不提供额外高环解题权。
  - 邪魔宗主：14级`炼狱放逐`有强单体惩罚，但次数和目标限制使其不超过顶级控制宗主。
- B:
  - 不朽者：14级自愈仍不足以弥补前期空洞，整体是正式魔契师中最弱子职之一。

## UA/合作方/第三方子职分阶段排行榜

Tier 1（1-4）:
- S:
  - 咒剑宗主（UA）：3-4级同时获得`护盾术`、新版`咒剑诅咒`和近战防御，明显压过多数正式宗主起步。
  - 契灵宗主（UA）：3-4级即有可复现伙伴、领域法术和治疗/诅咒/换位选项，功能面远超普通宗主。
  - 角之王宗主（歪曲之月）：3-4级`巫印`转移动作法术且`咒怨`可反应压关键检定，功能面已超过多数宗主。
- A:
  - 众首（狮鹫鞍包）：3-4级防突袭、死亡豁免修正和小额附伤让安全性明显高于普通魔契师。
  - 崇高猫猫庭（克苏鲁）：3-4级长时间猫形态提供极强潜入和位置优势，但战斗法术受猫咪法术表限制。
  - 巫王宗主（UA）：3-4级附赠动作`命令术`让它早期控制效率很高，但仍受豁免和免疫影响。
  - 愚神宗主（歪曲之月）：3-4级反应`恶言相加`在敌人受伤或失败后补劣势，早期干扰能力优秀。
  - 未来的你宗主（瓦尔达）：3-4级预置d20能改写关键D20，但误报风险让它不是无脑强。
  - 死灵宗主（UA）：3-4级仍以`战栗形态`压制敌人，始终准备法术让低级工具更顺。
  - 灯笼（胧忆岛）：3-4级`启明灵光`能用反应救关键豁免，团队支援价值高。
  - 猎食者（谦卑林）：3-4级`野兽之形`给临时生命和魅力天生武器，近战身份比普通魔契师明确。
- B:
  - 悲伤之母（黯潮之书）：3-4级毒素无视免疫很激进，但`猛毒印`只给每法术一次1d6，实际爆发有限。

Tier 2（5-10）:
- S:
  - 咒剑宗主（UA）：6级法术后附赠武器攻击与压低豁免让近战、施法、减益形成强循环。
  - 契灵宗主（UA）：6级短休回能让伙伴资源与魔契师节奏对齐，支援和召唤价值稳定。
  - 愚神宗主（歪曲之月）：6级诅咒池能目盲、减速、破隐或武器伤害减半，减益质量进入顶层。
  - 死灵宗主（UA）：6级无视暗蚀抗性后，输出与恐慌控制的稳定性明显超过正式版。
  - 角之王宗主（歪曲之月）：6级无专注`降咒`且豁免成功仍吃`咒怨`，减益可靠性极高。
- A:
  - 众首（狮鹫鞍包）：6级额外反应、临时生命和精神豁免重骰构成优秀防护工具包。
  - 崇高猫猫庭（克苏鲁）：6级近距附赠动作减益很好用，但需要猫形态和5尺距离。
  - 巫王宗主（UA）：6级施法触发鼓舞/恐慌，队伍节奏能力上升，但不是每场都能压住敌人。
  - 悲伤之母（黯潮之书）：6级反应中毒和扩展法术让减益能力上升，但体质豁免限制明显。
  - 未来的你宗主（瓦尔达）：6级反应+10 AC提供强防命中，短休回一次使可靠性不错。
  - 灯笼（胧忆岛）：6级灯笼化`灵体武器`与光源侦察拓展输出和信息获取。
  - 猎食者（谦卑林）：5-10级啃咬后双爪与`狩猎狂热`成型，近战续航好但附赠动作拥挤。

Tier 3（11-16）:
- S:
  - 咒剑宗主（UA）：10级按等级减伤大幅强化前线生存，诅咒目标战斗中极难有效反打。
  - 契灵宗主（UA）：10级团队抗性、免疫与0血保险让它具备顶级防御支援。
  - 崇高猫猫庭（克苏鲁）：10级`九命猫`大幅提高防死亡能力，是同级少见的强生存资源。
  - 巫王宗主（UA）：10级后控制、防御反击和高环魔契师法术叠加，成为强控场宗主。
  - 愚神宗主（歪曲之月）：10级反伤叠加高等级控制法术，继续强化惩罚敌人行动的定位。
  - 死灵宗主（UA）：10级防猝死和抗性强化前线/中线生存，控制循环更可靠。
  - 角之王宗主（歪曲之月）：10级王冠灵光每轮施压，兼具伤害、魅惑、劣势与恐慌。
- A:
  - 众首（狮鹫鞍包）：10级察觉/智力防护继续补短板，但没有进入顶级控制或输出。
  - 悲伤之母（黯潮之书）：10级毒/病免疫让对应题材战役表现更好，泛用战斗仍不顶级。
  - 未来的你宗主（瓦尔达）：10级先攻优势和首轮全抗让高难遭遇开局生存很强。
  - 灯笼（胧忆岛）：10级目盲爆发和更高环灯笼武器使战斗支援更完整。
  - 猎食者（谦卑林）：10级群体恐慌补控场，仍主要是近战战斗强项。

Tier 4（17-20）:
- S:
  - 咒剑宗主（UA）：14级强化诅咒爆发和恢复次数，终局仍是顶级战斗宗主，但未达到跨全职业S+。
  - 契灵宗主（UA）：14级高环召唤变身提供极高爆点，但每天一次和动作经济限制使其不达S+。
  - 崇高猫猫庭（克苏鲁）：14级反应力场伤害、目盲、倒地且可用契约槽重置，终局反打很强。
  - 巫王宗主（UA）：14级`绝对暴政`能把恐慌目标转化为自动失败`命令术`，对可受影响敌人压制极强。
  - 愚神宗主（歪曲之月）：14级小丑持续攻击可反复恐慌或倒地，但要占动作所以不达S+。
  - 死灵宗主（UA）：14级无成分咒法/死灵施法和飞行恢复让它成为顶级UA控制宗主。
  - 角之王宗主（歪曲之月）：14级把诅咒、传送和附伤分发给队友，团队资源价值顶级但仍不至S+。
- A:
  - 众首（狮鹫鞍包）：14级群体心灵伤害加失能是强爆点，长休一次限制合理。
  - 悲伤之母（黯潮之书）：14级长时麻痹很强，但需要命中和豁免失败，且单体次数有限。
  - 未来的你宗主（瓦尔达）：14级`祖父悖论`能单体震慑，但长休一次和智力豁免限制使其未进S。
  - 灯笼（胧忆岛）：14级常驻灵光与`寻路术`式导航强化探索，但不是顶级控场。
  - 猎食者（谦卑林）：14级反应变身反击提高防守反打，但终局解题面不如顶级宗主。

## 设计相关评分
| 设计项 | 评分 | 具体理由 |
|---|---|---|
| 设计质量 | A | 2024版把魔契、祈唤和宗主入口整理得更清楚，刃契不再只靠咒剑救场，职业强度来自可读的模块而非单一漏洞。 |
| 主题 | S | 与异界存在交易、用禁忌祈唤塑造自我，这个主题在机制上非常明确；几乎每个构筑选择都能解释成契约代价或恩赐。 |
| 能力设计质量 | A | `契约魔法`短休循环、`魔能祈唤`、`秘法奥秘`的边界都清楚；问题是高环法术固定选择会让后期构筑惩罚偏硬。 |
