## 队伍定位摘要
魔契师是短休法术位、`魔能爆`、`魔能祈唤`和魔契恩泽共同塑形的近满施法者。它的强项不是每日准备广度，而是把少量高环契约法术位、稳定无资源远程回合、祈唤模块和宗主增量压缩进同一个角色。

## 详细评价
1级`魔能祈唤`让魔契师开局就分线：`苦痛魔爆`/`推斥魔爆`提供稳定远程伤害和位移控制，`刃之魔契`让近战路线不再被旧版咒剑垄断，`链之魔契`提供便宜的特殊魔宠单位，`书之魔契`补仪式与戏法广度。`契约魔法`是职业最关键的资源结构：法术位少、环阶高、短休回满。T1-T2 若能获得一次短休，魔契师的有效高价值法术位投放会明显追上甚至压过普通满施法者；若短休稀少，连续遭遇中的容错仍低于法师、牧师、德鲁伊等准备满施法者。2级`邪魔活力`给固定12点临时生命，低级防御不是纸面数值；`链之魔契`也不是纯侦察，特殊形态能提供飞行/隐形/抗性/毒伤、帮助、触碰法术投递、占位和诱导敌人动作。`秘法回流`只是一次应急回槽，不等于稳定短休。

需要单独校准的是近战与短距多段命中。魔契师拥有`魂灵环绕`，但纯魔契师因魔契法术位机制通常最高只按5环施放，也就是每次命中额外`2d8`，不能像普通满施法者那样自然用7-9环法术位升到`3d8/4d8`。12级后`灭世魔刃`让刃契路线在攻击动作中作三次攻击；若三击全中，5环`魂灵环绕`提供额外`6d8`，平均27点，和一次5环`魔能斩`的额外`6d8`同量级。把二者叠在同一轮可形成很高爆发，但这需要贴近10尺、维持专注、先启动法术、命中多个攻击，并消耗魔契师最宝贵的契约法术位。它证明刃契魔契师不是低伤害笑话，但不等于拥有圣武士灵光或满施法者高环可调度性。

宗主法术表也要逐项看。`灼热射线`若由宗主法术表授予，能和`魂灵环绕`形成短距多段法术攻击爆发；5环`灼热射线`有六道射线，全命中时基础`12d6`，5环`魂灵环绕`再加`12d8`，但这要两枚契约法术位、前置专注、10尺距离和命中率，真实桌面不能当作稳定DPR。`咒火闪焰`本地速查显示是术士/法师向FR法术，不是魔契师默认法术；本轮只在宗主法术表确实授予时才计入，没有把它当作全体魔契师资源。

因此魔契师高阶综合强度进入`S-`而非普通满施法者式`S`：`玄奥秘法`是无需法术位的高环入口，但选择固定、通常长休一次，仍要动作、成分、专注和法术选择机会成本。宗主如果只给个人数值或风味工具，不能因为职业底盘强就自动上调；宗主若改变法术递送、团队资源、短休循环、伙伴行动或控制可靠性，才会真正推高评分。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | A+ | 1级`魔能祈唤`、短休`契约魔法`和`魔能爆`已经给出稳定远程/控制底线；2级`邪魔活力`固定12点临时生命，`链之魔契`还可提供便宜特殊魔宠单位。限制是法术位数量少、短休节奏敏感，宗主到3级才真正展开。 |
| Tier 2（5-10） | A+ | 5-10级射线数、祈唤数量、3-5环`契约魔法`和`法术反制`到位；一到两次短休会让高价值法术位次数非常竞争。它仍没有准备满施法者的每日调表，高影响法术选择和短休节奏决定上限，所以不自动进S-。 |
| Tier 3（11-16） | S- | 11-16级`玄奥秘法`提供6-8环无槽入口，12级`灭世魔刃`也让刃契三打成型；固定奥秘、短休依赖和专注/法术位竞争使其低于准备满施法者。 |
| Tier 4（17-20） | S- | 17-20级有9环`玄奥秘法`与`魔能掌控`，但无默认`祈愿术`、奥秘固定，`魂灵环绕`也通常止于5环魔契法术位，因此仍是S-而非S。 |

## 设计相关评分
| 设计项 | 评分 | 具体理由 |
|---|---|---|
| 设计质量 | A | 2024版把魔契、祈唤、宗主和契约魔法拆成清楚模块，刃契不再完全依赖咒剑补丁，构筑入口比2014健康；扣分点是短休槽、固定玄奥秘法和祈唤选择仍会制造较硬的构筑惩罚。 |
| 主题 | S | 与异界存在交易、以禁忌祈唤改造自身、用少量高压法术换取非标准魔法路径，是D&D里最有辨识度的职业主题之一。 |
| 能力设计质量 | A- | 契约魔法和魔能祈唤的边界清晰，短休节奏也有独特身份；但高等级玄奥秘法固定选择、祈唤机会成本和宗主差异会让后期可调度性低于准备施法者。 |
| 主题与能力关联度 | S | 魔能爆、魔契、宗主法术、祈唤和玄奥秘法都在表达“借来的力量如何重塑施法方式”，机制和叙事结合非常紧。 |
