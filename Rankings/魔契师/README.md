## 队伍定位摘要
魔契师是短休法术位、`魔能爆`、`魔能祈唤`和三种魔契路线拼出的模块化奥术职业。它不是满施法者的资源池，也不是纯武器职业；它更像一个可按队伍缺口改装的远程输出/控制/补位角色。

## 详细评价
1级的`魔能祈唤`让职业起跑就能分成远程魔爆、刃契近战、链契侦察和书契仪式等路线。`苦痛魔爆`与`推斥魔爆`仍然是最稳定的无消耗回合，`刃之魔契`则让近战魔契师不再完全依赖旧版咒剑。这个底盘很强，但强的方式不是“普通满施法者”：魔契师的法术位数量少，依赖短休回满；短休充足时可反复用高环`契约魔法`处理关键战斗，短休稀少时则比法师、牧师、德鲁伊更容易在连续遭遇中被耗空。`秘法回流`只是一次保险，不等于稳定短休。

Tier 1 的魔契师靠`魔能爆`、初始祈唤和3级宗主已经有稳定远程输出与少量控制。Tier 2 是职业最舒服的阶段：射线数增加、3-5环`契约魔法`、`法术反制`、`催眠图纹`、召唤和祈唤数量一起到位，能在短休节奏友好的桌面承担副主控或强输出。Tier 3-4 的`玄奥秘法`必须单独看：本地文本表明它是无需法术位施展的高环入口，因此不会消耗魔契法术位，也不触发“消耗法术位”的限制；但每个环阶选择固定、通常长休一次，仍要支付动作、成分、专注和已选法术机会成本。它给魔契师高等级解题资格，却不能让魔契师拥有准备满施法者的每日重配能力。

所以本轮不把魔契师按普通满施法者给分。职业底盘在高阶可进入`S-`：它有高环法术、稳定无资源远程回合、短休槽位和祈唤模块，但缺少法师的法术书/仪式生态、牧师/德鲁伊的每日准备广度，也没有默认`祈愿术`。宗主评分必须看增量：旧日支配者的无成分惑控、巨灵的容器/短休/`有限祈愿`、契灵的伙伴与领域法术、咒剑UA的近战压制，确实能把结果推到`S-`或`S`；只提供个人防御、风味工具或窄场景优势的宗主则不能因为底盘强自动进S。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | A | 1-4级靠`魔能爆`、初始祈唤和3级宗主稳定输出，但法术位极少，尚未形成高环解题或持续控制资源。 |
| Tier 2（5-10） | A+ | 5-10级射线数、祈唤数量、3-5环`契约魔法`和`法术反制`等关键法术到位；短休友好时很强，但仍不是每日重配满施法者。 |
| Tier 3（11-16） | S- | 11-16级`玄奥秘法`提供6-8环无槽高等级入口，叠加短休槽位和祈唤模块，已是高阶主力；限制是每环固定、长休一次和短休节奏依赖。 |
| Tier 4（17-20） | S- | 17-20级有9环`玄奥秘法`与`魔能掌控`，能参与终局法术问题；但无默认`祈愿术`、奥秘固定、契约槽结构和祈唤机会成本使其低于真正顶级准备满施法者。 |
