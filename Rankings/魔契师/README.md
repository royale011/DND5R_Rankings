## 队伍定位摘要
魔契师是短休`契约魔法`、`魔能爆`、`魔能祈唤`和宗主子职共同塑形的模块化近满施法者。它的强项不是每日准备广度，而是用少量高环短休法术位、稳定无资源远程回合、可替换祈唤和宗主增量，在每个遭遇里保持一个清晰的强动作。

## 详细评价
1级`魔能祈唤`让魔契师开局就分线：`苦痛魔爆`把`魔能爆`变成可靠主攻击，`推斥魔爆`把多段命中转成位移控制，`刃之魔契`用附赠动作铸造契约武器并允许用施法属性攻击，`链之魔契`给动作施放的特殊`寻获魔宠`入口，`书之魔契`补戏法和仪式广度。`契约魔法`是职业最关键的资源结构：法术位数量少、环阶高、短休回满。T1-T2 若能获得一次短休，魔契师的有效高价值法术位投放会明显追上甚至压过普通满施法者；若短休稀少，连续遭遇中的容错仍低于法师、牧师、德鲁伊等准备满施法者。

2级`秘法回流`是新版底盘的重要缓冲：1分钟仪式、1/长休、恢复一半已消耗魔契师法术位向上取整。它能救一次资源误判或短休失败，但不是稳定短休替代品。低级防御则主要来自祈唤选择而非职业表本身；`邪魔活力`按本地XPHB条目给动作版`虚假生命`，固定12点临时生命，是很强的早期容错，但占祈唤位，也会和其他低级祈唤路线竞争。`链之魔契`也不是纯侦察，特殊形态能提供飞行/隐形/抗性/毒伤、帮助、触碰法术投递、占位和诱导敌人动作。

需要单独校准的是近战与短距多段命中。魔契师拥有`魂灵环绕`，但纯魔契师因魔契法术位机制通常最高只按5环施放，也就是每次命中额外`2d8`，不能像普通满施法者那样自然用7-9环法术位升到`3d8/4d8`。12级后`灭世魔刃`让刃契路线在攻击动作中作三次攻击；若三击全中，5环`魂灵环绕`提供额外`6d8`，平均27点，和一次5环`魔能斩`的额外`6d8`同量级。把二者叠在同一轮可形成很高爆发，但这需要贴近10尺、维持专注、先启动法术、命中多个攻击，并消耗魔契师最宝贵的契约法术位。它证明刃契魔契师不是低伤害笑话，但不等于拥有圣武士灵光或满施法者高环可调度性。

宗主法术表也要逐项看。`灼热射线`若由宗主法术表授予，能和`魂灵环绕`形成短距多段法术攻击爆发；5环`灼热射线`有六道射线，全命中时基础`12d6`，5环`魂灵环绕`再加`12d8`，但这要两枚契约法术位、前置专注、10尺距离和命中率，真实桌面不能当作稳定DPR。`咒火闪焰`本地速查显示是术士/法师向FR法术，不是魔契师默认法术；本轮只在宗主法术表确实授予时才计入，没有把它当作全体魔契师资源。

9级`联络宗主`补一次安全的`异界探知`，更多是信息和剧情接口，不是战斗强度核心。11级后的`玄奥秘法`才是高阶评价的关键：6、7、8、9环各有一个无槽长休施放入口，每次升级还能换同环秘法。它让魔契师进入真正高阶施法区，但选择固定，不能像准备满施法者那样每天重排，也不自带`祈愿术`。20级`魔能掌控`把`秘法回流`改成恢复全部已消耗魔契师法术位，强但来得极晚，且仍围绕契约槽而非扩大每日高环调度。

社区校准方面，RPGBOT的2024 Warlock指南和常见公开讨论都很重视新版祈唤弹性、`魔能爆`/`推斥魔爆`控制、刃契路线可行性、`秘法回流`补强，以及短休节奏对上限的决定性影响；同时也普遍提醒`玄奥秘法`固定、契约位少、已准备法术/祈唤机会成本高。本轮独立结论与这一外部口径基本一致：魔契师是非常强的定制化短休施法者，高阶能进`S-`，但默认不是法师、牧师、德鲁伊式可每日重构的`S`级主引擎。宗主如果只给个人数值或风味工具，不能因为职业底盘强就自动上调；宗主若改变法术递送、团队资源、短休循环、伙伴行动或控制可靠性，才会真正推高评分。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | A+ | 1级`魔能祈唤`、短休`契约魔法`和`魔能爆`已经给出稳定远程/控制底线；2级`秘法回流`提供一次应急回槽，`邪魔活力`固定12点临时生命或`链之魔契`特殊魔宠都能显著改善低级容错。限制是法术位数量少、短休节奏敏感，宗主到3级才真正展开。 |
| Tier 2（5-10） | A+ | 5-10级射线数、祈唤数量、3-5环`契约魔法`、`法术反制`和9级`联络宗主`到位；一到两次短休会让高价值法术位次数非常竞争。它仍没有准备满施法者的每日调表，高影响法术选择、祈唤机会成本和短休节奏决定上限，所以不自动进S-。 |
| Tier 3（11-16） | S- | 11-16级`玄奥秘法`提供6-8环无槽入口，12级`灭世魔刃`也让刃契三打成型；固定奥秘、短休依赖和专注/法术位竞争使其低于准备满施法者。 |
| Tier 4（17-20） | S- | 17-20级有9环`玄奥秘法`、19级传奇恩惠和20级`魔能掌控`，但无默认`祈愿术`、奥秘固定，`魂灵环绕`也通常止于5环魔契法术位，因此仍是S-而非S。 |

## 官方宗主横向校准
本轮完成XPHB魔契师底盘与所有官方宗主重评后，官方宗主的横向口径调整为：`巨灵宗主`和`死灵宗主`是高阶官方顶层，前者靠团队短休、`有限祈愿术`与扩表`祈愿术`进入`S`，后者靠`战栗形态`、暗蚀递送、物理抗性、飞行穿墙和无成分咒法/死灵施法进入`S`。`咒剑`、`天界宗主`、`旧日支配者宗主`、`深海意志`、`至高妖精宗主`和`邪魔宗主`在高阶都归入`S-`：它们分别强化前线诅咒、副治疗/复原、隐秘控制/压豁免、触手控场、传送安全或爆破/单体移除，但没有巨灵式队伍资源引擎，也没有默认准备满施法者的每日重构能力。`不朽者`维持弱宗主定位，高阶主要靠魔契师底盘托底。
