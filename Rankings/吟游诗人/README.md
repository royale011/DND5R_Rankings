## 队伍定位总结
2024 吟游诗人的核心位置是主控/副治疗/技能中枢，而不是默认武器输出者。它用 `吟游诗人激励` 把关键攻击、豁免、检定和部分子职防线推过临界点，用 `专业技能` 与 `万事通` 接管社交、侦察、知识和工具任务，用治疗词类法术负责倒地拉人；在没有勇气、剑舞、舞蹈等子职支撑时，它不适合长期承担前排或无资源 DPR。

10级 `魔法奥秘` 是职业跃升，但不是“每日自由全表准备”。吟游诗人只能用有限准备数和升级替换来锁定牧师、德鲁伊、法师等列表中的关键答案；`法术卷轴`也只加深已经通过 `魔法奥秘` 实际准备的跨表法术，而不会让它临场读取所有法师/牧师/德鲁伊卷轴。因此高阶吟游诗人的真实强度取决于队伍缺口与选法规划：它能补 `法术反制`、复活、墙体、召唤、防御或输出组件，但不能同时把所有路线当作常备。

## 详细评价
Tier 1 的吟游诗人已经有清楚的队伍价值。`吟游诗人激励` 用附赠动作补关键 d20 测试，低环惑控、幻术和治疗法术让它能制造行动损失、社交突破和倒地救援；`专业技能` 则让它在非战斗场景比多数职业更稳定。短板也同样明显：护甲、生命骰、直接伤害和法术位都有限，若没有子职补护甲或机动，它更应该待在中后排。

Tier 2 的 5级 `激励之源` 是节奏核心，短休恢复激励并允许用法术位补激励，让它能把资源从“少数关键回合”扩展成整日支援。3-5环吟游诗人法术本身已经能覆盖控制、治疗、侦查和社交，10级末端的 `魔法奥秘` 再把跨表答案放入职业框架。这里必须按实际升级节奏评分：5-9级还没有跨表自由度，10级也只是开始锁定少数关键法术，不是立即完成所有优化表。

Tier 3-4 的吟游诗人是强满施法者，但和法师/牧师/德鲁伊的强法表日备不同。它能通过后续 `魔法奥秘` 和高环法术成为队伍第二主引擎，尤其擅长反制、救场、技能、控制和补位；但每个跨表选择都有机会成本。若拿 `咒唤微元素群`、`魂灵环绕` 或 `灼热射线` 等多段命中输出组件，就会挤压 `法术反制`、复活、墙体、召唤、防御、位移和解场等常见奥秘名额。评价吟游诗人时必须把这种锁位压力写进强度，而不是把所有社区优化路线叠在同一张角色卡上。

## 校准说明
本轮横向比较对象为法师、牧师、德鲁伊、术士、灵能使与秘术师。吟游诗人高阶仍按 S- 处理：魔法奥秘让它能拿到跨表答案，甚至在终局规划 祈愿术 等关键法术，但它不是每日全表准备，也没有法师法术书/仪式经济或牧师、德鲁伊的日备全表。子职若进入 S，必须靠提前跨表、豁免压制、团队先攻/行动经济或强防线提供实质增量，而不是只凭吟游诗人底盘。
