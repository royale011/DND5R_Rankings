# 调查员

所属职业：游荡者  
英文名 / 社区检索名：Inquisitive  
来源 / 版本：珊娜萨的万事指南兼容评价

## 队伍定位调整
调查员强化洞察、侦查、识破谎言和对单体弱点的判断。它在侦探、阴谋、社交和调查战役里很有价值；在纯战斗优化里偏弱。

## 详细评价
低级能力围绕洞察、调查和发现隐藏敌意展开。这些能力很贴合主题，也能让游荡者在信息战里更强。问题是 2024 游荡者底盘已经有 `专精` 和早到的 `可靠才能`，调查员的相对优势被压缩。

单体洞察战法让调查员在没有优势或队友贴身时仍能偷袭。它能解决偷袭触发问题，但不像风流剑客那样带机动，也不像魂刃那样补技能失败。

中级看穿幻象、发现隐藏和识破伪装在特定战役非常强。如果 DM 频繁使用骗局、伪装、隐形和暗杀，调查员会上浮；如果战役主要是正面打怪，它就显得没有子职。

高等级调查员仍是“找出真相”的专家，而不是正面战斗强者。它的综合评分应按战役型功能给中档，不按 DPR 给高分。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | B | 洞察和调查能力让它在信息战强，但正面战斗提升有限。 |
| Tier 2（5-10） | B | `可靠才能` 让调查员身份更稳，适合阴谋/社交战役。 |
| Tier 3（11-16） | C | 高等级仍缺少法术和强战斗工具，主要价值在侦查与识破。 |
| Tier 4（17-20） | C | 终局能发现问题，但不能像施法者那样解决大问题。 |
