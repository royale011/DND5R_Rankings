# 鬼魅

所属职业：游荡者
英文名 / 社区检索名：Phantom
来源 / 版本：塔莎的万事坩埚

## 队伍定位调整

鬼魅用亡魂知识、偷袭溅射、灵魂饰品和幽灵移动扩展游荡者。它成长较慢，但 Tier 3 后功能明显变宽。

## 详细评价

鬼魅3级的可变熟练和偷袭溅射让它比普通游荡者更灵活。可变熟练在技能密集战役不错，偷袭溅射则让游荡者少见地影响第二目标；但低级次数和触发都有限。

9级 `逝者遗物` 或同类灵魂饰品机制是转折点。它给资源、保命或询问死者式工具，让鬼魅终于不只是“多一点伤害”。不过饰品经济依赖有生物死亡，战役若少战斗或敌人类型特殊，资源会波动。

13级幽灵移动/穿透能力让鬼魅 Tier 3 评价上升。穿墙侦察、绕过门锁、从危险中撤离都是真功能，和游荡者潜入职责高度贴合。17级通常完成死亡主题，但仍不是复活或高环死灵法术。

因此 TCE 版鬼魅 Tier 3 给 A：它在中后期确实扩展了游荡者问题解决。终局仍是 B，因为高等级生态不只看侦察和穿墙。
