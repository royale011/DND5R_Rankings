# 游荡者总评

## 队伍定位摘要
游荡者是技能、侦察、潜入、巧手和无资源可靠性的专家。2024 版用 `武器精通`、`狡诈打击`、提前到 Tier 2 的 `可靠才能`、更丰富的偷袭消耗选项改善了手感，但没有把游荡者变成顶级 DPR 或高等级法术解题职业。它最强的队伍位置是“把不确定的任务变成稳定成功”，而不是在正面遭遇里替代战士、武僧或施法者。

## 详细评价
1级 `专精` 和 `偷袭` 决定职业身份。`专精` 让潜行、巧手、察觉、调查、欺瞒、游说等关键检定很早就超过多数职业；这不是小修小补，而是直接改变探索和社交遭遇的成功率。`偷袭` 给无资源单点伤害，但通常每轮一次，和战士三四次攻击、武僧多打、圣武士爆发不是同一条伤害曲线。

2024 的 `武器精通` 改善了偷袭触发稳定性。`干扰` 能制造后续优势，`迅击` 能支持双持节奏，远程武器也更容易保持安全输出。但这些是让游荡者“更稳”，不是把它推成同 Tier 最高伤害者。社区对新版游荡者的常见评价也是如此：更好玩、更可靠，但优化上限仍偏保守。

2级 `灵巧动作` 是生存和任务执行核心。附赠动作撤离、躲藏、冲刺让游荡者用位置而不是硬度活下来。5级 `狡诈打击` 是新版最大设计进步：你可以牺牲偷袭骰换缴械、投毒、绊摔、撤离等效果，让偷袭从纯伤害变成选择题。问题也很清楚：这些控制不是免费，你用本来就不夸张的伤害买控制，所以战斗评分不能过度上调。

7级 `可靠才能` 提前是实质增强。Tier 2 开始，游荡者很多熟练技能的 d20 低结果会被抬高，这让它在陷阱、潜入、追踪、社交和侦查中成为真正稳定器。11级 `强化狡诈打击`、14级 `诡诈打击` 让控制组合更灵活；15级 `心灵灵动` 补精神豁免；18级 `难以捉摸` 和20级 `幸运一击` 保证高等级个人稳定性。

高等级必须诚实：游荡者不会提供复活、传送、墙、召唤、反制、群控或现实改写。它在技能密集、潜入、侦查、机关和单目标任务中非常强；在大规模高魔法遭遇中，需要队友给法术框架。综合校准为 B/B/C/C：比旧文件的 Tier 3-4 D 更公平，但仍不能和满施法者或高功能半施法者同档。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | B | `专精`、`偷袭`、`灵巧动作` 和 `武器精通` 让游荡者早期非常可靠，但伤害和防御都不是顶级。 |
| Tier 2（5-10） | B | `狡诈打击` 与提前的 `可靠才能` 大幅提升技能和战斗选择；短板是控制要牺牲偷袭骰，队伍支援仍少。 |

## 分阶段子职排行榜

Tier 1（1-4）:
- A:
  - 诡术师：3级法术、`法师之手戏法` 与游荡者技能叠加，低级就获得防御、潜入和控制入口。
  - 魂刃：灵能骰、心灵刀刃和无声通信让低级任务可靠性极高，且不依赖装备。
- B:
  - 三神门徒：信仰/派系工具有场景价值，但默认战斗强度主要靠游荡者底盘。
  - 刺客：开场爆发强且比旧版稳定，但仍依赖先手和遭遇结构。
  - 斥候：反应撤离和自然技能让它安全、可靠，但战斗提升不高。
  - 盗贼：`快速之手` 和机动让它非常会处理物品/机关，但低级强度依赖可用装备。
  - 策士：远程 `协助` 和社交工具有用，但战斗强度偏软。
  - 调查员：洞察和调查能力让它在信息战强，但正面战斗提升有限。
  - 风流剑客：单挑偷袭和先攻改善近战手感，但功能面仍窄。
  - 鬼魅：3级有可变熟练和偷袭溅射，低级功能宽于纯输出游荡者。

Tier 2（5-10）:
- A:
  - 诡术师：2环法术和 `狡诈打击` 共存，能同时做技能专家、潜入者和小控制者，是游荡者最完整路线。
  - 魂刃：`可靠才能` 与灵能骰叠加后，魂刃成为最稳定的技能/潜入路线之一。
- B:
  - 三神门徒：`可靠才能` 与宗教/社交权限结合后适合剧情任务，但硬机制不如顶级子职。
  - 刺客：毒素、伪装和先手爆发有价值；持续战斗和队伍支援不如顶级子职。
  - 斥候：`可靠才能` 加野外技能让探索强，正面功能仍窄。
  - 盗贼：魔法物品和消耗品越丰富越强；默认战役不假设无限物品，所以不越过 B。
  - 策士：`可靠才能` 加社交/伪装能力让它在阴谋战役表现好，正面战斗仍弱。
  - 调查员：`可靠才能` 让调查员身份更稳，适合阴谋/社交战役。
  - 风流剑客：`可靠才能`、机动和社交让它很顺，但不如诡术师或魂刃扩展问题解决。
  - 鬼魅：`逝者遗物` 让鬼魅有稳定资源与灵魂工具，但仍依赖死亡触发。

Tier 3（11-16）:
- A:
  - 诡术师：更高环法术、反施法/偷法术能力和 S 级技能让它继续有问题解决面。
  - 鬼魅：`幽灵漫步` 加上饰品经济让侦察、生存和穿透地形能力明显上升。
- B:
  - 斥候：高移动和侦察继续有用，适合远程游荡者。
  - 盗贼：高等级物品使用和潜入能力扩大问题解决面，仍缺少自身法术进度。
  - 风流剑客：高级机动与单挑仍可用，战斗外社交稳定；缺少高魔法工具。
  - 魂刃：传送、隐形和技能保底仍强，但缺少法术级群控与反制。
- C:
  - 三神门徒：若没有明确高等级神术级工具，综合随游荡者高等级短板下滑。
  - 刺客：高等级暗杀能力有高光，但强敌防线和传奇资源让稳定性下降。
  - 策士：高等级功能仍偏社交防护，缺少硬控制和法术解题。
  - 调查员：高等级仍缺少法术和强战斗工具，主要价值在侦查与识破。

Tier 4（17-20）:
- B:
  - 盗贼：17级额外回合潜力很高，但强度高度依赖魔法物品生态和规则窗口。
  - 诡术师：终局仍不是满施法者，但在游荡者内部保留最高综合上限。
  - 鬼魅：终局死亡主题完整，但缺高环法术、团队救场和大范围控制。
  - 魂刃：终局任务执行能力优秀，正面高魔法问题解决不足。
- C:
  - 三神门徒：终局默认仍缺复活、反制和战役级神术。
  - 刺客：终局仍能爆发，但不能解决高魔法战役问题。
  - 斥候：终局缺高等级队伍工具，探索强度不足以支撑更高综合。
  - 策士：终局若不是政治/情报战役，机制贡献偏低。
  - 调查员：终局能发现问题，但不能像施法者那样解决大问题。
  - 风流剑客：终局仍主要是个人决斗路线，综合功能不足。

## UA/合作方/第三方子职分阶段排行榜

Tier 1（1-4）:
- A:
  - 厄运使者（鬼魅幽谷）：低级诅咒选项已经能改变敌方攻击、豁免或状态，是强减益开局。
  - 暗影潜行者（克苏鲁）：低级影像、斥候和换位让潜入与战斗机动都明显强于普通游荡者。
  - 窃法者（UA）：3级就以 `注能偷袭` 和 `吸取魔法` 获得反魔入口，低级功能明显宽于普通游荡者。
  - 蛛网追猎者（秘密尖塔）：3级就有蛛丝移动、远程物体操作、网绳和 `蛛网术`，低级战场分隔很强。
  - 蜡艺师（胧忆岛）：低级魔艺已经给烟雾、束缚、封门、临时生命和小制造，功能面远超普通游荡者。
- B:
  - 残戮行医（狮鹫鞍包）：低级治疗入口稀缺，但行动经济会和偷袭输出竞争。
  - 缚影使（黯潮之书）：低级视野与影子绑定有用，但还不是完整控场核心。
  - 罪人（歪曲之月）：低级厄运表和力场型厄运武器有用，但随机性让强度不稳定。
  - 鬼魅（UA）：UA 低级仍是技能弹性和偷袭溅射，但 `墓地哀嚎` 次数更可靠。

Tier 2（5-10）:
- S:
  - 窃法者（UA）：9级 `扰断法术` 能让目标施法动作被浪费，是游荡者极少见的硬反施法工具。
- A:
  - 厄运使者（鬼魅幽谷）：厄运数量和 `偷窃幸运` 让资源更耐用，队伍集火价值很高。
  - 暗影潜行者（克苏鲁）：暗影庇护和影子干扰让它成为优秀侦察/扰乱位。
  - 残戮行医（狮鹫鞍包）：战地医师式强化让它成为少见的治疗/输出混合游荡者。
  - 缚影使（黯潮之书）：无影攻击与黑暗魔法让潜入、减益和偷袭环境更稳定。
  - 蛛网追猎者（秘密尖塔）：攀墙和蛛网控场让它在垂直战场、潜入和追击中价值很高。
  - 蜡艺师（胧忆岛）：更多魔艺和烛心点让它成为控场/工具/制造混合位，队伍问题解决很宽。
  - 鬼魅（UA）：长休基础饰品与 `死亡之声` 让鬼魅不再完全依赖场上死亡，信息与资源更稳定。
- B:
  - 罪人（歪曲之月）：赌具、诅咒和失败修正提高容错，但不如可控减益子职稳定。

Tier 3（11-16）:
- S:
  - 窃法者（UA）：`奥秘面纱` 与 `进阶吸取魔法` 让反施法、潜入和偷取魔法同时扩张。
- A:
  - 厄运使者（鬼魅幽谷）：13级后诅咒投放更稳定，但仍受豁免、免疫和资源限制。
  - 暗影潜行者（克苏鲁）：`暗影之门` 强化空间能力，能改变潜入和战斗路线。
  - 残戮行医（狮鹫鞍包）：败血休克把医疗主题转成减益，队伍贡献明显高于纯输出。
  - 缚影使（黯潮之书）：高等级暗影工具继续强化侦察和减益，但场景依赖仍在。
  - 蛛网追猎者（秘密尖塔）：蜘蛛感应提高生存，蛛网与机动仍稳定；毒素路线开始受免疫影响。
  - 蜡艺师（胧忆岛）：`强化魔艺` 扩大持续时间、范围和同时数量，是第三方高功能路线。
  - 鬼魅（UA）：`幽灵漫步` 配合更可靠饰品经济，侦察和生存都强于 TCE 版。
- B:
  - 罪人（歪曲之月）：偷袭骰改命运很强，但消耗输出资源，且随机厄运仍不可控。

Tier 4（17-20）:
- A:
  - 窃法者（UA）：终局反魔仍强，但没有满施法者的全局控场和现实改写，不给 S+。
  - 蛛网追猎者（秘密尖塔）：麻痹毒液提供强终局单体控制，但体质豁免和传奇抗性限制 S+。
  - 蜡艺师（胧忆岛）：终局制造与控场仍强，但没有满施法者级传送、复活或现实改写，不给 S+。
- B:
  - 厄运使者（鬼魅幽谷）：终局仍是强减益游荡者，但缺高环法术和跨遭遇现实改写。
  - 暗影潜行者（克苏鲁）：真实暗影很强，但终局仍缺高环法术级问题解决。
  - 残戮行医（狮鹫鞍包）：终局仍缺复活链和高环治疗，只能算强副治疗。
  - 缚影使（黯潮之书）：终局仍缺高环法术，面对真视/盲视敌人会被压缩。
  - 罪人（歪曲之月）：终局风格完整但波动大，不能按最佳掷骰给高档。
  - 鬼魅（UA）：终局仍是高功能游荡者而非施法者；可靠性提高，但没有全生态顶级解题。

## 设计相关评分
| 设计项 | 评分 | 具体理由 |
|---|---|---|
| 设计质量 | B | 2024 游荡者比旧版更有选择，`狡诈打击` 是成功设计；但高等级仍没有足够队伍级工具。 |
| 主题 | A | 专家、潜入者、机会主义者、机关处理者和精准打击者主题非常清楚。 |

| 主题与能力关联度 | A | 绝大多数能力都围绕“用技巧而不是蛮力取胜”。 |

## 校准说明
本轮以 `DND5e_chm\玩家手册2024\角色职业\游荡者` 为底盘。公开社区评论普遍认为 2024 游荡者玩法质量提升，但伤害/高等级功能仍保守；本评分因此上调 Tier 2-4 的综合下限，但没有把它抬成顶级战斗职业。
