# 鬼魅（UA）

所属职业：游荡者
英文名 / 社区检索名：Phantom UA
来源 / 版本：新UA：贤者谏言 / 游荡者-鬼魅

## 队伍定位调整

UA 鬼魅比 TCE 版更稳定地处理灵魂饰品和死亡资源，整体可靠性略高。

## 详细评价

UA 鬼魅与 TCE 鬼魅的核心差异在资源稳定性。如果 UA 文本给长休基础饰品、较可靠的死亡之声或更顺的魂魄消耗，它会显著降低 TCE 版“等敌人死才有资源”的波动。

低级仍主要是技能弹性和第二目标伤害，强但不破格。Tier 2 开始，若饰品和死亡之声更稳定，鬼魅的信息、支援和资源回收会更像可计划工具，而不是战场副产物。

13级幽灵移动仍是 Tier 3 上调理由；UA 版若资源更稳，侦察和生存表现会比 TCE 更可靠。限制仍是高等级没有复活、反制和大范围控制。

因此 UA 鬼魅总体比 TCE 版略稳，仍不进入高等级 S。
