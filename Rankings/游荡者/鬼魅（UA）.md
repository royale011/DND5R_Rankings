# 鬼魅（UA）

所属职业：游荡者（Rogue）
英文名 / 社区检索名：Phantom UA update
来源 / 版本：新 UA：可怖子职，鬼魅更新

## 队伍定位调整
UA 鬼魅强化了官方鬼魅的资源可靠性：`墓地哀嚎` 次数改与敏捷相关，`逝者遗物` 长休给基础饰品，并新增 `死亡之声`。它比 TCE 版更稳定。

## 详细评价
本地 UA 明确写出相较 TCE 的主要改动：`墓地哀嚎` 可用次数与敏捷调整值相关，`逝者遗物` 每次长休提供一定数量灵魂饰品，新增 `死亡之声` 提供 `死者交谈` 类能力。9级饰品用法包括 `丧钟`、`生命精华` 与 `灵魂问询`，其中 `灵魂问询` 可用魔法动作消耗饰品施展 `卜筮术`。13级 `幽灵漫步` 仍给飞行、朦胧形态和虚体移动，17级 `死亡之友` 强化死亡收割与饰品恢复。UA 版的强点是把原本 DM/遭遇依赖的饰品经济变得可计划，弱点是仍没有真正反制、复活或群体控制。
