# 策士

所属职业：游荡者  
英文名 / 社区检索名：Mastermind  
来源 / 版本：剑湾冒险者指南 / 珊娜萨的万事指南兼容评价

## 队伍定位调整
策士是社交、伪装、语言和远程战术辅助路线。它很有“幕后操盘者”味道，但机制强度长期偏低，尤其在 2024 游荡者获得更好基础选项后，策士更显得旧。

## 详细评价
低级远程 `协助` 是策士最实际的战斗功能。它能把队友的关键攻击或检定推高，尤其适合队里有圣武士、战士、狙击手或高伤害单击构筑时使用。

问题是 `协助` 本身不是强到能支撑整个子职的机制。它占用附赠动作，而游荡者自己的 `灵巧动作`、双持、瞄准和物品互动也要附赠动作。策士经常在“帮队友”和“保自己”之间竞争。

中级伪装、语言和读心防护很适合社交战役。它能扮演间谍、外交官或犯罪顾问，但这些能力对战斗评分帮助很有限。

高等级策士如果战役重视政治和情报，会很有存在感；如果是高等级地下城战，它的机制强度明显落后于诡术师、魂刃和盗贼。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | B | 远程 `协助` 和社交工具有用，但战斗强度偏软。 |
| Tier 2（5-10） | B | `可靠才能` 加社交/伪装能力让它在阴谋战役表现好，正面战斗仍弱。 |
| Tier 3（11-16） | C | 高等级功能仍偏社交防护，缺少硬控制和法术解题。 |
| Tier 4（17-20） | C | 终局若不是政治/情报战役，机制贡献偏低。 |

## 校准说明
策士是典型“主题高于强度”的旧版子职，本轮不因游荡者底盘增强而抬高过多。
