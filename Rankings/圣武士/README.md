# 圣武士总评

## 队伍定位摘要

圣武士是 2024 规则下最强的非满施法队伍防御核心之一：前排、治疗、豁免灵光、坐骑机动、半施法支援和稳定武器伤害集中在同一个职业里。它的个人爆发被新版 `至圣斩` 明显压住，但 `防护灵光` 仍然足以让它在 Tier 2 成为队伍构筑中心。

## 详细评价

1级同时取得 `圣疗`、`施法` 和 `武器精通`，让圣武士从低级就不是单纯武器职业。`圣疗` 以附赠动作使用后，拉起倒地队友、补小额治疗、解除中毒都不再吞掉整回合动作；这在 Tier 1 的实战价值高于纸面治疗量。1级法术让 `祝福术`、`命令术`、`治愈伤口` 和防护类法术进入构筑考虑，但法术位少，专注也会和后续打击法术、誓言法术竞争。

2级 `战斗风格` 和 `圣武士斩击` 定义输出路线。新版 `至圣斩` 仍然是命中后补伤害的强按钮，但它是法术，通常每回合一次，并会吃附赠动作和法术位；因此不能再按 2014 的多次命中连环爆发来评分。`武器精通` 提高了普通攻击回合的战术底线，也让优势、倒地、推位、减速等来源变得更常见；这会削弱某些“给优势”子职特性的独占性。

3级取得子职和 `引导神力`，但底盘真正跃迁在 5-6级。5级 `额外攻击` 和 `忠诚坐骑` 给稳定武器输出、坐骑机动、战术占位和侦察入口；`寻找坐骑` 固定准备且每日一次免法术位，使圣武士天然拥有伙伴/坐骑生态。6级 `防护灵光` 是职业核心：把魅力调整值加入灵光内全体豁免，直接保护队伍免受控制、范围伤害、即死、恐惧、魅惑和许多法术的关键失败。

9级 `驱退敌人` 给多目标恐慌和行动限制，10级 `勇气灵光` 免疫恐惧，11级 `光耀打击` 给每次近战攻击和徒手打击补光耀伤害，14级 `复原之触` 把 `圣疗` 点数转成强状态清除。18级 `灵光扩展` 把核心防线扩到 30尺，是 Tier 4 仍有竞争力的关键。限制也很清楚：半施法进度、专注冲突、附赠动作压力、远程和大范围解题不足，使它无法像满施法者那样用传送、墙、召唤、反制链或现实改写主导所有高等级场景。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | A | `圣疗`、1级施法、`武器精通` 和重甲/盾牌让低级已经很全面，但还没有 `防护灵光`，队伍级压制力未成型。 |
| Tier 2（5-10） | S | 5级 `额外攻击` 与 `忠诚坐骑`、6级 `防护灵光`、9级 `驱退敌人`、10级 `勇气灵光` 叠加，圣武士成为 Tier 2 最强队伍防线之一。 |
| Tier 3（11-16） | A+ | 11级 `光耀打击`、14级 `复原之触` 和更高环法术让底盘继续增强；但半施法者缺少满施法者的高阶解题宽度，因此是 A+ 而非 S。 |
| Tier 4（17-20） | A | 18级 `灵光扩展` 与 5环圣武士法术仍强，但 Tier 4 比拼的是终局现实改写、传送、反制和群控，圣武士主要靠防线与近战存在感取胜。 |
