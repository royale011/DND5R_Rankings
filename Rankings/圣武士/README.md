# 圣武士总评

## 队伍定位摘要
圣武士是 2024 规则下最强的队伍防御核心之一：前排、治疗、豁免保护、坐骑机动、半施法支援和稳定武器伤害集中在同一个职业里。它不再是 2014 那种把多个法术位瞬间烧成爆发伤害的职业；新版 `至圣斩`是法术，通常每回合一次，动作经济更紧。但这个削弱没有碰到圣武士真正的队伍价值：`防护灵光` 仍然是全游戏最强的被动团队防御之一。

## 详细评价
1级同时取得 `圣疗`、`施法` 和 `武器精通`，让圣武士从开局就比多数武器职业更全面。`圣疗` 改成附赠动作后，战斗中拉起倒地队友或解除中毒不再吞掉整回合动作；这让它在低级尤其可靠。1级施法给 `祝福术`、`英勇术`、`治愈伤口` 等入口，`武器精通` 则让重武器、剑盾、投掷或长柄路线都有实际战术选择。

2级 `战斗风格` 和 `圣武士斩击` 决定输出方式。`至圣斩`仍是命中后补伤害的强按钮，但现在是法术并受附赠动作和每回合一次限制，不能再按旧版逻辑把多个命中全部转成爆发。这个改动让圣武士近战伤害从“瞬间爆炸”回到“高但受资源和动作限制”。因此本轮不再把基础圣武士评成全阶段 S 级伤害者。

3级 `引导神力` 的通用选项 `神圣感知` 偏侦测，9级 `驱退敌人` 才是战斗控制核心：它以多目标恐慌和行动限制提供真实减益，且使用 `引导神力`，短休/长休节奏下可重复。子职的 3级引导选项会决定早期风格，但圣武士底盘真正爆发在 5-6级。

5级 `额外攻击` 和 `忠诚坐骑` 是 Tier 2 的第一波跃迁。`寻找坐骑` 被职业固定准备且每日一次免法术位，意味着圣武士天然拥有伙伴、机动、侦察和战术占位工具。即便不骑乘，坐骑也能参与场景处理；若战役允许更高环施放，后期坐骑还会随法术环级提升。

6级 `防护灵光` 是圣武士评分的核心。把魅力调整值加到自己和灵光内盟友的所有豁免，直接压低控制、范围伤害、即死、恐惧、魅惑和大量法术的成功率。社区对 2024 圣武士的争议常集中在 `至圣斩`削弱，但优化评价仍普遍承认：只要 `防护灵光`存在，圣武士就是队伍防线的顶级职业。10级 `勇气灵光`、18级 `灵光扩展` 继续强化这一点。

11级 `光耀打击` 让每次近战攻击和徒手打击附加光耀伤害，保持武器回合厚度；14级 `复原之触` 用 `圣疗`点数移除目盲、魅惑、耳聋、恐慌、麻痹、震慑等状态，实际是把治疗资源转化为强力状态修复。高等级圣武士仍很强，但它的高等级问题解决主要是防护和支援，不是满施法者式传送、墙、群体召唤、反制链或跨位面解题。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | A | `圣疗`、1级施法、`武器精通` 和重甲/盾牌让圣武士低级已经是优秀前排与应急治疗者，但尚未取得 `防护灵光`，队伍级统治力还未成型。 |
| Tier 2（5-10） | S | 5级 `额外攻击`、免费准备并每日一次免槽的 `寻找坐骑`、6级 `防护灵光`、9级 `驱退敌人` 与10级 `勇气灵光` 叠加后，圣武士成为全队豁免和近战阵地的顶级核心。 |
| Tier 3（11-16） | A | `光耀打击`、`复原之触` 和更高环誓言/圣武士法术继续增强底盘，但半施法进度无法覆盖满施法者的传送、墙、群控和现实改写，综合从 Tier 2 的压制级回落。 |
| Tier 4（17-20） | A | 30尺灵光、5环法术、20级子职形态和高额 `圣疗` 仍非常强；短板是终局解题宽度不如满施法者，伤害也不再靠旧版连环至圣斩爆发压场。 |

## 分阶段子职排行榜

Tier 1（1-4）:
- S:
  - 巨灵贵族之誓：3级元素选项同时给伤害、推位/控制和短期防御，低级明显超过多数圣武士子职。
- A:
  - 古贤之誓：`自然之怒` 提供低级少见的多目标束缚，法术表有控制入口。
  - 复仇之誓：`仇敌誓言` 让单体命中和爆发最早成型，但队伍保护主要仍靠基础圣武士。
  - 奉献之誓：`神圣武器` 让魅力优先仍能可靠命中，低级防御法术也实用。
  - 守望者之誓：低级防精神豁免和反异界工具强但场景化，仍有圣武士底盘托底。
  - 征服之誓：早期恐惧和命中修正有压制力，但核心 `征服灵光` 尚未取得。
  - 荣耀之誓：`激励斩` 的可分配临时生命让低级队伍容错明显提高。
- B:
  - 救赎之誓：早期交涉和防护主题有用，但伤害和主动压制弱于多数圣武士子职。
  - 王冠之誓：早期防卫者概念能用，但动作成本和敌人配合问题让它低于新版强子职。
  - 破誓者：`恐惧面容` 和旧版 `操控亡灵` 上限高，但低级是否遇到可控不死完全看战役。

Tier 2（5-10）:
- S:
  - 守望者之誓：`哨兵灵光` 的全队先攻加值和 `反制法术` 入口让它在优化环境非常强。
  - 巨灵贵族之誓：元素抗性灵光、强法术表和圣武士底盘叠加，攻防覆盖面过宽。
  - 征服之誓：`征服灵光` 与恐惧效果形成速度归零控制，是圣武士内部最强 Tier 2 控场之一。
- A:
  - 古贤之誓：`守御灵光` 的暗蚀、心灵、光耀抗性与 `防护灵光` 叠加，防御价值高但覆盖面不是全伤害。
  - 复仇之誓：`迷踪步`、`加速术`、`无情复仇者` 与 `防护灵光` 叠加后追击能力很强，但不是顶级团队灵光子职。
  - 奉献之誓：`防护灵光` 与 `奉献灵光` 叠加后防魅惑很强，但常规控场和伤害不如征服/复仇/巨灵贵族。
  - 救赎之誓：承伤灵光、`反制法术` 和控制法术让它成为优秀保护者，代价是输出较低。
  - 破誓者：`憎恨灵光` 能把近战伤害推高，并给亡灵队伍明显收益；敌我风险和队伍适配阻止它进 S。
  - 荣耀之誓：机动灵光、`增强属性`、`加速术` 与圣武士底盘结合很实用，但缺少顶级控制或反制。
- B:
  - 王冠之誓：`防护灵光` 托底，但王冠额外能力不如征服、守望者或奉献直接。

Tier 3（11-16）:
- S:
  - 巨灵贵族之誓：15级反击/防御与高环元素法术继续扩大优势，是圣武士内部最强路线。
- A:
  - 古贤之誓：`不灭哨卫` 和抗性灵光让自己很难倒下，控制法术仍有用但主动压场不如征服。
  - 复仇之誓：`复仇之魂` 给稳定反应攻击，单体压力优秀；高等级综合受限于功能面偏窄。
  - 奉献之誓：`卫护斩` 把 `至圣斩` 转成全队短期半掩护，是非常明确的团队防护增益。
  - 守望者之誓：高等级施法者更多，守望者工具更有命中率；但非异界战役仍会浪费部分文本。
  - 征服之誓：恐惧锁仍强，但高等级免疫、传奇抗力和非地面敌人让稳定性下降。
  - 荣耀之誓：防御反应和临时生命使其成为可靠护卫，强度稳定但不压过守望者/征服。
- B:
  - 救赎之誓：自我恢复和保护仍强，但高等级需要主动解决威胁，救赎偏被动。
  - 王冠之誓：分担伤害和守线仍有价值，机制老旧导致高等级影响有限。
  - 破誓者：`超自然抗性` 稳定但偏旧版，恐惧/亡灵路线仍强烈依赖敌人类型。

Tier 4（17-20）:
- S:
  - 巨灵贵族之誓：20级飞行、悬停和小型许愿式成功保障触及全职业防御天花板，足以保持 S。
- A:
  - 古贤之誓：`上古斗士` 提升法术节奏和再生，终局强度稳定但不超越巨灵贵族。
  - 复仇之誓：`复仇天使` 的飞行和恐惧适合终局猎杀，但它仍主要解决“杀一个目标”，不是全面解题。
  - 奉献之誓：`神圣光辉` 对邪魔/不死和拥挤战场很强，但终局适用面仍不如巨灵贵族的泛用防御。
  - 守望者之誓：终局反异界/反施法仍强，默认不因场景化能力给 S。
  - 征服之誓：终局形态提升压迫与生存，对可恐惧敌人仍顶级；默认环境不按 S 处理。
  - 荣耀之誓：终局形态强化英雄冲锋与保护，仍属于优秀而非生态级最强。
- B:
  - 救赎之誓：终局防御形态很硬，但不能替代满施法者控场，也不能快速结束战斗。
  - 王冠之誓：终局形态符合主题但不足以改变与现代子职的差距。
  - 破誓者：`恐惧之主` 终局侵略性强，但综合解题仍不及顶级圣武士和满施法者。

## UA/合作方/第三方子职分阶段排行榜

Tier 1（1-4）:
- A:
  - 守护之誓（克苏鲁）：`肃卫干预` 和 `刻印打击` 低级就有保护与嘲讽，队伍防御强。
  - 川流之誓（胧忆岛）：`祝福之池` 是少见低级附赠动作群疗，`祝福术` 也很实用。
  - 惩戒之誓（歪曲之月）：`惩奸除恶` 低级可多目标速度归零，主动控制比普通反击路线强。
  - 护法之誓（UA）：`庇护之链` 和 `护法打击` 低级就给保护/反施法入口，但需要站位和反应。
  - 狂欢之誓（秘密尖塔）：`嗟，来饮！` 低级给豁免优势和临时生命，防控制价值高。
  - 狂热之誓（鬼魅幽谷）：`异端标记` 低级给扩展重击和反应攻击，单体压迫很强。
  - 破誓者（UA）：`唤现亡灵` 让低级有稳定伙伴和动作经济，`恐惧面容` 接到 `至圣斩` 后更顺。
  - 远洋之誓（塔尔多雷）：`海巡领域` 和 `浪涛之怒` 低级就能重写视线和位置。
  - 饮咒之誓（狮鹫鞍包）：`吸收魔法` 低级就是强打断和临时生命，明显超过普通圣武士反施法入口。

Tier 2（5-10）:
- S:
  - 守护之誓（克苏鲁）：`守护之仪` 能让低 AC 队友直接用你的 AC，保护价值很高。
  - 护法之誓（UA）：`沉默术`、`法术反制`、`解除魔法` 与 `专注灵光` 让 Tier 2 反施法和施法队友保护非常强。
  - 狂欢之誓（秘密尖塔）：`情谊灵光` 对多近战队伍放大极强，`催眠图纹` 也补控场。
  - 饮咒之誓（狮鹫鞍包）：`乱法灵光`、`解除魔法`、`缓慢术` 让 Tier 2 反施法/破专注非常强。
- A:
  - 川流之誓（胧忆岛）：`川流灵光` 让圣武士周围成为可控困难地形，兼具保护和推位。
  - 惩戒之誓（歪曲之月）：`火种灵光` 和 `法术反制` 让它从惩罚伤害扩展到抗性处理与反法。
  - 狂热之誓（鬼魅幽谷）：`明晰灵光` 对目盲/隐形场景价值高，配合追猎法术功能面宽。
  - 破誓者（UA）：`憎恨灵光` 只强化你和自家召唤物，进攻风险比旧版低，综合比 DMG14 更稳定。
  - 远洋之誓（塔尔多雷）：`解放灵光` 对擒抱/束缚很强，机动和控场价值稳定。

Tier 3（11-16）:
- S:
  - 守护之誓（克苏鲁）：`御者之护` 防止队友倒地并给反击，危机处理强。
  - 护法之誓（UA）：`破法之刃` 把贴身攻击和 `法术反制` 串联，反施法强度超过绝大多数圣武士。
  - 狂欢之誓（秘密尖塔）：`快乐崇拜` 失败不耗次数，能稳定保护关键 D20 检定。
  - 饮咒之誓（狮鹫鞍包）：`铠装心智` 使关键圣武士法术专注几乎不怕伤害，反法定位成型。
- A:
  - 川流之誓（胧忆岛）：`护卫之灵` 能把敏捷豁免减伤分享给队友，防范围伤害价值高。
  - 惩戒之誓（歪曲之月）：`炽诚审判` 提供反击优势和审讯信息，但仍依赖敌人触发。
  - 狂热之誓（鬼魅幽谷）：`强制告罪` 强化审讯和持续压力，但仍不是顶级团队保护。
  - 破誓者（UA）：`召唤亡灵`、恐惧和抗性路线持续有价值，但仍缺顶级队伍保护。
  - 远洋之誓（塔尔多雷）：`潮汐汹涌` 增加阵地惩罚，但功能仍偏战场位置而非全面支援。

Tier 4（17-20）:
- S:
  - 守护之誓（克苏鲁）：`守护天使` 的100尺共享 AC 和每回合恢复 `引导神力` 明显越过普通子职边界，但仍不等于全生态 S+。
  - 护法之誓（UA）：`永恒护法` 的全伤害抗性、对法术豁免优势和法术攻击劣势足以成为终局反法核心，但不因专精强就给 S+。
  - 狂欢之誓（秘密尖塔）：`派对嗨客` 给伤害、英雄激励和多状态免疫，是终局强团队灵光。
  - 饮咒之誓（狮鹫鞍包）：`奥法挪移` 是强终局法术防线和小范围治疗/伤害循环，但仍不到全生态 S+。
- A:
  - 川流之誓（胧忆岛）：`身化长川` 给全豁免优势和机动，终局稳定但不具备顶级反施法。
  - 惩戒之誓（歪曲之月）：`硫磺烈火` 的触及、倒地和速度0很强，但豁免与传奇抗性限制上限。
  - 狂热之誓（鬼魅幽谷）：`末世天启` 给真实视觉和队伍集火优势，终局很强但范围专精。
  - 破誓者（UA）：`恐惧之主` 可用5环位重置，终局压迫力强；但不是超过全职业天花板的 S+。
  - 远洋之誓（塔尔多雷）：`神话浪客` 强化单挑和机动，终局稳定但不越过全生态天花板。

## 设计相关评分
| 设计项 | 评分 | 具体理由 |
|---|---|---|
| 设计质量 | A | 2024 圣武士把治疗改为附赠动作、加入 `武器精通` 和固定 `寻找坐骑`，队伍定位更清楚；`至圣斩`法术化和每回合一次降低了旧版爆发过载。 |
