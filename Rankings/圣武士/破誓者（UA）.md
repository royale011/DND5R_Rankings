# 破誓者（UA）

所属职业：圣武士
英文名 / 社区检索名：Oathbreaker
来源 / 版本：其他 / 新UA / xge子职业更新

## 队伍定位调整
UA 破誓者把旧版最不稳定的 `操控亡灵` 改成短时召唤，并把恐惧、法术表和20级形态现代化。它比 DMG14 更可控、更适合常规战役，但依旧是进攻/恐惧/暗影路线。

## 详细评价
3级 `唤现亡灵` 用 `引导神力` 召唤骷髅或僵尸，持续1分钟且受你命令。这比旧版 `操控亡灵` 更稳定：不用等 DM 放合适不死生物，也不会出现24小时支配的高低落差。代价是上限降低，不能靠高价值不死生物滚雪球。

3级 `恐惧面容` 在你施展 `至圣斩` 后触发，以10尺光环压迫敌人并造成恐惧。这个改动把恐惧接到圣武士实际回合循环里，比旧版单独动作更顺；但范围缩小，面对高感知豁免或恐惧免疫敌人仍会打折。

`破誓者法术` 改为 `炼狱叱喝`、`巫术箭`、`疯狂冠冕`、`黑暗术`、`恐惧术`、`召唤亡灵`、`枯萎术`、`魅影杀手`、`疫病术`、`钢风斩`。这张表有更现代的召唤和爆发选项，但圣武士 DC 与法术位数量限制依旧存在。7级 `憎恨灵光` 只强化你和被你召唤的不死/邪魔，敌我风险比旧版小很多。

15级仍是对非魔法钝击、穿刺、挥砍抗性的 `超自然抗性`。20级 `恐惧之主` 注入你的守护灵光，给魔法黑暗、恐惧伤害和阴影打击，并能用5环法术位重置；这是强终局形态，但不是全生态 S+，因为它没有满施法者的跨场景压制。
