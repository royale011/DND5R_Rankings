## 构筑细节

默认核心是圣武士6 / 吟游诗人14，或为了某些誓言7级灵光改成圣武士7 / 吟游诗人13。圣武士段提供重甲盾牌、武器精通、附赠动作`圣疗`、`至圣斩`、5级`额外攻击`、`信实坐骑`和6级`守护灵光`；吟游诗人段提供`吟游诗人激励`、`专精`、`万事通`、`激励之源`、`反迷惑`、`魔法奥秘`和较高阶的吟游诗人施法。

关键规则是2024兼职施法：法术位按“吟游诗人全等级 + 圣武士等级一半向上取整”计算，但各职业准备法术仍受各自职业等级限制。圣武士6 / 吟游诗人14等同17级兼职施法者，能有9环法术位；圣武士7 / 吟游诗人13也等同17级兼职施法者。它们都不是17级吟游诗人，不能准备8环或9环吟游诗人法术。那些高环位主要用于升环治疗、控制、召唤、驱散/反制检定或低环高价值法术，而不是等同纯吟游诗人的`祈愿术`/9环终局包。

圣武士7分支取决于誓言。守望、古贤、征服、救赎、巨灵贵族等7级灵光或保护能力可能非常值；代价不是9环槽本身，而是推迟或失去吟游诗人14级子职特性，并让最终吟游诗人法术准备上限、已准备法术数量和激励骰进度更紧。若只是想要`守护灵光`，圣武士6通常更干净。

## 队伍定位

这是队伍容错平台，而不是DPR最大化路线。`守护灵光`把魅力调整值加到灵光内自己和盟友的所有豁免上，直接保护集中、控制豁免、范围伤害、即死、恐慌、魅惑和许多怪物能力；吟游诗人则补技能、失败后激励、短休激励恢复、治疗、控制和跨表准备。

它最适合缺少圣武士、缺少高魅力面位、经常遭遇法术/心智/豁免压力的队伍。若队伍已经有高魅力圣武士，或战役主要考验远程输出、跨位面机动和满施法者级现实改写，它的边际价值会下降。

## 详细评价

Tier 1是可靠但未成型。先圣武士能用重甲盾牌、`圣疗`、1级施法和武器精通站稳；先吟游诗人则有更多技能、`吟游诗人激励`和早期控制/治疗。问题是`守护灵光`、`激励之源`和中阶吟游诗人法术都未叠合，所以只是强起手，不是构筑主窗口。

Tier 2是构筑最漂亮的阶段。圣武士5-6给`额外攻击`、`信实坐骑`和`守护灵光`，吟游诗人3-5开始提供子职、`激励之源`、技能和2-3环法术。相比纯圣武士，它更会处理社交、技能、治疗和控制；相比纯吟游诗人，它显著更硬，并给全队豁免数学保险。它不进S，是因为吟游诗人3-5环法术和`魔法奥秘`被推迟。

Tier 3继续强。圣武士6 / 吟游诗人10之后，`魔法奥秘`让吟游诗人准备可以扩到牧师、德鲁伊、法师列表；同时`守护灵光`继续保护队伍集中和关键豁免。此时的缺口也很明显：若停圣武士6，就没有圣武士10-14的`勇气灵光`、`光耀打击`、`复原之触`；若继续圣武士，则吟游诗人的高阶法术、子职14和准备数继续推迟。

Tier 4仍是优秀团队保险，但不是终局主引擎。圣武士6 / 吟游诗人14有9环槽、7环吟游诗人法术、`魔法奥秘`和吟游诗人14级子职特性；圣武士7 / 吟游诗人13仍有9环槽和誓言7级节点，但少一个吟游诗人子职节点。两者都不能像纯吟游诗人17+那样准备8/9环法术，也没有纯圣武士18的30尺`守护灵光`或20级誓言形态。

## 分项理由

`团队增益`是主分项。`守护灵光`本身就是Tier 2开始的队伍级豁免引擎，吟游诗人的失败后激励、技能、控制和治疗让它从“强圣武士”变成“队伍容错平台”。T4降为S-，是因为30尺灵光、纯吟游诗人终局法术和纯圣武士终局形态都不完整。

`防御和生存`、`治疗和临时生命`很高但不越界。重甲盾牌、`圣疗`、吟游诗人治疗、`反迷惑`和高魅力豁免光环让角色很难被普通控制击穿；但生命值、专注、防线半径和高阶法术准备都仍有结构限制。

`法术伤害者`和`反制施法者`只到A，是因为高环槽不等于高环法术已准备。它可以升环、反制、驱散、控场和补位，但不能按纯吟游诗人17+的`祈愿术`、8/9环法术生态评分。

## 校准说明

社区对圣武士/吟游诗人兼职的常见评价，通常认可`守护灵光`是保留圣武士价值的核心，也提醒吟游诗人法术进度、属性点、额外攻击/施法路线和高阶已准备法术会被拖慢。本项目采纳这个问题框架，但最终按本地XPHB文本独立判断。本轮修正旧稿中“圣武士半等级向下取整”的表述：XPHB兼职施法对圣武士/游侠采用向上取整。

结论维持`A / S- / S- / A+`。它高于一般“圣武士混一点满施法者”的松散方案，因为吟游诗人确实能补技能、治疗、控制和跨表准备；但它低于纯强满施法终局，也低于某些有顶级7级灵光/反法/先攻专长的纯圣武士在各自最佳阶段的压制力。

## 开放改进问题

- 法术位与准备边界：兼职施法位不等于吟游诗人已知/准备高环法术；高环槽主要用于升环和检定，不应被描述成完整9环吟游诗人。
- 灵光半径与属性：圣武士6/7、魅力推进、战斗属性和吟游诗人法术DC之间要明确取舍。
- 队伍职责：构筑应先说明自己是灵光/支援/副控，而不是主DPR或准时满施法；否则容易高估终局。
