## 队伍定位摘要

Apothecary 是《Sebastian Crowe's Guide to Drakkenheim》的智力施法新基础职业。本轮依据本地 `5etools-homebrew\collection\Ghostfire Gaming; Sebastian Crowe's Guide to Drakkenheim.json` 中 `Apothecary|GuideDrakkenheim` 重查；同目录下还存在 `Dungeon Dudes; Sebastian Crowe's Guide to Drakkenheim.json` 的 `SCGtD` 平行对象，本轮采用清单对应的 `GuideDrakkenheim` 源标记。`DND5e_chm` 与 `5etools-cn` 未定位到可靠中文正式译名，因此本职业文件暂保留英文名。

在队伍中，Apothecary 是介于魔契师、奇械师、牧师和法师之间的“实验医学/毒素/禁忌研究”施法者。它有 d8 生命骰、智力/感知豁免、轻甲/中甲、简单武器、短剑、手弩、工具熟练、智力施法、长休准备法术，以及类似 pact magic 的短休高环槽。它的法术表非常宽，低环有 `Healing Word`、`Cure Wounds`、`Sleep`、`Grease`、`Web`、`Blindness/Deafness`、`Lesser Restoration`、`Silence`；中环有 `Dispel Magic`、`Fear`、`Haste`、`Hypnotic Pattern`、`Revivify`、`Slow`、`Polymorph`、`Death Ward`、`Greater Restoration`、`Hold Monster`、`Raise Dead`；高阶 `Greater Formula` 还能接触 `Forcecage`、`Plane Shift`、`Antimagic Field`、`Foresight`、`Mass Heal`、`Time Stop`、`True Polymorph`、`True Resurrection` 等终局法术。

这个底盘的强度不是来自某个单一爆点，而是来自“短休高环槽 + 长休准备宽表 + 治疗/复活/控制/污染法术 + Esoteric Theories”的堆叠。它比普通半施法者和多数第三方武职宽很多，也比魔契师默认更像可每日重配工具箱；但它仍受短休资源结构、准备数、具体法表、理论选择、子职路线和第三方法术稳定性的共同约束，不能简单等同于法师的法术书生态或牧师/德鲁伊的全表准备。

## 详细评价

1级 `Apothecary Magic` 是核心底盘。职业使用智力施法，准备数为 Apothecary 等级加智力调整值，并按长休重配；法术位采用 pact 进度，低级槽少但每个槽会随等级升高，短休恢复的节奏让每个遭遇都能投入更高环资源。与魔契师相比，Apothecary 的优势是准备制与治疗/复原/控制混合法表；劣势是没有默认 `Eldritch Blast` + 祈唤那种无资源远程伤害框架，职业输出更依赖法术、毒素/酸蚀法术和子职。

同为1级的 `Occult Practice` 让子职从开局就定义玩法。Alienist 偏心灵与力场，Chemist 偏酸毒爆破与瓶装法术，Exorcist 偏反邪/保护/治疗，Mutagenist 偏肉体变形近战，Pathogenist 偏疾病传播与毒素减益，Reanimator 偏尸造伙伴。基础职业评分只评价共同底盘；各实践会在后续文件独立重评。

2级起 `Esoteric Theories` 是 Apothecary 的第二成长轴。它类似一套职业理论/祈唤菜单，随着等级从2个增长到20级的11个。这个系统让 Apothecary 可以补侦察、工具、法术增强、毒素、污染或资源功能；强度取决于理论选择和桌面对 Drakkenheim 法术/污染规则的接受度。它提高了职业宽度，但也增加了构建复杂度和第三方法术校准压力。

11级后 `Greater Formula` 让 Apothecary 获得6环到9环法术入口，19-20级再追加额外6环与7环公式。这个设计非常关键：Apothecary 不是停在5环的半施法者，而是以魔契师式高阶秘法结构取得终局法术。由于高阶公式数量有限，它没有法师那种“抄录后按日换一整套高环答案”的上限；但只要选到 `Forcecage`、`Plane Shift`、`Antimagic Field`、`Foresight`、`Mass Heal`、`Time Stop`、`True Polymorph` 或 `True Resurrection` 这类法术，就足以把T3/T4表现推入强施法者区间。

20级 `Miraculous Recovery` 进一步强化续航。即使不把它写成无限循环，它也在终局给短休槽与高阶公式之外再补一层容错，使 Apothecary 在长冒险日里比普通高阶短休施法者更不容易被资源榨干。它不是 `Wish` 级规则所有权，但对治疗、复活、控制和实验法术混合职业很实用。

## 本轮重评

本轮社区校准检索覆盖 `Apothecary class Sebastian Crowe's Guide to Drakkenheim review balance 5e`、`Apothecary Drakkenheim 5e balance`、`Dungeon Dudes Apothecary class 5e`、Reddit、D&D Beyond 论坛与通用网页方向；未定位到稳定、同版本、可直接引用的强度共识。可见讨论多停留在职业概念、资源包或资料介绍层面，缺少像官方职业那样的大规模实战共识。因此本轮按本地 `GuideDrakkenheim` v1.1 文本、现有第三方新职业样本、魔契师/奇械师/灵能使/秘术师横向基准独立判断。

横向上，Apothecary 的底盘明显高于普通 A 级专家或半施法职业：它有中甲、治疗、复活、控制、短休高环槽、准备制和高阶公式，低中等级就很难被当作“窄职业”。但它也不像法师那样拥有法术书、仪式抄录、正式出版生态和 `Wish`；其许多强点要经过 Drakkenheim 自定义法术、Esoteric Theory 菜单和子职特性共同实现。综合定为 `A+ / S- / S / S`：低级强但资源槽少，中级进入强短休施法者线，高级因高阶公式与宽表进入 S，但不到 S+ 或 EX。

## 校准说明

Apothecary 按第三方/合作方新基础职业口径评分，不因来源自动降权，也不因拥有9环公式自动等同法师。它的合理比较对象是魔契师、奇械师、灵能使、秘术师与已重评的 Drakkenheim 子职生态：低级比多数专家底盘更强，中级靠短休高环槽与准备宽表进入 S-，高阶凭 `Greater Formula` 进入 S；缺少 `Wish`、法术书、正式扩展生态和低成本结果重写，所以不进入 S+。

六个实践横向校准后，Alienist、Exorcist、Pathogenist 是T3起的S+可靠性/防线组，分别靠关键豁免重骰取低、`Negation`动作否决和感染后Apothecary法术豁免劣势上探；Mutagenist 到18级解除变形施法/失败豁免限制后进入T4 S+；Chemist 与 Reanimator 高阶仍强，但前者偏爆破/类型/瓶装法术，后者偏伙伴占位/擒抱/信息与战术身体，因此停在S线。
