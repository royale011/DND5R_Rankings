# 装甲师

## 队伍定位摘要
装甲师是奇械师的前线装甲平台，也是新版正式子职里最稳定的防御路线。它能在无畏者、守护者和渗透者之间切换，把同一职业改造成推拉重装、嘲讽坦克或隐秘射手。

## 详细评价
3级`行业工具`给重甲训练、铁匠工具和护甲制作时间减半。`装甲师法术`质量很高：低级有`魔法飞弹`、`雷鸣波`、`镜影术`、`粉碎音波`，9级拿到`催眠图纹`和`闪电束`，后续还有`高等隐形术`、`穿墙术`、`力场墙`。这张表直接补足奇械师最缺的硬控、保命和战场分隔。

`奥能装甲`让盔甲成为法器，忽略力量需求并简化着卸甲。`装甲型号`是评分核心：无畏者的`力场破坏器`是1d10力场近战武器，命中能推拉小一号以上目标，`伟岸身姿`让体型和触及提升；守护者的`雷霆脉冲`让被命中目标攻击他人劣势，`防护场`在浴血时用附赠动作获得等级等量临时生命；渗透者的`闪电发射器`是90/300尺远程武器，速度+5并给隐匿优势。三种型号覆盖了奇械师最缺的“站在前线并改变敌人位置”的能力。

5级`额外攻击`让武器模式兑现；9级`精进装甲师`给额外护甲计划与一个限定护甲的额外仿制物品，并让特殊装甲武器+1。15级`完美装甲`让无畏者伤害变2d6、巨大化更强；守护者能反应用力量豁免把30尺内敌人拉到身边并顺带攻击；渗透者让闪电发射器变2d6，并让被电击目标攻击你劣势。附赠动作短暂飞行则解决高等级前线最怕的位移问题。社区通常把新版装甲师视为奇械师顶级子职之一，这与本地文本吻合：它不是最高爆发，但它的防线、型号切换、法术表和物品协同都太扎实。

## 分项理由
- **防御和生存**：重甲、盾牌、`镜影术`、守护者劣势、临时生命、15级拉人和高同调物品共同构成S级防线。
- **近战伤害者**：智力攻击、额外攻击、无畏者/守护者武器和9级+1武库使近战稳定，但终局不等于顶级武职爆发。
- **战场分隔**：`催眠图纹`、`力场墙`、推拉和守护者反应拉人让它比一般半施法前线更会控制空间。
- **技能使用**：渗透者的潜行优势和速度加值让装甲师在侦查场景也有可选路线。

| 阶段 | 综合强度 | 具体理由 |

| Tier 1（1-4） | A | 3级重甲、奥能装甲和三种装甲型号直接给防御、近战、远程、潜行或体型控制，低级比基础职业硬很多。 |
| Tier 2（5-10） | A | 5级`额外攻击`、9级额外护甲仿制和+1武库让装甲师成为奇械师最稳定的前线/副输出之一，但整体仍是半施法前线，不等于全局S档解题核心。 |
| Tier 3（11-16） | A | 15级前后守护者拉人、渗透者干扰、无畏者大体型与飞行爆发都让它在战术位置上非常全面，但高环法术上限不足以压过顶级满施法子职。 |
| Tier 4（17-20） | A | `力场墙`、高同调位、完美装甲和奇械师终局物品体系结合，四阶仍能承担强防御和战场分割职责；综合按A处理，避免把单项防御S误读成全局S。 |
