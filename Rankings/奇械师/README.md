## 队伍定位摘要
奇械师不是传统意义上的主控施法者，也不是单纯靠额外攻击吃饭的武器职业；它的核心价值是把工具、半施法、每天可指定的魔法物品、同调位和行动分配编成队伍后勤网络。EFA版比旧版更明确地把强度集中到`仿制魔法物品`、6级物品充能/汲取/转变、7级`灵光一闪`和11级`储法物品`。它能稳定抬高队伍装备线和关键检定下限，但上限仍受半施法环阶、方案已知数、同调位竞争、战役魔法物品生态和玩家执行复杂度约束。

## 详细评价
1级`施法`首先把奇械师锁在半施法节奏上：它准备法术、长休后可重配，并以`盗贼工具`、`修补工具`或熟练的工匠工具作为施法法器。这个限制很有主题，也有真实桌面影响：奇械师通常需要关心持物、工具可用性和被缴械后的施法状态。`工艺魔法`额外给`修复术`，并能用魔法动作临时制造滚珠、铁蒺藜、绳索、钩爪、提灯、油、火把、镣铐、捕兽夹等普通物品。它不是低级战斗压制能力，却能在照明、封路、攀爬、临时补给、陷阱和工具解题时把小失败变少。

2级`仿制魔法物品`是职业真正进入强度曲线的节点。奇械师长休后按已知方案制造一到数件临时魔法物品，若物品需要同调，自己可在制造时立即同调。低级方案已经能处理储物、夜视、水下行动、通讯、施法法器、命中和防御，后续等级再开放更高稀有度的防护、移动、施法和功能物品。这里的价值不是“DM可能给宝物”，而是职业本身每天指定一部分队伍装备基线；但它也不是无限物品池，方案已知数、制造件数、不同方案限制、同调位、队伍已有装备和战役魔法物品掉落都会影响边际收益。

6级`魔法物品工艺师`把仿制物品接进法术位经济：`充能魔法物品`用1环以上法术位给自造充能物品回血，`汲取魔法物品`每天一次把自造物品转成1环或2环法术位，`转变魔法物品`则用魔法动作把一件自造物品改成另一个已知方案。这些选项让奇械师能在当天需求变化时修正配置，尤其适合长冒险日、信息不完全和队伍装备缺口不断变化的战役。7级`灵光一闪`是中阶最可靠的队伍保险：30尺内可见生物失败属性检定或豁免后，用反应加智力调整值，覆盖关键豁免、解除机关、知识/工具检定和一些高难环境节点。它很强，但次数按智力、占反应、只能覆盖单次失败，因此不能等同于圣武士式常驻全队豁免光环。

11级`储法物品`是三阶质变。奇械师把一道1-3环、施法时间为动作、且没有被消耗材料的奇械师法术存入一件简易/军用武器或可作施法法器的物品；持有者用魔法动作产生法术效果，若法术需要专注则由持有者专注，次数等于智力调整值两倍。它把低环治疗、控制、位移、工具或增益法术的动作和专注交给队友、伙伴或前线单位承担，奇械师本人可以继续施放其他法术、命令炮台/伙伴、攻击或处理物品。14级`高级奇械`把同调位推到五个并让`灵光一闪`短休回一发，18级到六同调，19级获得`传奇恩惠`，20级`奇械之魂`可摧毁非普通或珍稀的仿制物品避免归零，并在有同调魔法物品时短休回满`灵光一闪`。终局奇械师资源厚、韧性强、队伍装备线极稳；问题是它仍只到5环法术，不能像满施法者那样用6-9环法术重写遭遇、旅行、情报和终局解题权。

## 分项理由
- **制造**：`工艺魔法`、`仿制魔法物品`、已知方案、每日自造物品、6级充能/汲取/转变和更高同调位共同构成职业授予的制造能力，因此长期为S；这不是把DM发宝物误算成职业强度。
- **团队增益**：自造物品能分配夜视、储物、通讯、命中、防御、移动、法器和部分充能工具，`灵光一闪`保护关键失败检定，`储法物品`还能把低环法术的动作与专注交给队友执行。
- **防御和生存**：中甲、盾牌、防御方案、多同调位、失败后补救的`灵光一闪`和20级`奇械之魂`让本职生存高于大多数半施法者；四阶因六同调和归零保护升到A+。
- **伤害能力**：本职没有稳定的高阶伤害主循环，近战/远程路线主要依赖子职、武器物品或伙伴，法术伤害也受半施法环阶限制；基础职业可以补足，不应按满施法炮台评分。
- **反制与控制**：奇械师能靠法术表、物品和`储法物品`补控制，但缺少法师/吟游诗人式反制宽度和高环硬控；它更擅长把已有低中环工具分发给队伍，而不是单人接管战场。
