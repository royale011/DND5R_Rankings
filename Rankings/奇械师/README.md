# 奇械师

## 队伍定位摘要
奇械师是半施法后勤核心、魔法物品工程师和队伍资源分配者。它不靠最高环法术改写现实，而靠工具、仿制魔法物品、`灵光一闪`、`储法物品`和子职模块把队伍短板补齐。

## 详细评价
1级`施法`给奇械师半施法底盘，准备法术可长休调整，法器必须是`盗贼工具`、`修补工具`或熟练的工匠工具。这个限制很重要：奇械师不是“少一点法术的法师”，而是把法术、工具、装备和队伍分工绑在一起运作。`工艺魔法`常备`修复术`，并能用修补工具临时做滚珠、绳索、钩爪、提灯、铁蒺藜、油、火把等普通物品；它不直接提高遭遇伤害，但低级探索、陷阱、攀爬、照明和临时物资处理都更稳。

2级`仿制魔法物品`是职业核心。长休后按已知方案制造临时魔法物品，并可立即同调。2级方案里已经有`炼金壶`、`次元袋`、`夜视镜`、`百变工具`、`+1盾牌`、`+1战法师魔杖`、`+1武器`、`+1神威绑带`等；6级、10级、14级继续开放更强的护甲、斗篷、魔杖、戒指和防御/移动/施法工具。它的强度不是“DM给了什么宝物”，而是职业自己稳定制造一部分队伍装备基线；但具体方案清单、队伍需求和战役中已有魔法物品仍会影响边际价值。

6级`魔法物品工艺师`把仿制物品从静态装备变成资源系统：`充能魔法物品`能用法术位补充仿制物品充能，`汲取魔法物品`能把物品转成法术位，`转变魔法物品`能把一件仿制物品换成另一项已知方案。7级`灵光一闪`用反应给30尺内失败的属性检定或豁免加智力调整值，是奇械师最强的队伍保险之一，尤其保护关键豁免、反制检定、解除检定和高难技能检定。

11级`储法物品`是奇械师三阶跃升的真正理由：它把一项1-3环、施法时间为动作、无消耗材料的奇械师法术存进武器或法器，让持有者以你的施法属性施放。它不等于高环法术，但它把行动、法术资源和专注/非专注工具分配给队友或伙伴，实际常能改变整天的资源流。14、18级增加同调位，19级`传奇恩惠`给终局通用强化，20级`奇械之魂`用分解仿制物品避免归零并在短休后恢复`灵光一闪`。所以本职在Tier 3应高于普通A档后勤；但四阶仍没有`祈愿术`、九环控制或满施法准备宽度，不能按顶级满施法者评分。

## 分项理由
- **制造**：`工艺魔法`、魔法物品方案、职业内置临时仿制物品、充能/汲取/转变和更多同调位让制造长期为S；这是职业授予能力，不是单纯期待DM发宝物。
- **团队增益**：`仿制魔法物品`可把夜视、防御、命中、移动、法术充能和工具优势分给队友，`灵光一闪`保护关键失败检定。
- **防御和生存**：中甲、盾牌、防御物品方案、多同调位、`灵光一闪`和20级`奇械之魂`让本职生存高于多数半施法者。
- **法术伤害者**：基础法术表和法术位环阶受半施法限制；真正伤害通常来自子职法术、炮台、武器、伙伴或物品，而非本职法表本身。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | B+ | 低级有中甲、盾牌、工具、长休换准备法术和2级`仿制魔法物品`，后勤、防御和队伍装备基线明显高于普通B；但战斗爆发和控场还没成型，所以不到A-。 |
| Tier 2（5-10） | A | 子职5级节点、6级`魔法物品工艺师`、7级`灵光一闪`和10级更多同调让奇械师成为稳定队伍资源核心；半施法限制使其停在A。 |
| Tier 3（11-16） | A+ | 11级`储法物品`把低环奇械师法术和行动经济分配给队友/伙伴，再叠5个同调位与高级仿制方案，实际团队杠杆高于普通A档，评为A+。 |
| Tier 4（17-20） | A | 18级六同调、19级`传奇恩惠`和20级`奇械之魂`很强，且物品工程仍可靠；但缺少六至九环法术和满施法者终局现实改写，综合为A。 |
