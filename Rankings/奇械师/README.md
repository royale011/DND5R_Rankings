# 奇械师总评

## 队伍定位摘要
奇械师是半施法后勤核心、魔法物品工程师和队伍资源分配者。它不靠最高环法术统治战场，而靠工具、仿制魔法物品、灵光一闪、储法物品和子职模块把队伍短板补齐。

## 详细评价
1级`施法`给半施法底盘，关键限制是必须手持`盗贼工具`、`修补工具`或熟练工匠工具作为法器。奇械师不是单纯法师，它的法术动作、工具熟练和装备配置必须一起考虑。`工艺魔法`始终给予`修复术`，并能用修补工具临时制造滚珠、绳索、钩爪、提灯、铁蒺藜、油、火把等常规物品；这不是战斗爆发，却让低级探索和临场解题很稳定。

2级`仿制魔法物品`是职业核心。长休后按已知计划制造临时魔法物品，并可立即同调；2级计划含`炼金壶`、`次元袋`、`夜视镜`、`百变工具`、`+1盾牌`、`+1战法师魔杖`、`+1武器`、`+1神威绑带`等。6级开始有`+1护甲`、`精灵靴`、`精灵斗篷`、`窃贼手套`、`法力恢复戒指`、`蛛网魔杖`等，10级和14级继续开放更高稀有度物品。这个系统让奇械师的强度高度依赖玩家是否会为队伍而不是只为自己选物品。

6级`魔法物品工艺师`把仿制物品从静态装备变成资源引擎：`充能魔法物品`可用法术位给物品补充次数，`汲取魔法物品`可把仿制物品转成法术位，`转变魔法物品`可把一个仿制物品换成另一项计划。7级`灵光一闪`用反应给30尺内失败的属性检定或豁免加智力调整值，是奇械师最强的团队保险之一。11级`储法物品`能把1-3环奇械师法术存进武器或法器，让持有者以你的施法属性使用；这不仅是续航，也是行动经济分配。14、18级增加同调位，19级`传奇恩惠`给终局通用强化，20级`奇械之魂`能分解仿制物品避免归零，并在短休后恢复`灵光一闪`。社区对新版奇械师的主要共识是：它终于更像2024规则下的完整职业，但真实强度仍取决于物品计划理解、队伍配合和DM对魔法物品经济的开放度。

## 分项理由
- **制造**：`工艺魔法`、魔法物品计划、药水/卷轴/武器/护甲/魔杖制作减时和多同调位让制造长期为S。
- **团队增益**：`仿制魔法物品`可把夜视、防御、命中、移动、法术充能和抗性分给队友，`灵光一闪`保护关键豁免。
- **防御和生存**：中甲、盾牌、魔法盾/护甲计划、多同调位和20级`奇械之魂`让本职防线高于普通半施法者。
- **法术伤害者**：奇械师法术位和环阶受半施法限制，真正伤害上限通常来自子职法术、炮台、武器或物品，而非基础法术表。

## 校准说明
本轮依据`DND5e_chm\艾伯伦：奇械锻炉\第一章`的奇械师与五个正式子职，以及`DND5e_chm\其他\新UA\可怖子职\苏生师.htm`重评。社区校准使用新版奇械师/子职评测与讨论：炼金师仍被认为相对保守，装甲师、魔炮师、战地匠师稳定强，制图师强在队伍协作而非个人伤害，苏生师UA按实际文本强度评分。2026-05-05二次校准：半施法子职的综合评分不再把单项防御、伙伴或队伍协调直接等同于全局S，除非其阶段贡献确实接近满施法顶层。

## 分阶段子职排行榜

Tier 1（1-4）:
- A:
  - 制图师：3级地图匣给多人先攻1d4、队友定位、无视视线/遮蔽锁定地图持有者、无槽`妖火`和传送门跃，低级团队功能极强；但它需要队伍主动配合，不能按无条件S处理。
  - 战地匠师：3级智力武器、军用武器和`钢铁守卫`让它低级就有双单位战斗与保护反应。
  - 装甲师：3级重甲、奥能装甲和三种装甲型号直接给防御、近战、远程、潜行或体型控制，低级比基础职业硬很多。
  - 魔炮师：3级`魔能炮台`在投火机、力场弩炮和防御者之间切换，低级伤害或临时生命都很强。
- B:
  - 炼金师：3级有`实验性灵药`、`治愈真言`和`致病射线`，支援面比基础奇械师宽，但免费药剂随机且战斗爆发一般。

Tier 2（5-10）:
- S:
  - 魔炮师：5级`奥法火器`、9级炮台火力提升和`火球术`让中阶远程输出/群体防护进入奇械师顶层。
- A:
  - 制图师：5级`精准引导`让地图法术或`妖火`目标伤害加智力，且受伤不破坏`妖火`专注，中阶稳定支持控场；整体是强A而非顶级控制者。
  - 战地匠师：5级额外攻击、守卫参与攻击、`护盾术`和9级`奥能震荡`让它成为优秀战斗型奇械师，但附赠动作指挥和半施法上限让综合停在A。
  - 装甲师：5级`额外攻击`、9级额外护甲仿制和+1武库让装甲师成为奇械师最稳定的前线/副输出之一，但整体仍是半施法前线，不等于全局S档解题核心。
- B:
  - 炼金师：5级`炼金术掌握`补治疗和酸/火/毒伤害，9级前资源仍紧，横向不如装甲师、魔炮师或战地匠师稳定。

Tier 3（11-16）:
- S:
  - 魔炮师：11级`储法物品`叠加炮台，15级双炮台和半掩护使它同时做输出、护盾和阵地防御。
- A:
  - 制图师：9级`灵机一动`把`灵光一闪`变成救援传送，11级`储法物品`叠加地图机制，团队机动与解题强度高，但仍受半施法和地图持有者规则限制。
  - 战地匠师：11级`储法物品`叠加守卫行动，15级守卫改良和4d6震荡让队伍保护与输出都很稳；这是强A，不是能和顶级满施法控制者并列的S。
  - 炼金师：9级免费`次级复原术`和更多灵药让状态处理、治疗与团队增益更像专职后勤，终于明显高于B档支援。
  - 装甲师：15级前后守护者拉人、渗透者干扰、无畏者大体型与飞行爆发都让它在战术位置上非常全面，但高环法术上限不足以压过顶级满施法子职。

Tier 4（17-20）:
- A:
  - 制图师：15级`万能地图匣`可在地图持有者归零时回血并传送，且有无槽`寻路术`；四阶是优秀队伍协调子职，不应和九环法术天花板同列S。
  - 战地匠师：四阶仍有优秀伙伴和物品体系，但单体武器/伙伴伤害相对顶级武职与满施法终局有所下滑。
  - 炼金师：15级`炼金专家`给抗性、免疫、额外力场伤害和无槽`塔莎泡泡坩埚`，终局功能不差但仍不是顶级控制/输出。
  - 装甲师：`力场墙`、高同调位、完美装甲和奇械师终局物品体系结合，四阶仍能承担强防御和战场分割职责；综合按A处理，避免把单项防御S误读成全局S。
  - 魔炮师：`力场墙`、双炮台、半掩护和高同调物品让四阶依旧是优秀远程阵地核心，但半施法者终局解题权不足以继续占全局S。

## UA/合作方/第三方子职分阶段排行榜

Tier 1（1-4）:
- A:
  - 苏生师（UA）：3级`再起伴兵`给完整亡灵伙伴，`电颤复律`让`维生术`兼顾拉人和范围闪电伤害，低级已强于普通支援。

Tier 2（5-10）:
- S:
  - 苏生师（UA）：5级`怪异修正`在法术借位增伤与伴兵双击之间选择，中阶伙伴路线实际输出和战术空间都很高。

Tier 3（11-16）:
- A:
  - 苏生师（UA）：9级`强化再起`给大型推离、恐慌光环或酸液反伤，控制和前线压迫明显超过多数伙伴子职；但作为UA半施法伙伴路线，综合仍不该长期占S。

Tier 4（17-20）:
- A:
  - 苏生师（UA）：15级死亡爆发升级、黯蚀无视抗性、复活减价和`性命流转`让终局依旧强，但复杂度、自爆站位和半施法终局上限使其回到A。
