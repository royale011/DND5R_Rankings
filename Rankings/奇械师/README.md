## 队伍定位摘要
奇械师在队伍里的位置不是传统施法主控，也不是单纯武器副手，而是把装备、工具、半施法和同调位变成可调后勤网络的职业。低级先补护甲、工具与临时魔法物品，中级靠`魔法物品工艺师`和`灵光一闪`稳定队伍关键检定，三阶以后用`储法物品`把低环法术交给队友或伙伴执行；限制则是法术环阶、准备宽度、方案清单和同调/物品生态都会决定它能把资源转成多少即时战力。

## 详细评价
1级`施法`把奇械师定在半施法节奏上：准备法术可在长休后重配，但施法法器必须是`盗贼工具`、`修补工具`或熟练的工匠工具，实际会影响持物、缴械、隐蔽施法和工具熟练的价值。`工艺魔法`让`修复术`成为常备，并能临时做滚珠、绳索、钩爪、提灯、铁蒺藜、油、火把等普通物品；这些东西不直接把遭遇变简单，却能在低级探索、照明、攀爬、陷阱和临时物资短缺时减少队伍失误。

2级`仿制魔法物品`是第一个强度断点。奇械师长休后按已知方案创造临时魔法物品，并可以立即同调；2级就能稳定给队伍夜视、储物、工具、命中、防御或施法法器，6级、10级、14级再逐步开放斗篷、护甲、戒指、魔杖和更多防御/移动/施法工具。这个能力的实战价值来自“职业每天能指定一部分装备基线”，不是期待DM发放宝物；但它也不是无限宝物池，已知方案数量、可制造件数、同调位、队伍已有物品和战役掉落都会改变边际收益。

6级`魔法物品工艺师`把仿制物品接入法术位经济：`充能魔法物品`用法术位补充充能，`汲取魔法物品`把一件仿制物品换成法术位，`转变魔法物品`则用魔法动作重排方案。7级`灵光一闪`是中阶队伍保险，用反应给30尺内失败的属性检定或豁免加智力调整值，覆盖解除机关、反制相关检定、关键豁免和高难技能节点；它的短板是次数按智力、吃反应，并且只有失败后才介入，不能替代真正的全队豁免光环。

11级`储法物品`让奇械师从装备工程进入行动分配。它把一道1-3环、施法时间为动作、无消耗材料的奇械师法术存入武器或法器，持有者用魔法动作触发，若法术需要专注则由持有者专注，最大次数等于智力调整值两倍。这意味着队友、伙伴或前线角色可以承担治疗、控制、增益或工具法术的动作与专注，奇械师本人继续做炮台、武器、物品或其他法术。14级与18级提高同调上限，19级给`传奇恩惠`，20级`奇械之魂`用非普通或珍稀仿制物品避免归零并在短休后回满`灵光一闪`；终局非常耐打且资源厚，但仍只到5环法术，无法用九环法术或满施法准备宽度接管所有场景。

## 分项理由
- **制造**：`工艺魔法`、魔法物品方案、职业内置临时仿制物品、充能/汲取/转变和更多同调位让制造长期为S；这是职业授予能力，不是单纯期待DM发宝物。
- **团队增益**：`仿制魔法物品`可把夜视、防御、命中、移动、法术充能和工具优势分给队友，`灵光一闪`保护关键失败检定。
- **防御和生存**：中甲、盾牌、防御物品方案、多同调位、`灵光一闪`和20级`奇械之魂`让本职生存高于多数半施法者。
- **法术伤害者**：基础法术表和法术位环阶受半施法限制；真正伤害通常来自子职法术、炮台、武器、伙伴或物品，而非本职法表本身。

| Tier 2（5-10） | A | 5级子职节点、6级物品充能/汲取/转变、7级`灵光一闪`和10级四同调让奇械师能连续影响防御、命中、豁免与工具解题；上限受半施法法术环阶和方案数量约束，稳定在A。 |
| Tier 3（11-16） | A+ | 11级`储法物品`把动作、专注和低环法术交给队友或伙伴承担，14级五同调与高级方案继续扩大装备网络；它能明显改变整天资源流，但缺少高环硬控与终局法术，因此是A+。 |
| Tier 4（17-20） | A | 18级六同调、19级`传奇恩惠`和20级`奇械之魂`让本职非常耐久且后勤极厚；问题是四阶评价看终局解题权，奇械师仍靠5环以内法术、物品方案和队伍执行，不足以维持A+。 |
