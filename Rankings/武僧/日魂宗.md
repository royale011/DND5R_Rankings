## 来源与版本
本轮评审对象为 XGE 正式版 `Way of the Sun Soul 日魂宗`，按 XPHB 2024 `武僧`底盘兼容评价。已核对 `5etools-src/data/class/class-monk.json`、`5etools-cn/data/class/class-monk.json`、`DND5e_chm\珊娜萨的万事指南\角色选项\武僧\日魂宗.html` 与 `DND2014_legacy`，确认子职特性为3级`耀阳箭`、6级`热能袭`、11级`焰阳爆`、17级`太阳盾`。

XGE文本仍使用旧版`ki`口径；在本项目2024兼容评审中按当前武僧`专注点`资源理解。另已确认本地资料中存在SCAG旧版/过时日魂宗条目，但本轮只评审XGE正式版。公开可检索社区讨论通常把日魂宗视为“主题强但数值和资源效率弱”的旧版武僧子职；本轮社区材料只作为弱校准，最终评分以已核对文本、XPHB底盘和本项目横向标准为准。

## 队伍定位调整
日魂宗试图把武僧改造成远程光辉输出者。它确实补了距离短板，让武僧能在30尺内用光耀攻击参与回合；但旧版数值、专注点效率和后续功能都不足，常常不能兑现太阳炮台的承诺。

## 详细评价
3级`耀阳箭`是日魂宗最重要的能力。它给你一种能在攻击动作中使用的30尺远程法术攻击，熟练，攻击和伤害加敏捷调整值，造成光耀伤害，伤害骰随武僧`武艺骰`成长；如果你在攻击动作中用过这个特殊攻击，还可以花1专注点用附赠动作再打两次。XPHB武僧的近战底盘已经很强，日魂宗的价值不是提高贴身输出，而是让武僧在不愿接触敌人、目标飞行、地面危险或需要光耀伤害时仍能保持多段攻击。

问题也从3级开始出现：`耀阳箭`距离只有30尺，不是弓手式安全距离；它是远程法术攻击，不承接很多武僧近战/徒手机制，也不能提供推摔、撤离、治疗、潜入或队友保护。附赠动作两箭的资源效率还算能用，但它本质上只是把`疾风连击`的一部分转换成远程光耀攻击，并没有额外控制。

6级`热能袭`让你在攻击动作后花2专注点以附赠动作施放`燃烧之手`，并可追加专注点升环，最多花费等于武僧等级一半。这个能力的动作接口看似顺，但资源效率差：`燃烧之手`射程/范围短，火焰伤害常见抗性也更多，2点起步会和连击、防御、震慑等武僧关键支出竞争。它能处理贴脸小群敌人，却不像四象武者那样把元素、位移和控制整合成稳定战术。

11级`焰阳爆`则是150尺内20尺半径体质豁免光耀爆发，基础失败2d6，可花最多3点专注点提高到8d6，且不透明全身掩护后目标无需豁免。它有少见的“无资源低伤害AoE”与可付费爆发，但高阶敌人体质豁免常强，2d6基础太低，花3点换8d6也只是一次普通范围伤害，缺少致盲、驱散、压制、留下区域或改变地形。17级`太阳盾`提供30尺明亮光照、额外30尺昏暗光照，并能在光照开启且被近战命中时用反应造成`5+感知调整值`光耀伤害；这个反伤数值到17级过低，还要求被近战命中，更多是主题展示而非终局能力。

因此本轮把日魂宗的早期从旧口径C上调到B：`耀阳箭`确实解决了武僧最常见的距离尴尬，且能用1点专注点打出远程多段光耀回合。中阶也勉强维持B，因为它至少有远程主循环和小范围燃烧；但11级后子职增长明显失速，T3仍是C。T4升回B主要来自终局武僧底盘，而不是`太阳盾`本身。

## 分项理由
| 项目 | 具体理由 |
|---|---|
| 近战伤害者 | 日魂宗不强化贴身近战，但XPHB武僧底盘仍能正常近战；高阶失速主要来自子职不继续放大近战回合。 |
| 远程伤害者 | `耀阳箭`提供30尺远程法术攻击，随`武艺骰`成长，并可1专注点附赠动作追加两箭；这是实际远程参与能力，但射程短、无控制、无武器精通。 |
| 法术伤害者 | `热能袭`相当于专注点版`燃烧之手`，`焰阳爆`是光耀范围爆发；它们像法术伤害但资源效率和控制附带远低于真正施法者。 |
| 防御和生存 | 远程攻击本身降低接触压力，终局武僧底盘很硬；`太阳盾`有光照和反伤，但反伤低且要被近战命中。 |
| 治疗和临时生命 | 子职没有治疗、临时生命或复原。 |
| 团队增益 | 光照有少量环境价值，但没有给队友加值、防护、移动或资源。 |
| 反制施法者 | 能远程追击或用武僧机动接触施法者，但没有沉默、反制、驱散或豁免压制。 |
| 减益 | 子职没有致盲、恐惧、减速、推拉或持续状态，仍靠底盘`震慑拳`。 |
| 战场分隔 | `焰阳爆`是瞬时伤害，不留下区域；没有墙、地形、召唤或稳定占位。 |
| 召唤 / 伙伴 | 没有召唤物或伙伴。 |
| 技能使用 | 没有技能特性；仅靠武僧底盘和光照在少数探索场景提供便利。 |
| 制造 | 没有制造、工具或装备经济能力。 |

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | B | `耀阳箭`给低级武僧30尺光耀远程主循环，能处理飞行、危险地面和不宜接触目标；但没有控制、治疗或队友保护。 |
| Tier 2（5-10） | B | `额外攻击`后可用多次`耀阳箭`维持远程输出，`热能袭`补贴脸小范围；但资源效率差，不能像强武僧子职那样扩展职责。 |
| Tier 3（11-16） | C | `焰阳爆`有150尺、20尺半径和光耀类型，但基础2d6太低，花点数也只是伤害；缺少能改变遭遇的控制、支援或战场分割。 |
| Tier 4（17-20） | B | `太阳盾`反伤很小，但终局武僧底盘有防御、机动和近战压制，综合不应低到C；它仍低于真正扩展职责的武僧子职。 |

## 设计相关评分
| 设计项 | 评分 | 具体理由 |
|---|---|---|
| 设计质量 | C | 日魂宗的设计目标很明确，却没有真正完成：它想让武僧成为太阳能量远程打击者，但后续伤害、资源效率和功能扩展支撑不起这个新定位。 |
| 主题 | A | 太阳气功、光辉弹幕和灼热灵气的画面很直观，辨识度比很多武僧子职更高。 |
| 能力设计质量 | C | `耀阳箭`是好入口，但`热能袭`、`焰阳爆`和`太阳盾`都偏低效；高阶没有把光、热、驱邪或区域压迫转成可靠规则。 |
| 主题与能力关联度 | B+ | 规则名称和伤害类型都贴着太阳主题，但主题承诺的是明亮、炽烈、压倒性的能量，实际机制更像低功率替代攻击和小范围爆发。 |
