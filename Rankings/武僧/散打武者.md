## 来源与版本

本轮评审对象为 XPHB 2024 正式版 `Warrior of the Open Hand 散打武者`。已核对 `5etools-src/data/class/class-monk.json`、`5etools-cn/data/class/class-monk.json` 与 `DND5e_chm\玩家手册2024\角色职业\武僧\散打武者.htm`：3级为`散打技巧`，含`慌神`、`推离`、`失衡`；6级为`混元体`；11级为`流星步`；17级为`渗透劲`。

这不是旧版2014散打宗的原样复述。当前11级不再是旧版`宁静 Tranquility`，而是`流星步 Fleet Step`：当你使用一个非`疾步如风`的附赠动作时，可在该附赠动作后立即使用`疾步如风`。17级`渗透劲`也使用当前XPHB 4点功力、10d12力场伤害和攻击动作替代一次攻击来结束暗劲的文本。

## 队伍定位调整

散打武者是“把武僧基本回合做得最纯”的子职。它不提供暗影的潜入传送、命流的治疗/状态处理或四象的远距触及；它把价值集中在`疾风连击`命中后的反应封锁、15尺推离、倒地，以及高等级把附赠动作与`疾步如风`叠在一起。

这条路线对可接触的单体目标非常烦人：它能把敌人推出危险或关键区域、让队友脱离借机攻击、制造倒地优势、逼目标花移动起身，或让自己在连击后继续冲刺/撤离。但它的答案面窄，面对飞行、远距施法、传送、巨大体型、高敏捷/力量豁免和传奇抗性时，会比暗影、命流、四象更依赖武僧底盘和队友补洞。

## 详细评价

3级`散打技巧`让每次`疾风连击`授予的攻击命中生物时，任选一个附加效果。`慌神`让目标到其下回合开始前不能使用借机攻击，这是稳定脱离和保护队友移动的工具；`推离`要求力量豁免，失败则推离最多15尺；`失衡`要求敏捷豁免，失败则倒地。它们都不直接增加伤害，但会让武僧的低级多段攻击从“多打几下”变成“打乱目标行动”。

Tier 2里，`额外攻击`与`震慑拳`会放大散打技巧。散打武者可以先用普通攻击或震慑试探，再用`疾风连击`附加推离/倒地/封借机，或者把倒地给后续近战队友和自身命中制造优势。6级`混元体`以附赠动作掷`武艺骰`+感知调整值回血，次数等于感知调整值、长休恢复；它是自保按钮，不是队伍治疗，但在武僧贴身作战时能减少被迫撤出战线的次数。

11级`流星步`是当前版散打武者的重要流畅度修复。当你使用非`疾步如风`的附赠动作后，可立即使用`疾步如风`，这意味着`疾风连击`、`混元体`或其他附赠动作后还能接冲刺/撤离。它把散打的“打完后移动”做得比普通武僧更顺，尤其适合打倒或推离目标后立刻换位、追后排或带队友移动。限制是`疾步如风`仍要按底盘消耗功力，附赠动作链条虽然顺，但资源池并没有变大。

17级`渗透劲`在徒手打击命中时花4点功力打入暗劲，持续天数等于武僧等级。之后可用动作结束，或在自己回合执行攻击动作时以其中一次攻击替换为结束暗劲；目标必须和你在同一位面，体质豁免失败受10d12力场伤害，成功半伤。它的优点是可先命中再投入资源，伤害类型优秀，也能在攻击动作中兑现；缺点是4点功力昂贵、体质豁免常是高阶怪强项，且它仍只是单体伤害爆点，不提供队伍级反制、复活、传送或大范围控场。

社区讨论方面，本轮能稳定检索到的版本匹配材料有限；可参考的讨论背景是2024散打武者通常被视为“最标准、最顺手”的PHB武僧路线之一，但也常被认为答案面不如命流的队伍容错、暗影的接入/潜入或四象的触及/地形适应。本文只把这些作为弱校准，最终评分按本地XPHB文本和武僧内部横向为准。

## 分项理由

| 项目 | 评分理由 |
|---|---|
| 近战伤害者 | 底盘多次攻击稳定，17级`渗透劲`提供10d12力场单体爆点；T4近战分项不应低于A，但综合仍受答案面限制。 |
| 远程伤害者 | 子职不补远程主循环；高机动可以追目标，但不能把角色变成远程输出者。 |
| 法术伤害者 | 子职没有法术伤害或施法体系。 |
| 防御和生存 | `慌神`、推离、倒地、`混元体`和`流星步`都能降低被围攻或吃借机攻击的风险；仍怕范围伤害、控制法术和远距火力。 |
| 治疗和临时生命 | `混元体`是附赠动作自我治疗，次数等于感知调整值/长休；它有用但数值小，且不能治疗队友。 |
| 团队增益 | 推离、倒地和封借机能给队友创造位置/攻击窗口；没有直接加值、灵光、防护或资源共享。 |
| 反制施法者 | 高速接敌、倒地、推离和`震慑拳`能干扰施法者站位与专注，但没有沉默、法术反制或解除魔法。 |
| 减益 | `散打技巧`把连击命中转成禁借机、推离或倒地，是子职核心；高阶仍主要是单体和命中后效果。 |
| 战场分隔 | 15尺推离、倒地和`流星步`后的移动能重排局部战场，T3-T4流畅度提高；但缺少墙、持续区域和召唤占位。 |
| 召唤 / 伙伴 | 子职没有伙伴、召唤物或独立占位实体。 |
| 技能使用 | 散打主要服务战斗，探索面来自武僧底盘速度和机动；子职自身没有额外技能/工具。 |
| 制造 | 子职没有制造、工具或物品经济能力。 |

## 综合评分

| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | A | 3级`散打技巧`把`疾风连击`命中转成`慌神`、15尺`推离`或`失衡`倒地，低级近战回合质量很高；限制是必须贴身并命中。 |
| Tier 2（5-10） | A | `额外攻击`、`震慑拳`和散打rider叠加后单体压制优秀，6级`混元体`补自我治疗；但没有队伍治疗、潜入传送、远距触及或队伍防线。 |
| Tier 3（11-16） | A | `流星步`让非`疾步如风`附赠动作后继续接`疾步如风`，连击、治疗和换位节奏更顺；限制是高阶战场开始考验飞行、传送和体型处理。 |
| Tier 4（17-20） | A- | `渗透劲`给4点功力的10d12力场单体爆点，且可在攻击动作中替代一次攻击引爆；但价值集中在可接触单体，不能替队伍提供反制、复活或远距投送。 |

## 设计相关评分

| 设计项 | 评分 | 具体理由 |
|---|---|---|
| 设计质量 | A+ | 散打武者把“最标准的徒手武僧”做得干净：连击带控制、自身内力疗伤、附赠动作后继续位移，终局以内劲制造单体爆点。 |
| 主题 | A | 空手格斗大师的主题朴素、直观、没有理解门槛；未给更高是因为宗派辨识度主要来自动作效果，而不是独特哲学、设定或强叙事钩子。 |
| 能力设计质量 | A+ | `散打技巧`、`混元体`、`流星步`和`渗透劲`都能自然接进武僧行动循环；扣分点是高等级仍在同一套贴身单体逻辑里放大，非战斗和队伍功能薄。 |
| 主题与能力关联度 | A+ | 规则几乎没有偏题：拳脚命中后造成慌神、推离、倒地和内劲暗伤，正是散打式控制武僧应有的表达。未给S是因为它更像基础武僧的纯化版，而不是另开一个强叙事分支。 |
