## 来源与版本

本轮评审对象为 SCAG 正式版 `Way of the Long Death 永亡宗`，按 XPHB 2024 `武僧`底盘兼容评价。已核对 `5etools-src/data/class/class-monk.json`、`5etools-cn/data/class/class-monk.json`、`DND5e_chm\剑湾冒险者指南\职业\武僧-永亡宗.html` 与 `DND2014_legacy`，确认子职特性为3级`往生咒`、6级`索命镰`、11级`握生死`、17级`长眠触`。

SCAG文本仍使用旧版`ki`口径；在本项目2024兼容评审中按当前武僧`专注点`资源理解。`索命镰`原文影响30尺内所有能看见你的生物，不区分敌我，这是本评审的重要限制。

## 队伍定位调整
永亡宗把武僧变成死亡边界上的顽抗者。它提供击杀临时生命、范围恐惧、濒死保护和终局单体暗蚀爆发，能让角色更难倒下，也能在合适战场制造压迫；但队伍功能、目标筛选和主动解题都偏窄。

## 详细评价
3级`往生咒`在你把5尺内生物打到0生命时，给予感知调整值+武僧等级的临时生命。它在清杂兵、连战或前线收割时很舒服，也和武僧多次攻击相容；问题是触发依赖近身击杀，面对少数强敌、敌人远离、召唤/持续伤害抢尾刀或队友抢尾刀时，子职存在感会下降。低级时它不像命流武者那样能主动救队友，也不像暗影武者那样改变地图行动。

6级`索命镰`用动作让30尺内能看见你的每个生物进行感知豁免，失败则恐慌到你的下回合结束。它的覆盖范围比很多武僧子职控制更大，配合高速接敌能扰乱近战群或后排；但它不区分敌我，占用动作，持续很短，且恐惧免疫、传奇抗性、视线和队友站位都会削弱稳定性。

11级`握生死`是它最有辨识度的生存点：降到0生命时花1专注点，无需动作，改为保留1生命。它和新版武僧的`拨挡攻击`、`拨挡能量`、`圆融自在`与高移动叠起来会很硬。代价也真实：保命、`疾风连击`、`震慑拳`、`疾步如风`和子职17级爆发都在用同一池资源。

17级`长眠触`给了终局爆发按钮：动作触碰5尺内一名生物，花1到10专注点，每点2d10暗蚀伤害，体质豁免成功半伤。它能把大量专注点转成单体伤害，但射程极短、体质豁免、伤害类型和资源消耗都很苛刻；用完后也会压缩武僧防御、震慑和位移空间。因此终局永亡宗能成为很难杀的前线骚扰者，但不是A档功能核心。

## 分项理由

| 项目 | 评分理由 |
|---|---|
| 近战伤害者 | 主要沿用武僧底盘；17级`长眠触`可把专注点转成单体暗蚀爆发，但动作、5尺触碰、体质豁免和高资源成本使其不能改写全阶段DPR评价。 |
| 远程伤害者 | 子职没有远程攻击主循环；恐惧范围不等于远程输出，终局触碰爆发也要求贴身。 |
| 法术伤害者 | 子职不提供法术、法表或施法资源。 |
| 防御和生存 | `往生咒`给击杀临时生命，`握生死`让11级后可花1专注点避免倒地，叠加XPHB武僧底盘后抗压很强；触发与资源竞争限制其上限。 |
| 治疗和临时生命 | 只有自身临时生命和保命，不会治疗队友，也没有复活或战后恢复。 |
| 团队增益 | 子职几乎没有直接队友增益；`索命镰`若站位好可以间接保护队友，但也可能误伤队友节奏。 |
| 反制施法者 | 武僧底盘负责追击和`震慑拳`，`索命镰`可以短暂恐慌施法者；但没有沉默、反制、驱散或远程打断。 |
| 减益 | `索命镰`是主要减益，范围大但敌我不分、动作成本高且持续到下回合结束；`震慑拳`仍来自底盘。 |
| 战场分隔 | 范围恐惧可制造短暂退让和接敌压力，效果受免疫、视线、队友站位和目标豁免影响。 |
| 召唤 / 伙伴 | 子职没有召唤或伙伴。 |
| 技能使用 | 除武僧底盘属性与机动外，子职没有技能扩展；死亡主题更多是叙事而非调查/工具规则。 |
| 制造 | 子职没有制造、工具或物品经济能力。 |

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | C | 武僧底盘强，`往生咒`击杀临时生命有用；但触发吃近身击杀，低级定位不如能主动治疗、传送或控场的子职。 |
| Tier 2（5-10） | B | `索命镰`范围恐惧和`往生咒`续航开始成型，适合杂兵和近战敌人；限制是恐惧免疫、敌我不分、动作竞争和触发不稳定。 |
| Tier 3（11-16） | B | `握生死`显著增强生存，叠加武僧防御很难杀；但会消耗专注点，且队伍功能仍窄。 |
| Tier 4（17-20） | B+ | 终局抗压很高，`长眠触`补单体爆发，临时生命和不倒机制叠加`无懈可击`后可靠；但功能面不足，不能达到A-。 |

## 设计相关评分
| 设计项 | 评分 | 具体理由 |
|---|---|---|
| 设计质量 | B | 永亡宗抓住了“见过死亡所以不畏死亡”的主题，但落到规则后主要是个人临时生命、恐惧和拒绝倒下，玩法宽度有限。 |
| 主题 | A | 亡魂、死亡凝视、恐惧与濒死顽抗都很有辨识度，适合阴郁或守墓风格武僧。 |
| 能力设计质量 | B | 临时生命、范围恐惧、濒死保护和终局触碰爆发都能落桌；问题是触发条件、敌我不分、动作成本和资源消耗较窄，很多能力不会改变武僧的默认战斗计划。 |
| 主题与能力关联度 | A | 每个能力都围绕死亡经验、恐惧和抗拒终结展开，主题没有明显漂移；扣分是机制表达偏个人防线，不够展开“亡者智慧”或队伍守护。 |
