## 来源与版本

本轮评审对象为 XPHB 2024 正式版 `Warrior of Shadow 暗影武者`。已核对 `5etools-src/data/class/class-monk.json`、`5etools-cn/data/class/class-monk.json` 与 `DND5e_chm\玩家手册2024\角色职业\武僧\暗影武者.htm`：3级为`暗影技艺`，分为`黑暗术`、`黑暗视觉`与`幻影术`；6级为`暗影步`；11级为`无影步`；17级为`幽影斗篷`，其下含`隐形`、`局部虚化`与`幽影连击`。

这不是 2014 `暗影宗 Way of Shadow` 的直接复述。当前版本没有从子职取得`沉默术`或`行动无踪`；强度核心转为可看穿、可移动的`黑暗术`、60尺暗处传送、11级花功力无视光照条件并附带一次`徒手打击`，以及17级暗处启动的隐形/穿格/免费`疾风连击`形态。

## 队伍定位调整

暗影武者把武僧推向潜入、遮蔽、后排切入和暗光环境作战。它不像命流武者那样补治疗，也不像四象武者那样补稳定触及；它的价值在于把武僧的高速、多次攻击和`震慑拳`放进“我决定从哪里开打”的位置优势里。

它最怕两类环境：一类是队友不能利用或承受黑暗遮蔽，导致`黑暗术`变成双方都要绕开的烟幕；另一类是高等级敌人的真视、盲视、明亮空场、反隐形和传奇机动压缩暗影玩法。标准队伍里它仍是强武僧子职，但不是无条件的全队解题器。

## 详细评价

3级`暗影技艺`比旧版更集中。`黑暗术`只花1点功力、无需成分、自己能看穿，并且持续期间每回合开始能把黑暗区域移动到自身60尺内空间；这让暗影武者可以制造掩护、切断视线、掩护撤离、保护自己接敌，或在低级战斗里用优势/劣势改变一次关键交换。`黑暗视觉`和`幻影术`补侦察、潜入和欺敌，但它们只是辅助；真正把评级推上去的是可控黑暗与武僧底盘的接入能力。

黑暗的限制也必须写清楚。2024版本并没有给全队看穿黑暗，也没有自动提供群体潜行加值。若队伍里没有盲视、魔鬼视界、范围控制配合，或战场要求队友远程看清目标，暗影武者的最强按钮会有明显队友成本。它强在“创造局部暗场并由自己利用”，不是强在无脑提高全队输出。

6级`暗影步`是 Tier 2 爆点：完全处于微光或黑暗时，以附赠动作传送到60尺内可见且同样处于微光或黑暗的未占据空间，随后本回合下一次近战攻击有优势。武僧本来就有高移动和多次攻击，传送把绕路、翻越、脱离和接后排合并成一个附赠动作；配合5级`额外攻击`、`震慑拳`和3级自造黑暗，它在地形复杂或光照可控的遭遇里非常凶。

11级`无影步`解决了暗影步最硬的环境条件：使用`暗影步`时可花1点功力移除起点与终点的微光/黑暗要求，并在传送后立刻进行一次`徒手打击`。这让暗影武者不再被明亮空场完全关机，也让附赠动作传送本身带一点进攻补偿。它没有给出范围控制、治疗、反制或复活，所以 Tier 3 仍是强A而不是跨职业S；但在武僧内部，它的目标接入质量仍然很高。

17级`幽影斗篷`需要在微光或黑暗中以`魔法`动作花3点功力启动，持续1分钟，直到失能或在明亮光照中结束回合。形态期间获得`隐形`，能以困难地形方式通过已占据空间，且`疾风连击`不再消耗功力。它把终局暗影武者变成极难卡位、很会穿后排、资源转换很顺的近战压制者；但启动动作、光照限制、反隐形手段和终局遭遇的广谱要求阻止它进入S级。

社区讨论方面，本轮能稳定检索到的版本匹配材料有限；可参考的讨论焦点主要仍是2024武僧整体翻身，以及暗影武者在PHB四子职中常被视为高位选择。本文只把这些作为弱校准，最终评分按本地源文确认的当前特性重建，尤其不沿用旧版`沉默术`与`行动无踪`印象。

## 分项理由

| 项目 | 评分理由 |
|---|---|
| 近战伤害者 | 子职不直接加伤害骰，但`黑暗术`、`暗影步`优势、11级传送后`徒手打击`和17级免费`疾风连击`都提高近战回合兑现率；T4因此不再低于A。 |
| 远程伤害者 | 底盘仍缺远程主循环；黑暗能保护远程撤离或压敌方视线，但不能让暗影武者自己成为优秀远程输出。 |
| 法术伤害者 | `黑暗术`和`次级幻象`不是伤害引擎，子职没有法术伤害循环。 |
| 防御和生存 | 可看穿黑暗、传送、隐形、穿过已占据空间和武僧底盘反应减伤共同提高生存；但范围豁免、反隐形和明亮光照仍能绕开核心防线。 |
| 治疗和临时生命 | 子职没有队友治疗、复活或临时生命，只有武僧底盘的自保和高等级生存。 |
| 团队增益 | 黑暗能掩护队友或切断敌方视线，但也会影响队友看目标；没有祝福、灵光、治疗池或稳定全队数学增益。 |
| 反制施法者 | 没有旧版`沉默术`，因此不能按硬反制施法评分；但高速传送、遮断视线、贴身压迫和`震慑拳`能对施法者形成实战压力。 |
| 减益 | 核心减益来自武僧底盘`震慑拳`与擒抱/推撞；子职提供的是优势接入和视线遮蔽，使这些减益更容易落到关键目标身上。 |
| 战场分隔 | 可移动`黑暗术`是低级就能改变视线和路线的战场工具，6级传送与17级穿格继续强化位置处理；限制是队友适配和特殊感官。 |
| 召唤 / 伙伴 | 没有召唤物、伙伴或独立占位单位。 |
| 技能使用 | 黑暗视觉、`次级幻象`、暗处传送和隐形让潜入、侦察、绕路与撤离明显强于普通武僧；明亮环境或反隐形会压缩收益。 |
| 制造 | 子职没有工具、制造或物品经济能力。 |

## 综合评分

| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | S | `暗影技艺`给1点功力的自看穿可移动`黑暗术`、`黑暗视觉`和`次级幻象`，低级就能改写视线、潜入和接敌路线；限制是黑暗会影响队友，且没有旧版`沉默术`。 |
| Tier 2（5-10） | S | 6级`暗影步`让武僧用附赠动作在暗处60尺传送并给下一次近战攻击优势，配合`额外攻击`和`震慑拳`形成顶级后排压制；仍依赖光照、视线和地图。 |
| Tier 3（11-16） | A | `无影步`可花1点功力无视暗处条件并在传送后补一次`徒手打击`，目标接入非常强；但它仍缺治疗、反制、复活和高环级群体控场。 |
| Tier 4（17-20） | A | `幽影斗篷`给隐形、穿过已占据空间和免费`疾风连击`，终局近战压制与潜入很强；但启动与光照限制、反隐形和高阶遭遇广度让它停在A。 |

## 设计相关评分

| 设计项 | 评分 | 具体理由 |
|---|---|---|
| 设计质量 | S | 新版暗影武者把旧版零散法术包收束成“制造并利用阴影”的行动循环：自看穿黑暗、暗处传送、无条件强化传送和终局幽影形态都服务同一玩法。 |
| 主题 | S | “在黑暗里藏身、从阴影中换位、以幽影形态穿过敌阵”的主题极清楚，既像忍者式潜入者，也像堕影冥界武术家。 |
| 能力设计质量 | A+ | 各等级能力有清晰升级线，且和武僧附赠动作、移动、`震慑拳`、`疾风连击`连接紧密；扣分点是光照/视线/队友适配仍会制造桌面摩擦，17级还需要魔法动作启动。 |
| 主题与能力关联度 | S | 机制几乎没有离开“影中行动”：制造黑暗、看穿黑暗、穿越阴影、借幽影隐形和穿格，主题与实战循环高度一致。 |
