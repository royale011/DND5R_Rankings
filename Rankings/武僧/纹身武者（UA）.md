# 纹身武者（UA）

所属职业：武僧（Monk）
英文名 / 社区检索名：Tattooed Warrior
来源 / 版本：新 UA：奥术二期；采用较新的纹身武者二期文本

## 队伍定位调整
纹身武者是多选项工具型 UA。它用 `魔法纹身`、`百兽之纹身`、`天象之纹身`、`山海之纹身` 和 `怪物之纹身` 给武僧追加戏法、技能补正、抗性、飞行、镜影和再生等能力。

## 详细评价
3级 `魔法纹身` 和 `百兽之纹身` 让武僧获得戏法与小型触发收益，例如 `神导术`、`光亮术`、`舞光术`、额外移动或 AC。6级 `天象之纹身` 可以花 `功力` 给搜索、躲藏、研究等检定加武艺骰，强化技能面。11级 `山海之纹身` 提供可切换抗性，真正抬高生存。17级 `怪物之纹身` 的选项跨度很大：眼魔提供飞行和多束力场射线，位移兽能把 `镜影术` 接入附赠动作，巨魔给流血期间回合开始回血。这个 UA 的问题是选项过散，不同纹身之间强度差距大；但按最合理选择，它确实比普通旧版武僧多很多工具。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | B | 低级靠戏法和纹身小收益扩展工具面，强度不低但缺硬控制。 |
| Tier 2（5-10） | B | `天象之纹身` 强化技能检定，能补武僧非战斗短板，但战斗贡献仍偏零散。 |
| Tier 3（11-16） | B | `山海之纹身` 的抗性让生存上升，整体仍是多功能而非压场子职。 |
| Tier 4（17-20） | B | `怪物之纹身` 有飞行、`镜影术` 和再生等强项，但选项不均衡且不提供满施法级解题。 |

## 设计相关评分
| 设计项 | 评分 | 具体理由 |
|---|---|---|
| 设计质量 | C | 选项多但强弱差距大，容易出现最优解固定。 |
| 主题 | A | 以纹身承载百兽、天象、山海和怪物力量，辨识度很高。 |
| 能力设计质量 | C | 很多单项有趣，但结构偏散，部分终局选项明显强于其他选项。 |
| 主题与能力关联度 | A | 所有能力都由纹身主题直接解释。 |
