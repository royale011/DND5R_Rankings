## 来源与版本

本轮评审对象为 FTD 正式版 `Way of the Ascendant Dragon 神龙宗`，按 XPHB 2024 `武僧`底盘兼容评价。已核对 `5etools-src/data/class/class-monk.json`、`5etools-cn/data/class/class-monk.json` 与 `DND5e_chm\费资本的巨龙宝库\玩家选项\武僧-神龙宗.html`，确认子职特性为3级`龙之息`与`龙之徒`、6级`龙翼翱翔`、11级`古龙姿`、17级`神龙身`。

FTD文本仍使用旧版`ki`口径；在本项目2024兼容评审中按武僧当前`专注点`资源理解，不因此额外降分，也不把后续UA或非正式改写当作本子职能力。

## 队伍定位调整
神龙宗给武僧元素伤害转换、短线/锥形吐息、短暂飞行、近身抗性光环和单体恐惧。它确实让武僧更会处理杂兵、抗性和垂直地形，但整体是保守的龙主题加厚，不是强控、治疗或支援路线。

## 详细评价
3级`龙之徒`提供威吓/游说失败后的反应重骰、龙语或任选语言，以及徒手伤害转强酸、寒冷、火焰、闪电或毒素。它能避开普通物理伤害尴尬，也让武僧主题更鲜明；但社交重骰只有在把失败转成功后才长休刷新，战斗强度主要仍靠底盘。

同级`龙之息`允许在攻击动作中用一次攻击替换20尺锥形或30尺线形吐息，造成两次`武艺骰`伤害，11级升至三次`武艺骰`，使用次数为熟练加值/长休，耗尽后可花2专注点再用。低级能清杂兵、打弱点和绕开贴身限制；但伤害骰数保守、敏捷豁免半伤、形状受站位限制，且花2点再用的机会成本很高。

6级`龙翼翱翔`把`疾步如风`接上本回合飞行速度，次数同样为熟练加值/长休。武僧本来就快，短暂飞行能解决垂直地形、危险地面和部分飞行敌人；问题是它只持续到回合结束，且必须使用`疾步如风`，没有暗影武者那种传送优势，也没有四象武者的持续远距触及。

11级`古龙姿`用附赠动作开10尺光环，1分钟内二选一：给自己和光环内盟友一种龙息伤害抗性，或用创建时/后续附赠动作指定光环内一名生物进行感知豁免，失败则恐慌。这个能力终于有队友价值，但半径短、伤害类型窄、恐惧一次只点一个目标，还会继续抢武僧附赠动作。

17级`神龙身`包含增强吐息、10尺盲视和开`古龙姿`时选择可见光环内任意数量生物吃3d10元素爆发。它让终局神龙宗不至于只靠武僧底盘，但增强吐息仍只是4次`武艺骰`，爆发绑定光环启动，盲视半径也很短。总体结论仍是：主题强、功能直观、执行成本不太复杂，但每个按钮都偏浅，不能和治疗、传送、强控或法术生态型武僧竞争。

## 分项理由

| 项目 | 评分理由 |
|---|---|
| 近战伤害者 | 主要沿用武僧底盘；`龙之徒`能改徒手伤害类型，减少抗性/免疫尴尬，但不增加命中、次数或常驻伤害。 |
| 远程伤害者 | `龙之息`给20尺锥形/30尺线形处理距离与群体的能力，17级可扩大到60尺锥形或90尺线形；伤害保守且次数有限，所以只是B级补洞。 |
| 法术伤害者 | 子职不是施法者，吐息是职业能力而非施法生态；元素伤害有用但没有法术位、法表或高环爆发。 |
| 防御和生存 | 底盘防御很强；`古龙姿`的抗性光环和17级盲视有实战价值，但抗性只覆盖五种元素之一且半径10尺。 |
| 治疗和临时生命 | 子职没有治疗、复活或临时生命，完全依赖武僧底盘和队友。 |
| 团队增益 | `古龙姿`抗性光环是主要团队增益，能在对龙息/元素主题敌人时救场；平时范围短、类型窄，不能稳定等同灵光或祝福。 |
| 反制施法者 | 仍靠武僧速度、`震慑拳`和擒抱/推撞接触施法者；神龙宗自身没有沉默、反制、驱散或远程打断。 |
| 减益 | 底盘`震慑拳`是主要减益，`古龙姿`恐惧补一个10尺内单体感知豁免；附赠动作与半径限制让它不能成为强控核心。 |
| 战场分隔 | 吐息形状、短暂飞行和单体恐惧能制造局部压力；没有墙、持续危险地形、召唤占位或可靠群控。 |
| 召唤 / 伙伴 | 子职没有伙伴或召唤实体。 |
| 技能使用 | `龙之徒`提供语言和一次成功型威吓/游说重骰，低阶社交增量清楚；其它技能主要靠武僧属性与移动。 |
| 制造 | 子职没有制造、工具或物品经济能力。 |

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | B | 武僧底盘强，`龙之徒`和`龙之息`补伤害类型、语言/社交小工具和小范围处理；但吐息次数与伤害保守，不能压过强控制或治疗子职。 |
| Tier 2（5-10） | B | `龙翼翱翔`改善接敌、垂直地形和危险地面处理，元素适应也更顺；但子职没有稳定控制、队伍救急或强资源转换。 |
| Tier 3（11-16） | B | `古龙姿`的恐惧与抗性光环扩展主题，综合仍偏个人/近身战斗适应；高阶横向不如能提供传送、治疗或范围控制的武僧。 |
| Tier 4（17-20） | B+ | 终局武僧底盘很硬，`神龙身`带来扩大吐息、10尺盲视和开光环爆发；但吐息、龙威与抗性没有成长为队伍级控场、反制或复活工具。 |

## 设计相关评分
| 设计项 | 评分 | 具体理由 |
|---|---|---|
| 设计质量 | B+ | 神龙宗把龙裔式能力移植到武僧上，方向清楚、易上手，但很多内容仍是龙主题的标准件：元素、吐息、飞行、威慑。 |
| 主题 | A | “修行者模仿巨龙”很好理解，视觉也强，适合想要龙息拳脚和短暂腾空的玩家。 |
| 能力设计质量 | B+ | 元素伤害转换、吐息、短飞、近身光环和终局爆发都有用途，且没有严重破坏武僧主循环；限制是数值、次数、半径和附赠动作压力都偏保守。 |
| 主题与能力关联度 | A | 规则一直围绕龙性展开，契合度不错；扣分在于它更多是把龙模板套到武僧动作上，宗派哲学和武术解释略薄。 |
