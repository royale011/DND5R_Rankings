## 来源与版本

本轮评审对象为 XGE 正式版 `Way of the Drunken Master 醉拳宗`，按 XPHB 2024 `武僧`底盘兼容评价。已核对 `5etools-src/data/class/class-monk.json`、`5etools-cn/data/class/class-monk.json`、`DND5e_chm\珊娜萨的万事指南\角色选项\武僧\醉拳宗.html` 与 `DND2014_legacy`，确认子职特性为3级`附赠熟练`、3级`醉拳技巧`、6级`酣醉若狂`下的`鲤鱼打挺`与`借力打力`、11级`醉汉机运`、17级`酣醉若狂（Intoxicated Frenzy）`。

XGE文本仍使用旧版`ki`口径；在本项目2024兼容评审中按当前武僧`专注点`资源理解。注意本地CN数据中6级总特性`Tipsy Sway`与17级`Intoxicated Frenzy`都显示为`酣醉若狂`，本文在17级处加英文括注以避免混淆。

## 队伍定位调整
醉拳宗把武僧变成更滑、更难被围住的混战游击手。它的主要价值是`疾风连击`后的撤离/加速、倒地修正、借敌人攻击打敌人、取消劣势和群战多目标连打，而不是治疗、强控、反制或远程解题。

## 详细评价
3级`附赠熟练`给表演和酿酒工具熟练，更多是主题与社交风味。真正的战斗核心是`醉拳技巧`：每当你使用`疾风连击`，你获得撤离动作收益，且步行速度到本回合结束增加10尺。XPHB武僧已经把`疾风连击`整合进`武僧武功`，因此这个接口依然顺滑；低级武僧打完容易留在危险位置，醉拳宗能把连击后的站位调整做得更自然。

6级`酣醉若狂`包含两项：`鲤鱼打挺`让倒地起身只花5尺移动；`借力打力`则在有生物近战攻击检定未命中你时，花1专注点和反应让该攻击改为命中你5尺内可见且不是攻击者的另一个生物。这个能力很有醉拳味，也能在多人近战里反打；但它要求敌人近战攻击你、打空、旁边还有合适目标，还会和武僧底盘反应防御竞争。

11级`醉汉机运`允许你在进行属性检定、攻击掷骰或豁免且具有劣势时，花2专注点取消该次劣势。它覆盖面很广，能救关键豁免或关键命中；但2点成本不低，且只是在劣势存在时把你拉回普通掷骰，不是优势、重掷或成功保证。

17级`酣醉若狂（Intoxicated Frenzy）`让你使用`疾风连击`时最多额外进行三次攻击，总计最多五次`疾风连击`攻击，但这些攻击必须各自指定不同生物。它在被杂兵包围时很爽，能把醉拳宗的混战幻想推到极致；可面对单体传奇、远程施法者、飞行敌人或开阔场地时，子职文本的戏剧性会明显下降。所以醉拳宗不是废，但它更像风格化生存/游击包，而非终局功能核心。

## 分项理由

| 项目 | 评分理由 |
|---|---|
| 近战伤害者 | 低中阶主要沿用武僧底盘；17级`酣醉若狂（Intoxicated Frenzy）`可在多目标混战里把`疾风连击`扩到最多五击，但必须分打不同生物，不能提高单体爆发。 |
| 远程伤害者 | 子职没有远程攻击、投射或远距触及；移动增强帮助接敌，不等于远程输出。 |
| 法术伤害者 | 子职没有法术、法表或施法资源。 |
| 防御和生存 | `醉拳技巧`提供连击后的撤离和加速，`鲤鱼打挺`降低倒地惩罚，`借力打力`可把近战失手转给旁边目标，`醉汉机运`能取消关键劣势；都很实用但多为个人防线。 |
| 治疗和临时生命 | 子职不提供治疗、复活或临时生命。 |
| 团队增益 | 主要通过拉扯敌方位置、诱导误击间接帮队友，没有直接增益、保护光环或资源共享。 |
| 反制施法者 | 速度和撤离让武僧更容易接触施法者，但子职没有沉默、反制、驱散或远程打断。 |
| 减益 | 子职本身不施加稳定负面状态，仍靠底盘`震慑拳`；`借力打力`是惩罚敌人失手，不是主动减益。 |
| 战场分隔 | 高移动和撤离改善站位，`借力打力`能惩罚围攻；但没有持续区域、墙、推拉锁定或召唤占位。 |
| 召唤 / 伙伴 | 子职没有召唤或伙伴。 |
| 技能使用 | 表演与酿酒工具熟练有风味，`醉汉机运`也能覆盖关键属性检定劣势；其它技能仍靠武僧底盘。 |
| 制造 | 酿酒工具是风味熟练，不构成稳定制造/物品经济能力。 |

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | B | 3级`醉拳技巧`把`疾风连击`接到撤离和+10尺移动，低级混战很舒服；但没有治疗、额外控制DC或队友保护。 |
| Tier 2（5-10） | B | `鲤鱼打挺`和`借力打力`让混战生存更好，能减少被围攻风险；仍缺命流救急、暗影传送或四象距离处理。 |
| Tier 3（11-16） | B | `醉汉机运`能取消关键劣势，和误导攻击都很有风味；但依赖劣势、敌方站位与攻击方式，难稳定抬高综合。 |
| Tier 4（17-20） | B+ | 终局武僧防御叠加醉拳脱离、误导和多目标`酣醉若狂（Intoxicated Frenzy）`后很难缠；但面对远距施法和单体传奇敌人时，混战技巧会缩水。 |

## 设计相关评分
| 设计项 | 评分 | 具体理由 |
|---|---|---|
| 设计质量 | B+ | 醉拳宗的核心想法很好：打完就滑走、让敌人误判、借混乱反打。它比纯数值子职更有动作感，但整体仍偏防御和位移修正。 |
| 主题 | A | 醉拳步法、摇晃闪避、借力误导敌人的主题鲜明，和武僧底盘天然合拍。 |
| 能力设计质量 | B | 撤离加速、低成本起身、误导攻击、取消劣势与群战连击都顺着醉拳动作感展开；问题是触发条件和收益都不稳定，很多时候只是让武僧更安全而非改变战局。 |
| 主题与能力关联度 | A | 大多数能力都像醉拳身法的规则化表达，主题没有跑偏；未给更高是因为设计没有进一步展开醉拳的控制、欺敌或团队配合潜力。 |
