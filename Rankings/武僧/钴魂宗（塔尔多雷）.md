# 钴魂宗（塔尔多雷）

所属职业：武僧（Monk）
英文名 / 社区检索名：Way of the Cobalt Soul
来源 / 版本：塔尔多雷，合作方 / 第三方

## 队伍定位调整
钴魂宗把武僧转向情报、弱点识破和反应压制。它不是最高伤害子职，但能把命中转化为抗性/易伤信息、语言/技能知识和额外反应，是典型知识武僧。

## 详细评价
3级 `问诊详情` 命中后花资源读取目标信息，是钴魂宗的核心：知道抗性、易伤、免疫或豁免强弱能改变全队输出选择。6级 `拷问真相` 与 `神秘博学` 扩展审讯、语言和知识技能，让它在非战斗中比普通武僧更有事做。11级 `水银之心` 提供更多反应，17级 `衰弱连击` 能制造更重的破绽。它的短板是强度高度依赖队伍是否利用这些信息；若队友只想单点打血，它不如暗影或命流直接。
