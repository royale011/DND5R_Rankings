## 来源与版本

本轮评审对象为 TCE 正式版 `Way of the Astral Self 星我宗`，按 XPHB 2024 `武僧`底盘兼容评价。已核对 `5etools-src/data/class/class-monk.json`、`5etools-cn/data/class/class-monk.json` 与 `DND5e_chm\塔莎的万事坩埚\玩家选项\职业\武僧（TCE）\星我宗.html`，确认子职特性为3级`星我之臂`、3级`你的星我形态`、6级`星我之容`、11级`星我之躯`、17级`星我觉醒`。

TCE文本仍使用旧版`ki`口径；在本项目2024兼容评审中按当前武僧`专注点`资源理解。星我宗不是施法子职，所有形态收益按职业/子职能力评价。

## 队伍定位调整
星我宗把武僧变成中距离、感知驱动、力场伤害和形态强化的个人战斗者。它缓解部分多属性与触及问题，提供不错的感官、社交和自保能力；但主要强化自身，不明显承担治疗、硬控制、复活或队伍防线。

## 详细评价
3级`星我之臂`给出最重要的差异：附赠动作花1专注点召唤10分钟，登场时让10尺内自选可见生物进行敏捷豁免，失败吃两次`武艺骰`的力场伤害。形态存在期间，你可以用感知替代力量进行力量检定和力量豁免；用星我之臂做徒手打击时触及增加5尺，并可用感知替代力量或敏捷进行攻击和伤害掷骰，伤害类型为力场。对多属性紧张的武僧来说，这是真正改变手感的能力。

6级`星我之容`同样花1专注点持续10分钟，也可以和`星我之臂`同一个附赠动作一起启动。它给120尺魔法/非魔法黑暗正常视觉、洞悉和威吓优势，以及单体60尺私语或600尺扩音。这个阶段的星我宗不只是打得远一点，也更适合侦察、谈判压迫和黑暗环境行动；但它仍不替队友挡伤，也不制造大范围控制。

11级`星我之躯`要求同时有手臂和面容，随后无需动作显现完整躯体。它带来两项重要加厚：受到强酸、寒冷、火焰、力场、闪电或雷鸣伤害时可用反应减1d10+感知调整值；每回合一次用`星我之臂`命中时额外造成一颗`武艺骰`伤害。和神龙宗、永亡宗相比，这个等级的星我宗增量更稳定，所以本轮把T3从普通B校准到B+；但它仍是个人战斗加厚，不是队伍级工具。

17级`星我觉醒`花5专注点一次召出手臂、面容和躯体并觉醒10分钟，给+2 AC，并把`额外攻击`的两次星我臂攻击改为三次。它使终局星我宗成为非常完整的个人近中距武僧形态：力场伤害、触及、额外攻击、AC、感官和部分能量减伤都在线。问题也同样明确：5点启动很贵，输出仍是武僧式单体/近中距输出，队伍不会因此获得复活、反制、传送链或战场塑形。

## 分项理由

| 项目 | 评分理由 |
|---|---|
| 近战伤害者 | `星我之臂`让徒手打击获得+5尺触及、感知攻击/伤害和力场伤害，11级每回合一次额外`武艺骰`，17级觉醒后`额外攻击`可变三次星我臂攻击；限制是启动耗点且主要仍是单体武僧输出。 |
| 远程伤害者 | 触及增加不是远程主循环，仍需要接近战场；没有弓/投射强化或远程法术。 |
| 法术伤害者 | 子职没有法术、法表或施法资源；力场伤害是职业能力而非施法生态。 |
| 防御和生存 | 底盘防御加上11级指定元素/力场/雷鸣反应减伤、17级+2 AC很扎实；但反应会和`拨挡攻击`/`拨挡能量`竞争，且启动需要资源。 |
| 治疗和临时生命 | 子职不提供治疗、复活或临时生命。 |
| 团队增益 | 主要是个人形态；600尺扩音和60尺定向传声有场景价值，但不是战斗数值支援。 |
| 反制施法者 | 高感知、10尺触及、黑暗视觉和武僧底盘能帮助接触并震慑施法者；子职自身没有沉默、反制、驱散或强制断法。 |
| 减益 | 子职没有独立强减益，主要依靠底盘`震慑拳`；登场力场伤害和触及改善命中位置，但不提供新控制状态。 |
| 战场分隔 | 触及增加、力量检定/豁免用感知和武僧机动有利于推拉擒抱与站位，但没有墙、持续区域或召唤占位。 |
| 召唤 / 伙伴 | 星我形态不是独立实体，不能占位、承伤或独立行动。 |
| 技能使用 | 120尺黑暗视觉、洞悉/威吓优势、定向传声/扩音和感知主属性让侦察与社交压迫明显好于普通武僧。 |
| 制造 | 子职没有制造、工具或物品经济能力。 |

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | B | 3级`星我之臂`缓解触及和属性压力，底盘仍负责主要输出；启动与专注点压力让它低于直接救急或强控子职。 |
| Tier 2（5-10） | A | 触及、感知驱动、力场伤害和`星我之容`感官/社交工具成型后，星我宗能更安全地执行武僧压制；但缺少队伍救急和地形级工具。 |
| Tier 3（11-16） | B+ | `星我之躯`同时给特定伤害反应减免和每回合额外`武艺骰`，个人攻防比普通B子职更稳；但队伍功能仍窄。 |
| Tier 4（17-20） | B+ | `星我觉醒`叠加+2 AC和三次星我臂攻击，配合终局武僧底盘明显高于普通B；但仍主要强化自身，缺少队伍级控制、复活或反制。 |
