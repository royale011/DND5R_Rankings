## 来源与版本
本轮评审对象为 XGE 正式版 `Way of the Kensei 剑圣宗`，按 XPHB 2024 `武僧`底盘兼容评价。已核对 `5etools-src/data/class/class-monk.json`、`5etools-cn/data/class/class-monk.json`、`DND5e_chm\珊娜萨的万事指南\角色选项\武僧\剑圣宗.html` 与 `DND2014_legacy`，确认子职特性为3级`剑圣之途`（含`剑圣武器`、`灵巧格档`、`持弓审固`、`书画之道`）、6级`人剑合一`（含`剑气`、`巧劲`）、11级`锋锐剑影`、17级`精妙剑法`。

XGE文本仍使用旧版`ki`口径；在本项目2024兼容评审中按当前武僧`专注点`资源理解。社区可检索讨论通常把剑圣宗视为“武器武僧/弓僧”的稳定但偏窄路线；当前公开讨论中对XPHB底盘下剑圣宗的版本匹配深度材料有限，因此本文只把该方向作为辅助校准，最终评分仍以已核对文本、2024武僧底盘和本项目横向标准为准。

## 队伍定位调整
剑圣宗让武僧更稳定地走武器路线，尤其适合想用刀剑、长弓或特定魔法武器表达武僧身份的角色。它的增量偏个人攻防加厚：更高AC、更稳命中、更顺的武器伤害，而不是新的队伍职责。

## 详细评价
3级`剑圣之途`首先给两类`剑圣武器`，其中近战与远程各一类，范围是不具重型和特殊属性的简易或军用武器，长弓例外可选。这在2014环境里很有价值；但放到XPHB后，武僧底盘与武器精通已经让武器回合更顺，剑圣宗的“我能用武器”不再是稀缺卖点。真正有战斗价值的是`灵巧格档`：你在攻击动作中打出徒手打击且手持近战剑圣武器时，到下回合开始获得+2 AC。新版武僧徒手打击本来就是主循环的一部分，因此这个条件并不难满足，能把前线武僧的承压做得更稳。

同级的`持弓审固`用附赠动作让本回合的远程剑圣武器命中额外造成1d4同类型伤害。它可以服务长弓/飞镖等远程姿态，但附赠动作会和武僧的`武僧武功`、`疾风连击`等核心动作经济竞争，且数值只随命中次数线性增加，不会变成真正的远程压制路线。`书画之道`则是风味与工具熟练，主题很好，但硬功能有限。

6级`人剑合一`包含`剑气`和`巧劲`。`剑气`让剑圣武器攻击视为魔法攻击以突破非魔法攻击/伤害抗性或免疫，在2024兼容环境里主要是遗留保障；`巧劲`每回合一次，在剑圣武器命中后花1专注点追加一个武艺骰伤害。它稳定、简单、不会吃动作，但只是个人伤害加厚，不能像命流的救急、暗影的传送或四象的距离处理那样改变队伍打法。

11级`锋锐剑影`是剑圣宗最强的中高阶节点：附赠动作花最多3点专注点，让接触的一件剑圣武器在1分钟内获得等额攻击与伤害加值。若战役没有稳定的+1/+2/+3武器，这相当于自带高质量武器强化，命中与伤害都很实在；但它不能作用于已经提供攻击与伤害加值的魔法武器，因此在魔法武器供给慷慨的战役里会明显折损。它还吃附赠动作启动和专注点，不能解决武僧高阶最缺的反制、远距投送、复活或战场结构改写。

17级`精妙剑法`每回合一次，当你在自己回合用武僧武器攻击失手时可以重骰。它和`锋锐剑影`、终局武僧高次数攻击一起提高可靠性，能让剑圣宗作为个人武器手保持稳定；但依旧没有队伍级工具。最终评价是：剑圣宗可玩、顺手、有明确武器幻想，但在XPHB武僧横向里更像个人攻防强化包，而不是能扩展队伍职责的高位子职。

## 分项理由
| 项目 | 具体理由 |
|---|---|
| 近战伤害者 | `灵巧格档`鼓励近战武器+徒手混打，`巧劲`提供每回合一次武艺骰附伤，`锋锐剑影`在无加值魔法武器时显著提高命中和伤害，17级`精妙剑法`补可靠性；但没有额外攻击次数或单体爆发窗口。 |
| 远程伤害者 | `剑圣武器`可选长弓，`持弓审固`补1d4，`锋锐剑影`和`精妙剑法`也可服务远程武僧武器；问题是附赠动作竞争严重，远程路线不能承接许多武僧近战压制。 |
| 法术伤害者 | 子职没有施法、戏法或法术伤害入口。 |
| 防御和生存 | `灵巧格档`的+2 AC很稳定，终局武僧底盘也强；但子职没有抗性、治疗、传送脱离或队友保护。 |
| 治疗和临时生命 | 没有治疗或临时生命；只靠武僧底盘自保间接降低治疗压力。 |
| 团队增益 | `锋锐剑影`只能强化自身接触的剑圣武器，整体几乎不支援队友。 |
| 反制施法者 | 能用武僧速度和命中去追击施法者，但子职本身没有沉默、反制、驱散、视野剥夺或豁免压制。 |
| 减益 | 仍主要依赖武僧底盘`震慑拳`，剑圣宗自身没有可靠状态施加；更高命中率能帮助底盘命中但不是独立减益。 |
| 战场分隔 | 没有墙、区域、推拉、召唤占位或地形控制，只有武僧机动与个人威胁区。 |
| 召唤 / 伙伴 | 没有召唤物或伙伴。 |
| 技能使用 | `书画之道`提供书法工具或画家工具熟练，主题强但覆盖面窄。 |
| 制造 | 有工具熟练但无制造减时、配方、装备制作或魔法物品生产规则。 |

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | B | 3级`剑圣武器`、`灵巧格档`和长弓入口让低级武僧更稳，但主要是个人攻防适配，没有治疗、传送、强控或队友保护。 |
| Tier 2（5-10） | B+ | `人剑合一`后武器伤害和抗性穿透更可靠，`灵巧格档`也继续加厚前线；但贡献仍集中在个人武器回合，低于能扩展职责的强武僧。 |
| Tier 3（11-16） | B+ | `锋锐剑影`在缺加值魔法武器时很强，能用专注点换1分钟命中/伤害加值；但会被+X魔法武器稀释，也缺少队伍级控制、治疗或位移工具。 |
| Tier 4（17-20） | B+ | `精妙剑法`提高终局武器命中可靠性，叠加武僧底盘很稳；但子职仍只强化自身武器表现，不能达到A-级队伍功能。 |

## 设计相关评分
| 设计项 | 评分 | 具体理由 |
|---|---|---|
| 设计质量 | B | 剑圣宗定位清楚，但大量能力像给旧版武僧补武器接口；2024底盘已经改善武器使用后，它的独特性和队伍贡献被压缩。 |
| 主题 | A | 把刀剑、弓矢或其他武器当作修行对象的主题很清楚，能很好承载侠客、剑客、弓僧等幻想。 |
| 能力设计质量 | B | `灵巧格档`、`巧劲`、`锋锐剑影`和`精妙剑法`都能运行，但动作经济和魔法武器排斥让高阶体验很吃战役装备环境。 |
| 主题与能力关联度 | A | 能力始终围绕“武器与武僧技艺合一”，方向没有漂移；扣分是玩法多为数值加厚，较少呈现新的剑术决策。 |
