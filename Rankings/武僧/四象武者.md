# 四象武者

所属职业：武僧
英文名 / 社区检索名：Warrior of the Elements / Way of the Four Elements
来源 / 版本：玩家手册2024

## 队伍定位调整

四象武者不再是旧版“花大量气放弱法术”的失败设计，而是把元素触及、推拉、范围和机动接进武僧攻击回合。

## 详细评价

3级元素能力的关键是动作经济：它不是让武僧停下来模仿半吊子法师，而是把元素触及、伤害类型、推拉和范围处理接入武僧攻击。这样它既能维持 `疾风连击` / `额外攻击` 的核心循环，又比基础武僧更会处理距离、飞行、小群敌人和地形。

6级、11级和17级能力继续扩大元素机动、范围爆发或抗性/伤害适应。和散打相比，四象更会处理“够不到”和“敌人不站一条线”的问题；和暗影相比，它不那么依赖光照；和命流相比，它缺少治疗和复活。

必须压低的部分是法术伤害评分。四象有元素表现，但不是满施法 blaster；它的强项是武僧回合上的元素控制，而不是用火球、墙和召唤接管遭遇。若把它当法师评分会高估。

综合上，它是 2024 武僧最成功的改造之一：主题终于兑现，强度稳，但没有 S 级的队伍救急或暗影传送级低成本战场改写。
