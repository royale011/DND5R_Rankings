## 来源与版本

本轮评审对象为 XPHB 2024 正式版 `Warrior of the Elements 四象武者`。已核对 `5etools-src/data/class/class-monk.json`、`5etools-cn/data/class/class-monk.json` 与 `DND5e_chm\玩家手册2024\角色职业\武僧\四象武者.htm`：3级为`元素同调`与`掌控元素`，6级为`元能爆发`，11级为`四象遁术`，17级为`四象神通`，其下含`伤害抗性`、`破灭奔行`与`真力注拳`。

这不是旧版 2014 `四象宗 Way of the Four Elements` 的昂贵“元素法术包”。当前版本的核心循环是：花1点功力在回合开始启动10分钟`元素同调`，让`徒手打击`获得额外10尺触及、可改成强酸/寒冷/火焰/闪电/雷鸣伤害，并在造成这些伤害时迫使目标力量豁免，失败则推离或拉近10尺。

## 队伍定位调整

四象武者把武僧从纯贴身压制扩展成中短距元素控位者。它仍是武僧，不是小法师；它的主要贡献是让多次`徒手打击`更容易触及目标、更容易选择伤害类型，并把每次命中变成可能的10尺推拉。

它补的是武僧最实际的战斗问题：敌人不总在5尺内，危险光环和地形会惩罚贴脸，飞行/水域/垂直地形会让普通武僧绕路。它不补治疗、复活、反制、召唤或战役级法术，所以综合强度保持强A，而不是跨到S。

## 详细评价

3级`元素同调`非常关键。1点功力、回合开始启动、持续10分钟，使武僧在整个遭遇甚至连续探索段落里获得额外10尺徒手触及。这个触及不是单次爆发，而是覆盖攻击动作、`疾风连击`和多数徒手打击机会，直接降低接敌压力。`元素注拳`还能把伤害改为五种元素类型，并在造成这些类型时尝试10尺推拉；这让四象武者可以把武僧的多段命中转成拉近后排、推离危险目标、推出光环或整理站位。

3级`掌控元素`给`四象法门（Elementalism）`，以感知施法。它是主题和探索小工具，不是强度主轴；不能把它按完整施法者工具箱评分。四象武者低级强在“攻击回合变宽”，而不是获得真正的低环法术列表。

6级`元能爆发`是一个2点功力、`魔法`动作、120尺内一点、20尺半径球状范围的敏捷豁免爆发，伤害为3个`武艺骰`，可选强酸、寒冷、火焰、闪电或雷鸣。它给武僧少见的远距范围清杂入口，尤其能处理集中弱敌或无法安全贴脸的目标；但动作成本与2点功力都重，伤害也按武艺骰而非法术位成长，不能和同级满施法者的强控/大伤害法术等量。

11级`四象遁术`在`元素同调`激活期间给等同速度的飞行速度和游泳速度。由于武僧速度高，这个能力很实在：垂直地形、飞行敌人、水域战和危险地面都会被明显改善。它的优势是和已有10分钟同调绑定，不需要重新开一个短时飞行按钮；限制是同调仍要花功力启动，并且飞行本身不等于队伍投送或战役移动。

17级`四象神通`把同调升级成终局形态。`伤害抗性`可在每回合开始切换五种元素抗性；`破灭奔行`让`疾步如风`本回合速度+20尺，并让你进入自选生物5尺内时造成一次`武艺骰`元素伤害；`真力注拳`每回合一次给徒手命中额外一个`武艺骰`同类型伤害。它让终局四象武者在高速穿场、多目标擦身伤害和单体输出上都更顺，但仍主要是个人战斗性能，不是队伍级终局答案。

社区讨论方面，本轮能稳定检索到的版本匹配材料有限；可参考的共同背景是2024四象被普遍视为对旧版四元素武僧的重大修复，讨论焦点多在更便宜的常驻元素打击、触及、推拉和后期飞行上。本文只把这些作为弱校准，评分仍以本地源文和武僧内部横向为准。

## 分项理由

| 项目 | 评分理由 |
|---|---|
| 近战伤害者 | 额外10尺触及、元素伤害类型、推拉和17级每回合额外`武艺骰`都提高近战兑现；T4不应低于A，但仍缺战士四攻或圣武士爆发。 |
| 远程伤害者 | 10尺额外触及和120尺`元能爆发`能处理部分远距问题；但常规主循环仍围绕徒手打击，不是稳定远程输出职业。 |
| 法术伤害者 | `元能爆发`是魔法动作范围伤害，但不是法术位体系，伤害和资源效率都低于真正施法者。 |
| 防御和生存 | 武僧底盘提供反应减伤、机动和高阶自保；11级飞行/游泳和17级可切换元素抗性改善特定环境，但没有队伍防线。 |
| 治疗和临时生命 | 子职没有治疗、临时生命或复活，只有武僧底盘自我恢复。 |
| 团队增益 | 推拉可帮队友整理位置，`元能爆发`可清杂，但没有直接给队友加值、防护或资源。 |
| 反制施法者 | 高速、触及、飞行和推拉能压施法者站位，底盘`震慑拳`仍是主要打断手段；没有沉默、反制或解除魔法。 |
| 减益 | 每次元素徒手命中可迫使力量豁免并推拉10尺，和多次攻击叠加后很有战术价值；但不是失能、定身或大范围控制。 |
| 战场分隔 | 触及、推拉、远距20尺球状爆发、飞行和高速度能改变局部站位；缺少墙、持续地形和召唤占位。 |
| 召唤 / 伙伴 | 子职没有召唤、伙伴或独立战斗单位。 |
| 技能使用 | `四象法门`、飞行、游泳和元素主题互动能补探索小工具；但没有专精、语言/社交包或稳定调查能力。 |
| 制造 | 子职没有制造、工具熟练或物品经济。 |

| Tier 1（1-4） | A | `元素同调`用1点功力给10分钟额外10尺徒手触及、元素伤害和命中后的10尺推拉，显著缓解低级接敌；但缺治疗、保护和硬控。 |
| Tier 2（5-10） | A | `额外攻击`、`疾风连击`和`震慑拳`会放大元素触及与推拉，6级`元能爆发`又补120尺范围清杂；限制是功力、动作和附赠动作都紧张。 |
| Tier 3（11-16） | A | `四象遁术`让同调期间获得等同速度的飞行和游泳，复杂地形处理很强；但它仍没有满施法者的墙、召唤、反制或战役级工具。 |
| Tier 4（17-20） | A | `四象神通`给可切换元素抗性、`疾步如风`穿场伤害和每回合额外`武艺骰`，终局战斗适应性很强；仍是元素武术而非生态级法术引擎。 |
