## 来源与版本

本轮评审对象为 XPHB 2024 正式版 `Monk 武僧`。已核对 `5etools-src/data/class/class-monk.json`、`5etools-cn/data/class/class-monk.json` 与 `DND5e_chm\玩家手册2024\角色职业\武僧\武僧.htm` 中的职业特性顺序、等级与本地中文译名：`武艺`、`武艺骰`、`敏捷攻击`、`武僧武功`、`疾风连击`、`坚强防御`、`疾步如风`、`运转周天`、`拨挡攻击`、`震慑拳`、`真力注拳`、`反射闪避`、`飞檐走壁`、`升阶专注`、`返本还元`、`拨挡能量`、`圆融自在`、`明镜止水`、`无懈可击`与`天人合一`。

本文件只评价当前 XPHB 底盘，不把旧版 2014 `气 Ki`、旧版 `Stillness of Mind` / `Diamond Soul` / `Empty Body` 机制或旧译名当作现行能力。旧版武僧社区印象可作为历史对照，但不作为当前评分证据。

## 队伍定位摘要
武僧在2024规则中是高速接敌、单体压制和自保反应很强的非施法职业。它的核心贡献不是站桩DPR，而是用移动、附赠动作、反应减伤、震慑和高等级豁免去追击后排、打断敌人节奏、逼关键目标交资源。

它的短板同样明确：没有高环法术、没有稳定群体控制、没有复活链、没有召唤生态，也很少提供队伍级数学增益。横向比较时，武僧应被看作战斗内执行力极高的武系压制者，而不是能替代满施法者的万能解题者。

## 详细评价
1级的`武艺`、`武艺骰`、`敏捷攻击`和`无甲防御`决定了武僧的底色。新版`武艺骰`从d6起跳，`附赠徒手打击`让低级输出不再像2014版那样贫弱；`敏捷攻击`还把徒手擒抱/推撞的豁免DC接到敏捷，令武僧更容易把多次攻击转成位置压力。代价也清楚：AC依赖敏捷与感知，生命值又要求体质不能太低，所以多属性压力仍是真实成本。

2级是新版武僧的关键修复点。`武僧武功`把`疾风连击`、`坚强防御`和`疾步如风`集中到同一短休资源池，`运转周天`又给了每长休一次的战斗内回满专注点并回血窗口。它让武僧在输出、防御和位移之间有真实选择，而不是旧版那种“花气才像职业”的窘境。限制也在这里：低级专注点仍少，附赠动作也同时被攻击、防御、冲刺、撤离和不少子职能力争抢。

3级`拨挡攻击`让武僧从“冲进去容易死”变成能吃下一次关键钝击/穿刺/挥砍攻击命中的前线穿插者；若减到0，还能花1专注点把伤害转给附近或远程可见目标。13级`拨挡能量`把覆盖面扩展到任意伤害类型攻击后，高等级武僧面对远程火力和部分能量攻击更稳。它不是无限防线：反应只有一次，多段攻击、豁免伤害、范围控制和控制法术仍会绕过这条防御。

5级`额外攻击`和`震慑拳`是Tier 2的主要强峰。新版`震慑拳`即使目标通过豁免也不完全空转，这让武僧的专注点转化率比旧版健康；但它仍是单体、体质豁免、消耗资源的近身控制，不能和`催眠图纹`、墙类法术、召唤或反制法术等满施法者工具混为一谈。

7级以后，`反射闪避`、`飞檐走壁`、`升阶专注`、`返本还元`、`圆融自在`、`明镜止水`、`无懈可击`和`天人合一`把武僧推成高机动高韧性的终局武系。它能追施法者、逼传奇敌人分配反应和豁免、在危险地形中换位；但面对跨位面、预言、复活、群体防线和现实改写类问题时仍缺少职业内答案。因此武僧高阶综合是强武系，而不是生态级施法引擎。

## 分项理由

| 项目 | 评分理由 |
|---|---|
| 近战伤害者 | `武艺骰`从d6起跳、附赠徒手打击、`疾风连击`和5级`额外攻击`让T1-T3近战输出稳定；T4仍能打，但缺少战士第四次攻击、圣武士终局爆发或施法者高环伤害生态，因此回落为B。 |
| 远程伤害者 | 职业底盘没有可靠远程主循环，只能依靠投掷/短弓、移动接敌或个别子职补洞；高阶生存改善不能自动变成远程火力。 |
| 法术伤害者 | 武僧底盘不是施法者，也没有职业内法术伤害引擎；评分只保留来自通用魔法物品或子职以外资源的极低可能性。 |
| 防御和生存 | `无甲防御`起步受属性压力限制；从`拨挡攻击`、`反射闪避`、`返本还元`、`拨挡能量`、`圆融自在`到`无懈可击`后，武僧逐步成为非常难被常规攻击和豁免压倒的前线穿插者。 |
| 治疗和临时生命 | 底盘只有`运转周天`自我回血和10级`坚强防御`临时生命，缺少队友治疗、复活或持续恢复；命流武者等子职另计。 |
| 团队增益 | 底盘主要服务自身行动质量，10级`疾步如风`可携带一名自愿大型或更小盟友移动，是少数队友位移增量；缺少灵光、祝福、群体防线或资源共享。 |
| 反制施法者 | 高速接敌、`震慑拳`、擒抱/推撞和高阶豁免让武僧能压迫施法者站位与专注；但它没有`法术反制`、沉默光环、反魔法或远程硬反制，故只到B。 |
| 减益 | `震慑拳`是核心单体减益，即使豁免成功也会减速并给下一次攻击优势；多次徒手打击可反复尝试擒抱/推撞，但范围与目标数仍有限。 |
| 战场分隔 | 高移动、撤离/冲刺、推撞、擒抱和10级携带盟友移动能制造位置差；缺少墙、地形持续区、召唤占位和大范围控制。 |
| 召唤 / 伙伴 | 底盘没有召唤物、伙伴或可持续占位实体。 |
| 技能使用 | 敏捷/感知主属性、速度、攀爬液面移动和自我状态清除提升潜入、追踪、侦察和复杂地形执行；但熟练项和专家级技能支持有限。 |
| 制造 | 职业底盘没有制造、工具扩展或物品经济能力。 |

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | A | T1武僧已经有`武艺骰`、附赠徒手打击、专注点、`运转周天`和3级子职，低级战斗执行力明显强于旧版；限制是AC/体质压力、专注点少，以及缺少远程和群体工具。 |
| Tier 2（5-10） | A | 5级`额外攻击`和`震慑拳`给出可靠单体压制，7-10级防御、机动与自我恢复继续加厚；它是优秀战斗职业，但没有满施法者的群控、反制、召唤和战役解题。 |
| Tier 3（11-16） | A | 11级子职能力、13级`拨挡能量`、14级`圆融自在`与15级`明镜止水`让武僧在高压战斗里很稳；横向仍输给能用高环法术改写遭遇结构的职业。 |
| Tier 4（17-20） | A- | 18级`无懈可击`和20级`天人合一`让终局武僧比普通武器职业更能存活、接敌和持续压制；但它仍缺复活、反制、跨场景法术和大范围控制，所以从A回落到A-。 |

## 校准说明

本轮按 `DND5e_chm` 中 2024 `武僧` 底盘、对应子职来源文本、`不全书.wcp` 合作内容分割线和当前项目评分标准重写。当前可检索公开讨论材料有限；既有社区讨论的共同焦点主要是2024武僧从2014低谷大幅翻身，但仍保留多属性需求、附赠动作压力、专注点续航、近战接触和高等级非施法解题限制。评分按非满施法者标准处理，不因没有高环法术自动扣分，也不因多次攻击或单体控制自动升到满施法者级别。

官方武僧与官方子职本轮横向校准已完成。当前官方梯度为：`命流武者`和`暗影武者`在Tier 1-2保持S档，分别代表队伍容错/状态处理与潜入/接入/暗场战术；`四象武者`与`散打武者`是强A档，前者补触及、推拉、范围和机动，后者强化贴身单体控制与动作流畅度；`剑圣宗`、`星我宗`、`神龙宗`、`醉拳宗`、`永亡宗`和`日魂宗`按XPHB底盘兼容后维持B到B+为主，只有日魂在Tier 3因`焰阳爆`失速落到C。未发现需要在本次横向校准中进一步改动综合评分的官方武僧子职。

武僧UA子职本轮横向校准已完成。当前UA梯度为：`秘术武者（UA）`凭三分之一术士施法、低环防御/控制和专注点/法术位转换成为最高位混合路线，T2达S-但不按满施法者升S；`Warrior of Venom（UA）`低级无豁免行动压缩非常锋利，因中后期毒素/中毒免疫、体质豁免和被动触发回落到A；`净土宗（5E UA）`用巨大治疗池和`疾风连击`替换治疗占据A+/A+救急线；`纹身武者二期（UA）`在T2因天象纹身偏技能插件降到A-，T4靠怪物纹身回到A；`醉拳僧（UA）`比官方醉拳更顺，依靠魔法酒、反应附打和多目标`疾风连击`进入B+/A/A/A-。本轮第三方补评已纳入`绵龙牧者（胧忆岛）`：其源文不是伙伴/召唤子职，而是用`羊羔牧者`、`劲风射击`、`灵活截击`、`腾风`和`牧人指引`做队伍位移与柔性护送，横向定位为B+/A-/A/A-，明显高于旧伙伴误读下的B线，但低于高伤害、高控制或高阶施法型武僧。
