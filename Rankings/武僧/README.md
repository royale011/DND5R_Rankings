# 武僧（Monk）总评

## 队伍定位摘要

武僧在 2024 规则中从旧版弱势武系变成高速、耐打、资源更顺的单体压制者。它的队伍位置是机动突击、单体控制、前后排穿插、豁免型生存和少装备依赖的近战专家；弱项是治疗、召唤、制造、群体法术、战役级信息与高等级现实改写。

## 详细评价

1级 `武术` 和 `无甲防御` 让武僧从开局就有附赠动作徒手打击和无甲 AC。新版武术骰从 d6 起跳，武器和徒手攻击的底线明显比 2014 可靠；但 AC 同时依赖敏捷和感知，体质又不能太低，所以社区常说的多属性压力仍然存在。武僧不像战士或圣武士那样能靠重甲、盾牌和单一主属性解决前线问题。

2级 `武僧的专注`、`疾风连击`、`耐心防御`、`踏风步` 和 `不可思议代谢` 是 2024 武僧真正翻身的原因。`不可思议代谢` 让低等级不再完全被短休节奏掐住；`耐心防御` 和 `踏风步` 可以不花专注点先提供基本撤离/冲刺，再花点数升级成闪避或更强移动，动作经济比旧版健康得多。限制也必须写清：专注点仍是短休资源，低等级不能每回合都开满，附赠动作还要在输出、防御、位移和子职能力之间排队。

3级 `偏斜攻击` 是防御曲线的核心。它能把一次武器攻击伤害大幅削掉，并在条件满足时反击；13级 `偏斜能量` 又把覆盖面扩展到能量伤害。这让武僧面对单个强敌或远程火力时比旧版硬很多，但它消耗反应，面对多段命中、豁免伤害、控制法术和群体压力时并不是绝对防线。

5级 `额外攻击` 与 `震慑拳` 决定 Tier 2 强度。新版 `震慑拳` 不再是完全成败二元：即使目标通过豁免，也会受到一定干扰，因此它比旧版更少把专注点扔进空洞。它仍然是体质豁免、每回合触发受限、消耗专注点的单体控制，不能当作法师的大范围失能或圣武士光环。

7级 `反射闪避`、10级 `高度专注` 与 `自我复原`、14级 `纪律幸存者`、18级 `卓越防御` 和20级 `身心合一` 让武僧高等级的防御、豁免和机动非常扎实。它能追击施法者、逼敌人交豁免、穿越战线和保持单体压力；但它没有 `法术反制`、墙、召唤、复活链、跨位面、预言或改写行动经济的高环法术。高等级综合因此保守：武僧很强，但仍是武系强者，不是生态级施法引擎。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | A | `武术`、附赠动作攻击、`不可思议代谢` 和3级子职让 Tier 1 武僧远强于旧版；但低级 AC/体质压力和专注点数量仍限制连续爆发。 |
| Tier 2（5-10） | A | 5级 `额外攻击` 与 `震慑拳` 是早段强峰，7-10级防御和机动继续加厚；综合已是优秀武系，但不是满施法者级解题核心。 |
| Tier 3（11-16） | A | 11-16级武术骰、`偏斜能量`、`纪律幸存者` 和子职 11级能力让武僧在战斗内很稳；限制是缺少群体控制、反制和战役级工具。 |
| Tier 4（17-20） | B | 18级 `卓越防御` 与20级 `身心合一` 很硬，但 Tier 4 环境更看重祈愿、复活、跨场景法术和大范围控制，纯底盘综合落到 B。 |
