## 来源与版本
本轮评审对象为 XPHB 2024 正式版 `Warrior of Mercy 命流武者`，不是TCE旧版`Way of Mercy 命流宗`。已核对 `5etools-src/data/class/class-monk.json`、`5etools-cn/data/class/class-monk.json` 与 `DND5e_chm\玩家手册2024\角色职业\武僧\命流武者.htm`，确认子职特性为3级`夺命之手`、3级`予命之手`、3级`操命本事`、6级`生死之触`、11级`生杀予夺`、17级`命极之手`。

XPHB英文文本使用`Focus Point`，本地中文为`功力`；本项目武僧总评中常用`专注点`指同一资源池。社区可检索讨论通常把2024命流武者列为武僧强子职之一，原因集中在“治疗与输出嵌入疾风连击、6级状态处理、11级免费化、17级复活”这些版本匹配特性；本轮仍以本地源文和横向评分标准独立判断。

## 队伍定位调整
命流武者把武僧从单纯突击手扩展为近战救急者。它仍靠武僧底盘贴身、连击和震慑，但子职把功力/专注点改造成治疗、附伤、负面状态处理和终局复活入口，是武僧内部最稳定的队伍容错来源。

## 详细评价
3级的`夺命之手`与`予命之手`决定它的强度。`夺命之手`每回合一次，在你用`徒手打击`命中并造成伤害时花1点功力，追加一枚`武艺骰`+感知调整值的暗蚀伤害；它不是爆发核弹，但它直接嵌入武僧最常用的命中窗口。`予命之手`则可用魔法动作花1点功力触碰治疗一名生物，治疗量同样是一枚`武艺骰`+感知调整值；更关键的是，当你使用`疾风连击`时，可以把其中一次徒手打击替换成一次免费`予命之手`。这让低级武僧在继续执行前线回合的同时具备拉人和止血能力。

3级`操命本事`给`洞悉`、`医药`和`草药工具`熟练，补上“云游医者”的探索/社交侧身份。它不是战斗核心，却让命流武者比普通武僧更容易承担诊断、看护、尸检、草药和病症相关场景。低级评分高的原因并不是这个工具包，而是治疗和附伤都接在武僧短休资源上；低级队伍法术位少，能用武僧资源做救急，边际价值很高。代价也真实：每一次主动治疗、附伤或防御都在和`疾风连击`、`坚强防御`、`疾步如风`与后续`震慑拳`竞争同一资源池。

6级`生死之触`把两只手都升级：`夺命之手`附带让目标`中毒`到你下个回合结束；`予命之手`则在治疗时能结束目标身上的`目盲`、`耳聋`、`麻痹`、`中毒`或`震慑`之一。这一步让命流武者不只是小号回血。中毒、麻痹和震慑这类状态会直接毁掉队友回合，命流武者能把一部分状态压力从神术准备位和法术位转到武僧短休资源上；同时，附带中毒也给它补了一个无额外豁免DC的近战减益按钮，虽然中毒免疫不少，且目标必须先被`夺命之手`命中。

11级`生杀予夺`是资源效率的关键跃迁。使用`疾风连击`时，你可以把每一次徒手打击都替换为免费`予命之手`；同时，当你用`疾风连击`的徒手打击造成伤害时，可以免费把`夺命之手`接到那次打击上，仍受每回合一次限制。这些收益每长休可用次数等于感知调整值（至少一次）。它不是无限免费，但在关键战斗中能让命流武者同一轮完成治疗、附伤、状态处理和武僧底盘压制，比大多数武僧子职更能影响队伍容错。

17级`命极之手`让它获得非施法武僧极少见的复活入口：魔法动作、花5点功力，触碰24小时内死亡的生物尸体，使其以4d10+感知调整值生命值复活，并移除死前的`目盲`、`耳聋`、`麻痹`、`中毒`和`震慑`，每长休一次。这是真正的终局队伍功能，但仍不是牧师式复活链或高环恢复生态：它需要尸体、触碰、5点功力和长休冷却，不能覆盖灵魂囚禁、尸体缺失、群体死亡或战役级恢复问题。

所以它在T1-T2很高，是因为低中级最缺这种低成本容错；T3-T4仍强，但高环法术、群体治疗和复活链开始拉开差距。它是顶级武僧子职，不是满施法支援的替代品。

## 分项理由
| 项目 | 具体理由 |
|---|---|
| 近战伤害者 | `夺命之手`把一次徒手命中转成武艺骰+感知的暗蚀附伤，11级还能在`疾风连击`中若干次免费触发；它提高近战回合质量，但不提供额外攻击次数。 |
| 远程伤害者 | 子职没有远程攻击或投送手段，仍靠武僧底盘、投掷武器或机动接敌。 |
| 法术伤害者 | 没有法术伤害；暗蚀附伤来自武僧特性而非施法。 |
| 防御和生存 | 通过`予命之手`、状态解除和终局复活提高队伍与自身容错，个人防御仍主要来自武僧底盘。 |
| 治疗和临时生命 | `予命之手`低级即能治疗，`疾风连击`可免费替换一次，11级可替换每次连击，17级有复活；治疗规模小但频率和动作接口优秀。 |
| 团队增益 | 主要是救急、解除`目盲`/`耳聋`/`麻痹`/`中毒`/`震慑`、复活和保队友行动；没有命中/豁免光环或全队增伤。 |
| 反制施法者 | 能用武僧速度接触施法者、用`震慑拳`和`夺命之手`压回合；子职本身没有反制法术、沉默或驱散。 |
| 减益 | 6级`夺命之手`附带`中毒`到下回合结束，叠加底盘`震慑拳`，是武僧中较稳定的状态压力之一；但受中毒免疫影响。 |
| 战场分隔 | 没有墙、区域、召唤或推拉锁场；靠武僧机动和状态压制影响单体。 |
| 召唤 / 伙伴 | 没有召唤物或伙伴。 |
| 技能使用 | `操命本事`给`洞悉`、`医药`和`草药工具`熟练，能支撑治疗者、诊断和社交判断身份。 |
| 制造 | 有`草药工具`熟练，但没有制药减时、配方、魔法物品或稳定制造经济。 |

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | S | `夺命之手`和`予命之手`让低级武僧把功力转成附伤或治疗，且`疾风连击`可免费带一次治疗；低级队伍缺治疗与容错，这个增量压过多数武僧子职。 |
| Tier 2（5-10） | S | `生死之触`把治疗升级为解除`目盲`、`耳聋`、`麻痹`、`中毒`或`震慑`，并让`夺命之手`附带中毒；配合`额外攻击`和`震慑拳`，同级武僧里功能最完整。 |
| Tier 3（11-16） | A | `生杀予夺`让关键战斗中的连击治疗和夺命附伤大幅省资源，底盘防御也更稳；但它仍是小范围、短休/长休资源，不具备满施法者的群体恢复或战役级解题。 |
| Tier 4（17-20） | A | `命极之手`给非施法武僧稀缺复活入口，且能移除若干状态；但5点功力、触碰尸体、24小时限制和长休冷却让它低于高阶施法支援。 |

## 设计相关评分
| 设计项 | 评分 | 具体理由 |
|---|---|---|
| 设计质量 | S | 命流武者是武僧子职里最完整的功能型设计之一：附伤、治疗、状态清除与复生都服务于“用内劲调节生死”的同一命题，而且不会把武僧拖出近战节奏。 |
| 主题 | A+ | 疗愈者、处刑者、点穴与生死边界被压在同一个形象里，比普通“慈悲僧”更有锋利度；未给S是因为文本仍偏职业功能说明，缺少更强的宗派或世界观钩子。 |
| 能力设计质量 | S | `予命之手`、`夺命之手`、`生死之触`、`生杀予夺`和`命极之手`都能嵌入武僧资源循环，既没有严重动作经济冲突，也让功力拥有非纯伤害用途。 |
| 主题与能力关联度 | A+ | “命流”几乎逐级落到规则上：一手救人、一手截断生命、再到把同伴从死亡线上拉回。扣分主要来自高阶治疗在顶级伤害面前仍有限，主题比数值覆盖略宽。 |
