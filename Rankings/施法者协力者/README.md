## 队伍定位摘要

施法者协力者（Spellcaster Sidekick）是《塔莎的万事坩埚》协力者规则中的低复杂度施法NPC职业。本轮按本地 `5etools-src\data\class\class-sidekick.json` 与 `5etools-cn\data\class\class-sidekick.json` 重查，中文名采用本地翻译“施法者协力者”。它不是给玩家角色正常选用的完整PC职业，而是给低CR、有语言能力的生物叠加的协力者职业；协力者起始等级与队伍平均等级相同，随队伍平均等级提升，并在遭遇难度估算中按一名角色计算。

施法者协力者的共同底盘是：1级选择 Mage、Healer 或 Prodigy 角色之一，分别使用法师、牧师+德鲁伊、吟游诗人+魔契师法术列表，并以智力、感知或魅力施法；拥有半施法者式法术位进展，1级2个戏法和1个1环已知法术，20级达到4个戏法、11个已知法术和最高5环法术位。6级`强力戏法`把施法属性加入任意戏法伤害，14级`增强法术`给一个法术学派的消耗法术位伤害或治疗加入施法属性，20级`专注施法`让受到伤害不能打断专注。

在队伍里，它最像“多出来的一名半施法补位NPC”。Mage 可以用法师表补控制、侦测、护盾类工具和战场塑形；Healer 可以用牧师/德鲁伊表补治疗、复原、祝福、动物/自然工具；Prodigy 可以用吟游诗人/魔契师表补`魔能爆`、`治疗真言`、社交和少量控制。若作为额外协力者，它比普通NPC强很多；若替代一名完整PC，它又明显落后于真正的吟游诗人、牧师、德鲁伊、法师、术士或半施法职业，因为它已知法术少、没有子职、没有完整高环生态，也没有真实职业表的额外防线和资源。

## 详细评价

1级`协力者职业`要求获得施法者职业的生物数据板中至少有一门能说的语言。协力者规则通常限定CR 1/2或更低的生物数据板，因此施法者协力者的上限会被初始数据板影响：高施法属性、好移动、好防御、飞行或特殊感官的数据板会显著提高实际强度；普通人形NPC则更接近“轻甲半施法助手”。

1级`附赠熟练`给智力、感知或魅力豁免熟练之一，两项从奥秘、历史、洞悉、调查、医药、表演、宗教中任选的技能熟练，并熟练轻甲和简易武器。这个起点比专家协力者窄得多，但足够把它放在安全后排并补一个知识/医药/社交位。它没有盾牌、中甲、反应防线或完整PC生命骰，因此低级很怕被敌人识别为施法者后集火。

1级`施法`是全部价值核心。Mage 走法师表、智力施法；Healer 走牧师和德鲁伊表、感知施法；Prodigy 走吟游诗人和魔契师表、魅力施法。法术已知进展非常慢，但列表很宽，且每次升级可替换一个已知法术。低级最重要的不是伤害，而是把额外行动经济变成`祝福术`、`治疗真言`、`妖火`、`纠缠术`、`护盾术`、`魔能爆`、`雷鸣波`、`命令术`、`侦测魔法`等关键按钮。这里必须强调：它只有已知法术，不是每日准备完整表；一个1级协力者只知道1个非戏法法术，不能同时承担所有想象中的法术位功能。

5级开始有2环法术位，9级有3环，13级有4环，17级有5环。这个节奏比全施法者慢很多，但作为额外身体仍然强：第二个专注来源、第二套治疗/控制动作、额外反制目标压力和额外侦测/功能法术都能改变小队容错。Mage路线在T2-T3尤其看重`蛛网术`、`催眠图纹`、`法术反制`、`火球术`、`放逐术`、`石墙术`等候选；Healer路线看重`治疗真言`、`祝福术`、`次级复原术`、`复活术`、`群体治疗真言`、`防死结界`、`强效复原术`等候选；Prodigy路线则看重`魔能爆`、`治疗真言`、`妖火`、`暗示术`、`反制法术`、`放逐术`、`怪物定身术`等候选。具体强度会随已知法术选择大幅波动。

6级`强力戏法`让施法属性加入任意戏法伤害。对Prodigy的`魔能爆`尤其友好，因为多段戏法在等级成长后能把固定属性值多次兑现；对其他路线则让`圣火术`、`火焰箭`、`冻寒之触`等低资源输出更接近可用。它不等于术士完整射线生态：没有魔能祈唤、没有专属子职，也没有高阶魔契资源，但“额外身体每回合用戏法打人”本身已经有可观行动经济价值。

14级`增强法术`选择一个学派；协力者消耗法术位施放该学派法术时，可把施法属性加到伤害或治疗掷骰。这个特性对Evocation伤害、Conjuration/Abjuration等视具体法术而定，对Healing类治疗法术若学派匹配也能补一点数值。它的短板是只选一个学派，且高阶真正强的施法价值常常来自控制、位移、召唤、信息和状态，而非伤害/治疗掷骰本身。

20级`专注施法`很强但来得太晚：受到伤害不能打断协力者专注。它不能防止失能、昏迷、死亡、`解除魔法`、`反制法术`、反魔法场、非伤害式强制掉专注或直接击杀，但在普通伤害压力下维护`祝福术`、`妖火`、`蛛网术`、`放逐术`、`石墙术`、`怪物定身术`等非常舒服。问题是20级才取得，而终局队伍真正关心的往往已经是9环法术、传奇抗力、位面级机动和现实改写。

## 本轮重评

本轮社区校准检索覆盖 `Tasha sidekick Spellcaster Sidekick review guide 5e`、`Spellcaster Sidekick Tasha's Cauldron review reddit`、`D&D 5e sidekick rules Tasha spellcaster healer mage prodigy`、D&D Beyond 与通用网页方向。公开可定位材料多集中在“Sidekicks是给人数不足队伍或跟团NPC降低复杂度的规则”“协力者会增加行动经济和桌面管理量”，逐特性、同版本、可引用的施法者协力者强度共识很稀薄。因此本轮只把社区信息作为使用场景校准：施法者协力者常被视为小队补位/低复杂度NPC，而不是完整PC职业；具体评分按本地TCE文本、专家协力者当前评分和项目横向表独立判断。

横向上，施法者协力者比专家协力者更吃已知法术选择和初始数据板，但上限更高：额外专注、额外治疗/控制动作、5环以内的牧师/德鲁伊/法师/吟游诗人/魔契师工具，都能让小队容错明显提高。它仍不能按完整施法者评分，因为半施法进展、已知法术少、无子职、无高环、无完整职业资源都是真限制。综合定为 `A / A+ / A+ / A`：作为额外队友很强，作为完整PC替代则明显不足。

## 校准说明

施法者协力者按协力者口径评分，不按玩家完整基础职业评分。若它替代一名PC，它低于真正的法师、牧师、德鲁伊、吟游诗人、术士和强半施法者；若它作为额外随队协力者，额外专注、额外治疗、额外控制和额外戏法输出会显著提高队伍稳定性。项目默认其按规则计入遭遇难度，不假设玩家能自由堆叠多个协力者或挑选最优特殊数据板套利。

三类协力者横向校准后：施法者协力者综合为 `A / A+ / A+ / A`，技能和稳定协助弱于专家，但T2-T3上限最高，因为法术表能直接解决治疗、控制、反制和信息问题；专家协力者综合为 `A- / A / A / A-`，强在附赠动作协助、技能覆盖和可靠才能；武者协力者综合为 `A- / A / A- / A-`，强在低复杂度战斗和护卫，但高阶功能最窄。施法者T4回落到A，是因为没有6-9环法术和完整PC职业资源的缺口会重新变得很大。
