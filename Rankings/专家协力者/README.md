## 队伍定位摘要

专家协力者（Expert Sidekick）是《塔莎的万事坩埚》协力者规则中的技能/支援型NPC职业。本轮按本地 `5etools-src\data\class\class-sidekick.json` 与 `5etools-cn\data\class\class-sidekick.json` 重查，中文名采用本地翻译“专家协力者”。它不是给玩家角色正常选用的完整PC职业，而是给低CR、有语言能力的生物叠加的协力者职业；协力者起始等级与队伍平均等级相同，随队伍平均等级提升，并在遭遇难度估算中按一名角色计算。

专家协力者的核心价值是“多一个低复杂度队伍成员 + 每轮附赠动作协助”：1级`乐于助人`可用附赠动作执行协助动作，2级`灵巧动作`补机动，3级`专精`给两项技能双熟练，6级`协同攻击`把协助攻击的目标距离扩到30尺并让专家自己命中时追加2d6，11级`鼓舞协助`让被协助者的d20或命中后伤害再加1d6，14级`可靠才能`强化所有带完整熟练加值的属性检定，20级鼓舞骰提高到2d6。

因此它最适合小队人数不足、需要一个侦察/社交/工具手、或想让某个NPC持续跟团但不想管理完整PC职业的桌。若把它当作替代一名玩家角色的主职业，它明显不如游荡者、吟游诗人或游侠；若把它当作额外随队单位，它的强度来自行动经济和协助支援，而不是个人爆发。

## 详细评价

1级`协力者职业`要求获得专家职业的生物数据板中至少有一门能说的语言；这排除了大量纯野兽式随从，也让“专家”更偏智能NPC、雇员、学徒、斥候或社交助手。协力者规则本身还要求通常使用CR 1/2或更低的生物数据板；这意味着专家的强弱会被初始数据板显著影响。一个敏捷高、有好技能和好移动方式的数据板会比普通人形平民强得多。

1级`附赠熟练`给敏捷、智力或魅力豁免熟练之一，5项任选技能熟练、轻甲熟练；若是类人生物或数据板中已有简易/军用武器，还获得所有简易武器和两种工具熟练。这个起点非常适合做“队伍缺口填补”：察觉、隐匿、调查、洞悉、游说、盗贼工具等都能被补上。它的局限是没有施法、没有专长式专门机制，且战斗攻击仍受原数据板和装备限制。

1级`乐于助人`是强度核心：专家可以用附赠动作执行协助动作。低级战斗中，这等于把专家的附赠动作稳定转化为友方一次攻击优势或关键检定帮助。限制也要明确：普通协助攻击仍要求专家能在合适位置干扰目标，低级时专家通常要靠近敌人，容易被范围伤害、机会攻击或集火打掉；而且协助只改善一次d20，不等于提供持续增益或群体防线。

2级`灵巧动作`让专家以附赠动作疾走、撤离或躲藏。它和`乐于助人`竞争同一个附赠动作，但能让专家在危险战场中活下来、重选位置、或转入潜行支援。3级`专精`给两项技能双熟练，15级再给两项；这让专家在侦察、潜入、社交或工具位上非常稳定，尤其适合不想让主PC牺牲职业选择去补技能的队伍。

4、8、10、12、16、19级有六次`属性值提升`，若DM允许也可选专长。协力者属性提升会同步调整数据板相关攻击、技能、豁免和DC。这个成长很实用，但也带来桌面差异：有些DM会限制协力者专长，有些会允许强专长或飞行/特殊数据板组合，强度跨度会很大。

6级`协同攻击`是专家的战斗成形点。当专家用`乐于助人`协助盟友攻击某生物时，目标可距专家最多30尺；并且专家在本回合结束前下一次用攻击掷骰命中该生物时额外造成2d6同类型伤害。这个特性同时解决低级协助要贴脸的问题，并给专家一个“附赠动作协助 + 动作攻击”的基础回合循环。它仍不是游荡者的偷袭：没有随等级增长的伤害骰，也不保证命中，且要先把协助目标和自己的攻击目标对齐。

7级`反射闪避`显著提高专家对火球、龙息、陷阱等敏捷半伤效果的生存。11级`鼓舞协助`让受协助者获得1d6加到d20；若该掷骰是攻击，受协助者也可以放弃命中加值，在命中后把骰值加到一次伤害。这个节点让专家从“给优势”升级为“给优势外加小骰”，对高价值攻击、豁免式能力检定、反制/解除相关检定或关键技能检定都很舒服。20级提升到2d6，但来得太晚，且仍只是单次协助增量。

14级`可靠才能`把专家所有包含完整熟练加值的属性检定中d20为9或更低视为10；由于专家本身有大量技能熟练、两到四项专精，以及可能的工具熟练，这个能力让高阶非战斗可靠性很强。18级`明锐心智`再给智力、感知或魅力豁免熟练之一，补一点高阶心智防线。问题是高阶战斗生态已经进入高环法术、传奇抗力、飞行/传送、强范围控制和多层防线，专家没有施法、反制、群体治疗、强控或真正的高阶伤害成长；它更像可靠助手，而不是高阶问题解决者。

## 本轮重评

本轮社区校准检索覆盖 `Expert Sidekick Tasha 5e review`、`Tasha sidekick classes expert guide`、`Expert Sidekick Inspiring Help`、`Coordinated Strike Expert Sidekick`、Reddit 与 D&D Beyond 相关方向。可定位讨论以“协力者适合小队人数不足、低复杂度NPC跟团、会增加行动经济和桌面管理量”为主，未定位到稳定同版本、逐特性优化评测的专家协力者共识。因此本轮把社区信息作为使用场景校准：专家协力者常被讨论为小队补位/辅助NPC，而非完整PC职业；具体评分仍按本地TCE文本与项目横向表独立判断。

横向上，专家协力者低阶比普通NPC强得多：每轮附赠动作协助、五技能、专精和灵巧动作对小队非常实用。中阶后，30尺协助攻击、2d6跟进、反射闪避和1d6鼓舞协助让它成为优秀支援件。但它没有施法、没有多目标控制、没有真正个人DPR曲线，也高度依赖初始生物数据板和DM是否允许协力者专长/特殊种族能力。综合定为 `A- / A / A / A-`：这是很好的“额外支援身体”，不是能和完整PC职业同台争顶的主职业。

## 校准说明

专家协力者不按“玩家用完整基础职业”满额加权。若它替代一名PC，它明显低于游荡者、吟游诗人、游侠、奇械师等完整技能/支援职业；若它作为额外随队协力者，它的每轮附赠动作协助、技能覆盖和额外行动经济会显著提高队伍稳定性。项目评分采用中间口径：默认桌面正式允许协力者，并按遭遇难度把它视为一名角色，但不假设玩家能随意叠加多个专家或挑选最优怪物数据板来套利。

三类协力者横向校准后：专家协力者是最好的技能/协助支援，综合为 `A- / A / A / A-`；施法者协力者凭额外专注、治疗、控制、反制和信息法术拥有最高上限，综合为 `A / A+ / A+ / A`；武者协力者最擅长低复杂度战斗和护卫，但高阶功能最窄，综合为 `A- / A / A- / A-`。专家的高分来自队伍增益与技能可靠性，而非伤害或战役级规则所有权。
