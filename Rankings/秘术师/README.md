# 秘术师（5E UA）

## 队伍定位摘要
秘术师是5E UA时代的灵能全能职业：用`灵力点`驱动`灵能灵术`，以`心灵集中`取得可切换的常驻小增益，再用不同修会强化情绪控制、读心侦查、肉体改造、传送探索、魂刃武斗或元素塑形。它不是2026 UA`灵能使`，也不能按现代满施法者直接类比；它更像一个法术点施法者、半个技能职业和半个自定义职业框架的混合体。

本评估对象是 5E UA `秘术师 Mystic`，不是 2026 新UA的`灵能使`。按项目兼容补丁：19级`属性值提升`按5.5e终局基线替换为`传奇恩惠`，不额外获得`武器精通`；所有1级修会特性按3级获得来评估。

## 详细评价
1级本体只有1个`灵能天赋`、1个`灵能灵术`、4点灵力点和2点灵力限制。按5.5e兼容规则，原本1级修会赠送的额外灵术、护甲或魂刃等都推迟到3级，因此1-2级不再是旧UA文本中那种开局就拿一大包修会资源的形态。但从3级开始，灵术已知数、额外灵术和14点灵力点叠起来，职业开始明显压过普通低级角色的横向功能。

2级`秘术恢复`让每次消耗灵力点后可用附赠动作回血，`心电感应`给120尺可见目标无语言心灵交流。4级`心智之力`能在短休或长休后把感知豁免熟练替换成力、敏、体或魅力之一；这不是传统“全豁免”，但能按冒险信息切换关键防线，尤其对高压地城有价值。8级`烈性灵能`让武器每回合一次加心灵伤害，并让伤害天赋加智力调整值，给无资源回合保底。

真正的问题在5-10级成型。灵力限制到5-7点，灵力点在5级27点、9级57点、10级64点，且很多灵术是动作、附赠动作、反应分别触发，能在同一轮把位移、防御、爆发、控制或治疗压缩到一个角色身上。`迅捷爆发`的附赠动作闪避/额外武器攻击、`野兽形态`的飞行和+2 AC、`灵能复原`的治疗/复活、`预知心术`的长时先攻和防突袭、`漫游神智`的浮动熟练、`心灵突袭`/`心灵扰乱`的高效心智攻击，都解释了社区为什么长期说秘术师低中级“不是一个正常职业”。

10级`燃烧心火`可以用当前和最大生命值代替灵力点，数值不可减免且长休恢复，表面上有代价，但它把关键战斗的资源上限再抬一次。11级`灵能精通`给特殊灵力点池，并允许用这些特殊点同时维持多个需专注灵术；13、15、17级增加次数，15级特殊点池升到11点。这让秘术师在高阶仍能打出强爆发和多专注组合，但它仍停留在灵术7点限额附近，没有真正的`异界传送`、`拟像术`、`祈愿术`、大规模长期召唤/绑定和顶级法术经济。因此高阶强，但不按法师/牧师顶端生态给S+。

20级`灵能之躯`给钝击/穿刺/挥砍抗性、不衰老、免疫疾病/毒素伤害/中毒，并有死亡后50%概率1d3天重现。这是强生存终局，但来得太晚，且不能替代高阶法术的主动解题能力。

## 设计相关评分
| 设计项 | 评分 | 具体理由 |
|---|---|---|
| 设计质量 | D | 概念有野心，但资源、灵术包、跨修会选择、动作经济和多专注没有被同一套边界约束，低中级极易抢走队友职责。 |
| 主题 | A | 心灵训练、怪癖、修会、心灵集中和灵术都很好地呈现了非传统施法的灵能职业。 |
| 能力设计质量 | D | 单个灵术常像一组法术包，成本、动作、专注和伤害/控制尺度不统一；许多能力需要DM逐项判定，发布质量不足。 |
| 主题与能力关联度 | A- | 大多数能力确实服务“心智改变现实”，但全职业覆盖太宽，让主题从“灵能大师”滑向“什么都能做”。 |

## 校准说明
社区评价方面，Reddit、D&D Beyond 旧帖和指南讨论长期把 5E UA 秘术师称为“过度万能”“低中级太强”“高等级掉队”，也有少数玩家指出它并非在每个单项都超过专精职业，而是取决于灵术选择、灵力点消耗和冒险日长度。本轮不照搬任一极端说法：本地文本显示其核心问题是灵术包跨修会选择、灵力点近似法术点、心灵集中免费切换、10级生命值换灵力、11级起`灵能精通`允许多专注爆发；同时它没有真正6-9环法术生态、没有`祈愿术`、高阶解题宽度低于顶级满施法者。
