# 蜕变者（UA）

## 队伍定位摘要
蜕变者是灵能使的肉体改造/近战子职。二期把它改得更像“满施法者带可用近战形态”，而不是完整武职；它能打，但不能把它当野蛮人、武僧或月亮德鲁伊替代品。

## 详细评价
3级`蜕变者法术`给变形、肉体控制和补位工具，真正改变队伍定位的是`可变形态`和`有机武器`。`可变形态`用灵能骰换临时生命和身体适应，`有机武器`让它用智力进行近战，并能选择不同有机攻击形态。这个节点让灵能使第一次有了“站到前线也不完全浪费”的理由。

6级`额外攻击`和`血肉编织者`是关键成型点：它能在保持满施法者身份的同时拥有两次攻击和更好的变形收益。10级`强化可变形态`延长持续时间并提供更强适应，14级`扭命武器`把命中与黯蚀伤害/恢复挂钩，强化自持。

问题同样明显。社区对蜕变者的常见质疑是：一个d6生命骰满施法者想进近战，却缺少真正稳定的未着甲防御或重甲底盘。这个问题没有被二期完全解决。起源专长能给`护盾术`或其他补丁，但灵能使本来法表已有`护盾术`，这不是蜕变者独占优势。蜕变者强在“满施法者能临时肉搏”，不是“最强近战伤害者”。

本轮校准：蜕变者不因UA身份降权，但它的子职增量主要是个人近战、自持和肉体适应。灵能使本职高阶已经是 S- 级满施法者，蜕变者高等级可以随底盘进入 S-；但它没有念动使的硬分隔、裂空使的群体重排，也没有传心师的队友结果修正，因此不进 S。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | A | 3级`可变形态`和`有机武器`给近战入口，同时保留灵能使低级控制；但d6生命骰和防御底盘不支持长期前排。 |
| Tier 2（5-10） | A+ | 6级`额外攻击`与`血肉编织者`让近战形态成型，满施法底盘仍在；限制是近战、专注和施法动作互相抢节奏。 |
| Tier 3（11-16） | S- | 10级`强化可变形态`和14级`扭命武器`提高自持，高环灵能使法术撑起综合；子职增量偏个人强化，所以为 S-。 |
| Tier 4（17-20） | S- | 终局主要靠灵能使高环法术和20级资源，蜕变者提供肉体主题与生存补充；不是最佳高阶控制/支援子职。 |

| 主题 | A | 肉体异化、骨刃、伸展肢体和生命扭曲主题鲜明，是四个子职里视觉冲击最强的一个。 |
| 能力设计质量 | B | 有机武器和额外攻击给了清晰玩法，但资源、防御和满施法行动经济之间仍有张力。 |
| 主题与能力关联度 | A | 所有能力都围绕肉体改造展开，几乎没有脱题能力。 |
