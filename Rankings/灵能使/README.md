## 队伍定位摘要
灵能使是UA满施法职业，最新基础职业、才赋选项、法术表、蜕变者、念动使、传心师均采用`其他\新UA\灵能二期`；`裂空使`使用一期`其他\新UA\灵能`，因为二期说明没有重测该子职，社区讨论也普遍理解为它测试反馈已经足够好，暂未在二期重写。它不是正式出版职业，评分按实际功能处理，不因UA身份降权。

## 详细评价
灵能使的核心不是“另一张法师表”，而是用`灵能施法`、`灵能力量`、`灵能才赋`和专属心灵/念动法术制造一个偏控制、心灵沟通、位移和特殊资源管理的满施法者。它的法术表非常强：低环有`护盾术`、`睡眠术`、`塔莎狂笑术`、`侦测思想`、`沉默术`、`塔莎心灵鞭`；中阶有`催眠图纹`、`恐惧术`、`反制/解除类工具中的解除魔法`、`召唤星界实体`、`放逐术`、`困惑术`、`任意门`、`支配人类`、`怪物定身术`、`心灵静爆`、`心灵遥控`、`传送法阵`；高阶有`力场监牢`、`海市蜃楼`、`传送术`、`迷宫术`、`支配怪物`、`心灵屏障`、`预警术`、`变身术`、`时间停止`和`心灵尖啸`。没有`祈愿术`使它的终局上限低于最顶层法师，但它仍是货真价实的高阶满施法者。

1级`灵能施法`让许多施法不依赖通常的语言/材料表现，实桌中会强化隐蔽施法、束缚环境施法和社交场景施法。`灵能力量`提供会随等级成长的灵能骰，2级`灵能才赋`让它每回合能把骰子换成额外功能。5级`灵能复原`和7级`灵能浪潮`解决资源续航与爆发，18级`灵能储备`和20级`源火生命`则让高阶资源更稳定。

社区对二期灵能使的整体反馈比一期更正面：常见评价是“这版方向对了，终于像一个能落桌的灵能满施法者”，但也有人批评它仍然更像“带灵能皮肤的施法者”，而不是完全独立的灵能系统。这个批评成立：它的强度主要来自优秀法表加灵能骰，而不是一套彻底区别于法术位的核心机制。

本轮校准重点是把灵能使放回当前满施法者压缩标准中：UA身份不自动降权，但文本仍可能在正式出版前变化；同时，灵能使没有默认`祈愿术`，也没有法师法术书/仪式经济或牧师/德鲁伊日备全表那种生态宽度。它的优势是强控制法表、心灵/念动主题法术、较隐蔽的`灵能施法`表现和灵能骰资源；这些足以让它成为高阶强满施法者，但本职默认不应压过真正拥有更广泛生态重写能力的顶级法师。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | A | 1-4级已有`护盾术`、`塔莎狂笑术`、`睡眠术`、`侦测思想`、`灵能力量`和2级`灵能才赋`，低级控制和隐蔽施法很强；但灵能骰、法术位和已准备法术规模仍小。 |
| Tier 2（5-10） | S- | 3-5环控制、召唤、位移和信息法术上线，`灵能复原`与`灵能浪潮`提高资源流；但本职没有`法术反制`级反魔法强项，也没有法师式法术书宽度，默认压缩为 S-。 |
| Tier 3（11-16） | S- | `力场监牢`、`传送术`、`迷宫术`、`心灵屏障`、子职10/14级特性让它能改写遭遇；限制是法表生态窄于法师且没有`祈愿术`。 |
| Tier 4（17-20） | S- | `预警术`、`变身术`、`时间停止`、`心灵尖啸`和20级资源回流保持强满施法者地位；UA不降权，但终局缺少`祈愿术`和成熟出版生态，仍为 S-。 |

## 校准说明
灵能使按最新 UA 满施法职业横向比较，不因 UA 身份自动降权，也不因像施法者而自动等同法师。它的强项是强控制法表、心灵/念动主题、较隐蔽的灵能施法、灵能骰与部分无专注/位移子职增量；限制是没有默认祈愿术，没有法术书/仪式经济，也缺少成熟出版生态带来的广泛规则支撑。因此本职高阶保持 S-，念动使、裂空使等能明显改变战场位置或专注压力的子职可进入 S。
