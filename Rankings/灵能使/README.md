# 灵能使（UA）总评

## 队伍定位摘要
灵能使是UA满施法职业，最新基础职业、才赋选项、法术表、蜕变者、念动使、传心师均采用`其他\新UA\灵能二期`；`裂空使`使用一期`其他\新UA\灵能`，因为二期说明没有重测该子职，社区讨论也普遍理解为它测试反馈已经足够好，暂未在二期重写。它不是正式出版职业，评分按实际功能处理，不因UA身份降权。

## 详细评价
灵能使的核心不是“另一张法师表”，而是用`灵能施法`、`灵能力量`、`灵能才赋`和专属心灵/念动法术制造一个偏控制、心灵沟通、位移和特殊资源管理的满施法者。它的法术表非常强：低环有`护盾术`、`睡眠术`、`塔莎狂笑术`、`侦测思想`、`沉默术`、`塔莎心灵鞭`；中阶有`催眠图纹`、`恐惧术`、`解除魔法`、`召唤星界实体`、`放逐术`、`困惑术`、`任意门`、`支配人类`、`怪物定身术`、`心灵静爆`、`心灵遥控`、`传送法阵`；高阶有`力场监牢`、`海市蜃楼`、`传送术`、`迷宫术`、`支配怪物`、`心灵屏障`、`预警术`、`变身术`、`时间停止`和`心灵尖啸`。没有`祈愿术`使它的终局上限低于最顶层法师，但它仍是货真价实的高阶满施法者。

1级`灵能施法`让许多施法不依赖通常的语言/材料表现，实桌中会强化隐蔽施法、束缚环境施法和社交场景施法。`灵能力量`提供会随等级成长的灵能骰，2级`灵能才赋`让它每回合能把骰子换成额外功能。5级`灵能复原`和7级`灵能浪潮`解决资源续航与爆发，18级`灵能储备`和20级`源火生命`则让高阶资源更稳定。

社区对二期灵能使的整体反馈比一期更正面：常见评价是“这版方向对了，终于像一个能落桌的灵能满施法者”，但也有人批评它仍然更像“带灵能皮肤的施法者”，而不是完全独立的灵能系统。这个批评成立：它的强度主要来自优秀法表加灵能骰，而不是一套彻底区别于法术位的核心机制。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | A | 1级已有`护盾术`、`塔莎狂笑术`、`睡眠术`、`心灵之削`、`灵能力量`和2级`灵能才赋`，控制与自保都强；但法术位和灵能骰规模还小，不能压过最顶级低级职业。 |
| Tier 2（5-10） | S | 3环与4环控制、召唤和位移进入核心区间，`灵能复原`、`灵能浪潮`和子职6级能力叠加后，队伍解题能力接近满施法顶层。 |
| Tier 3（11-16） | S | `力场监牢`、`传送术`、`迷宫术`、`心灵屏障`、子职10/14级特性让它具备高阶遭遇改写能力；没有`祈愿术`，但仍明显高于普通A档。 |
| Tier 4（17-20） | S | `预警术`、`变身术`、`时间停止`、`心灵尖啸`和20级资源回流让它保持S；只是缺少`祈愿术`导致不进入S+。 |

## 设计相关评分
| 设计项 | 评分 | 具体理由 |
|---|---|---|
| 设计质量 | B | 二期明显比一期更清楚，资源、才赋、子职节点都能读懂；扣分在于它仍大量依赖传统法术表，灵能与法术的边界不够锋利。 |
| 主题 | A | 心灵、念动、变形、空间折叠四个方向辨识度很强，职业主题比普通法师更集中。 |
| 能力设计质量 | B | 灵能骰、无语材施法和才赋提供了新手感，但高等级仍靠满施法法表支撑，部分专属法术和资源消耗需要实桌验证。 |
| 主题与能力关联度 | A | 心灵沟通、隐蔽施法、移动物体、位移和自我改造都能直接解释为灵能表现，主题与功能关系很紧。 |

## 校准说明
本轮按UA灵能使混合最新边界评估：基础职业、灵能才赋、法术表、蜕变者、念动使、传心师采用`其他\新UA\灵能二期`；裂空使采用一期`其他\新UA\灵能`，因为二期没有替换它。UA内容按实际功能评分，不因测试稿身份刻意压低；若未来正式版发布，应重新按正式文本评估。

## 分阶段子职排行榜

本职业本身为UA职业，因此本排行榜列出当前灵能使生态内的全部子职；文件名仍保留`（UA）`标记，避免未来正式版混淆。

Tier 1（1-4）:
- A:
  - 蜕变者（UA）：3级`可变形态`和`有机武器`让它有近战入口，同时仍是灵能满施法者；但低级生命值和防御底盘不支持硬扛。
  - 念动使（UA）：3级`念动技法`和念动扩表已经能稳定推拉、补伤害和控制位置，但S级关键工具还没到。
  - 传心师（UA）：`心灵渗透`和`信号干扰`在3级就能影响侦查、社交和战斗命中，低级功能很实。
  - 裂空使（UA）：3级`裂空推送`已经能把失败豁免转化为空间惩罚，低级战场分隔强于多数施法子职。

Tier 2（5-10）:
- S:
  - 念动使（UA）：`缓慢术`、`力墙术`、6级`回弹力场`和`毁灭迷狂`让它同时有控制、防御和输出增强，是灵能使二期最强成型窗口。
- A:
  - 蜕变者（UA）：6级`额外攻击`和`血肉编织者`让近战形态真正成型，但它仍要在施法、专注和站位风险之间取舍。
  - 传心师（UA）：6级`强韧心智`和10级`心灵援助`提高队伍防线，但缺少念动使那种直接战场封锁天花板。
  - 裂空使（UA）：6级`传送战斗`和10级`虚假目标`提供优秀位移动作经济和防御换位，但还未达到群体战场改写。

Tier 3（11-16）:
- S:
  - 念动使（UA）：14级无槽且无专注的`心灵遥控`非常强，能持续改写战场；强度接近顶级控制子职但不够S+。
  - 传心师（UA）：14级`扰乱心灵`是无槽大范围友方豁免`困惑术`，配合高阶法表和队友补正，进入S级控制/支援。
  - 裂空使（UA）：14级`群体瞬步移动`能一次重排多名敌我位置，配合高环法术和地形非常强，是裂空使最好的阶段。
- A:
  - 蜕变者（UA）：10级`强化可变形态`和14级`扭命武器`提高自持，满施法法表也进入高阶；但近战收益不够把它推到S。

Tier 4（17-20）:
- S:
  - 念动使（UA）：高环灵能使法术叠加念动控制仍是S；没有`祈愿术`和时间魔法式越界，所以不上S+。
  - 裂空使（UA）：高环满施法加群体传送仍保持S；但强度依赖地图和传送裁定，不给S+。
- A:
  - 蜕变者（UA）：终局强度主要来自灵能使高环法术，蜕变者提供肉体主题和自持补充，不是高阶环境的最佳解题子职。
  - 传心师（UA）：终局仍强，但精神控制容易受免疫、传奇抗力和高豁免影响，综合略低于念动使的硬控制。
