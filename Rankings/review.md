## Goal
Rewrite official / partner class and subclass reviews so `详细评价` reads like the homebrew reviews:

- Analyze the class chassis feature by feature.
- Analyze each subclass feature by feature.
- Mention how spell lists, weapon mastery, 起源专长（Origin Feats）, magic items, Circle Magic, and tier breakpoints affect ranking.
- Let the prose naturally state whether a feature is strong, weak, balanced, narrow, over-budget, or poorly designed.
- Avoid repeating aspect-ranking summaries as the main body of `详细评价`.
- Keep all standards: honest, objective, unbiased, brutally clear.

## Current Calibration Notes
- 起源专长（Origin Feats） are assumed available. If a class/subclass gains something commonly accessible through an 起源专长（Origin Feat）, the review should credit saved build cost but not treat it as exclusive power.
- S+ is reserved for the best of the whole ecosystem, not merely the best within one class.
- For martial classes, Weapon Mastery improves floor and play quality but does not by itself solve Tier 3-4 full-caster gap.
- For spellcasters, spell list and action economy usually matter more than isolated subclass ribbon features.
- For legacy 2014 subclasses in 2024 environment, judge both text power and interface friction.
