## 队伍定位摘要

Treasure Hunter 是《The Lord of the Rings Roleplaying》的10级潜入、盗宝、侦察和弱点打击型新基础职业。本轮依据本地 `5etools-homebrew\collection\Free League; The Lord of the Rings Roleplaying.json` 中 `Treasure Hunter|TLotRR` 重查；`DND5e_chm`、`5etools-cn` 与既有 `Rankings` 未定位到可靠中文正式译名，因此本职业文件暂保留英文名。

这个职业的共同底盘是 d8 生命骰、敏捷/智力豁免、轻甲、简单武器/短剑/TLotRR剑、4个技能、工具入口、`Expertise`、标准条件的 `Sneak Attack`、`Cunning Action`、5级 `Uncanny Dodge`、9级第二轮专精、10级 `Reliable Talent`，以及4/6/8级的 Virtue/Craft 选择。它比 Messenger 更像完整盗贼替代品：偷袭按1-10级推进到5d6，2级就有附赠动作 Dash/Disengage/Hide，10级还拿到可靠才能。它仍不是2024游荡者：没有武器精通、狡诈打击、子职高阶进展，也没有11级以上原生职业表。

在队伍里，Treasure Hunter 是TLotRR低魔生态里最直接的专家/潜入/DPR混合位。它的T1已经有4技能、双专精、Cunning Action和2d6偷袭；T2有3-5d6偷袭、Uncanny Dodge、第二轮专精和Reliable Talent，技能可靠性明显高于多数非施法专家。若战役重视潜入、侦查、锁具、盗宝、弱点判断和低魔资源管理，它很容易成为队伍问题解决核心；若直接进入普通1-20高魔生态，10级后的部分需要另行延伸；本评审不把无施法、无高阶控制/反制的空白外推成T3/T4评分。

## 详细评价

1级 `Expertise` 让任意两个技能或工具熟练获得双倍熟练，9级再选两个。配合起始4技能和工具入口，Treasure Hunter 的低级技能面非常宽。源JSON在工具熟练上存在一个小不一致：文字写作从制图工具、游戏用具或盗贼工具中选择一个，结构化 `toolProficiencies` 则是选择两个。本轮按文字入口保守理解，并在实际桌面建议DM先确认采用哪一处；即使按一个工具处理，核心技能强度仍然成立。

1级 `Sneak Attack` 条件与传统游荡者接近：每回合一次，使用灵巧或远程武器命中，若攻击具有优势，或目标5尺内有另一个未失能敌人且你不处于劣势，则追加伤害。成长是1-2级1d6、3-4级2d6、5-6级3d6、7-8级4d6、9-10级5d6。这个进度明显快于 Messenger 的慢偷袭，足以让 Treasure Hunter 在1-10级低魔战斗中维持真实DPR；限制是没有额外攻击、没有施法增伤，也没有2024游荡者的狡诈打击把偷袭转化为控制。

2级 `Cunning Action` 给每回合附赠动作 Dash、Disengage 或 Hide。这是职业的行动经济核心：它让 Treasure Hunter 能稳定换位、脱离、找遮蔽或制造优势，直接提高偷袭兑现率和生存。5级 `Uncanny Dodge` 再用反应把可见攻击者命中造成的伤害减半，和轻甲/d8组合后让它能承担危险侦察和游击位，但仍害怕范围豁免、控制、抓取、震慑和无法隐藏的开阔战场。

4/6/8级 `Virtue` / `Craft or Virtue` 是TLotRR共同成长轴，可补属性、文化特长、工艺工具或低魔子系统能力。Treasure Hunter 对这些选择的依赖低于 Scholar，因为自身有完整技能/偷袭/机动底盘；但若战役采用Journey、Craft、文化Virtue和中洲资源压力，6级Craft入口会进一步扩展盗宝、旅行和问题解决。若放进普通5e，Virtue/Craft价值仍要看DM如何接入。

10级 `Reliable Talent` 是基础职业的高点：任何能加入熟练加值的能力检定，d20掷出9或更低都视为10。它与4技能、四项专精、Cunning Action Hide、盗贼工具/制图/调查/潜行等组合后，让Treasure Hunter在1-10级封顶环境中拥有近似“不会失手”的专家身份。问题也很清楚：职业表到10级结束，后续没有11级后游荡者特性、狡诈打击升级、额外ASI、强防线、施法、传送、反制或战役级工具。

## 本轮重评

本轮社区校准检索覆盖 `The Lord of the Rings Roleplaying Treasure Hunter class review`、`TLotRR Treasure Hunter Burglar Spy`、`Lord of the Rings Roleplaying 5e Treasure Hunter calling`、`Treasure Hunter Weak Spot Lord of the Rings Roleplaying`、Reddit、D&D Beyond论坛与通用网页方向。可稳定定位的公开材料主要是TLotRR规则包、上架信息和“5e低魔10级Calling”背景介绍，未定位到同版本、逐项讨论Treasure Hunter强度或Burglar/Spy优化的稳定共识。因此本轮只把外部材料作为低魔10级结构背景，最终评分按本地TLotRR文本、2024职业生态、Captain/Champion/Messenger/Scholar组和第三方新基础职业横向样本独立判断。

横向上，Treasure Hunter 的T1-T2应明显高于 Messenger 基础线：它有正常偷袭成长、2级Cunning Action、更多专精和10级Reliable Talent；但它不应越过Champion/Slayer那类纯战斗峰值，也不能按完整2024游荡者的1-20成长、武器精通和狡诈打击评价。综合定为 `A / A+`；T3/T4因TLotRR原生止于10级不评估。

## 校准说明

Treasure Hunter 按第三方新基础职业口径评分，不因非WotC来源自动降权；T1-T2评价其TLotRR原生1-10级内容，T3/T4因TLotRR原生止于10级不评估。它比Messenger更强在完整偷袭进度、Cunning Action和Reliable Talent，比Scholar更强在个人DPR与潜入执行；但它不像Champion那样在T2获得三打或强战斗Way，也不像完整施法者/完整20级职业那样拥有高阶问题解决。

本组横向校准后，Spy 是家族中更强的实战Speciality：`Weak Spot`给单体偷袭准入，`Cunning Eye`和`Sharp-Eyed`强化侦查/搜索，能把Treasure Hunter的核心输出和侦察身份结合起来。Burglar 更贴近盗宝、攀爬和物件互动，探索强但战斗增量较低。两者T3/T4不评估，不是因为低级能力失效，而是TLotRR 10级Calling结构缺少11级后的职业表。
