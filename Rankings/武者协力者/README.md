## 队伍定位摘要

武者协力者（Warrior Sidekick）是《塔莎的万事坩埚》协力者规则中的低复杂度战斗NPC职业。本轮按本地 `5etools-src\data\class\class-sidekick.json` 与 `5etools-cn\data\class\class-sidekick.json` 重查，中文名采用本地翻译“武者协力者”。它不是给玩家角色正常选用的完整PC职业，而是给低CR、有语言能力的生物叠加的协力者职业；协力者起始等级与队伍平均等级相同，随队伍平均等级提升，并在遭遇难度估算中按一名角色计算。

武者协力者的共同底盘是：1级`附赠熟练`给力量/敏捷/体质豁免之一、两项武职技能、所有护甲；若是类人生物或数据板已有简易/军用武器，还获得盾牌、简易武器和军用武器熟练。1级`战斗职能`在Attacker与Defender之间选择，前者所有攻击检定+2，后者可用反应让5尺内攻击者攻击非自己目标时劣势。随后有2级`回气`、3级`精通重击`、6级`额外攻击`、7级`备战`先攻优势、10级AC+1、11/18级`不屈`、15级三次攻击，以及多次ASI/可选专长。

在队伍里，它最像“多出来的一名简化战士/护卫”。Attacker路线能稳定补命中和武器DPR；Defender路线能保护后排或主人；多次ASI、护甲盾牌和三次攻击让它比普通NPC耐用得多。它的短板也很直白：没有施法、反制、强控、技能可靠性、战役级解题或真正子职；强度高度依赖初始数据板、装备、DM是否允许协力者专长，以及协力者作为额外行动经济而非PC替代的使用方式。

## 详细评价

1级`协力者职业`说明该职业意图用于NPC协力者。协力者规则通常使用CR 1/2或更低、能说至少一种语言的生物数据板，因此武者协力者的真实强度会被初始数据板拉开：原本就有高力量/敏捷、好感官、好移动、飞行或多攻击的数据板会比普通卫兵强很多。项目评分不假设玩家能任意挑选最优怪物数据板，也不把它当作完整PC职业替代。

1级`附赠熟练`给一个大豁免熟练、两项战斗/野外技能、所有护甲，并在条件满足时给盾牌和全武器熟练。这是武者低级耐用性的来源：一个原本脆弱但有语言的数据板可以立刻穿甲持盾，或者使用合适军用武器。它的限制是没有技能宽度、工具、法术或特殊战术；如果初始生物属性差，单靠熟练并不会变成真正战士。

1级`战斗职能`决定基础路线。Attacker的所有攻击检定+2非常稳定，尤其对低CR数据板和多攻击/远程武器路线价值高；Defender的反应劣势很像简化保护风格，可保护5尺内被攻击的其他目标。Attacker更适合补DPR，Defender更适合护卫NPC或小队后排。两者都很低复杂度，但也都缺少战斗大师式控制、回音骑士式站位或圣武士式光环。

2级`回气`用附赠动作恢复 `1d10 + 武者协力者等级`，短休/长休恢复，20级变为每休两次。3级`精通重击`让攻击在19-20重击，6级`额外攻击`给两次攻击，15级变为三次。若数据板已有Multiattack，它每回合只能在Extra Attack和Multiattack中选一个，不能叠加。这条限制很重要：武者不能把怪物多重攻击和职业额外攻击无脑合并成越界行动经济。

7级`备战`给先攻优势，10级`精通防御`AC+1，11级`不屈`失败豁免重掷一次，18级变两次。再加上4/8/12/14/16/19六次ASI或DM允许的专长，武者协力者可以逐步修正攻击属性、体质、AC或拿战斗专长。问题是这些多半是被动数值成长：它确实更能打、更能站，但不会获得施法者协力者那种控制/治疗/反制工具，也不会获得专家协力者的可靠才能与每轮协助增益。

总体上，武者协力者是三类协力者中最容易运行、最直观的战斗补位。它对小队人数不足、需要一个前排、护卫、弓手或低复杂度战斗NPC的桌很有用；但在高阶生态中，纯武器额外身体会逐渐被飞行、传送、范围控制、传奇抗性、高阶施法和战役级问题拉开。

## 本轮重评

本轮社区校准检索覆盖 `Tasha Warrior Sidekick review 5e`、`Warrior Sidekick Tasha's Cauldron review reddit`、`D&D 5e Warrior Sidekick rules review`、`D&D Beyond Warrior Sidekick sidekick rules` 等方向。可定位公开讨论多集中在“Sidekicks适合人数不足队伍或跟团NPC”“协力者会增加行动经济与桌面管理量”，未定位到稳定同版本、逐特性优化评测的武者协力者共识。因此本轮只把社区信息作为使用场景校准，具体评分按本地TCE文本、专家/施法者协力者当前评分和项目横向表独立判断。

横向上，武者协力者比专家协力者更会打、更硬、更容易保护队友；但它没有专家的附赠动作协助、技能宽度、可靠才能，也没有施法者协力者的额外专注、治疗、控制、反制和信息法术。作为额外随队身体，它很有用；作为完整PC职业替代，它明显缺乏高阶规则所有权。综合定为 `A- / A / A- / A-`。

## 校准说明

武者协力者按协力者口径评分，不按玩家完整基础职业评分。若它替代一名PC，它低于战士、野蛮人、游侠、圣武士等完整武器职业；若它作为额外随队协力者，额外身体、护甲、武器攻击和保护反应能显著提高小队低中阶稳定性。项目默认其按规则计入遭遇难度，不假设玩家能自由堆叠多个协力者或挑选最优特殊数据板套利。

三类协力者横向校准后：施法者协力者最高上限来自额外专注、治疗、控制、反制和信息法术，综合为 `A / A+ / A+ / A`；专家协力者最擅长技能、协助和可靠执行，综合为 `A- / A / A / A-`；武者协力者最稳定直接战斗，但高阶功能最窄，综合为 `A- / A / A- / A-`。
