# 德鲁伊总评

## 队伍定位摘要
德鲁伊是原初满施法者，队伍定位是治疗、召唤、区域控制、自然探索和战术资源转换。2024版削弱了旧荒野变形生命池，但让荒野变形与法术位、伙伴、子职资源的关系更清楚。

## 详细评价
1级`施法`给德鲁伊完整原初施法底盘，准备法术来自德鲁伊法术列表，能使用德鲁伊法器。`德鲁伊语`和始终准备`动物交谈`让它在动物、自然和探索场景里有明确身份。1级`原初职能`则允许选择`术师`或`卫士`：前者给额外德鲁伊戏法，并把感知调整值加入智力（奥秘）和智力（自然）检定；后者给军用武器熟练和中甲训练。这个选择会影响早期防线与技能定位，但不会改变德鲁伊作为满施法者的核心。

2级`荒野变形`在2024版变成附赠动作，使用次数有限，短休恢复1次，长休恢复全部。变形时不再获得野兽完整生命池，而是获得等同德鲁伊等级的临时生命；身体数据替换为野兽，但保留生命值、智力/感知/魅力、职业特性、语言和专长，且不能施法但可维持专注。这个改动大幅削弱旧版“靠野兽生命池反复坦伤害”的玩法，也让月亮结社以外的德鲁伊更倾向把荒野变形当作工具资源。`荒野伙伴`则可花法术位或荒野变形无材料施放`寻获魔宠`，补上侦查和小型伙伴价值。

5级`荒野复苏`把荒野变形与法术位接起来：没有荒野变形次数时，可花法术位恢复一次；也可花一次荒野变形恢复一环法术位。7级`元素之怒`在`强力施法`与`原力蛮击`间选择，前者把感知调整值加入德鲁伊戏法伤害，后者让武器或野兽形态攻击每回合追加元素伤害；15级继续强化。18级`兽形施法`和20级`大德鲁伊`让高等级变形、施法和资源转换更自由。整体上，德鲁伊不是最强反施法者，也不是最强单体爆发者；它的真实位置是治疗、召唤、区域控制、自然探索和长期资源的满施法核心。

## 分项理由
- **治疗和临时生命**：德鲁伊法术表与多个子职都能提供治疗，荒野变形也能在非月亮路线中作为资源转换入口。
- **战场分隔 / 减益**：缠绕、地形、植物、天气、召唤和高环自然法术使德鲁伊很擅长改变战场形状。
- **召唤 / 伙伴**：`荒野伙伴`加上德鲁伊召唤法术使伙伴能力从低级就存在。
- **反制施法者**：基础德鲁伊通常没有`法术反制`一类奥术硬反制，更多依赖`解除魔法`、沉默/地形/视线控制和召唤压迫。

## 校准说明
本轮依据`DND5e_chm\玩家手册2024\角色职业\德鲁伊`及各正式/UA子职本地文本重评。社区校准采用2024德鲁伊与子职讨论：星辰、野火、大地评价偏高，月亮被普遍视为较2014版健康化，海洋中后期稳定，恩护UA按实际功能而非测试稿身份评分。

## 分阶段子职排行榜

Tier 1（1-4）:
- S:
  - 星辰结社：3级`星图`、免费`光导箭`和三种星耀形态直接覆盖伤害、治疗与专注稳定，是官方德鲁伊低级顶层。
- A:
  - 大地结社：3级可每日选择地貌并获得额外法术，`大地之援`兼具小范围伤害与治疗，低级稳定但未破格。
  - 月亮结社：3级高CR荒野变形、13+感知AC和三倍等级临时生命让低级近战/承伤强，但不再是2014式额外生命池。
  - 海洋结社：3级`瀚海之怒`提供附赠动作小范围冷冻伤害和推离，低级控位不错但半径很小。
  - 牧人结社：按2024兼容，`精魂图腾`和`林地之语`视为3级；熊/鹰/独角兽图腾低级非常强。
  - 野火结社：按2024兼容，旧2级特性视为3级；野火灵魄提供伙伴、传送和小范围伤害，低级很实用。
- B:
  - 孢子结社：按2024兼容，旧2级特性视为3级；四倍等级临时生命很厚，但近战增伤与德鲁伊本职行动模式冲突。
  - 梦境结社：按2024兼容，旧2级治疗池视为3级；低级附赠动作远程治疗好用，但战斗控制没有额外提升。

Tier 2（5-10）:
- S:
  - 大地结社：6级`自然恢复`提供无槽结社法术和短休回法术位，资源量明显高于基础德鲁伊。
  - 星辰结社：6级`宇宙预兆`可给关键d20加减d6，强化队友或压敌人豁免，功能面继续保持S。
  - 野火结社：6级`强化链接`把火焰伤害与治疗都加d8，并允许法术从灵魄起点施放，战术价值进入S。
- A:
  - 月亮结社：6级野兽攻击可改光耀伤害并给体质豁免加感知，专注更稳，仍是A档前线施法混合体。
  - 海洋结社：6级光环扩大到10尺并获得游泳速度，水域战强，普通地面战仍是A档。
  - 牧人结社：6级召唤兽与妖精强化仍有价值，但2024召唤法术生态削弱了旧版海量召唤滥用。
- B:
  - 孢子结社：6级僵尸反应有战术价值，但1生命随从在中级战斗中很脆。
  - 梦境结社：6级休息结界提升潜行和察觉，偏探索/扎营，不足以把战斗综合推到A。

Tier 3（11-16）:
- S:
  - 大地结社：10级抗性与免疫中毒让专注和生存更稳，结社法术仍覆盖伤害、控制和移动。
  - 星辰结社：10级星座可每回合切换，射手/圣杯变2d8，巨龙给飞行悬停，灵活度非常高。
  - 野火结社：10级死亡生物留下的火焰可治疗或伤害，配合灵魄传送形成优秀团队支援。
- A:
  - 月亮结社：10级`月光飞步`补机动与优势，但野兽攻击成长开始低于真正武职和顶级控制子职。
  - 梦境结社：10级附赠动作传送自己或触摸传送队友，机动和救援价值明显提高。
  - 海洋结社：10级光环期间飞行和三种抗性显著提高生存与机动，但输出没有进入顶级。
  - 牧人结社：10级图腾区域内召唤物回血很好，但强度取决于战役采用的召唤法术与怪物管理方式。
- B:
  - 孢子结社：10级扩散孢子能做区域伤害，但会占用原本的反应孢子，强度仍偏中等。

Tier 4（17-20）:
- S:
  - 大地结社：14级`自然庇护`把半掩护和元素抗性投给队友，高等级团队防线优秀。
  - 星辰结社：14级星耀形态期间获得钝击/穿刺/挥砍抗性，高等级仍是最全面官方德鲁伊之一。
  - 海洋结社：14级可把光环给盟友或双光环，配合高环德鲁伊控制时团队价值明显上升。
  - 野火结社：14级`炽烈回生`用灵魄换半血复起，高等级保命能力非常强。
- A:
  - 孢子结社：14级免疫多种状态且免暴击，防御明显提高；高等级仍靠德鲁伊法术表抬综合。
  - 月亮结社：14级每回合光耀加伤和带人传送很好，但高等级德鲁伊的最优行动仍多是施法而非野兽普攻。
  - 梦境结社：14级短休后无槽`托梦术`、`探知术`或特殊`传送法阵`，高等级剧情/旅行功能不错但不顶级。
  - 牧人结社：14级濒死自动召唤四只野兽有救场性，但作为旧法术文本兼容，桌面执行负担很高。
