## 队伍定位总结
德鲁伊是原初满施法核心，主要负责治疗与复原、召唤和伙伴压场、地形/天气/植物系区域控制、自然探索，以及把荒野变形在侦察、承压、魔宠和法术位之间转换。它能用日备法术适应环境，长日韧性很强；短板是默认缺少`法术反制`式硬反制、单体爆发不如专精奥术施法者，许多强项还吃专注、地形、召唤执行和敌人移动类型。

## 详细评价
1级`施法`给德鲁伊完整原初准备施法：治疗、解除、地形控制、召唤和自然探索都从法术表展开，长休换准备让它能按当天地区、敌人类型和旅行需求改装。`德鲁伊语`与始终准备`动物交谈`让动物、植物、荒野信息和自然社交成为稳定入口。`原初职能`把早期路线分成`术师`与`卫士`：前者强化戏法与自然/奥秘知识，后者给军用武器和中甲，决定的是站位与技能侧重，不改变“先用法术塑形战场”的核心。

2级`荒野变形`是附赠动作有限资源，短休只回1次、长休回满。2024文本不再给野兽完整生命池，而是给等同德鲁伊等级的临时生命；变形时不能施法但可以维持专注，身体数据替换、心智属性和职业特性保留。这让普通德鲁伊更常把变形用于侦察、移动、承压、穿越和子职资源，而不是长期当主坦。`荒野伙伴`可用法术位或荒野变形施放`寻获魔宠`，把侦察、帮助、送触及法术和小型占位接入职业资源。

5级`荒野复苏`把荒野变形和法术位互换：缺变形时可花法术位补一次，也可花变形回一环法术位。这个转换提高长日弹性，但不是免费资源循环，因为它直接竞争侦察、伙伴和子职启动。7级`元素之怒`在`强力施法`与`原力蛮击`之间选择，前者服务戏法远程回合，后者服务武器或野兽形态接触线，15级再强化。18级`兽形施法`和20级`大德鲁伊`把高等级变形、施法和资源转换放开；终局德鲁伊很难被长日拖垮，但仍要面对专注位有限、反制手段偏软和高环问题解决不如`祈愿术`体系宽的限制。

## 分项理由
- **治疗和临时生命**：德鲁伊法术表与多个子职都能提供治疗，荒野变形也能在非月亮路线中作为资源转换入口。
- **战场分隔 / 减益**：缠绕、地形、植物、天气、召唤和高环自然法术使德鲁伊很擅长改变战场形状。
- **召唤 / 伙伴**：`荒野伙伴`加上德鲁伊召唤法术使伙伴能力从低级就存在。
- **反制施法者**：基础德鲁伊通常没有`法术反制`一类奥术硬反制，更多依赖`解除魔法`、沉默/地形/视线控制和召唤压迫。

|---|---|---|

| Tier 2（5-10） | S- | 5-10级三环法术、召唤、植物/地形控制和`荒野复苏`让德鲁伊成为高位长日施法核心；它靠控制战场、治疗复原和资源转换赢，但缺少奥术硬反制与最宽泛的爆发解题，所以停在S-。 |
| Tier 3（11-16） | S- | 11-16级高环自然法术、召唤和区域控制覆盖面极宽，`元素之怒`强化常规回合；实际上限受专注竞争、召唤执行、地形/视线和敌人机动影响，不能默认按稳定S级遭遇主引擎处理。 |
| Tier 4（17-20） | S- | 17-20级九环自然法术、`兽形施法`和`大德鲁伊`让长日弹性和变形施法很强；但终局仍没有默认`祈愿术`、稳定硬反制或无视战场条件的全局解题权，综合停在S-。 |

## 设计相关评分
| 设计项 | 评分 | 具体理由 |
|---|---|---|
| 设计质量 | A | 2024版把荒野变形从额外生命池改成临时生命、熟悉形态和资源转换，旧版最失控的承伤问题被收窄；保留日备满施法、变形、魔宠和自然工具箱也让职业上手信息量仍高。 |
| 主题 | S | 自然祭司、变形者、治疗者、召唤者和原初施法者身份都很清楚，德鲁伊语与动物交流强调秘传传统，荒野变形和原初法术表强调人与自然边界的流动。 |
| 能力设计质量 | A | `荒野变形`、`荒野伙伴`和`荒野复苏`把法术位、短休次数、魔宠和形态资源接成同一套循环，边界比2014版清楚；扣分点是野兽资料、准备法术、专注和召唤执行仍会带来较高桌面管理成本。 |
| 主题与能力关联度 | S | 常用玩法就是用自然法术改变地形、召唤或治疗，再用荒野变形和伙伴解决侦察、位移或承伤问题；机制循环直接表达自然守护者与万物形态互通的主题。 |

## 校准说明
本轮横向比较后，德鲁伊维持高阶 S-：它有日备原初满施法、治疗、召唤、地形/天气控制、探索和荒野变形资源转换，但缺少默认祈愿术，硬反施法弱于法师，许多强项还受专注、地形、召唤执行和桌面管理影响。能进入 S 的结社必须实际提高召唤可靠性、团队防线、行动经济或战场控制兑现率；只提供个人耐久、主题工具或窄伤害线的结社不因基础职业强度自动上探。
