## 来源与版本
- 版本：`Druid`（XPHB, 2024-09-17）。
- 规则核对：`5etools-src/data/class/class-druid.json`、`5etools-cn/data/class/class-druid.json`、`DND5e_chm\玩家手册2024\角色职业\德鲁伊\德鲁伊.htm`。
- 本轮重评逐项核对了`德鲁伊语`、`原初职能`、`术师`、`卫士`、`荒野伙伴`、`荒野形态`、`荒野复苏`、`元素之怒`、`强力施法`、`原力蛮击`、`元素之怒精通`、`兽形施法`与`大德鲁伊`。公开讨论常把2024德鲁伊的争点集中在荒野形态临时生命取代旧生命池、熟悉形态数量、`荒野复苏`资源转换、强专注法术和召唤/地形执行成本；本评审把这些作为背景，最终仍按本地XPHB文本和本项目横向标尺独立评分。

## 队伍定位总结
德鲁伊是原初满施法核心，主要负责治疗与复原、召唤和伙伴压场、地形/天气/植物系区域控制、自然探索，以及把`荒野形态`在侦察、承压、魔宠和法术位之间转换。它能用日备法术适应环境，长日韧性很强；短板是默认缺少`法术反制`式硬反制、单体爆发不如专精奥术施法者，许多强项还吃专注、地形、召唤执行和敌人移动类型。

## 详细评价
1级`施法`给德鲁伊完整原初准备施法：治疗、解除、地形控制、召唤和自然探索都从法术表展开，长休换准备让它能按当天地区、敌人类型和旅行需求改装。`德鲁伊语`与始终准备`动物交谈`让动物、植物、荒野信息和自然社交成为稳定入口。`原初职能`把早期路线分成`术师`与`卫士`：前者强化戏法与自然/奥秘知识，后者给军用武器和中甲，决定的是站位与技能侧重，不改变“先用法术塑形战场”的核心。

2级`荒野形态`是附赠动作有限资源，短休恢复1次、长休回满。2024文本不再给野兽完整生命池，而是给等同德鲁伊等级的临时生命；变形时不能施法但可以维持专注，生命值、生命骰、心智属性、职业特性、语言和专长等关键角色数据保留，兽形数据主要接管身体与攻击/移动等面向。这让普通德鲁伊更常把变形用于侦察、移动、承压、穿越和子职资源，而不是长期当主坦。`荒野伙伴`可用法术位或`荒野形态`施放无材料的`寻获魔宠`，魔宠为妖精且在完成长休时消失，把侦察、帮助、送触及法术和小型占位接入职业资源。

5级`荒野复苏`把`荒野形态`和法术位互换：每回合一次、无需动作，若没有`荒野形态`次数可花法术位补一次，也可花一次`荒野形态`回一环法术位且每长休一次。这个转换提高长日弹性，但不是免费资源循环，因为它直接竞争侦察、伙伴和子职启动。7级`元素之怒`在`强力施法`与`原力蛮击`之间选择，前者给德鲁伊戏法加感知调整值伤害，后者让武器或野兽形态攻击每回合一次追加元素伤害，15级再分别强化射程或伤害骰。18级`兽形施法`允许在野兽形态中施法，但仍排除昂贵或消耗性材料成分；20级`大德鲁伊`让开战无次数时回1次`荒野形态`，并能把未用次数无动作转成法术位。终局德鲁伊很难被长日拖垮，但仍要面对专注位有限、反制手段偏软和高环问题解决不如`祈愿术`体系宽的限制。

本轮额外校准两个容易低估的伤害入口：`荆棘丛生`不是单纯减速地形，配合推拉、拖拽或堵门会把低环专注转成持续穿刺伤害；`咒唤林地卫士`按2024文本是自身10尺光环，移动时能让光环进入敌人空间并触发5d8力场豁免。前者更依赖队伍和地图，后者更依赖专注、移动和近身风险；它们提高德鲁伊的法术伤害评价，但不改变德鲁伊仍被专注竞争和硬反制短板限制的事实。

## 分项理由
- **治疗和临时生命**：德鲁伊法术表与多个子职都能提供治疗，`荒野形态`也能在非月亮路线中作为资源转换入口。
- **战场分隔 / 减益**：缠绕、地形、植物、天气、召唤和高环自然法术使德鲁伊很擅长改变战场形状。
- **召唤 / 伙伴**：`荒野伙伴`加上德鲁伊召唤法术使伙伴能力从低级就存在。
- **反制施法者**：基础德鲁伊通常没有`法术反制`一类奥术硬反制，更多依赖`解除魔法`、沉默/地形/视线控制和召唤压迫。

## 校准说明
本轮横向比较后，德鲁伊维持高阶 S-：它有日备原初满施法、治疗、召唤、地形/天气控制、探索和`荒野形态`资源转换，但缺少默认`祈愿术`，硬反施法弱于法师，许多强项还受专注、地形、召唤执行和桌面管理影响。能进入 S 的结社必须实际提高召唤可靠性、团队防线、行动经济或战场控制兑现率；只提供个人耐久、主题工具或窄伤害线的结社不因基础职业强度自动上探。

## 本轮补评横向校准

本轮补评的三个非第一方结社各自强在不同轴线：`猛兽结社（5E UA）`是最强的双体行动路线，14级`二体一心`让满施法动作和伙伴攻击同回合兑现，因此Tier 3可以到S；`溪泉结社（5E UA）`是恢复链条最完整的旧UA路线，`魔水`在5.5e附赠动作小瓶口径下低级很强，但高阶仍主要是容错和复原而不是主动控场；`落英结社（胧忆岛）`低中阶只是稳定攻守与短时兽灵，高阶凭无专注、免疫伤害、可指定通行的花墙进入S-。三者都不能因为德鲁伊满施法底盘直接上探S+：猛兽受旧野兽数据和伙伴生存限制，溪泉受治疗后置性限制，落英受每日一次与墙体裁定限制。
