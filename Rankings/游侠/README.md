# 游侠总评

## 队伍定位摘要
游侠在 2024/5R 环境里的客观位置是“强度可用、手感争议大”的半施法武器专家。它能远程输出、侦察、潜入、追踪、治疗补位、召唤/伙伴、地形控制和野外推进；但职业文本过度围绕 `猎人印记`，让集中和附赠动作长期拥挤。社区对 2024 游侠的批评主要集中在设计，而不是“完全不能打”。

## 详细评价
1级 `青睐敌人` 给免费 `猎人印记`，同时游侠有 `武器精通`、技能熟练、1环法术和中甲/盾牌。低级游侠很强，因为同级环境还没有高环法术压场，`猎人印记`、长弓、双武器、`治疗伤口`、`纠缠术`、`云雾术` 等工具足以覆盖很多场景。问题也从1级开始：`猎人印记`需要集中，转移目标要附赠动作，和很多更有影响力的法术互斥。

2级 `战斗风格` 让射手、双武器、德鲁伊战士路线成型。远程游侠通常是最稳选择，因为它更容易保集中；近战游侠能打，但生命、护甲和集中检定压力让它不像战士、野蛮人、圣武士那样适合长期挨打。`起源专长` 可以用 `魔法入门（法师）` 拿 `护盾术` 等补丁，但这不是游侠独有强点，不能过度抬分。

5级 `额外攻击` 和2环法术是 Tier 2 的核心。`行踪无迹` 会改写潜入和遭遇开启，`荆棘丛生` 是同等级极强的地形伤害与战场分隔，`召唤野兽` 或伙伴型子职能显著提高动作经济。这个阶段最客观的结论是：游侠好用，但最优打法经常不是无脑维持 `猎人印记`，而是在标记、地形控制、召唤、治疗和隐蔽之间选择。

Tier 3 的游侠仍可靠。更高环召唤、机动、隐蔽、感知和子职成长让它能继续做副输出、斥候、控场补位和自然专家。问题是高等级职业特征仍在给 `猎人印记`补丁：13级免受伤失去集中、17级命中优势、20级加感知修正到伤害。这些能力不是没用，而是来得太晚，且没有提供 `反制法术`、复活、传送、墙体、群体免疫或现实改写。

Tier 4 评分降到 C 是横向比较结果。游侠仍能完成冒险，尤其在技能、远程输出和召唤/伙伴上稳定；但 17-20 级的队伍问题通常由高环法术、强光环、传奇级防御或战役级工具解决，游侠很少能独自定义遭遇胜负条件。

| 阶段 | 综合强度 | 具体理由 |

## 分阶段子职排行榜

Tier 1（1-4）:
- A:
  - 妖精漫游者：`恐怖打击` 免费叠伤且不抢集中，社交能力让低级功能面明显变宽。
  - 幽域追猎者：先攻、首轮机动和暗处侦察让低级遭遇开启质量很高。
  - 集群牧者：攻击附带推位/移动让低级就能操纵战场。
  - 驯兽师：3级伙伴给额外身体、伤害和侦察，低级队伍价值明显高于基础游侠。
  - 龙兽守卫：3级龙兽伙伴给额外身体和元素主题，低级很有价值。
- B:
  - 凛冬行者：寒冷和地形主题明确，但默认战役中环境命中率不稳定。
  - 怪物杀手：识破怪物和单体标记有用，但动作经济与基础 `猎人印记` 冲突。
  - 猎人：早期武器伤害和猎物情报可用，但没有伙伴、先攻或社交扩展。
  - 边界行者：`位面战士` 有力场伤害但附赠动作成本高，传送门侦测场景化。

Tier 2（5-10）:
- A:
  - 妖精漫游者：2环法术、精神防线和社交/技能优势并行，队伍位置比猎人更完整。
  - 幽域追猎者：`额外攻击` 和暗处优势仍强，但爆发不再压过所有子职。
  - 集群牧者：`蛛网术`、`荆棘丛生` 与集群推位配合，控场质量很高。
  - 驯兽师：`额外攻击`、伙伴攻击和2环法术并行，能稳定输出、占位和控场。
  - 龙兽守卫：伙伴、抗性和基础游侠法术稳定，但效率略低于驯兽师。
- B:
  - 凛冬行者：减速、抗性和游侠2环法术能组成中档控场。
  - 怪物杀手：豁免补强和单体猎杀稳定，仍缺少大范围影响。
  - 猎人：基础游侠的 `额外攻击` 与2环法术很强，猎人只是小幅加厚。
  - 边界行者：`迷踪步`、`加速术` 等法术改善机动，但资源少且来得不早。

Tier 3（11-16）:
- A:
  - 妖精漫游者：召唤、传送和魅惑/恐惧互动继续成长，是高功能游侠路线。
  - 集群牧者：集群强化和机动防御继续支持位置玩法，是高功能游侠。
  - 驯兽师：伙伴成长与高环法术让它仍有多单位优势，是游侠内部最稳定路线。
  - 龙兽守卫：龙息、飞行和更强伙伴让主题与强度同时成型。
- B:
  - 凛冬行者：冰雪场地和减速能力更完整，但不突破半施法上限。
  - 幽域追猎者：输出可靠性和生存提高，缺少新的高等级队伍解题工具。
  - 怪物杀手：打断施法/传送让它有稀缺反制价值，但触发窄。
  - 边界行者：短距传送和多目标攻击终于成型，综合维持中上。
- C:
  - 猎人：高等级能力偏被动，难与功能型子职竞争。

Tier 4（17-20）:
- B:
  - 妖精漫游者：终局功能仍宽，但半施法上限无法和满施法者比较。
  - 幽域追猎者：终局仍是优秀侦察/先手游侠，但不能接管高魔法战场。
  - 集群牧者：终局仍好用，但半施法控场无法和高环墙/群控比较。
  - 驯兽师：终局仍强，但伙伴无法等同高环召唤军团或满施法者解题。
  - 龙兽守卫：终局伙伴仍强，但不具备满施法者级别的遭遇改写。
- C:
  - 凛冬行者：终局仍吃地形和伤害类型，综合下滑。
  - 怪物杀手：终局单体猎杀可用，综合解题面不足。
  - 猎人：终局仍能射、能追踪、能补法术，但子职增量太浅。
  - 边界行者：终局防御可用，但半施法位面主题不能和真正传送/位面施法者比较。

## UA/合作方/第三方子职分阶段排行榜

Tier 1（1-4）:
- A:
  - 侵蚀行者（胧忆岛）：`第一性状` 可用附赠动作自给标记，低级爆发高于普通游侠但和 `猎人印记` 抢附赠动作。
  - 噩兆先驱（歪曲之月）：`死兆星闪` 同时给额外伤害和噩兆伙伴，短休恢复且可用法术位补次数。
  - 幽邃戍卫（UA）：`荒野之怒` 低级就用 `猎人印记` 换感知加 AC 和近身动作压制，强度明显高于普通猎人。
  - 荒径守护者（克苏鲁）：`协同战术` 低级就提供团队反应位移和额外伤害，队伍战术价值高。
- B:
  - 天鹏旅人（狮鹫鞍包）：`指引之风` 和 `知识之语` 低级稳定但不爆炸，主要是轻伤害和可变熟练。

Tier 2（5-10）:
- A:
  - 侵蚀行者（胧忆岛）：`第三性状` 增加命中和反应减伤，输出/生存都比基础游侠更稳。
  - 噩兆先驱（歪曲之月）：`灾劫使者` 让暗蚀无视抗性，伙伴与本体集火更稳定。
  - 幽邃戍卫（UA）：`荆棘丛生`、感知加体质豁免和浴血回血让它成为硬度很高的压迫型游侠。
  - 荒径守护者（克苏鲁）：`行动无踪`、`灵体卫士` 与 `战术突袭` 让潜入、范围压制和防护同时成型。
- B:
  - 天鹏旅人（狮鹫鞍包）：`行云之风` 强化机动与环境适应，但战斗增量仍偏温和。

Tier 3（11-16）:
- S:
  - 荒径守护者（克苏鲁）：`协同奋战` 给队伍先攻加团队骰，配合 `灵体卫士` 和反应位移是顶级战术支持。
- A:
  - 侵蚀行者（胧忆岛）：`第四性状` 的病症选项给盲视、反应封锁或额外控制，功能面明显扩张。
  - 噩兆先驱（歪曲之月）：`宿命难逃` 能逼目标以劣势重骰成功豁免，是很强的队友控制放大器。
  - 天鹏旅人（狮鹫鞍包）：`天鹏之翼` 给短时飞行，配合远程游侠能显著改善站位和探索。
  - 幽邃戍卫（UA）：`枯朽残虐` 给无视抗性的灵光伤害和额外精通词条，控制与伤害都上升。

Tier 4（17-20）:
- S:
  - 荒径守护者（克苏鲁）：`群体围攻` 能把团队骰加到多名队友的命中和伤害，人数越多越接近越界团队增益。
- A:
  - 侵蚀行者（胧忆岛）：`第五性状` 的额外两次攻击和最大标记爆发很高，但二次使用副作用阻止它进 S。
  - 噩兆先驱（歪曲之月）：`宿命已定` 可制造单体易伤和更高伙伴伤害，但受传奇抗性与魅力豁免限制。
  - 天鹏旅人（狮鹫鞍包）：`飓风之护` 同时减伤、击倒、伤害和传送，是很强的短休反应防线。
  - 幽邃戍卫（UA）：`亘古不息` 提供归零续命和力竭免疫，但仍受 `猎人印记` 专注与半施法者资源限制。

## 设计相关评分
| 设计项 | 评分 | 具体理由 |
|---|---|---|
| 设计质量 | C | 2024 游侠强度不低，但设计把职业身份压在 `猎人印记` 上，和集中、附赠动作、召唤、控制法术长期冲突，读起来比实际表现更糟。 |
| 主题 | B | 荒野斥候、猎手、追踪者和自然半施法者主题仍成立，但 2024 文本削弱了探索身份，把“标记一个目标”放得过重。 |
| 能力设计质量 | C | `武器精通`、半施法、技能和伙伴/召唤都好用；问题是多个强循环互相踩资源，高等级职业特征没有提供足够终局回报。 |
| 主题与能力关联度 | C | 很多能力能解释成游侠能力，但 `猎人印记`、召唤、地形、潜行、技能和武器输出之间缺少一个清晰统一的机制主轴。 |

## 校准说明
本轮以 `DND5e_chm\玩家手册2024\角色职业\游侠` 为职业底盘，优先采用 2024 PHB 已覆盖的子职；旧版或第三方子职按本地索引来源保守接入。社区校准采用 RPGBOT 2024 游侠/子职指南、Dungeon Dudes 2024 游侠子职排名，以及 Reddit 对 `猎人印记`集中和附赠动作冲突的讨论。结论是：游侠实战不弱，但 2024 设计口碑差是有充分机制原因的。
