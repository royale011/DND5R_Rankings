## 队伍定位摘要
2024游侠是远程/近战武器、技能、探索、半施法和有限伙伴/召唤的混合职业。它的低中级队伍位置很实在：能打、能追踪、能潜行、能用自然法术补控制和探索，还能根据子职走伙伴、妖精社交、暗处伏击或战场推位。

它的问题不是“不能玩”，而是职业身份被`猎人印记`拉得太重。`猎人印记`能提供稳定单体伤害和追踪，但专注、附赠动作和可替代性让它很难承担整个职业的高等级设计核心。高阶游侠仍有5环半施法和技能价值，却缺少满施法者级别的反制、复活、跨场景法术和现实改写。

## 详细评价
1级`青睐敌人`、武器熟练、技能和1环法术让游侠开局很全面。远程武器、双持、盾牌路线和`武器精通`都能工作，`猎人印记`给了低级稳定单体伤害。限制从一开始也存在：标记吃专注和附赠动作，和`荆棘丛生`、召唤、治疗、伙伴指令或其他专注法术之间会互相排队。

Tier 2是游侠最舒服的阶段。`额外攻击`、2环法术、`行踪无迹`、`荆棘丛生`、治疗/复原、召唤或伙伴路线让游侠比普通武器职业更宽。此时它不是最高DPR，但能把侦察、伏击、地形和持续输出组合起来。问题是这些好工具往往不能和`猎人印记`同时维持，玩家必须在伤害、控制、潜行、召唤和支援之间选择。

Tier 3后，游侠的半施法和技能仍有用，子职11级也常给真实增量；但职业高等级开始把大量设计预算花在改善`猎人印记`。这会产生结构矛盾：如果标记被更强专注法术、专长或魔法物品替代，高等级职业特性就显得像补丁；如果坚持标记，又会牺牲高阶法术和战术选择。

Tier 4游侠依旧能做远程输出、追踪、斥候、自然法术补位和部分召唤/伙伴工作。横向看，它不该被压到纯弱武系；但它也不能像法师、牧师、德鲁伊、顶级圣武士或奇械师那样用高环法术、光环、道具生态或复活链定义终局遭遇。

## 分项理由
- **远程伤害者**：远程武器、`武器精通`、`额外攻击`和`猎人印记`构成最稳定输出路线；近战会承受更高集中和生存压力。
- **法术伤害者**：半施法进度和少量法术位限制爆发，法术更多用于控制、潜入、治疗和召唤，而不是当主炮。
- **召唤 / 伙伴**：Tier 2 后召唤法术和伙伴型子职使该项显著上升，但仍受集中、指令动作和伙伴耐久限制。
- **反制施法者**：基础游侠缺少`法术反制`和稳定反魔法工具，只能靠沉默、视线、击杀或个别子职间接处理施法者。
- **技能使用**：技能熟练、专精、探索法术和高感知让游侠长期是优秀斥候/探索者。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | A | `青睐敌人`、`武器精通`、1环法术、技能和早期子职让低级游侠很全面；限制是`猎人印记`从开局就占用专注与附赠动作，和其他强工具互斥。 |
| Tier 2（5-10） | A- | `额外攻击`、2环法术、潜行/地形工具、召唤或伙伴入口让游侠明显强于普通B档武系；但最佳回合经常在标记、控制、召唤、治疗和伙伴指令之间冲突，所以低于真正A档半施法核心。 |
| Tier 3（11-16） | B+ | 3-4环半施法、技能、探索和子职11级仍有真实贡献；但`猎人印记`的高阶补丁不能解决可替代性和专注竞争，也缺少反制、强墙体、复活链和战役级解题。 |
| Tier 4（17-20） | B | 终局游侠仍能远程输出、侦察和用5环半施法补位；但高阶职业特性没有把它推成遭遇主引擎，和满施法者、顶级灵光/道具职业相比只算可靠补位。 |
