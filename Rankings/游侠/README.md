# 游侠总评

## 队伍定位摘要
游侠是中高可用性、低设计口碑的半施法武器/探索职业。它擅长远程输出、斥候、潜入、野外推进、治疗补位、地形控制和伙伴/召唤，但不应被误判为满施法者式的遭遇定义者。

## 详细评价
2024 游侠的底盘不能再按“不能玩”处理。1级就有`法术施展`、`青睐敌人`、`武器精通`、技能熟练和中甲/盾牌，远程路线能在低级用长弓或双武器稳定输出，技能和自然法术也让它能承担斥候、追踪和探索职责。问题不是数值完全失败，而是职业身份被`猎人印记`过度绑定：它需要专注，转移目标通常要附赠动作，和`纠缠术`、`荆棘丛生`、召唤法术、部分子职伙伴指令或双武器循环长期冲突。

Tier 1 的游侠非常实用。`青睐敌人`给免费`猎人印记`，低级敌人生命不高时额外伤害很明显；`治疗伤口`、`云雾术`、`纠缠术`等法术让它不是纯射手。`起源专长`会抬高优化底线，例如`魔法学徒（法师）`给`护盾术`能帮助保集中，但这不是游侠独有优势，不能把职业本体直接抬成更高档。

Tier 2 是游侠最健康的阶段。5级`额外攻击`把武器输出稳定下来，2环法术中的`行踪无迹`和`荆棘丛生`分别能改写潜入与战场地形，召唤/伙伴路线也从此开始有真实动作经济。这个阶段最客观的判断是：游侠强，但最优回合经常不是维持`猎人印记`，而是根据战斗需要在标记、控场、召唤、治疗和潜入之间切换。

Tier 3 的游侠仍然可靠，但短板开始明显。13级以后`猎人印记`更难因受伤失去专注，17级和20级继续给标记补丁；这些能力有用，却来得晚，而且没有解决`猎人印记`可替代性问题。`妖精触碰`可取得符合学派限制的一环法术，如`猎人印记`或`脆弱诅咒`；部分物品或合作内容也能复制类似专注增伤。游侠的优势在于免费次数、职业后续支持和子职联动，而不是“只有游侠能做标记增伤”。

Tier 4 的评分下降是横向比较结果。游侠仍能射、能潜入、能追踪、能召唤/带伙伴和治疗补位，但同级法师、牧师、德鲁伊、吟游诗人、魔契师和顶级圣武士开始通过高环法术、强光环、复活、反制、传送和战役级工具定义遭遇。游侠缺少`法术反制`、高环墙体、群体免疫、复活链或现实改写，因此终局更多是可靠队员，而不是主要胜利引擎。

## 分项理由
- **远程伤害者**：远程武器、`武器精通`、`额外攻击`和`猎人印记`构成最稳定输出路线；近战会承受更高集中和生存压力。
- **法术伤害者**：半施法进度和少量法术位限制爆发，法术更多用于控制、潜入、治疗和召唤，而不是当主炮。
- **召唤 / 伙伴**：Tier 2 后召唤法术和伙伴型子职使该项显著上升，但仍受集中、指令动作和伙伴耐久限制。
- **反制施法者**：基础游侠缺少`法术反制`和稳定反魔法工具，只能靠沉默、视线、击杀或个别子职间接处理施法者。
- **技能使用**：技能熟练、专精、探索法术和高感知让游侠长期是优秀斥候/探索者。

| Tier 2（5-10） | B | `额外攻击`、`行踪无迹`、`荆棘丛生`、召唤/伙伴和技能专精都强，但最佳选择经常与`猎人印记`互斥，综合为可用但不顶级的B。 |
| Tier 3（11-16） | B | 13级后`猎人印记`可靠性上升，子职和召唤仍有价值；但缺少反制、高环解题和战役级工具，维持B而非A。 |

## 设计相关评分
| 设计项 | 评分 | 具体理由 |
|---|---|---|
| 设计质量 | C | 2024 游侠可玩但不优雅：底盘有武器、技能、半施法和伙伴/召唤，强度并不低；但职业身份被`猎人印记`集中与附赠动作绑定，导致很多强功能互相挤压。 |
| 主题 | B | 荒野猎手、斥候、追踪者和自然半施法者主题清楚，问题是“标记一个目标”被放得过重，压扁了不同游侠风格。 |

| 主题与能力关联度 | C | 大多数能力能解释成游侠能力，但标记、召唤、地形、潜行和武器输出之间缺少一个足够统一且不互相踩资源的机制主轴。 |

## 校准说明
本轮按 `DND5e_chm\不全书.wcp` 的游侠条目核对范围：2024 PHB 的游侠、驯兽师、妖精漫游者、幽域追猎者、猎人；XGE/TCE/FTD/FRHoF 兼容子职；`其他\新UA\可怖子职\幽邃戍卫.htm`；以及合作/第三方的狮鹫鞍包、胧忆岛、歪曲之月、火炬光下的克苏鲁条目。社区校准参考了 RPGBOT 2024 游侠职业/子职指南、Dungeon Dudes 2024 游侠子职排名，以及围绕 2024 `猎人印记`集中和附赠动作冲突的讨论；评分仍以本地文本、动作经济、集中、资源和同项目横向比较为准。
