# 猎人

所属职业：游侠
英文名 / 社区检索名：Hunter
来源 / 版本：玩家手册2024

## 队伍定位调整
猎人是最朴素的游侠子职：加厚武器输出、防御和对猎物信息的读取。它容易上手，但缺少改变队伍位置的能力，是“可用但保守”的典型。

## 详细评价
3级 `猎物` 系列能力给额外伤害或对受伤/多目标敌人的效率，`猎人学识` 通过 `猎人印记`读取抗性、免疫和易伤信息。这个情报能力很有猎人味，但实际问题是游侠往往主要造成穿刺或武器伤害，知道抗性后不一定有手段绕开。它也进一步把猎人推向“维护 `猎人印记`”的玩法，而标记类增伤在 2024 环境并不稀有：`妖精触碰` 可拿 `猎人印记` 或 `脆弱诅咒`，部分魔法物品也能提供同类效果，所以猎人的独特性不能只按“有标记”来算。

猎人的低维护是优点。你不需要管理伙伴、恐惧转移、妖精召唤或暗处规则，只要把游侠底盘的武器、`猎人印记`、`武器精通`、`行踪无迹`、`荆棘丛生`用好即可。很多收益是被动或随攻击触发，不会额外抢附赠动作、反应或专注；这使它比文本看起来更顺手。对新玩家友好，也不容易玩错。

问题是它的选择经常只是小数值。防御战术能补生存，多重攻击能在敌人站位合适时赚效率，高等级防御让长期冒险更稳；但这些能力很少让队伍因为你是猎人而改变战术。相比驯兽师的多单位、妖精漫游者的社交/精神线、幽域追猎者的先手，猎人缺少鲜明队伍位置。

因此猎人综合给 B/C 曲线：早期好用，Tier 2 还能跟住基础游侠强度，Tier 3-4 因高等级增量偏浅而下滑。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | B | 早期武器伤害和猎物情报可用，且多数收益不额外抢动作；但没有伙伴、先攻或社交扩展。 |
| Tier 2（5-10） | B | 基础游侠的 `额外攻击` 与2环法术很强，猎人只是小幅加厚。 |
| Tier 3（11-16） | C | 高等级能力偏被动，难与功能型子职竞争。 |
| Tier 4（17-20） | C | 终局仍能射、能追踪、能补法术，但子职增量太浅。 |

## 校准说明
RPGBOT 对 2024 猎人的评价偏低，认为很多选择收益小且像“选择的幻觉”。本轮保留可用性，但综合低于 PHB 其他强子职。
