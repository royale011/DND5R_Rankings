# 牧师总评

## 队伍定位摘要
牧师是 2024/5R 中最强的队伍支援与神术解题职业之一。它不是“只能治疗”的职业：`祝福术`、`灵体卫士`、解除/复原、复活、召唤、预言、群体治疗和高环神术让它同时承担治疗、增益、减益、范围伤害、防御和信息职责。领域决定风格，但牧师底盘已经足够强。

## 详细评价
1级 `神圣秩序` 是 2024 牧师的重要改动。`保护者` 让任何领域都能拿重甲和武器路线，`奇术师` 则强化戏法和宗教/奥秘知识。这解决了旧版“领域绑定武装熟练”的一部分问题，也让生命、光明、诡术、战争以外的旧领域接入时更容易统一评价。

低级牧师强在 `祝福术`、治疗、控制和防御选择。`祝福术` 在 Tier 1 对命中和豁免的影响巨大，`治疗真言` 或类似远程救场能把倒地队友拉回行动经济，`命令术`、`曳光弹`、`庇护术` 等法术都能改变回合。牧师低级不是最高法术伤害者，但综合已经是 A。

Tier 2 是牧师进入顶级的阶段。`灵体卫士` 提供移动范围压制、持续伤害和近战阵地控制；`复活术`、`解除魔法`、`防死结界`、`群体治疗真言`、`昼明术` 等法术让牧师同时处理战斗和灾难后果。社区评价牧师时常说“牧师不只是治疗者”，核心就在这里。

Tier 3-4 牧师继续强。高环神术提供召唤、防护、信息、群体恢复和 `神圣干预` 式问题解决。按本地 PHB2024 文本，10级 `神圣干预`用一个魔法动作施展五环或更低、施法时间非反应的牧师法术，且无需法术位和材料；20级 `进阶神圣干预`可选择 `祈愿术`。这让 `治疗祷言`、`圣居`、`守卫刻纹` 等长施法/材料限制法术出现动作压缩讨论，但这类收益必须标注桌裁敏感性，并保留法术本身的目标、持续、触发、位置和后续限制。它在反制施法者上不如法师，因为缺少通用 `法术反制`；在制造上也不是奇械师。但在治疗、支援、减益、神术解题和队伍韧性上，它长期处于全职业顶档。

需要诚实指出：部分领域越界。和平领域和暮光领域不是“比其他牧师强一点”，而是用低成本、低互动、广覆盖能力压低失败率或伤害压力，足以评为 S+。这类 S+ 是全生态评价，不只是牧师内部第一名。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | A | 1级满施法、`神圣秩序`、治疗与 `祝福术` 让牧师低级已经是队伍核心；短板是早期法术位少，防御/伤害定位依赖选择。 |
| Tier 2（5-10） | S | 3环法术后 `灵体卫士`、`复活术`、`解除魔法`、`群体治疗真言`、`防死结界` 等工具把牧师推到队伍支援和控场顶级。 |
| Tier 3（11-16） | S | 6-8环神术、`神圣干预`和更强治疗/召唤/防护让牧师继续承担队伍问题解决核心；动作压缩型用法很强，但按桌裁敏感项处理，不自动抬高所有领域到 S+。 |
| Tier 4（17-20） | S | 9环法术、终局领域能力和`进阶神圣干预`让牧师仍是全职业顶级支援/治疗/神术解题者；`祈愿术`入口提高上限，但2d4长休冷却和具体复制法术限制仍要计入。 |

## 分阶段子职排行榜

Tier 1（1-4）:
- S:
  - 和平领域：`鼓舞纽带` 可与 `祝福术` 叠加且不需要专注，Tier 1 就系统性抬高命中、检定和豁免。
  - 暮光领域：`夜之眼`、`警觉祝福` 和早期 `暮光庇护` 已经显著提高整队先手、暗视和伤害缓冲。
- A:
  - 光明领域：`黎明曙光` 低级清场和 `守御之光` 防御反应都很实用，但法术位少时仍需在治疗、增益和爆破间取舍。
  - 坟墓领域：`生死轮回` 让低级倒地救援更可靠，`坟墓之眼` 有侦测价值，但主动战斗增量等到引导神力后才完整。
  - 奥秘领域：法师戏法和奥秘技能扩大低级功能，但 `奥秘阻绝` 目标类型窄，战斗爆点还未成型。
  - 战争领域：`战争祭司` 在低级给真实额外攻击，`导引打击` 能保住关键命中，但法术位和引导神力仍有限。
  - 死亡领域：低级死灵戏法扩散和 `死亡之触` 让进攻强于普通牧师，但支援增量少。
  - 生命领域：`生命门徒` 和固定准备的 `祝福术`/治疗法术让 Tier 1 容错极高，但战斗主动权仍主要来自牧师通用法术。
  - 知识领域：技能和信息能力让低级调查/宗教/奥秘场景很强，战斗则主要靠牧师通用法术。
  - 秩序领域：`权威之声` 低级就能把治疗/增益转成队友反应攻击，若队伍有高价值武器攻击者会非常赚。
  - 自然领域：3级取得的德鲁伊戏法和自然技能让低级适应面不错，但重甲不计为子职增量，`魅惑动物和植物` 目标类型仍过窄。
  - 诡术领域：`诡术祝福` 和 `召现分身` 提供低级潜入与安全施法角度，但爆发和治疗仍靠牧师通用资源。
  - 锻造领域：3级 `锻造祝福` 能把武器或护甲变成 +1，低级命中/AC收益实际；重甲入口不计为子职收益，制造能力提供战斗外价值。
  - 风暴领域：3级取得的 `风暴狂怒` 和早期雷鸣法术让低级反击/爆发强；重甲和军武不计入子职强度，反应触发也偏被动。

Tier 2（5-10）:
- S+:
  - 和平领域：`鼓舞纽带`、`和平慰藉`、`守护纽带` 与3环神术叠加，队伍失败率和集火风险被明显压低。
  - 暮光领域：`暮光庇护` 每轮刷新临时生命且不需要专注，可与 `灵体卫士` 同时运作，防线明显越界。
- S:
  - 光明领域：`灵体卫士` 与领域爆破/反黑暗能力叠加，让光明牧师能同时清场、保护自己并维持牧师支援职责。
  - 坟墓领域：`往墓之途` 易伤窗口和 `死门哨卫` 反重击同时在线，配合3环神术形成强爆发/防御支援。
  - 奥秘领域：3环神术托底，6级治疗驱散能把救场和解除法术压制合并，反魔法价值明显。
  - 战争领域：无专注 `虔诚护盾`/`灵体武器`、`导引打击` 和3环神术叠加，使战争牧师成为强前线支援而非普通武器职业。
  - 生命领域：`维持生命`、`神祝医者`、`援助术` 和3环神术让它同时承担控场后的群体止血、倒地救援和状态复原。
  - 秩序领域：`灵体卫士` 底盘叠加 `权威之声` 与 `秩序要求`，让牧师同时提供控场、支援和队友动作经济。
  - 诡术领域：`灵体卫士`、幻象源点施法和 `诡诈换位` 形成强机动支援，能用站位优势保护专注并扩大法术覆盖。
  - 锻造领域：`锻炉之魂`、高 AC 和 `灵体卫士` 让它成为很强的前线牧师，能稳定维持阵地。
  - 风暴领域：`毁灭狂怒` 最大化雷鸣/闪电伤害与3环神术同期，清场和爆发都达到官方领域高档，但仍受引导神力和伤害类型约束。
- A:
  - 死亡领域：`灵体卫士` 托底，暗蚀爆发仍可用，但领域机制更多服务个人伤害而非队伍防错。
  - 知识领域：3环神术保证战斗强度，知识领域主要补侦测、识别和战斗外解题，不直接放大队伍动作经济。
  - 自然领域：`灵体卫士` 托底，`抑制元素` 能保护队友吃元素伤害，但领域主动控制仍偏场景化。

Tier 3（11-16）:
- S+:
  - 和平领域：高环神术本已强，和平领域继续用传送分伤和加骰覆盖关键失败，强度超过正常官方领域边界。
  - 暮光领域：高环神术、飞行和持续临时生命叠加后，多数伤害型遭遇很难真正压垮队伍。
- S:
  - 光明领域：高环神术托底，`守御之光` 升级后能保护队友，光明领域仍是强范围伤害与防御混合体。
  - 坟墓领域：高等级敌人的重击和关键豁免更危险，坟墓领域仍能用反重击与易伤窗口改变首领战关键轮。
  - 奥秘领域：高环神术加治疗驱散和异界阻绝，使奥秘领域在魔法/异界战役中问题解决面很宽。
  - 战争领域：高环神术托底，`导引打击` 继续保护队伍高价值攻击，战争领域的反应资源仍有决定性意义。
  - 生命领域：中高环治疗与复原法术被 `生命门徒` 放大，生命领域在连续战斗和灾难恢复中明显优于普通牧师。
  - 秩序领域：高环支援法术继续触发队友攻击，秩序领域的价值随队友单次攻击质量提高而上升。
  - 诡术领域：高环神术叠加幻象战术后，诡术领域在潜入、救援、错位施法和战场机动上仍有独特价值。
  - 锻造领域：火焰抗性、重甲 AC 加值和高环神术叠加，使锻造领域在防御/制造/前线站场上进入高档。
- A:
  - 死亡领域：忽略暗蚀抗性和高环神术保持可用性，但无法补足反制、团队增益或广谱控制。
  - 知识领域：高环神术与信息工具结合，在调查和政治战役价值高，但纯战斗影响低于顶级支援领域。
  - 自然领域：高环神术让牧师仍强，自然领域本身主要提供元素减伤和荒野主题，不足以进入顶级支援。
  - 风暴领域：高环神术仍强，但风暴领域后续增量依赖特定伤害类型和推离目标，泛用性低于顶级支援领域。

Tier 4（17-20）:
- S+:
  - 和平领域：`扩展纽带` 加范围和抗性后成为终局全队防错系统，属于全生态顶级团队防护/增益选项。
  - 暮光领域：`暮光帷幕` 再给庇护范围半掩护，终局仍是全生态最强防御领域之一。
- S:
  - 光明领域：`光冕` 对火焰/光耀法术和 `黎明曙光` 施加豁免劣势，终局仍有明确的范围压制价值。
  - 坟墓领域：`引魂明灯` 补足击杀续航，高环牧师法术再叠加易伤爆发，终局仍是强官方支援领域。
  - 奥秘领域：17级从法师列表取得6-9环法术，终局解题能力进入官方牧师领域顶档。
  - 战争领域：`战争化身` 加强物理生存，高环牧师法术与命中保障让它终局仍能承担前线支援核心。
  - 生命领域：`极效治疗` 让高环治疗与引导神力稳定取满，是官方领域里最可靠的治疗终局。
  - 秩序领域：`秩序之怒` 继续把自身命中转化为队友伤害，高环牧师支援让它终局仍是顶级团队增幅领域。
  - 诡术领域：`精通分身` 提升终局幻象价值，满施法牧师再加安全施法角度，综合仍在官方强领域行列。
- A:
  - 死亡领域：终局死灵扩散有主题价值，综合仍主要靠牧师高环法术而非领域本身。
  - 知识领域：终局仍是强施法者；知识领域的额外贡献更偏信息优势，而不是和平/暮光式遭遇数学压制。
  - 自然领域：`自然之主` 仍受动物/植物目标限制，终局综合主要靠牧师法术而不是领域特性。
  - 锻造领域：17级火焰免疫和物理抗性很硬，但它主要保护自己，团队影响低于和平、暮光、生命、秩序等顶级支援领域。
  - 风暴领域：终局飞行和抗压有用，但领域本身没有和平、暮光、奥秘那种改变高等级问题解决面的工具。

## UA/合作方/第三方子职分阶段排行榜

Tier 1（1-4）:
- A:
  - 丰收领域（歪曲之月）：低级牧师底盘强，子职早期特性提供明确主题增量；群体英雄激励和反应重掷治疗非常强，但受长休/阶段和引导神力约束。
  - 力量领域（阿芒凯）：低级牧师底盘强，子职早期特性提供明确主题增量；力量专精场景强，泛用性低于战争/秩序。
  - 团结领域（阿芒凯）：低级牧师底盘强，子职早期特性提供明确主题增量；治疗、命中保障和终局治疗最大化让它明显高于普通旧版武器领域。
  - 团队领域（谦卑林）：低级有食物治疗、恐慌/中毒处理和牧师底盘，支援价值明确。
  - 坟墓领域（UA）：浴血额外暗蚀和濒死治疗取满让低级更稳定，但核心减益资源仍有限。
  - 天启领域（克苏鲁）：低级牧师底盘强，子职早期特性提供明确主题增量；重击窗口与豁免劣势强，但队伍构成依赖明显。
  - 奥秘领域（UA）：奥秘专精和 `修饰魔法` 低级有用，但引导神力次数少，主要仍靠牧师基本法术。
  - 审判领域（鬼魅幽谷）：低级牧师底盘强，子职早期特性提供明确主题增量；对法术环境高度强势，但专精面窄。
  - 宴饮领域（狮鹫鞍包）：低级牧师底盘强，子职早期特性提供明确主题增量；治疗道具和抗性稳定，但中毒/毒素常遇免疫。
  - 暗影领域（黯潮之书）：低级牧师底盘强，子职早期特性提供明确主题增量；潜行与束缚实用，但不是持续全队强支援。
  - 月亮领域（塔尔多雷）：低级牧师底盘强，子职早期特性提供明确主题增量；双专注限定在领域法术内，强但不等于任意双专注。
  - 热忱领域（阿芒凯）：低级牧师底盘强，子职早期特性提供明确主题增量；中级爆发强，但伤害类型和领域法术限制使其后期不稳。
  - 监管领域（黯潮之书）：低级牧师底盘强，子职早期特性提供明确主题增量；临时生命加反应攻击的动作经济很强，17级濒死半血救援也很硬。
  - 知识领域（阿芒凯）：低级牧师底盘强，子职早期特性提供明确主题增量；本质是知识领域复用，不额外上浮。
  - 野心领域（阿芒凯）：低级牧师底盘强，子职早期特性提供明确主题增量；分身施法强，但不是2024诡术的完整升级。
  - 鲜血领域（塔尔多雷）：低级牧师底盘强，子职早期特性提供明确主题增量；生命换法术位和操血控制很强，但目标限制与自伤阻止其进入 S+。
  - 黑夜领域（谦卑林）：早期防御反应和范围目盲很强，但依赖光照与目盲有效性。

Tier 2（5-10）:
