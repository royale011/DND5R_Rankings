## 来源与版本

- 版本：`Cleric`（XPHB，2024）。
- 规则核对：`5etools-src/data/class/class-cleric.json`、`5etools-cn/data/class/class-cleric.json`、`DND5e_chm\玩家手册2024\角色职业\牧师\牧师.htm`。
- 本轮重评重点核对了`圣职`、`保护者`、`奇术使`、`引导神力`、`神圣火花`、`驱散亡灵`、`灼净亡灵`、`受祝击`、`神圣干预`与`进阶神圣干预`。公开讨论通常集中在新版`神圣干预`动作压缩、20级`祈愿术`接口、`灵体卫士`仍然是核心强度锚点，以及和平/暮光等强领域是否越界；以下结论以本地XPHB文本和已收录领域横向比较为准。

## 队伍定位摘要
牧师是2024/5R里最可靠的队伍韧性与神术解题职业之一。它的核心不是“每回合治疗”，而是用准备施法、`祝福术`、`灵体卫士`、复原/复活链、领域特性和`神圣干预`把失败局面拉回可控范围。它缺少法师式通用反制和奇械师式制造底盘，但在治疗、团队增益、减益、持续阵地、神术支援和高阶救场上长期处于全职业顶层。

## 详细评价
牧师的新版底盘不是单纯治疗者，而是“准备施法 + 队伍韧性 + 神术问题解决”的组合。`圣职`把武装路线和施法路线从领域中拆出，`保护者`取得军用武器与重甲训练，`奇术使`取得额外戏法并强化宗教、奥秘检定；这使旧版领域接入2024底盘时不再因为重甲或军用武器熟练反复加分。`引导神力`现在明确承载`神圣火花`和`驱散亡灵`，前者是30尺内治疗或光耀/黯蚀伤害的小型即时按钮，后者配合5级`灼净亡灵`给反不死场景更稳定的群体压制与伤害。

Tier 1 的牧师强在低级回合经济：`祝福术`能同时提高命中和豁免，远程治疗能把倒地队友拉回行动序列，`命令术`、`曳光弹`、`庇护术`、`疗伤术`等法术分别处理控制、命中、保护和救场。这个阶段的短板是法术位少，牧师必须在专注增益、救人和进攻之间作选择。

Tier 2 是牧师真正进入顶级的阶段。`灵体卫士`把移动、站位、持续伤害和减速合在一起，`复活术`、`解除魔法`、`次级复原术`、`防死结界`、`群体治疗真言`让队伍能从坏局面里恢复。这里还要注意 TCE 扩展法术：`魂灵环绕`属于牧师可用资源，若领域或兼职提供多段命中接口，例如光明领域的`灼热射线`，它会抬高近距离法术DPR；但它需要专注、10尺距离和启动动作，不能当作无风险常驻输出。

Tier 3-4 的牧师强在高环神术、复原链和`神圣干预`。按本地XPHB文本，10级`神圣干预`用一个魔法动作选择一个5环或更低、施法时间不是反应的牧师法术，并作为该动作的一部分施展，且无需法术位和材料；20级`进阶神圣干预`可在使用`神圣干预`时选择`祈愿术`，但如此使用后需等2d4次长休才能再次使用`神圣干预`。这会引出`治疗祷言`、`圣居`、`守卫刻纹`等长施法或材料法术的动作压缩讨论，也让20级牧师获得终局级复制法术接口；但仍要保留目标、持续、触发、位置、法术文本、`祈愿术`复制限制与桌裁限制。牧师不如法师那样有通用`法术反制`生态，也不等同奇械师的制造底盘；它的顶级位置来自治疗、增益、复原、神术解题和队伍抗压的总和。

领域之间差异很大。和平领域和暮光领域的EX来自非专注或低互动的全队数学扭曲；秩序、奥秘、诡术、光明等领域则分别用动作经济、跨表法术、施法角度或爆破曲线提高上限。较窄领域不能因为牧师本体强就自动进入S，必须说明领域增量到底解决了什么问题。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | A | 低级牧师已经有`祝福术`、远程救场、`圣职`、`神圣火花`和准备施法，能同时支援命中、豁免和倒地队友；限制是法术位少，进攻、治疗和专注增益不能全部同时满负荷。 |
| Tier 2（5-10） | S | 3环法术后`灵体卫士`、`复活术`、`解除魔法`、`群体治疗真言`、`防死结界`让牧师成为队伍支援与阵地核心；`魂灵环绕`等扩展法术也给部分领域近距离DPR路线，但专注和距离限制仍真实存在。 |
| Tier 3（11-16） | S | 6-8环神术、复原链、召唤/防护工具与10级`神圣干预`让牧师能处理战斗后果和部分长施法动作压缩；它强在综合解题，不因动作压缩讨论自动把所有领域抬成S+。 |
| Tier 4（17-20） | S | 9环神术和`进阶神圣干预`的`祈愿术`入口把牧师留在终局顶层；2d4长休冷却、复制法术限制、桌裁敏感点和缺少常规`法术反制`使职业整体停在S而非S+。 |

| T1 | A | 风暴领域 | `风暴狂怒`和雷鸣/闪电法术让低级反击与爆发强；反应触发偏被动，旧武装/重甲熟练不重复计分。 |

| T3 | S- | 战争领域 | 高环神术负责解题，领域继续保护偷袭、惩击、战士爆发或关键法术攻击；缺口是没有通用反制或全队防错。 |

| T3 | S- | 锻造领域 | 高环神术叠加制造/装备法术后仍很稳，`火墙术`、`活化物件`和`造物术`扩大工具面；缺少主动S级团队增益或反制生态。 |
| T3 | A+ | 死亡领域 | 高环牧师本体抬高综合下限，忽略暗蚀抗性也提高可靠性；但领域没有补出反制、团队防错、跨表法术或广谱控制短板。 |
| T3 | A+ | 自然领域 | 高环神术让成品牧师强，`抑制元素`在元素战役继续有用；但领域没有提供反制、强控或队伍级防错。 |
| T3 | A+ | 风暴领域 | 高环牧师仍强，但领域后续主要是推离和特定伤害增幅，泛用性低于顶级支援/控制领域。 |
