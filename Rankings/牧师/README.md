## 队伍定位摘要
牧师是2024/5R里最可靠的队伍韧性与神术解题职业之一。它的核心不是“每回合治疗”，而是用准备施法、`祝福术`、`灵体卫士`、复原/复活链、领域特性和`神圣干预`把失败局面拉回可控范围。它缺少法师式通用反制和奇械师式制造底盘，但在治疗、团队增益、减益、持续阵地、神术支援和高阶救场上长期处于全职业顶层。

## 详细评价
牧师的新版底盘不是单纯治疗者，而是“准备施法 + 队伍韧性 + 神术问题解决”的组合。`神圣秩序`把武装路线和施法路线从领域中拆出，使旧版领域接入2024底盘时不再因为重甲或军用武器熟练反复加分；`引导神力`、`驱散不死生物`、`受祝击`、复原/复活法术和准备施法共同构成了职业核心。

Tier 1 的牧师强在低级回合经济：`祝福术`能同时提高命中和豁免，远程治疗能把倒地队友拉回行动序列，`命令术`、`曳光弹`、`庇护术`、`疗伤术`等法术分别处理控制、命中、保护和救场。这个阶段的短板是法术位少，牧师必须在专注增益、救人和进攻之间作选择。

Tier 2 是牧师真正进入顶级的阶段。`灵体卫士`把移动、站位、持续伤害和减速合在一起，`复活术`、`解除魔法`、`次级复原术`、`防死结界`、`群体治疗真言`让队伍能从坏局面里恢复。这里还要注意 TCE 扩展法术：`魂灵环绕`属于牧师可用资源，若领域或兼职提供多段命中接口，例如光明领域的`灼热射线`，它会抬高近距离法术DPR；但它需要专注、10尺距离和启动动作，不能当作无风险常驻输出。

Tier 3-4 的牧师强在高环神术、复原链和`神圣干预`。按本地PHB2024文本，10级`神圣干预`用一个魔法动作施展五环或更低、施法时间非反应的牧师法术，且无需法术位和材料；20级`进阶神圣干预`可选择`祈愿术`。这会引出`治疗祷言`、`圣居`、`守卫刻纹`等长施法或材料法术的动作压缩讨论，但仍要保留目标、持续、触发、位置、法术文本和桌裁限制。牧师不如法师那样有通用`法术反制`生态，也不等同奇械师的制造底盘；它的顶级位置来自治疗、增益、复原、神术解题和队伍抗压的总和。

领域之间差异很大。和平领域和暮光领域的S+来自非专注或低互动的全队数学扭曲；秩序、奥秘、诡术、光明等领域则分别用动作经济、跨表法术、施法角度或爆破曲线提高上限。较窄领域不能因为牧师本体强就自动进入S，必须说明领域增量到底解决了什么问题。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | A | 低级牧师已经有`祝福术`、远程救场、`神圣秩序`和准备施法，能同时支援命中、豁免和倒地队友；限制是法术位少，进攻、治疗和专注增益不能全部同时满负荷。 |
| Tier 2（5-10） | S | 3环法术后`灵体卫士`、`复活术`、`解除魔法`、`群体治疗真言`、`防死结界`让牧师成为队伍支援与阵地核心；`魂灵环绕`等扩展法术也给部分领域近距离DPR路线，但专注和距离限制仍真实存在。 |
| Tier 3（11-16） | S | 6-8环神术、复原链、召唤/防护工具与10级`神圣干预`让牧师能处理战斗后果和部分长施法动作压缩；它强在综合解题，不因动作压缩讨论自动把所有领域抬成S+。 |
| Tier 4（17-20） | S | 9环神术和`进阶神圣干预`的`祈愿术`入口把牧师留在终局顶层；2d4长休冷却、复制法术限制、桌裁敏感点和缺少常规`法术反制`使职业整体停在S而非S+。 |

## 设计相关评分
| 设计项 | 评分 | 具体理由 |
|---|---|---|
| 设计质量 | A- | 2024牧师底盘的设计方向比旧版更干净：`神圣秩序`把武装/施法路线从领域中拆出，领域选择不再被重甲和武器熟练绑架，`引导神力`与满施法框架也能稳定承载治疗、反不死和神圣干预。扣分点是牧师通用法术表过强，`灵体卫士`等核心法术会压缩领域差异，让部分领域实际玩起来像同一个强底盘的不同皮肤。 |
| 主题 | A | 神术代理人、信仰领域、治疗/祝福、驱散不死、神圣干预和领域法术共同构成非常清楚的职业身份；牧师能覆盖战斗祭司、学者、治疗者、审判者和邪神仆从等多种神职幻想。 |
| 能力设计质量 | B+ | 底盘资源、准备施法和领域入口结构扎实，2024兼容性也比旧版好；主要问题是强法术和少数强领域会让设计生态失衡，而弱领域很难靠子职特色盖过通用牧师最优回合。 |
| 主题与能力关联度 | A | 绝大多数职业能力都把信仰、神恩和领域权能转成实际桌面选择；少数问题来自通用强法术稀释领域差异，而不是职业主题本身漂移。 |
