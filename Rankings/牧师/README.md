# 牧师总评

## 队伍定位摘要
牧师是 2024/5R 中最强的队伍支援与神术解题职业之一。它不是“只能治疗”的职业：`祝福术`、`灵体卫士`、解除/复原、复活、召唤、预言、群体治疗和高环神术让它同时承担治疗、增益、减益、范围伤害、防御和信息职责。领域决定风格，但牧师底盘已经足够强。

## 详细评价
1级 `神圣秩序` 是 2024 牧师的重要改动。`保护者` 让任何领域都能拿重甲和武器路线，`奇术师` 则强化戏法和宗教/奥秘知识。这解决了旧版“领域绑定武装熟练”的一部分问题，也让生命、光明、诡术、战争以外的旧领域接入时更容易统一评价。

低级牧师强在 `祝福术`、治疗、控制和防御选择。`祝福术` 在 Tier 1 对命中和豁免的影响巨大，`治疗真言` 或类似远程救场能把倒地队友拉回行动经济，`命令术`、`曳光弹`、`庇护术` 等法术都能改变回合。牧师低级不是最高法术伤害者，但综合已经是 A。

Tier 2 是牧师进入顶级的阶段。`灵体卫士` 提供移动范围压制、持续伤害和近战阵地控制；`复活术`、`解除魔法`、`防死结界`、`群体治疗真言`、`昼明术` 等法术让牧师同时处理战斗和灾难后果。社区评价牧师时常说“牧师不只是治疗者”，核心就在这里。

Tier 3-4 牧师继续强。高环神术提供召唤、防护、信息、群体恢复和 `神圣干预` 式问题解决。按本地 PHB2024 文本，10级 `神圣干预`用一个魔法动作施展五环或更低、施法时间非反应的牧师法术，且无需法术位和材料；20级 `进阶神圣干预`可选择 `祈愿术`。这让 `治疗祷言`、`圣居`、`守卫刻纹` 等长施法/材料限制法术出现动作压缩讨论，但这类收益必须标注桌裁敏感性，并保留法术本身的目标、持续、触发、位置和后续限制。它在反制施法者上不如法师，因为缺少通用 `法术反制`；在制造上也不是奇械师。但在治疗、支援、减益、神术解题和队伍韧性上，它长期处于全职业顶档。

需要诚实指出：部分领域越界。和平领域和暮光领域不是“比其他牧师强一点”，而是用低成本、低互动、广覆盖能力压低失败率或伤害压力，足以评为 S+。这类 S+ 是全生态评价，不只是牧师内部第一名。

## 综合评分
| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | A | 1级满施法、`神圣秩序`、治疗与 `祝福术` 让牧师低级已经是队伍核心；短板是早期法术位少，防御/伤害定位依赖选择。 |
| Tier 2（5-10） | S | 3环法术后 `灵体卫士`、`复活术`、`解除魔法`、`群体治疗真言`、`防死结界` 等工具把牧师推到队伍支援和控场顶级。 |
| Tier 3（11-16） | S | 6-8环神术、`神圣干预`和更强治疗/召唤/防护让牧师继续承担队伍问题解决核心；动作压缩型用法很强，但按桌裁敏感项处理，不自动抬高所有领域到 S+。 |
| Tier 4（17-20） | S | 9环法术、终局领域能力和`进阶神圣干预`让牧师仍是全职业顶级支援/治疗/神术解题者；`祈愿术`入口提高上限，但2d4长休冷却和具体复制法术限制仍要计入。 |

| T3 | A+ | 风暴领域 | 高环牧师底盘仍强，但领域后续主要是推离和特定伤害增幅，泛用性低于顶级支援/控制领域。 |
