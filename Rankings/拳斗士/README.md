## 队伍定位总结

拳斗士是强韧、短休资源型、贴身压迫的近战职业。它用 d10 生命骰、体质 AC、附赠动作徒手攻击、锐气、浴血反弹和无甲/轻甲前线循环，把自己放在危险距离里持续换血；3 级后命中徒手打击同时造成伤害并追加擒抱或推撞，形成比普通武器攻击更强的地面单体控制入口。

它的强项很清楚：低级多一次附赠攻击，中级最大化命中伤害、力场伤害、每轮额外伤害/挑衅、状态清理与三项大豁免优势，高级则靠筋疲力尽换续航、先攻重置和濒死反弹维持战斗存在感。短板同样清楚：它几乎没有远程生态、反制施法、范围控制、召唤、制造或战役级解题；面对飞行、传送、巨大体型、传奇抗力、远程风筝和需要跨位面/情报/反魔法的局面，仍需要队友接力。

## 详细评价

Tier 1 的拳斗士已经能打：`斗殴术` 给附赠动作徒手打击、d8 拳斗骰、临时武器 Sap 精通；`铁下巴` 用体质撑 AC；2 级 `锐气` 提供临时生命、两次附赠徒手或打一下后疾走/撤离；`浴血奋战` 在受伤时用反应回满锐气，浴血时还给大量临时生命。问题是它的防线要靠主动吃伤害换资源，反应与附赠动作很紧，远程和飞行目标会显著压低发挥。

Tier 2 进入成熟近战期。`重拳手` 让徒手命中同时伤害并擒抱/推撞，`万钧痛击` 在命中时最大化伤害且命中后返还锐气，6 级力场伤害绕过大量抗性，9 级 `硬派学院` 在伤害、下一次攻击最大化和挑衅之间选择，10 级还能清理目盲、魅惑、麻痹、束缚、震慑等关键状态。这个阶段是它最像“可靠前线执行者”的时间段，但它仍是地面近战答案，不是战场改写者。

Tier 3 的价值来自“不容易倒下”。`深挖潜能` 让硬派学院每轮两次，`坚不可摧` 给力量、敏捷、体质豁免优势并可花锐气重掷失败豁免，`斗志昂扬` 在先攻时消除疲乏并重置多个长休按钮。它能非常稳定地站在前线、制造单体压力和保护队友，但贡献仍主要围绕自己触及范围内的目标。

Tier 4 继续强化生存而非扩大解题面。18 级 `斗魂` 的 0 血反弹、半最大生命临时生命、回满锐气和除力场外全伤抗性很硬，20 级提高力量/体质上限并在短休回血。它在终局物理压力下比多数非施法武职更难被打穿，但缺少高阶施法者那种改写场景、战略移动、反制和范围封锁，所以顶端评分不能再上探。

## 综合评分

| 阶段 | 综合强度 | 具体理由 |
|---|---|---|
| Tier 1（1-4） | A- | 附赠徒手、体质 AC、锐气临时生命、受伤回满锐气与 3 级命中后擒抱/推撞，让低级拳斗士是优秀近战压迫者；但它必须进入危险距离，远程、飞行和法术控制会直接限制发挥。 |
| Tier 2（5-10） | A | 额外攻击、万钧痛击、力场伤害、硬派学院与状态清理把它抬成稳定地面前线，能同时做伤害、挑衅和单体控制；限制是范围、反制、目标接入和非战斗解题仍明显不足。 |
| Tier 3（11-16） | A | 深挖潜能、三项大豁免优势、失败豁免重掷和先攻重置让它在长战里很难被普通压力打崩；档位停在 A 是因为高阶贡献仍围绕触及范围内的目标，不能替代施法者级别的战场改写。 |
| Tier 4（17-20） | A- | 斗魂和顶峰状态提供优秀濒死反弹、抗性、锐气回收和休息恢复，终局前线稳定度很高；但它没有获得反魔法、远程飞行处理或战略级工具，所以比真正终局核心低一档。 |
